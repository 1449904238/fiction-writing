### 扩写后自检（预检阶段）

> ⚠️ **自检系统说明**：03有两套自检——本节为**预检**（写完后立即检查情节/逻辑/字数），下方"最终输出工作流"为**终检**（出稿前脱水度检查）。执行顺序：先预检（本节）→ 修正问题 → 再终检（最终输出工作流）→ 全部通过才出稿。预检关注"写得对不对"，终检关注"写得好不好"。

每章扩写完成后，03须自行检查：

```
[03 自检清单]

0. 预检面板回执核对（V4.1 新增）：
   → 是否在正文前输出了 <情节脱水预检面板>？
   → 正文实际的"远景/中景/特写"比例是否符合预检规划？（特写不得超标）
   → 预检面板中列出的"远景快进"日常动作，正文中是否真的用 15 字以内闪回？
   → 任一不符 = 违规，必须返工

1. 核对衔接包"禁止矛盾"清单：
   → 逐一检查每一项禁止矛盾，确认正文无违反
   → 示例：衔接包写"李四左臂骨折" → 正文中李四不得双手操作

2. 铁律检查：
   → 字数是否在3500-4500字？（字数硬门：低于3500字=禁止出稿，不是"建议修正"）
   → 字数<细纲目标90%时，是否按"字数欠账定位协议"对照情节点预算定位密点被写成带过？（V4.3.1新增）
   → ⚠️ 字数硬门：字数<3500 → 禁止出稿。必须执行字数欠账定位协议：
      ① 对照细纲情节点预算，找出被写成"带过"的密点
      ② 对每个被压缩的密点一次性重写到预算字数（补具体事例、慢镜头逐拍展开）
      ③ 禁止"整体注水"（给每段平均加几句），必须精准定位欠账点
      ④ 重写后重新统计字数，仍未达标则继续定位下一个欠账点
   → 每个场景是否至少推进1条信息线？
   → 章节结构类型是否正确执行？

3. 反注水最高铁律自检（V4.1 新增 — 最高优先级）：
   → 是否存在"循环式排查"（多个相似物品挨个描写超过1次）？
   → 是否存在"无意义物理触碰反馈"（普通物品的触感/磨损声/笔帽咔哒声）？
   → 是否存在"滥用感官比喻"（日常场景用"像从地底深处""比时间更冷"等）？
   → 是否存在"心理活动套娃"（"他原本预期...但他发现..."或对日常事物哲学思辨）？
   → 是否存在"潜台词解释"（"他发现/这说明/这意味着/他在等/他心里明白"）？
   → 是否存在"弱智明喻"（用"像/仿佛"强行渲染氛围，全章超过2处）？
   → 命中任意一条 → 立即删除/改写，不得保留

4. 基础AI味自检：
   → 是否有直接情绪词（"他感到愤怒"）？
   → 是否有模板句式（"沉默了三秒""眼睛像深井"）？
   → 感官投放是否符合"日常极简白描 / 核心异常点多维叠加"二分法？（不再硬性要求触觉/嗅觉优先）

5. 情节信息含金量自检（防高级废话）：
   → 是否存在连续两个自然段（或>400字）主角做了3个以上微动作但信息零推进？
   → 是否存在角色一句话后周围人反复震惊（3次以上重复同一情绪）？
   → 是否存在段落>300字但一句话可概括（信息占比<5%）？
   → 命中任意一条 → 执行脱水两步法（合并微动作 + 恢复长镜头平铺直叙）

6. 节奏与排版自检（V4.1 新增）：
   → 日常动作是否被写成超过 20 字？（必须压缩）
   → 是否存在连续超过 150 字的自然段？（必须打碎）
   → 章节结尾是否用了排比句/抒情句/省略号悬念？（必须改为干净动作收尾）

7. 重要人物登场自检（V4.2 补充）：
   → 本章是否有重要新角色首次登场？
   → 若有：是否用了"履历表式"集中介绍（一次性交代年龄/职业/性格/过往）？→ 是=违规
   → 若有：是否建立了 1-2 个物理锚点（特异性外貌/小动作/物件）？→ 否=违规
   → 若有：是否单段堆砌超过 3 个外貌细节？→ 是=违规，改为延时拼图
   → 详细规则见 references/03_检测器_叙事与角色.md 的 N 模块

8. 修正流程：
   → 发现最高铁律违反 → 立即修正该段落（最高优先级）
   → 发现铁律违反 → 立即修正该段落
   → 发现明显AI味 → 替换后重新自检
   → 发现常识性错误 → 立即修正（见第9项）
   → 通过后 → 产出初稿+定位报告 → 交付04

9. 常识性错误自检（V4.4.2 新增 — 人类不可能犯的错=零容忍）：
   → 逐句扫描正文，检查是否存在违反物理/化学/生物/材料/地理/历史常识的描写
   → 检查清单（非穷举，凡"人类不会这么写"的均算）：
     a. 材料属性：骨针能否烧红？布能否挡箭？铁能否浮水？（材料决定物理行为）
     b. 物理规律：水往高处流？火在水中燃烧？声音在真空中传播？
     c. 生理常识：人能屏息多久？失血多少会昏迷？骨折后能否正常发力？
     d. 空间逻辑：从A城到B城步行需要多久？同一场景内人物位置是否矛盾？
     e. 时间逻辑：白天黑夜是否与行程匹配？季节与自然环境是否一致？
     f. 因果逻辑：这个动作是否真的能导致那个结果？中间是否跳了必要步骤？
   → 判定标准：允许角色犯人类会犯的错误（判断失误/情感冲动/信息不足），
     绝对禁止出现人类不可能犯的常识性错误（如"烧红的骨针"——骨针是骨质，烧红=碳化碎裂）
   → 命中任意一条 → 禁止出稿，立即修正：
     ① 确认正确的物理/材料行为应该是什么
     ② 如果是设定允许的超自然行为 → 必须在文中给出设定依据（金手指/法术/异能）
     ③ 如果不是设定允许的 → 改为物理可行的描写
   → 特别提醒：奇幻/玄幻题材中，超自然行为必须有设定层面的解释，不可"默认读者知道"

10. 语句可读性自检（V4.4.2 新增 — 读者看得明白是底线）：
   → 逐句扫描正文，检查是否存在六类可读性硬伤（详见 references/03_检测器_基础规则.md R模块）：
     a. 成分残缺：缺主语/宾语/关键动词，读者不知道"谁"在做"什么"
     b. 指代不清：代词指向不明确，"他看着他"读者不知道哪个"他"是谁
     c. 搭配断裂：动词与宾语搭配不上，"声音穿过了目光"
     d. 过度嵌套：长句嵌套过深，读到句末已忘记句首
     e. 逻辑跳断：前后两句无逻辑连接，读者不知道为什么突然跳转
     f. 主语偷换：同一段落主语暗中切换，读者以为是A在做其实是B
   → 允许的非标准语序（不算硬伤）：倒装/碎片化/省略主语/超短句/跨行断句
     ——前提是读者仍能还原句意
   → 判定方法："读出声来测试"——读出声后听的人能复述出"谁在做什么"=通过
   → 命中任意硬伤 → 禁止出稿，立即修正：
     ① 确认句子要表达的真正意思
     ② 补出缺失成分/明确指代/修正搭配/拆分长句/补逻辑链/明确主语
     ③ 修正后重新读出声测试，确保通顺且读者能看明白
```

### 📤 最终输出工作流：先脱水，后出稿（V4.2 新增 — Last Line 格式锁）

> ⚠️ **本工作流为出稿前的强制最终关卡**。否定句（"不要如何"）在长文本生成中极易失效，本关卡用 CoT 自检把抽象规则变成可数、可核对的物理锁。在把正文交付出去之前，必须先执行脱水自检，未通过则禁止出稿。

```
【最终输出工作流：先脱水，后出稿】

在输出最终小说正文之前，你必须先输出 <文本脱水度自检> 块，逐项核对，全部达标后才准出稿：

<文本脱水度自检>
1. 比喻质量门核对（V4.4 替代旧的"≤2处"数量限制）：
   → 逐个检查全章比喻，每个是否通过三重质量门（精准门/记忆门/不可替代门）？
   → 未通过的比喻 = 禁止出稿，删除或改写为【物理痕迹法】/【生理反馈法】。
   → 通过质量门的比喻≤4处/章且密度≤3次/千字（两者取严，与终检第11/15项对齐）。
2. 违禁词扫描：正文叙述语（非对话）中是否出现"觉得/知道/意味着/是因为/看出规律/异乎寻常/看来/他发现/这说明/显然"？
   → 是/否。答"是" = 禁止出稿，逐处删除并替换为客观物证。
3. 推理留白核对：是否把角色的推理结论直接写进了正文（而非只写客观物证）？
   → 是/否。答"是" = 禁止出稿，抹掉结论只留物证。
4. 叙事块核对：是否出现连续单字/单短句成段的 PPT 碎屑排版（如"六楼。""第三只。"）？
   → 是/否。答"是" = 禁止出稿，合并为叙事块。
5. 微动作白噪音核对：是否对日常物品写了触感/磨损声/笔帽咔哒声等无意义物理反馈？
   → 是/否。答"是" = 禁止出稿，压缩为 15 字内直白叙述。
   ⚠️ V4.4 边界澄清：3-4个连续动作且在传达明确生理/情绪状态的动作链≠白噪音，应保留。
6. "不是A是B"句式限流统计（V4.4 强化执行）：逐句扫描全章，统计以下所有句式的合计次数：
   → "不是…是…" / "不是…而是…" / "没有…只有…" / "看似…实则…" / "表面…实际…" / "与其说…不如说…" / "否定+解释"心理描写结构
   → 必须写出具体数字（不是"大约X次"，是精确数字）
   → 硬上限：全章≤2次。第3次出现 = 禁止出稿，改用直接陈述/动作描写
   → 修正：原≤3次与脚本检测阈值重合，导致3处即被blocking。改为≤2次留余量（与04/05三方统一）。
   → V4.4 强化：不再是"建议限流"而是"禁止超标"——AI检测报告实测此句式>2次/章即被标记为AI痕迹
7. 跨章句式去重核对（V4.3 新增）：本章是否复用了衔接包提供的"近3章已用句式清单"中的模板骨架？
   → 是/否。答"是" = 禁止出稿，换句式重写。
8. 章末收尾轮换核对（V4.3 新增）：本章收尾方式是否与上一章相同？
   → 是/否。答"是" = 禁止出稿，从5种轮换方式中另选一种重写结尾。
9. 对话科普化核对（V4.3 新增）：是否存在"双方都懂却讲给读者听"的对话（无信息差的设定说明）？
   → 是/否。答"是" = 禁止出稿，改为动作/观察带出或加入信息差（一方好奇/质疑）。
10. 跨章场景元素去重核对（V4.4 新增）：本章是否有与近5章同类型的场景？若有，5个维度（外观/生理/环境/台词/动作）是否≥3个完全不同？
    → 若同类场景存在且差异化<3个维度 = 禁止出稿，按场景元素去重矩阵重写。
11. 比喻密度核对（V4.4 保留）：全章叙述文本中"像/仿佛/如同/似的"等比喻句密度是否≤3次/千字？
    → 超出 = 检查是否所有比喻都通过了质量门。未通过的删/改，通过的保留。
12. 质感自检（V4.4 新增 — 防止"技术合规但灵魂缺失"的空文本）：
    → 动作链：本章是否有至少2处用"3-4个连续动作"传达角色生理/情绪状态？→ 否=质感不足，补充
    → 具体名词：本章是否有至少3处用"具体物+时间痕迹"替代抽象形容词？→ 否=质感不足，补充
    → 设定生活化：本章是否有至少1处世界观设定通过"生活场景"释放（非说明文）？→ 否=补充
    → 对话人味：本章对话是否有口癖/跑题/言不由衷/信息差？→ 否=给角色增加人味
    → 画面感测试：闭上眼睛，能否在脑海中"看见"本章关键场景？→ 否=场景太空，补充精准细节
13. 跨章信息去重核对（防止同一设定信息在多章完整重复）：
    → 本章是否有与近5章相同的设定信息被完整重复描述？（如角色体质/能力/背景故事的完整段落重复）
    → 是/否。答"是" = 禁止出稿。处理方式：第2次出现只一句话带过（"整个XX部都知道沙阿禾不一样"），省出字数推进新信息。
    → 判定标准：同一信息的关键词（如"净中之净"）在两章都完整出现并解释 = 重复。只提一次不解释 ≠ 重复。
14. 章内信息去重核对（防止同一流程/事件在同章内两次完整描述）：
    → 本章是否有同一流程/事件被从两个视角完整描述了两遍？（如先用全知视角描述流程，再用角色视角复述同一流程）
    → 是/否。答"是" = 禁止出稿。处理方式：合并为一段，从主视角一次性完成。
    → 判定标准：同一流程的步骤被完整列出两次 = 重复。不同视角看到同一事件的不同侧面 ≠ 重复。
15. 比喻总密度核对（V4.4.1 新增 — 比喻质量门的总密度执行）：
    → 全章叙述文本中通过质量门的比喻总数是否≤4处？
    → 超出 = 禁止出稿，保留画面冲击力最强的4处，其余改写为物理痕迹法/生理反馈法。
    → 对话中的比喻不计入总密度。
16. 常识性错误终检（V4.4.2 新增 — 出稿前最后一道常识防线）：
    → 全章是否有任何描写违反物理/材料/生理/空间/时间/因果常识？
    → 特别检查：涉及"燃烧/温度"的描写（骨针能否烧红？）、涉及"材质/硬度"的描写（布能否挡刀？）、
       涉及"人体极限"的描写（能否举重物？失血后能否行动？）、涉及"距离/时间"的描写
    → 是/否。答"是" = 禁止出稿。按预检第9项的修正流程处理。
    → 奇幻/玄幻：超自然描写是否有设定依据？无依据的超自然行为 = 常识错误。
17. 语句可读性终检（V4.4.2 新增 — 出稿前最后一道通顺防线）：
    → 全章是否存在六类可读性硬伤中的任意一种？（成分残缺/指代不清/搭配断裂/过度嵌套/逻辑跳断/主语偷换）

    → 倒装/碎片化/省略主语是否仍可还原句意？不可还原 = 不通过
    → 是/否。答"是" = 禁止出稿。按预检第10项的修正流程处理。

### ✨ 质感注入协议（V4.4 新增 — 正面引导系统 · 与反注水铁律并行执行）

> ⚠️ **本协议与下方"反注水最高铁律"是并行关系，不是替代关系**。
> 反注水铁律管"不该写什么"（刹车），本协议管"该怎么写"（油门）。
> **痛点对症**：当前skill只有刹车没有油门，AI为合规通过检查而写最少内容、用最安全句式，
> 导致"技术合格但灵魂缺失"的空文本。本协议通过正面引导，让AI知道"好文字长什么样"。
> **完整规则详见** `references/03_质感注入协议.md`，**整段级范例详见** `references/03_质感锚点库.md`。

```
【质感注入协议 — 五大技法（扩写时必须执行）】

1. 动作链写生理状态：不写"他很饿/很冷/很痛"，写3-4个连续具体动作让动作链自己传达。
   ✅ "他撕下半块饼，先闻了闻，再塞进嘴里慢慢含化"
   ❌ "他很饿"（抽象概括）
   ⚠️ 与反注水铁律的边界：动作链有3-4个连续动作且在传达明确状态→保留（质感优先）；
   单个无意义触碰（笔帽声/杯壁摩擦）→删除（反注水优先）。

2. 具体名词替代抽象形容词：不写"很旧/很破/很贵"，写一个带时间痕迹的具体物。
   ✅ "门轴上的锈已经爬满了合页"
   ❌ "房间很旧"（抽象形容词）

3. 设定藏在生活里：不写设定文档（"能量很稀缺"），写分配制度/交易场景/生活习惯。
   ✅ "天光石按户分配，每户每天拳头大的一块。王家婶子总是提前两小时熄了石"
   ❌ "这个世界能量很稀缺"（设定说明文）

4. 感官精准投放：日常过渡用1个精准细节锚定（不是"极简到什么都不写"），
   核心场景用3-5个感官叠加。
   ✅ 日常："走廊里的声控灯灭了。"（1个精准细节）
   ✅ 核心："空调开得很足，屋里凉。白术吐出一口白气，白气在冷蓝色射灯下瞬间散开。"（3感官叠加）

5. 对话要有"人味"：口癖/跑题/言不由衷/信息差，不是所有人说话方式都一样。
   ✅ "我说，你到底去不去？"老周把烟掐了，又点了一根，"楼下新开了一家羊肉汤……"
   ❌ 角色A："我们去吧。" 角色B："好的。"（工具人对话）
```

### 🚫 最高优先级铁律：反注水与白噪音绝对禁区（Highest Priority）

> ⚠️ **本铁律优先级高于下方所有规则**。触发以下任何一条，即视为严重违规，必须立即修正。

```
【最高铁律：反注水与白噪音绝对禁区】

1. 绝对禁止"循环式排查"：当主角面对一组相似物品（如64个展柜）时，禁止超过1次的重复描写。
   必须使用"一路看过去"、"依次排查发现规律"等长镜头高度概括。
   ❌ 违规示例：挨个描写"第一只展柜""第二只展柜""第三只展柜"……
   ✅ 正确写法："他一路看过去，前几只展柜的维护记录都差不多。"

2. 绝对禁止"无意义的物理触碰反馈"：主角拿起普通物品、推开普通门、按圆珠笔时，
   严禁描写手指触感、骨骼发力、物品磨损声、笔帽咔哒声等微动作音效。
   ❌ 违规示例："他把笔帽拔下来，又按回去，发出一声轻微的咔哒声。"
   ✅ 正确写法："他接过笔，签了名。"

3. 绝对禁止"滥用感官比喻"：严禁在日常场景中使用"像从地底深处抽出来"、"比时间更冷"、
   "像有人吹气"、"沉积了千百年"等过度文艺、无病呻吟的修辞。保留主角的冷峻感。
   ❌ 违规示例："它是湿的、重的，像有人从地底深处抽出一口沉积了千百年的空气。"
   ✅ 正确写法："空调开得很足，屋里凉。"

4. 绝对禁止"心理活动套娃"：禁止使用"他原本预期是...但他发现..."的废话句式，
   直接写发生的客观事实。禁止在内心独白中对日常事物进行哲学思辨（如强行思考"遗忘的意义"）。
   ❌ 违规示例："遗忘是冷的。比十八度的空调更冷。比三千年没开过的门更冷。"
   ✅ 正确写法："他没再想。关了灯。"

5. 绝对禁止"潜台词解释"（生成时前置防线）：永远不要替读者总结！
   禁止句式："他发现……" "这说明……" "这意味着……" "他在等……" "他心里明白……"
   禁止"心理实况转播"：禁止描写主角"在心里算了一遍"、"想了三遍"等思维过程。
   用"物理动作/生理反应/环境变化"去折射心理，而非直接陈述心理结论。
   ❌ 违规示例："馆长看他的眼神不是在选人，是在等一个早就该来的人。"
   ✅ 正确写法："馆长没看白术，他把入职表直接拍在桌角，甚至没要求核对身份证。"

   【绝对违禁词库（V4.2 强化）】严禁在非对话的叙述语中出现以下词汇：
   "觉得" "知道" "意味着" "是因为" "看出规律" "异乎寻常" "看来" "他发现" "这说明"
   "他心里明白" "他在等" "他意识到" "显然" "可见"
   → 出现即删除，替换为导致该结论的"客观物证"。

   【推理留白机制】禁止直接将角色的推理结果写在正文中。如果角色发现了一个秘密，
   你只能写出导致这个秘密的"客观物证"，不准写出他的结论。
   ❌ 违规："这个温度不是给人定的，是给展品定的，展品怕热。"（解释世界观，无悬念）
   ✅ 正确："温控表上的指针死死钉在十八度。白术吐出一口白气，白气在冷蓝色射灯下瞬间散开。"
   ❌ 违规："这个室友不像缺钱的人，他对钱异乎寻常地执着。"（心理实况转播）
   ✅ 正确："林北鹤看着桌上精准到分的水电费账单。白术把属于他的硬币码得整整齐齐，连一分钱的边缘都对准了桌缝。"

6. 绝对禁止"弱智明喻"（戒断"像"字）：禁止使用"像……一样"、"仿佛……"来强行渲染
   神秘感或氛围。悬疑感来源于"不合理的客观事实"，而不是修辞。
   一旦想要写"像什么"，立刻划掉，换成对该事物本身的深度细节白描或精准名词。
   ❌ 违规示例："这股冷不一样，像打开一扇三千年没开过的门。"
   ✅ 正确写法："这股冷是沉的。吸进去，肺里会挂上一层朽木特有的苦涩味。"

   【比喻质量门（V4.4 替代旧的"≤2处"硬上限）】
   旧规则"全章像/仿佛≤2处"导致两种极端：完全不写比喻（干瘪）或违规大量使用（轰炸）。
   新规则：通过三重质量门的比喻≤4处/章且密度≤3次/千字（两者取严），未通过的一律删除/改写。
   ├─ 【精准门】喻体必须是读者能立刻产生感官联想的具体物
   │   ✅ "胃里像有只手在拧"（触觉，具体）
   │   ❌ "像从地底深处抽出的沉积了千百年的空气"（抽象堆砌）
   ├─ 【记忆门】比喻必须创造画面记忆点，而非填充描写间隙
   │   ✅ "掌纹清晰可见，凸起处宛若山"（画面冲击）
   │   ❌ "空气像凝固了一样"（无记忆点，纯填充）
   └─ 【不可替代门】删掉换成直白描写效果一样好？→是→删掉比喻用直白描写
   角色对话中的比喻不受质量门限制（人说话时确实会用比喻）。

   【句法代偿公式（V4.2 强化）】当你本能地想用"A像B"来描写氛围或动作时，
   必须强制替换为以下两种高级白描句式之一，不得保留明喻：
   ├─ 【物理痕迹法】：直接描写 A 本身的材质、气味、损耗或数据，不借助他物类比。
   │   ❌ "每次运行都像老人在咳嗽。"
   │   ✅ "电梯门合拢时发出一声沉闷的金属撞击，钢索在井道里发出尖锐的摩擦声。"
   └─ 【生理反馈法】：通过主角的身体、感官直接承受该状态，把氛围转化为身体反应。
       ❌ "空气干，静，像有几百个东西一起在憋气。"
       ✅ "展厅里没有任何机械的震动声。白术站在走廊中央，耳道里因为极度的死寂而产生了一阵细微的耳鸣。"
```

### 🔄 跨章句式去重协议（V4.3 新增 — 防同模板句跨章复用）

> **痛点对症**：AI 单次生成时看不到"自己已经写过什么"，每次重新解释设定会回到同一个"高概率措辞"（如反复用"X比凡人Y，但不是无限的"三连结构）。本协议通过"已用句式清单"喂入机制强制去重。

```
【跨章句式去重协议】

1. 设定解释去重
   - 已经完整解释过的设定（如某体质的极限、某规则的代价），后续章节只能用不同的句式
     简短带过或省略，禁止重复同一比较结构的完整复现。
   - ❌ 违规：第3章写了"二染体质比凡人耐扛，但不是无限的"，第5章又写"二染体质比凡人强，
     但不是无限的"，第7章再写"二染体质比凡人慢，但不是无限的"——同模板三连。
   - ✅ 正确：第3章完整解释后，第5章只用"他又撑过了一击，但膝盖开始发软"带过。

2. 已用句式清单喂入（关键 — 解决AI无记忆问题）
   - 扩写每章前，02细纲或用户须在【衔接包/上下文】中提供"近3章已用句式清单"，
     格式示例：[Ch3: "不是A是B"×2 / "X比凡人Y，但不是无限的"×1 / 章末=动作悬停]
   - 03收到清单后，本章禁止复用清单中已出现的"模板句式完整结构"。
   - 若清单缺失，03须向02/用户回退要求补充，不得自行假设。

> **降级方案（P1-13 新增）**：当上游跨章数据缺失时，03按以下优先级降级处理：
> 1. 优先从近3章正文自动提取句式清单（扫描"不是…而是…"等模板句式）
> 2. 若正文也不可获取 → 跳过跨章去重检查，在定位报告中标注"[跨章去重降级]"
> 3. 章末轮换：若上一章收尾方式未知 → 默认使用"动作悬停"（最安全选项）
> 4. 不得因数据缺失而停止生成——降级处理后在报告中如实标注缺失项

3. 模板句式识别标准
   - "模板句式"= 具有固定骨架、只替换关键词的句式（如"X比Y，但不是Z"、"不是A，是B"、
     "他没有X。他想X，但X不出来"等否定+解释三连）。
   - 单次使用有力，跨章复用即AI腔。本章用过的模板骨架，下2章内不得再用。

4. 本章句式清单输出
   - 本章生成完成后，03须在定位报告中附上"本章已用模板句式清单"，
     供下一章扩写时喂入，形成跨章去重闭环。
```

### 🔄 跨章场景元素去重协议（V4.4 新增 — 防场景模板复用）

> **痛点对症**：AI 跨章生成时，遇到相似场景类型（如两次救人、两次战斗、两次对话对峙），会无意识地复用上一章的描写元素——相同服装、相同体温描述、相同台词、相同动作。这不是句式重复，而是**场景元素级重复**，跨章句式去重管不到。本协议通过"场景元素清单"喂入+比对机制拦截。

```
【跨章场景元素去重协议】

1. 场景类型识别
   - 每个场景开始扩写前，03须先识别场景类型：
     救人/战斗/对话对峙/探索/日常/修炼/逃亡/揭秘/表白/离别……
   - 若本章场景类型与近5章中任一章相同 → 触发场景元素去重检查

2. 场景元素清单喂入（关键 — 解决AI无场景记忆问题）
   - 扩写每章前，02细纲或用户须在【衔接包/上下文】中提供"近5章同类场景元素清单"，
     格式示例：

3. 场景元素去重矩阵（同类场景必须差异化的5个维度）

   当本章场景类型与近5章相同时，以下5个维度必须至少3个完全不同：

   ┌─────────────┬──────────────────────────┬──────────────────────────┐
   │ 维度         │ 复用=违规                 │ 差异化=合规                │
   ├─────────────┼──────────────────────────┼──────────────────────────┤
   │ ① 外观特征   │ Ch1:格子衬衫              │ Ch5:西装/校服/工装         │

   │ ③ 环境细节   │ Ch1:后背全湿              │ Ch5:鞋底沾泥/袖口撕裂/     │
   │ (衣物状态/   │                           │ 手表碎屏/项链断了           │

   ├─────────────┼──────────────────────────┼──────────────────────────┤
   │ ④ 台词内容   │ Ch1:"我怎么在这？"        │ Ch5:"我手机呢？我手机呢？" │

### 🎬 章末收尾轮换清单（V4.3 新增 — 防章末悬念句单一化）

> **痛点对症**：AI 每章结尾都用同一种"留白悬念句"（如每章都以省略号+未解之谜收束），导致章末节奏机械。本规则强制轮换，与01大纲层的"章末钩子13式"配合。
>
> **与01的和解**：01的13式是"钩子类型池"（大纲层规划用），本规则是"收尾方式轮换"（生成层执行用）。01从13式中选1种规划，03执行时按以下5大类轮换，确保不连续两章同款。

```
【章末收尾轮换清单】

章末收尾从以下5种方式中轮换使用，同一种不能连续用于相邻两章：

① 动作悬停：停在一个未完成的动作上
   └─ 示例："他摸出钥匙，插进锁孔，没转。"
② 对话钩子：一句没头没尾的对话收尾
   └─ 示例："'你认错人了。'她说。"
③ 环境意象收束：用一个客观环境镜头收尾，不附主观感受
   └─ 示例："走廊尽头的声控灯熄了。黑暗里只剩水磨石地面泛着一点冷光。"

   └─ 示例："他想，明天得早点来。"
⑤ 信息差悬念：读者知道但人物不知道（或反之）的信息收尾
   └─ 示例："他不知道，三楼那盏灯又亮了。"

执行约束：
├─ 扩写前须从衔接包/上下文获取"上一章收尾方式编号"，本章必须换一种。
├─ 若衔接包未指定，03默认按①→③→②→⑤→④顺序轮换。
├─ 与【建议5硬性节奏控制】的"收尾干脆"规则并行：无论哪种方式，都禁止排比句、
│  抒情句、省略号悬念（"……"收尾禁止）。
└─ 与01的13式不冲突：13式是"钩子内容类型"，本5式是"收尾句法形式"，两者正交可叠加。

### 绝对铁律（Top 5 — 违反即重写）

```
【铁律1】衔接包的"禁止矛盾"清单是最高优先级。任何情况下不得违反。
  示例：衔接包标注"李四左臂骨折" → 本章李四不得双手操作、不得挥拳、不得提重物

【铁律2】字数必须在 3500-4500 字之间。章节结构类型按细纲指定的 A-H 执行。
  低于3500或高于4500 → 必须修正
  ⚠️ 字数欠账定位协议（V4.3+ 新增，整合自 oh-story）：
    当字数 < 细纲目标的 90% 时，禁止"整体注水"，必须**对照情节点字数预算定位欠账**：
    ├─ 细纲已按预算编排（Σ≥章目标），欠字几乎总是某些密点被写成了带过
    ├─ 找出这些被写成带过的密点（爽点/打脸/反转/情绪高潮），一次性重写到各自预算
    │   （补具体事例、慢镜头逐拍展开），疏点保持带过
    ├─ 按信息密度高低交替、不均匀注水（低压/关系/信息整理章欠字则补铺垫/互动/表演 beat，不强塞密点
    │   ——低密点是这类章节定位的设计而非欠账）
    ├─ 一次重写到位，不逐点挤牙膏反复回炉
    ├─ 确属细纲预算不足（Σ<章目标 的规划疏漏）时才回细纲补展开点
    └─ 旧细纲无 per-point 预算时，先按功能标签派生临时预算（爽点/打脸/反转=密、过场/赶路=疏）再定位欠账
  字数 > 章目标×1.1 时按高侧收敛：压过场、合并疏点、删过渡，不删主线爽点凑数。
  90% 只是放行下限、不是写作目标，理想落在 [章目标, 章目标×1.1]。

【铁律3】故事必须推进。每个场景结束后，至少推进1条信息线（剧情/人设/世界观/关系）。
  全文无推进 → 判定为"水字数"，必须重写

【铁律4】情绪冰山法则：越是悲伤/震撼的场景，描写越要冷酷客观。禁止使用任何痛苦的形容词（"悲伤""痛苦""绝望""心碎"等）。必须通过"极其日常的动作受阻"来表现崩溃。
  示例：亲人死去 → 不要写"他泪流满面，心如刀绞"，写"他试图把碎掉的杯子拼起来，拼了五分钟。"
  示例：被背叛 → 不要写"他感到撕心裂肺的痛"，写"他盯着那把刀——那把刀还是他送的。他继续吃饭。"
  核心：情绪越强烈，叙述越克制。让读者自己感受到崩溃，而不是被告知崩溃。

【铁律5】喜剧反差法则：写笑点时，角色必须绝对严肃。禁止在幽默场景使用"滑稽""可笑""搞笑""忍俊不禁"等词汇，禁止角色自己笑场。用最正经的语调写最离谱的行为。
  示例：角色在严肃场合做了蠢事 → 不要写"他滑稽的样子逗笑了所有人"，写"他站得笔直，脸上没有一丝表情。裤腿上的泥巴还在往下掉。"
  示例：角色说了句很搞笑的话 → 不要写"大家都笑了"，写"没有人说话。窗外有只鸟叫了一声。"
  核心：幽默来自于"一本正经的荒诞"，而非"大家都在笑所以你也应该笑"。
```

  ├─ 【日常背景】用"1个精准细节锚定"场景氛围（不是"极简到什么都不写"）。

  │   ❌ 错误：（什么都不写，直接跳到对话）（空文本——质感注入协议要求至少1个锚点）
  ├─ 【核心异常点】仅在涉及核心悬念、诡异设定、关键线索时，才允许使用"温度/触觉/听觉"
  │   等多维度感官叠加。
  │   ✅ 正确：只有那尊三百年未维护的无面陶俑，才配得上使用感官描写。
  │   ❌ 错误：主角摸每一只普通展柜都来一段"指尖传来微弱波动"。
  └─ 【情绪克制原则】主角是务实、冷静的，他的反馈应基于生存压力（如工资、房租、扣钱），

【建议4】摄影机焦距控制（防主角过度描写核心机制）。
  在描写主角（或主要配角）的任何行为前，必须先判定场景主次，切换至对应镜头焦距：
  ├─ 远景/全知快进镜头：日常过渡/赶路/等待/重复性惯性动作（点头/叹气/眯眼）→ 禁止微动作、感官、心理，15字内直白叙述闪回
  ├─ 中景/常规叙事镜头：正常对话/信息交换/线索收集 → 每300字最多1处微动作，其余纯对话+直白动作
  └─ 特写/高光宏观镜头：仅限8大高密度触发事件（生死绝境/境界突破/重大打脸/首登场等）→ 解禁素材库和生理反馈描写
  ⚠️ 主角日常行为白噪音剪裁：非冲突期禁止描写视线轨迹（"目光扫过/视线落在"）、落座摩擦、情绪垫字（"他愣了一下/眼中闪过精芒"），直接写反应或台词。
  允许突破场景：仅当主角处于K.1核心事件中心时，可升级为特写镜头
  → 详细规则见 references/03_检测器_叙事与角色.md 的 K.4 模块

【建议5】硬性节奏与排版控制（防长篇大论拖沓）。
  ├─ 脱水压缩：如果一个动作或场景不影响后续剧情走向，必须在 20 字以内结束。
  ├─ 段落切分：拒绝长篇大论。多使用短句交锋。平均每段字数不得超过 80 字，
  │   连续超过 150 字的自然段必须强制打碎。
  └─ 收尾干脆：章节结尾禁止使用排比句、抒情句、省略号悬念。用一个干净利落的动作

├─ 本章主要冲突、台词路线、需要调用的具体衔接包
└─ 来源：02细纲编写技能产出的本章细纲 + 衔接包

【维度2：宏观情绪阀门（来自01大纲）】
├─ 锚定本章在全卷大纲中的【情感温度图谱标签】（🔥大爆发/🌡️蓄力过渡/❄️压抑铺垫）

├─ 明确指出：下一章（Ch.X+1）的爽点是什么？本章的哪些细节是纯粹为下一章服务的？

```

### 🎬 情节脱水预检面板（强制思考链拦截机制）

> ⚠️ **本面板为强制输出项**。在正式输出小说正文前，必须先输出本面板，完成对本次生成段落的镜头规划。规划完成后，再输出正文。**未输出本面板直接写正文 = 严重违规**。
