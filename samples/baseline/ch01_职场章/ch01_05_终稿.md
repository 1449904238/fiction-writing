【小说正文】

"林默，赵总让你去三号会议室。"

HR小李站在工位旁边。她平时笑起来挺大声的，今天嘴唇抿成一条线，眼神往旁边飘。

林默心里咯噔一下。面上没动。

"行吧。"他关掉屏幕上的代码，手机揣进裤兜。站起来的瞬间腿有点麻，坐了四个小时没动。他活动了一下膝盖，跟着小李往走廊走。

路过开放区的时候，扫了一眼四周。平时这个点工位上至少还坐着七八个人，今天只剩三四个，其他位置都空着，椅子推进桌子里，屏幕黑着。上周赵总宣布"组织优化"之后，整层楼的气氛就不太对，大家说话声音都小了，好像怕吵醒什么不该吵醒的东西。

三号会议室在走廊尽头，磨砂玻璃门，看不清里面。林默走过去的时候，扫了一眼苏晚的工位——空的。桌上那盆多肉还搁着，叶片上沾着水珠，不知道谁浇的。苏晚被叫进去是什么时候？她没跟他说过。

推开门。

赵总坐在长桌对面。手边一台笔记本电脑，投影仪开着，蓝莹莹的光打在他脸上，那张笑脸有点失真。桌上摆着两份文件，封面写着"绩效改进计划"。旁边一杯茶，茶面浮着枸杞，冒着热气——赵总到得比他早。

"小林来了，坐坐坐。"赵总抬手示意。每次赵总笑着说"小林"的时候，后面都没好事。

林默坐下。赵总旁边坐着一个法务的人，不认识，戴眼镜，面前摊着本子，手里捏着笔，已经摆好了记录的架势。林默心里又沉了一下——叫法务来，这不是谈话，是走流程。

三个人坐成三角。空调出风口正对着林默后脖子吹。他把椅子往前挪了一点。

"今天叫你来，聊聊。"赵总双手交叉放在桌上，拇指无意识地搓着，"上周线上那个事故，你负责的模块，排查结果出来了。影响挺大的，你知道的。"

点了一下笔记本。投影跳出一张截图——一段代码，红框标着几行，旁边注释写着"内存泄漏导致服务中断47分钟"。下面还标了一行小字："直接损失：SLA违约罚金38万，客户投诉12单。"

林默看着那段代码。

不对。

他写内存回收用的是对象池模式。这段代码里变成裸new，连try-catch都没有。命名风格也不对——他习惯驼峰，截图里全是下划线。有人改了他的代码，或者干脆重写了一份，贴上他的名字。38万罚金，12单投诉——这锅扣得够狠的。

"赵总，"他开口，"这段代码不是我写的。"

"小林，"赵总笑了。那个笑容林默太熟了，"排查报告是运维出的，git记录也对得上。你看——"点开下一页，"commit记录，提交人是你。"

投影跳出一行git log。author字段写着"Lin Mo"。

林默盯着那行字。三秒。

低头的时候，余光扫到桌角一小片水渍——没拖干净的。他想起前天在茶水间接水，老王拖完地经过，没头没脑来了一句："小林，有些东西删了不代表存在。"

当时没当回事。

git记录可以伪造，代码风格不会说谎。做个diff分析就能看出来。但他手里没证据。而且他现在被三个人围着，手机如果拿出来查，赵总会立刻制止。他太了解赵总了——这个人最擅长的就是用流程碾压你，在你还没反应过来之前把结果定死。

"我理解你的感受，"赵总的声音还在响，和和气气的，"公司也不希望是这样。所以准备了绩效改进计划，你看看。"

HR小李把文件推过来。林默扫了一眼——PIP，绩效改进，三个月考察期，降薪百分之三十，不通过自动解除合同。最后一页留了签字栏，旁边印着"本人已阅读并同意"。

标准的逼退流程。他见过别人签，最后都没撑过两个月。隔壁组的老张签了PIP之后被调去做测试，每天点一百遍登录按钮，点了一个月自己辞了。赵总这套流程玩得很熟。

林默没说话。手伸进裤兜，摸到手机。手机壳是硅胶的，边角磨秃了，他习惯性用拇指摩挲壳边——紧张时的小动作。

赵总还在说："其实小林，你能力是有的，就是有时候想法太多。公司给你这个机会……"

林默没在听。

他打开手机，切到代码编辑器。不是公司配的那个——自己写的，黑色界面，一个光标在闪。这个编辑器只有他自己知道，装在加密文件夹里，名字叫"dev_null"。

开始打字。

不是业务代码。不是任何正常人会写的程序。这是"具现代码"。三天前他才发现自己能写这种东西——加班到凌晨两点，整个楼层只剩他一个人，头顶日光灯管闪了一下，他盯着屏幕发呆，手指无意识地敲了一行让咖啡杯悬浮的函数。杯子真飘起来了。悬在半空，离桌面十厘米，杯里的咖啡一点没洒。他以为自己做梦，掐了一下大腿，疼。又试了几次，只要语法对、逻辑闭环，写什么就发生什么。花了两个晚上摸清了规则。

范围有限。他只能影响小物件。杯子、文件、门锁。有一次试椅子，只动了两厘米，太阳穴痛了半小时。改投影仪的数据流。刚好够。

赵总的声音还在继续。投影蓝光打在墙上。林默低着头，手机屏幕的冷白光照着他的指节。

一行一行。

```
function overrideProjection(sourceFile, targetData) {
  redirect stream: accident_report → fund_transfer_log;
  unlock read_only;
  execute;
```

最后一行。

拇指悬在回车上方。太阳穴已经开始跳了——预兆。每次用能力都头痛，像大脑被拧了一把。这回还没按下。

"……所以你看，签了这个，对大家都好。"

林默抬头。

"赵总，投影能切一下吗？我想再看一遍事故报告。"

赵总愣了一下。随即笑了："当然可以。"点了一下笔记本。

事故报告重新投上去。

林默按下回车。

投影闪了一下。

蓝光灭了半秒。

亮回来的时候，屏幕上的代码没了。取而代之的是一张表格——密密麻麻的数字，表头写着"云栈科技·技术中心·项目资金拨付记录"。

赵总的笑容碎了一下。

会议室安静了。法务推了推眼镜，凑近屏幕。

表格在动。

数字一行行刷新，像有人在往里录入。林默的代码在自动检索赵总电脑里的财务文件，把挪用记录一条条提出来，投到大屏上。投影仪的风扇忽然响了一声，像是负载过重——这段代码写的查询范围比他预想的大，赵总电脑里的文件太多了。太阳穴又跳了一下，他咬着牙忍住了。

"云栈·智能客服项目，拨付金额320万，实际到账140万。"

林默端起桌上那杯不知道谁的凉白开，喝了一口。

"差了180万。赵总，这180万去哪了？"

赵总的脸从红变白。他伸手去按笔记本——关投影。林默又敲了一行。投影仪锁定。物理锁定。按键无效。

"别费劲了。"林默把杯子放回去，"从你电脑里直接读的。"

他站起来。投影上还在刷新——转账记录、关联公司资金流向、日期、金额，一条接一条。有些数字林默自己都没料到——不只是智能客服项目，还有两个其他项目的资金也对不上。赵总挪的不止180万。林默站在投影旁边，看着那些数字往上滚，忽然觉得有点荒谬——他写这段代码的时候只是想把事故报告换成资金记录，没想到赵总电脑里的账比他想象的烂得多。

赵总嘴唇动了动。没声音。手按在笔记本上，指节发白。他看了法务一眼，又看了HR小李一眼，最后看向林默。那个眼神里没有愤怒，只有一种被扒光之后的赤裸感——他算好了一切，就是没算到这一步。

法务已经在掏手机拍照了。HR小李的嘴张着，看了看投影，又看了看赵总，手里的笔不知道什么时候掉了。

林默往门口走。经过赵总身边，停了一下。

"赵总，事故报告里的代码不是我写的，commit记录是伪造的。"他抬下巴指了指投影，"但这个是真的。您要不解释一下那180万？"

赵总没说话。他的手还按在笔记本上，指节发白，嘴唇抿成一条线——和HR小李进来之前的表情一样。

林默拉开门。出去。

走廊没开空调，闷热。走了十几步，太阳穴的跳痛变成钝痛，一锤一锤往里敲。他扶着墙，胃里翻涌。干咽了两下。后背衬衫汗湿了一片，贴在脊椎上，冰凉的。

用能力的代价。每次都这样。这回写得多，比平时痛。他靠在墙上，听见自己心跳在耳朵里咚咚响，比平时快。

靠在墙上，闭眼。

走廊另一头传来拖把擦地的声音。沙沙的。很慢。这声音他太熟了——每天晚上十点以后老王开始拖地，从这头拖到那头。林默平时加班经常听见，从来没在意过。

睁眼。老王从拐角出来，蓝色工装洗得发白，腰间钥匙串叮当响。老王步伐很稳，拖把在地面画出一道道水痕，在日光灯下反着光。

老王走到离他三米远的地方，停了。

拖把杵在地上。他歪头，看了林默一眼。林默靠在墙上，一手扶着太阳穴，脸色不太好。老王大概以为他是加班加的——正常人也只会这么想。但那个眼神不是平时那种浑浊的、发呆的眼神。

那一下很清。

然后低头，拖把尖在地面划出一道水痕，沙沙沙走过去了。

林默盯着背影。钥匙串叮叮当当，越走越远。

刚才那段代码，老王不可能知道。会议室门关着。

但那个眼神。

他摇头。行吧。可能是想多了。

站直身子，往工位走。路过苏晚工位——她在。坐在那儿，抬头看着他，嘴唇动了动。整个楼层只剩他们两个了，空调也停了，安静得能听见机房风扇的嗡嗡声。

"你没事吧？"苏晚问。声音压得很低。她平时说话不这样，是那种会直接拍你桌子问"这个接口什么时候好"的风格。今天压着声音，像怕吵醒什么。

林默看了她一眼。眼镜后面的眼睛有点红，不知道是熬夜还是别的。她面前的屏幕还亮着，上面开着一个表格，满屏红色标记。

"没事。"他说，"别签那个PIP。"

苏晚愣了一下。

"什么PIP？"

林默没解释。走回工位，坐下，打开电脑，开始写业务代码。屏幕亮起来的时候，他看了一眼右下角的时间——十二点十分。赵总在会议室里还坐着没出来。法务的拍照声隔着磨砂玻璃传出来，咔嚓咔嚓的，像在拍遗照。

正常人的代码。

太阳穴还在跳。但没那么痛了。
