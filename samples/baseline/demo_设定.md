## 二、世界观设定

### 2.1 背景设定

现代都市，故事发生在某二线城市的互联网公司"云栈科技"。表面是普通的996职场，暗面存在一个"代码具现者"的隐秘群体——极少数人能通过编写特定代码影响现实物理世界。

职场规则与现实的规则并行：白天写业务代码，晚上写"具现代码"。公司内部的权力斗争与能力的隐秘使用交织。

### 2.2 力量体系：代码具现

| 等级 | 能力范围 | 表现 | 代价 |
|------|---------|------|------|
| 1级·物件 | 影响小物件（移动、变形、修复） | 咖啡杯悬浮、文件自动整理、门锁自动开关 | 轻微头痛，持续30秒-2分钟 |
| 2级·规则 | 改变局部物理规则（重力、温度、光线折射） | 让一段走廊重力减半、让屏幕数据实时篡改 | 鼻血、短暂视觉模糊，恢复需2-4小时 |
| 3级·空间 | 创造/折叠局部空间 | 制造"看不见的房间"、空间折叠瞬移 | 剧烈呕吐、昏迷风险，连续使用可能永久损伤神经 |

**核心限制**：
- 具现代码必须语法正确，逻辑闭环——写错一行，能力反噬
- 只能影响"代码覆盖范围内"的物理对象，无法影响人的意识
- 每次使用消耗精神力，连续使用会叠加副作用
- **隐藏规则（后期揭露）**：每次具现都会在现实留下"代码残影"，被同级别以上能力者感知

## 三、角色档案

### 3.1 主角：林默

| 属性 | 内容 |
|------|------|
| 年龄 | 28岁 |
| 职业 | 云栈科技·后端开发工程师（P6） |
| 外貌 | 中等身高，常年黑眼圈，格子衬衫收藏家，头发开始战略性后撤 |
| 性格 | 表面佛系内里精明，嘴贱但护短，擅长用自嘲消解压力，关键时刻果断 |
| 口头禅 | "行吧"、"这需求我得再理解理解"（潜台词：这什么破需求） |
| 能力 | 代码具现（Ch.1觉醒1级，Ch.6升级2级） |
| 动机 | 表层：保住饭碗，不被裁；深层：证明"普通人的代码也能改变世界" |
| 缺陷 | 过度依赖能力后开始忽视真实的人际沟通；对苏晚的感情不敢表达 |

### 3.2 配角：苏晚

| 属性 | 内容 |
|------|------|
| 年龄 | 26岁 |
| 职业 | 云栈科技·产品经理（与林默同部门） |
| 外貌 | 短发干练，戴细框眼镜，笑起来有酒窝，工位永远放着多肉植物 |
| 性格 | 逻辑清晰但偶尔犯轴，嘴硬心软，对不合理需求敢于当面怼，私下喜欢看推理小说 |
| 与林默关系 | 同事+暧昧对象，互相有好感但都没捅破 |
| 功能 | 情感线锚点、林默能力的"人性拉扯"——她代表林默不该用能力捷径解决的真实关系 |

### 3.3 反派：赵总（赵鹏）

| 属性 | 内容 |
|------|------|
| 年龄 | 41岁 |
| 职业 | 云栈科技·技术中心总监（林默的上级的上级） |
| 性格 | 笑面虎，表面和气实则冷酷，擅长用流程和规则碾压人，挪用项目资金填补私人投资窟窿 |
| 动机 | 推进"组织优化"裁员，真实目的是裁掉知道内情的老员工，安插自己人 |
| 弱点 | 过度自信，认为一切都在掌控中；对"超自然力量"毫无防备 |

### 3.4 隐藏高手：老王（王守正）

| 属性 | 内容 |
|------|------|
| 年龄 | 58岁 |
| 职业 | 云栈科技·保洁（实际是公司创始团队早期成员，隐退后留在公司） |
| 外貌 | 瘦小精干，头发花白，总穿洗得发白的蓝色工装，腰间别着一串钥匙 |

| 真实身份 | 上一代代码具现者（3级），十多年前因过度使用能力导致神经损伤，主动"降级"隐退 |
| 功能 | 导师型角色，但不是主动型——他等林默自己发现问题；Ch.7身份暴露，揭露能力代价的残酷真相 |

## 四、关系矩阵

```
林默 ──(暧昧)── 苏晚
  │                 │
  ├──(对立)── 赵总 ──(利用/打压)
  │
  └──(师徒/隐藏)── 老王
                    │
                    └──(知情者)── 赵总不知道老王的真实身份
```

信息差设计：
- 林默：知道自己有能力，不知道老王也有
- 苏晚：不知道林默的能力，不知道老王的身份
- 赵总：不知道任何超自然力量的存在
- 老王：知道林默觉醒了能力，一直在暗中观察，等待林默发现代价

## 五、第1卷大纲（10章）

| 章 | 标题 | 定位 | 核心事件 |
|----|------|------|---------|
| 1 | 那行代码不该写 | 觉醒 | 林默加班写bug，意外触发代码具现，咖啡杯悬浮 |
| 2 | 试运行 | 试错 | 林默偷偷测试能力边界，赵总宣布"组织优化"计划 |
| 3 | 裁员陷阱 | 对峙 | 赵总设局栽赃林默，林默首次用能力反击，揭露赵总挪用资金 |
| 4 | 暗流 | 推进 | 赵总暗中调查林默，老王第一次给出意味深长的提示 |
| 5 | 天台夜话 | 日常 | 林默和苏晚加班后的深夜对话，差点暴露能力 |
| 6 | 突破 | 升级 | 林默在危机中能力升级到2级，改变局部物理规则 |
| 7 | 钥匙串 | 反转 | 老王身份暴露，林默得知能力代价的真相，情绪爆发 |
| 8 | 裂痕 | 冲突 | 林默与老王理念冲突，赵总发动最后一击 |
| 9 | 代码战争 | 高潮 | 林默vs赵总（赵总调动资源围剿），2级能力正面交锋 |
| 10 | 注释 | 收束 | 林默击败赵总，但付出代价；老王留下最后一行"注释"后离开 |

情绪曲线：好奇 → 试探 → 爽快 → 暗流涌动 → 温暖 → 紧张 → 震撼 → 割裂 → 燃 → 余韵
