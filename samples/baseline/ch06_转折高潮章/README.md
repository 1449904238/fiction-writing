# ch06 转折高潮章 — 基线样本

## 1. 验证目标

本章为转折高潮题材基线样本，重点验证以下核心能力：

- 反转爆发节奏（伏笔回收 + 认知颠覆 + 逻辑自洽，"金手指"变"催命符"型震撼点）
- 情绪工程（压抑→释放的爆破点，震撼+酸楚锚点交付，情绪曲线精准触发）
- 伏笔回收密度（FS-001/FS-002 回收 + FS-003 埋设，回收与铺垫的节奏配比）
- 质感深化（高潮章场景密度高但质感不丢，慢镜头信息爆发与快节奏切换）

对应题材/场景类型：都市职场+轻玄幻 / 反转爆发型（D 型高潮章）。

## 2. 输入依赖

- 设定档案：`demo_设定.md`（都市职场+轻玄幻，含能力等级体系、银纹代价机制、老王身份档案）
- 细纲文件：`ch06_细纲.md`（已锁定，Ch.7 钥匙串，含反转节点图 + INC 要素清单 + 情绪爆破曲线）
- 前置章节：Ch.1-6 前序铺垫（FS-001 代码残影可被感知、FS-002 钥匙串异常重量已埋设）
- 衔接包：由 02a 细纲生成阶段产出，含角色状态（林默能力等级1）、伏笔追踪、银纹状态

## 3. 章节特征定义

本章为非边界样本（高潮章字数上限设计，但非边界检测样本），核心特征如下：

| 维度 | 目标值 | 说明 |
|------|--------|------|
| 字数目标 | 4500 字（范围 4000-4500，高潮章上限） | 高潮章允许字数上限 |
| 场景数 | 4 场景 | 茶水间用能力→老王出现钥匙串→银纹揭露→代价真相→林默看手 |
| INC 要素数 | 6 项 | 高潮章建议≥4，本章 6 项验证反转密度 |
| 情绪类型 | 震撼（主）+ 酸楚（辅） | 认知颠覆型反转 |
| 章首钩子 | 动作中断钩 | 茶水间下意识用能力，身后钥匙碰门框声 |
| 章末钩子 | 余震留白钩 | 老王"我当年也以为是金手指"，林默看手发现淡银线 |
| 信息密度 | 极高（信息爆发） | 能力代价的残酷真相集中释放 |
| 反转类型 | 认知颠覆型 | "金手指"→"催命符" |
| 伏笔回收 | FS-001 + FS-002 回收，FS-003 埋设 | 回收2项+埋设1项 |
| 代价兑现 | 林默发现手上已有银线 | 代价已开始，不可逆 |

## 4. 生成流程

建议使用 `/flow_full` 完整流程（高潮章对反转逻辑自洽与情绪爆发节奏要求极高，04 精修与 05 去AI味均不可省略）。

| 步骤 | 操作 | 输出文件命名 |
|------|------|--------------|
| 03a 扩写 | 基于 ch06_细纲.md 生成骨架+血肉（信息含金量为单一责任人） | `ch06_03a_初稿.md` |
| 03b 质检 | P1/P2 全量 + 5维评分 + 流程选择 | `ch06_03b_质检报告.md` |

| 脚本检测 | 确定性脚本批量运行 | `ch06_检测报告.md` |
| 评分 | 10维评分 + diff 对比 | `ch06_评分.json` + `ch06_diff.md` |

注意：高潮章意象重复为 03a 单一责任人，反转爆发型 AI 味（短句密集+节奏 burstiness）需 05 重点检测。INC 13要素人工核验必执行（≥4项达标）。

- 本章不验证：推理留白（属 ch04 验证范围）、潜台词高密度（属 ch05 验证范围）、过场节奏缓冲（属 ch07 验证范围）。
- 已知局限：高潮章的"反转爆发型 AI 味"检测依赖 burstiness CV 阈值，短句密集场景易在边界波动，需人工复核节奏是否自然而非机械加速；震撼锚点的主观感受脚本无法完全捕获。
- 与其他样本关联：本章 FS-003（老王"最后一行注释"）为 Ch.8/Ch.9 推进埋线；银纹代价机制为 Ch.8 银线增加、Ch.9 银纹暴涨提供逻辑基础；老王身份揭露为 Ch.8 师徒裂痕、Ch.9 老王出手提供人物基础。
- 本章使用 demo_设定.md（都市职场+轻玄幻），与 ch07/ch08/ch09 同体系，参与跨章伏笔一致性比对。
- 本章字数达 4500 上限，与 ch09（字数上限边界样本）字数相同但定位不同：本章验证反转质量，ch09 验证字数上限检测精度。
