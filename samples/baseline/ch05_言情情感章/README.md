# ch05 言情情感章 — 基线样本

## 1. 验证目标

本章为言情情感题材基线样本，重点验证以下核心能力：

- 情感推进节奏（克制→升温→决堤→余韵的螺旋式拉扯，欲言又止/进退博弈/情绪反转）
- 潜台词对话密度（表层语义 vs 深层意图分离，"不用你管"式嘴硬、沉默式守护）
- Show Don't Tell（微表情/微动作/生理反应分层呈现，情绪颗粒度细化）
- 防线崩塌节奏（病中脆弱=防线最薄时，崩堤节点精准触发，不提前不延后）

对应题材/场景类型：言情情感 / 情感爆发型（D型）+ 日常质感型（F型）混合，以 D 为主。

## 2. 输入依赖

- 设定档案：`demo_言情设定.md`（含情感曲线图、关系推进节点表、人物情绪档案）
- 细纲文件：`ch05_细纲.md`（已锁定，Ch.6 凌晨三点的门铃，含五段式内容概括与多线情节安排）
- 前置章节：Ch.1-5 前序克制压抑章节（甲方解围"早点回去"伏笔 FS-005 已埋设）
- 衔接包：由 02a 细纲生成阶段产出，含角色关系状态、情感推进进度、伏笔追踪

## 3. 章节特征定义

本章为非边界样本，核心特征如下：

| 维度 | 目标值 | 说明 |
|------|--------|------|
| 字数目标 | 4000 字（范围 3500-4500） | 标准情感章字数 |
| 场景数 | 5 场景 | 深夜加班→病重独处→门铃响→屋内决堤→守夜天亮 |
| INC 要素数 | 7 项 | 言情章≥3 达标，本章 7 项验证情感密度上限 |
| 情绪类型 | 心疼（主）+ 心动（辅） | 全书情感转折点，前5章克制后的第一次崩堤 |
| 章首钩子 | 环境意象钩 | 凌晨三点的台灯与城市寂静 |
| 章末钩子 | 留白余韵钩 | 外套咖啡渍味，江晚埋脸进去 |
| 信息密度 | 极高但极克制 | 情感决堤但绝不挑明，为 Ch.8-9 蓄力 |
| 潜台词密度 | 高 | "不用你管"/"嗯"/"你为什么现在才来" |
| 关系推进 | 暗流涌动→情感决堤 | 第一次非工作场景独处，第一次顾屿克制出现裂痕 |
| 伏笔推进 | FS-004推进/FS-006埋设/FS-005回收 | 草稿窗口习惯动作、药盒手写说明、住址来源留白 |

## 4. 生成流程

建议使用 `/flow_full` 完整流程（言情章对潜台词精度与情绪颗粒度要求高，04 精修是核心，不宜走快速通道）。

| 步骤 | 操作 | 输出文件命名 |
|------|------|--------------|
| 03a 扩写 | 基于 ch05_细纲.md 生成骨架+血肉 | `ch05_03a_初稿.md` |
| 03b 质检 | P1/P2 全量 + 5维评分 + 流程选择 | `ch05_03b_质检报告.md` |
| 04 精修 | Show Don't Tell / 潜台词深化 / 情绪颗粒度细化 | `ch05_04_精修稿.md` |
| 05 去AI味 | 11 Gate 全量执行（重点：情绪标签堆砌 Gate） | `ch05_05_终稿.md` |
| 脚本检测 | 确定性脚本批量运行 | `ch05_检测报告.md` |
| 评分 | 10维评分 + diff 对比 | `ch05_评分.json` + `ch05_diff.md` |

注意：言情章为心理直给/解释腔单一责任人（04 唯一执行点），情绪标签堆砌需 05 重点检测。情绪工程是首要目标。

## 8. 已知限制

- 本章不验证：动作场景描写能力（言情章无打斗）、推理留白（属 ch04 验证范围）、爽点爆发（属 ch06/ch09 验证范围）。
- 已知局限：言情章的"情绪颗粒度"评判存在主观性，脚本只能检测标签堆砌，微表情/微动作的精准度需人工复核；"心疼"锚点的命中度依赖读者主观感受，脚本 --anchor 参数只能验证关键词覆盖。
- 与其他样本关联：本章 FS-004（顾屿未发送草稿习惯动作）为 Ch.8 草稿揭露埋线，Ch.8 须验证草稿伏笔回收；FS-006（药盒手写说明涂改）为后续犹豫线索；FS-005（"早点回去"）本章回收。
- 本章使用独立言情设定档案（demo_言情设定.md），与 ch06/ch07/ch08/ch09 的 demo_设定.md 体系不同，不参与跨设定伏笔一致性比对。
- 本章定位为全书情感转折点，对 04 精修的依赖度最高，03a 初稿评分预期偏低，04 精修后预期跃升至 80+。
