# 产出样本验证体系

> 本目录用于存放03→04→05三阶段的标准产出样本，作为规则有效性的验证基线和AI对标参考。

## 目录结构

每个样本章节包含以下文件：
- `XX_细纲.md` — 输入：细纲+衔接包
- `XX_03_初稿.md` — 03扩写产出
- `XX_03_自检报告.md` — 03自检报告
- `XX_04_精修.md` — 04精修产出
- `XX_04_精修意图包.md` — 04交接包
- `XX_05_终稿.md` — 05去AI味产出
- `XX_05_检测报告.md` — 05的11 Gate检测报告
- `XX_diff对比.md` — 三阶段逐段对比标注（红删绿增+对应规则编号）

## 样本用途

1. **验证规则有效性**：对比03初稿→04精修→05终稿的质量提升，验证规则是否有效
2. **AI对标参考**：03扩写时参考样本中的写法，05去AI味时参考样本中的改法
3. **回归测试基线**：规则迭代后，用同样输入重新生成，对比是否产生退化

## 样本选择标准

- 选择涵盖多种场景类型的章节（战斗/对话/日常/情绪/新角色登场）
- 选择有典型AI味问题的初稿（验证05去AI味效果）
- 样本一旦建立，不随规则迭代修改（作为历史基线）
