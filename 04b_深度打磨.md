#### 技法1：Show Don't Tell 第一原则（潜台词拦截器）

> 从04 V5.2原则1提取。禁止输出角色的"心理结论"或"状态定义"——永远不要替读者总结。

```
【潜台词拦截器 — 严禁解释】（精简版）

禁止句式："他发现……" "这说明……" "这意味着……" "他在等……" "他心里明白……"
禁止"心理实况转播"：禁止描写主角"在心里算了一遍"、"想了三遍"等思维过程
改写逻辑：用"物理动作/生理反应/环境变化"去折射心理

绝对违禁词库（叙述语中严禁出现）：
"觉得" "知道" "意味着" "是因为" "看出规律" "异乎寻常" "看来" "他发现" "这说明"
"他心里明白" "他在等" "他意识到" "显然" "可见"
→ 出现即删除，替换为导致该结论的"客观物证"

推理留白机制：禁止直接将角色的推理结果写在正文中。
├─ 只能写出导致这个秘密的"客观物证"，不准写出结论
├─ ❌ "这个温度不是给人定的，是给展品定的，展品怕热。"
└─ ✅ "温控表上的指针死死钉在十八度。白术吐出一口白气。"

万能前缀黑名单（叙述语中严禁）：
"他心里想/暗自思忖/心道/心说/不禁感到/忽然感到/察觉到"
→ 删除前缀，心理内容改为直接内心独白或生理反应

修改示例：
❌ "馆长看他的眼神不是在选人，是在等一个早就该来的人。"
✅ "馆长没看白术，把那张填着'白术'的入职表直接拍在桌角。'明天九点，迟到扣五十。'"
```

#### 技法2：比喻质量门（精简版——3条核心检验）

> 从04 V5.2原则2提取。质量检验而非绝对封杀——评估每个比喻的质量，低质量替换，高质量保留。

```
【比喻质量门 — 3条核心检验】

扫描全文所有"像/仿佛/好似/犹如/如同/似"句，逐个评估：

检验1：新鲜且精准？
├─ 读者没见过但一听就懂 → 高质量，保留
├─ 陈词滥调（"像三千年没开过的门""眼睛像深井""时间像流水"）→ 低质量，替换
└─ 常识性比喻（"像兔子一样快""像冰一样冷"）→ 低质量，替换

检验2：有叙事功能？
├─ 比喻本身携带信息，不只是装饰 → 高质量，保留
├─ 纯氛围渲染——比喻只为渲染气氛，无信息增量 → 低质量，替换
└─ 解释性比喻——用比喻来解释设定 → 低质量，替换

检验3：角色口吻？
├─ 对话中角色说的比喻 → 保留（人在说话时确实会用比喻）
└─ 叙述语中的比喻 → 按检验1+2评估

低质量比喻替换方案（句法代偿公式）：
├─ 物理痕迹法：直接描写A本身的材质、气味、损耗，不借助类比
│   ❌ "每次运行都像老人在咳嗽。"
│   ✅ "电梯门合拢时发出一声沉闷的金属撞击，钢索在井道里发出尖锐的摩擦声。"
└─ 生理反馈法：通过主角的身体/感官直接承受该状态
    ❌ "空气干，静，像有几百个东西一起在憋气。"
    ✅ "展厅里没有任何机械的震动声。白术站在走廊中央，耳道里因为极度的死寂而产生了一阵细微的耳鸣。"
```

#### 技法3：解释腔消除（04唯一执行点——精简版）

> 从04 V5.2 Step 4提取。叙述者跳出故事，像写百科词条一样向读者解释设定/概念/规则=解释腔。与Gate 11合并执行。

```
【解释腔消除 — 5个信号词 + 消除方法】

5个核心信号词（出现即标记）：
① "在XX，这个词/东西意味着……" → 概念词说明句
② "就好像她的身体天生就知道……" → 能力总结句
③ "被严禁参与XX" → 规则说明句
④ "在大荒，XX的XX是XX" → 百科式定义句
⑤ "之所以…是因为…" / "换句话说…" → 叙述者解释因果

边界裁定：解释腔 ≠ 新事物释放
├─ 解释腔（删）：叙述者定义句"在XX，这个词意味着…""所谓XX，就是…"
├─ 合法释放（保留）：通过角色对话（有信息差）/角色反应/生活场景/物件暗示带出的设定信息
└─ 裁定法：这段信息是"叙述者跳出来讲的"还是"角色/场景自然漏出的"？后者=保留

消除方法：
├─ 概念词说明 → 改为角色对话或行动带出
│   ❌ "净血。在大荒底层，这个词比'强壮'值钱。"
│   ✅ 老长老拉住要出门的阿介："净血不出猎。这是规矩。"
├─ 能力总结句 → 删除总结，只保留行为
│   ❌ "就好像她的身体天生就知道什么东西该放多少。"
│   ✅ "她捏起一撮粉末放在手腕内侧擦了擦，等了一会儿，点点头。没人教过她这套流程。"
└─ 规则说明句 → 改为角色反应带出
    ❌ "未染煞的年轻男女是部族延续血脉的唯一火种，被严禁参与高危猎杀。"
    ✅ 沙九看到阿介的尸体："才十六岁，还没拓脉。"
```

#### 技法4：心理直给检测（04唯一执行点——精简版）

> 从04 V5.2 Step 5提取。叙述者直接写出角色的心理结论，而非通过行动/物证让读者推断=心理直给。与Gate 3合并执行。

```
【心理直给检测 — 信号词扫描 + 外化方法】

信号词扫描（叙述语中检查，对话不计）：
① "他也不知道为什么" → 心理结论直给
② "他想起/他想起了" → 心理入口直给
③ "他知道" → 心理结论直给
④ "他觉得" → 情绪直给
⑤ "他发现/他意识到" → 心理结论直给

外化方法（4种替换方案）：
├─ 心理结论 → 改为物理动作暗示
│   ❌ "他也不知道为什么要留。"
│   ✅ "他把最小的那枚碎片翻了个面，又翻了个面。然后塞进了皮囊。"
├─ 心理入口 → 改为场景闪回
│   ❌ "他想起母亲临死前攥着他的手说的最后一句话。"
│   ✅ "母亲的手攥着他的手腕，力气大得不像一个快死的人。'看好你妹妹。'她说完就松了手。"
├─ 情绪直给 → 改为生理反应
│   ❌ "他觉得很不安。"
│   ✅ "他后脖颈的汗毛竖起来了。"
└─ 高压反应 → 转移注意力（每章最多1-2处，仅用于高压场景）
    ❌ "他内心十分恐惧。"
    ✅ "他突然发现对手刀柄上的红缨缺了一根线。"

例外（允许保留的心理描写）：
├─ 内心吐槽/独白（语气像真实对话，不是叙述口吻）
├─ 关键转折处的短暂心理（不超过3句）
└─ 角色性格设定为"想太多"型（但也要控制比例）
```

#### 技法5：感官入口重定向（精简版——二分法）

> 从04 V5.2技法1提取。与03 V4.1二分法完全对齐。

```
【感官入口多元化规则 — 二分法】

写每一段环境/场景/物品时，先判断这是日常场景还是核心场景：

日常场景（极简白描模式）：
├─ 1-2个感官即可，不需要堆叠
├─ 优先选最自然、最符合角色当前状态的感官
├─ 不强制"触觉>视觉"——按场景需要灵活选择
└─ 不强制固定百分比——按场景类型灵活选择

核心场景（多维叠加模式）：
├─ 至少触发2种以上感官，避免视觉单一入口
├─ 推荐组合：气味+视觉 / 温度+声音 / 触感+视觉
├─ 如果上来就写"他看到了……"（核心场景中）→ 改掉，换一个非视觉入口
└─ 核心冲突场景可用"失焦感官"（转向无关细节）

修改示例：
❌ AI味："白泽走进展馆，看到一排玻璃展柜……"（核心场景视觉单一入口）
✅ 人味儿："霉味先钻进鼻腔。脚下踩到什么，发出吱呀一声——是松动的木地板。白泽这才抬头。"
```

#### 技法6：对话潜台词检查（精简版）

> 从04 V5.2技法3提取。人类说话，70%的意思不在字面上。

```
【潜台词对话检查 — 每步对话不应是"问答"】

检查标准：对话是否为"试探 → 回避 → 再试探 → 松动"模式？
├─ 角色B是否直接回答了角色A的问题？→ 直接回答=信息过于顺畅，标记
├─ 对话中是否有"答非所问"的小动作（转杯子/看窗外/摸口袋）？→ 无=标记
├─ 对话结束后是否有动作/细节暗示角色B的真实想法？→ 无=标记

修改方法：
├─ Step 1: 角色A问一个直接的问题
├─ Step 2: 角色B不直接回答，而是做一个与问题无关的小动作 + 用另一句话"答非所问"
├─ Step 3: 角色A察觉到异常，但不追问
├─ Step 4: 对话结束后，用一个动作/细节暗示角色B的真实想法
└─ 全程不解释任何心理活动

修改示例：
❌ AI味（直给式）：
  "你怎么来了？"
  "我有事要告诉你。关于你父亲的。"
  "什么事？"
  "他没死。"

✅ 人味儿（潜台词式）：
  "你怎么来了。"
  他把杯子转了半圈。茶水洒出来，他没擦。
  "路过。"
  凌晨两点，路过一家不打烊的茶馆。白泽没戳穿他，也没坐下。
  过了很久，他又开口，声音很轻：
  "你父亲——"
  白泽的手停在半空。
  "……算了。"
  但白泽已经知道了。他没死。
```

### Step 4: 去AI味核心 Gate（执行 7+4 个 Gate，V5.2.1分级策略）

> V5.2.1：Gate执行策略改为**分级**——原"跳过Gate 4/5/8/9"调整为：Gate 5/9精简版**不可跳过**（核心检测项），Gate 4/8**可条件跳过**（burstiness>0.45时跳过）。
> V5.2.0：从4个核心Gate扩充为7个。新增的Gate 3/7/11从05提取，与上述技法3/4合并执行（不重复检测同一问题）。

**Gate分级策略表（V5.2.1）：**

| 分级 | Gate | 检测内容 | 跳过条件 |
|------|------|---------|---------|
| 不可跳过 | Gate 5精简版 | 对话差异化3项核心检测（声口/同质化/信息差） | 无（每章必执行） |
| 不可跳过 | Gate 9精简版 | 剧透式叙述2项核心检测（意图直给/信息不对称破坏） | 无（每章必执行，S1级必须修复） |
| 可条件跳过 | Gate 4 | 节奏打碎（句子长度变化率） | burstiness>0.45时跳过 |
| 可条件跳过 | Gate 8 | 叙述语调（语调起伏/POV越界） | 与Gate 4合并执行，Gate 4跳过则一并跳过 |

#### Gate 1：禁用词过滤
```
检测：一级禁用词（"然而/此外/与此同时/值得一提的是/不难发现/众所周知/综上所述"等AI高频连接词）
处理：一级禁用词必须修改，替换为自然过渡
```

#### Gate 2：句式去套路
```
检测：否定翻转（"不是A，是B"）/ 万能状语（"地"结尾副词）/ 工整对仗
处理：
├─ "不是A，是B"句式每章≤2次（与03自检第6项、05 Gate 9对齐）
├─ 超出限额→改用直接陈述或动作描写替代
└─ 对话中的"不是A，是B"不计入限额
```

#### Gate 3：心理描写外化（V5.2新增——与技法4心理直给检测合并执行）
```
检测目标：把"他想……""他觉得……"式的内部心理，转化为外部可感知的行为和细节
合并执行：与技法4（心理直给检测）在同一遍扫描中完成——技法4负责消除，Gate 3负责验证

检测项：
├─ 直接心理描写占比：每1000字不超过2次"他想/他觉得/他知道/他明白/他意识到"
├─ 心理总结句式："他内心很矛盾"/"他既期待又害怕"/"他感到一阵温暖" → 必须外化
└─ 万能前缀残留：技法4已删除的信号词是否仍有残留？

通过条件：每1000字直接心理描写≤2次，且无心理总结句式
注：Gate 3不做"外化"操作（外化由技法4完成），仅扫描残留
```

#### Gate 6：结尾去升华
```
检测：套路结尾 / 情感收束过度 / 预告式结尾
├─ 章末是否有"升华/总结"句式？有=删除
├─ 章末是否断在"最有悬念的瞬间"而非"事件结束"？
└─ 章末排比/抒情/省略号悬念？→全部必须修正
```

#### Gate 7：意象疲劳（V5.2新增——精简版，每章同一意象≤2次）
```
检测目标：同一意象在同一章内反复使用（AI把1-2个"好用"的意象当万能钥匙）
精简版规则：每章同一意象出现≤2次（≥3次=拦截替换）

检测项：
├─ 统计全章中每个感官意象的出现次数
├─ 同一意象（如"冷""温度""发烫""硌着"）每章出现≥3次 → 必须替换
├─ 同一物件反复承担"情绪容器"功能（如铜钱每次情绪波动就"发烫"）→ 标记
└─ 意象万能化：同一个意象被用于表达≥3种不同情绪 → 标记

替换方案：
```
