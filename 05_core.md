## 核心 Gate 1：禁用词过滤

> 全量扫描一级+二级禁用词。这是最基础的去AI味步骤。

### 一级禁用词（必须删除或替换）
- "不禁"、"缓缓"、"微微"、"淡淡"、"静静"、"默默"
- "竟然"、"忽然"、"似乎"、"仿佛"、"犹如"
- "一丝"、"一缕"、"一股"、"一阵"（情绪/感觉描写中）
- "嘴角微微上扬"、"眼中闪过一丝"、"心中一动"

### 二级禁用词（标记后视上下文替换）
- "不可思议"、"难以置信"、"前所未有"
- "与此同时"、"话分两头"、"另一边"
- "刹那间"、"瞬间"、"霎时"（连用时）
- "无可奈何"、"不由自主"、"情不自禁"

### 替换原则
1. 禁用词 → 删除（如果删除后语义不变）
2. 禁用词 → 白描动作（如"微微一笑"→"嘴角动了动"）
3. 禁用词 → 感官细节（如"一阵寒意"→"后脖颈的汗毛竖了起来"）
4. 保留：角色口头禅中合理使用的不禁/缓缓等（白名单豁免）

## 核心 Gate 2：句式去套路 + 对话去腔调

> 检测AI最典型的句式套路和对话腔调。

### 句式套路检测（~19项）
1. **万能前缀**："他知道/他明白/他清楚"开头的句子 → 替换为行为展示
2. **万能转折**："然而/不过/但是"连续出现 → 打破转折链
3. **万能递进**："不仅...而且...更..." → 拆分为独立句
4. **万能总结**："总而言之/总的来说/换句话说" → 删除
5. **排比三连**：三个结构相同的短句连排 → 打破其中一个
6. **情绪套娃**："他感到[情绪]，这让他更加[情绪]" → 外化为动作
7. **转折过度**：一段话中超过2个转折词 → 减少
8. **时态跳跃**：过去时和现在时混用 → 统一
9. **被动泛滥**：连续3+被动句 → 改主动
10. **比喻连缀**：一段话中超过2个比喻 → 保留最佳
11. **形容词堆叠**：名词前3+形容词 → 精简到1-2个
12. **副词修饰情绪**："愤怒地/悲伤地/开心地" → 删副词+加动作
13. **程度副词链**："非常/十分/极其/特别" → 删除或具体化
14. **主语重复**：连续3句同一主语 → 省略或换主语
15. **因果过度**："因为...所以..." → 暗含因果
16. **条件假设**："如果...那么..." → 拆分
17. **定义式句子**："[名词]是[定义]" → 改为场景展示
18. **解释式插入**："——也就是说——" → 删除
19. **列举式描写**："有A，有B，有C" → 改为有层次的描写

### 对话去腔调检测（~6项，原Gate 5并入）
1. **翻译腔**："你是在说...吗？" → "你意思是...？"
2. **说明腔**：对话中直接解释设定 → 拆分到叙述中
3. **翻译式倒装**："我不得不说..." → "我得说..."
4. **万能回应**："我明白了/原来如此/原来是这样" → 替换为具体反应
5. **情绪标签对话**："[情绪]，"他说 → 删情绪标签+加动作
6. **信息转述**：对话中复述前文已写的事件 → 删除

## 核心 Gate 3：心理描写残留扫描

> 04核心Step 2已做外化操作，05仅扫描残留。不做外化（避免重复改写）。

### 扫描项（~8项）
1. "他感到[情绪]" 残留 → 标记（不替换，交回04处理）
2. "他心想[想法]" 残留 → 标记
3. "他觉得/他认为/他意识到" 残留 → 标记
4. "一股[情绪]涌上心头" 残留 → 标记
5. "内心深处" 残留 → 标记
6. 情感连贯性检查（朱雀C融入）：情绪变化是否有因果链
7. 情绪标签密度：每千字情绪直给词≤3个
8. 残留标记≥5处 → 建议回退04重新处理

## 核心 Gate 4：结尾去升华

> 与04核心Step 3的标点收尾互补——04管标点，05管结尾模式。

### 检测项（~12项）
1. 哲理升华结尾 → 替换为动作/感官收尾
2. 感悟总结结尾 → 替换为对话/物件收尾
3. 时间跳瞰结尾 → 替换为当下时刻收尾
4. 命运修辞结尾 → 删除
5. 情绪标签结尾 → 替换为生理反应
6. "未完待续"式结尾 → 删除（由断章钩子替代）
7. 读者喊话结尾 → 删除
8. 预告式结尾 → 删除
9. 对仗工整结尾 → 打破对仗
10. 主题点题结尾 → 删除
11. 感叹号收尾 → 替换为句号
12. 省略号收尾 → 限制使用（每3章≤1次）

## 核心 Gate 5：剧透式叙述与信息不对称检测

> AI最隐蔽的问题：叙述者替读者下结论，提前剧透角色尚未感知的信息。

### 检测项（~15项）
1. **意图直给**："他这样做是为了..." → 删除意图解释，让行为自己说话
2. **结果预告**："他不知道的是，这个决定将改变..." → 删除预告
3. **全知插入**：叙述者知道角色不知道的信息 → 删除或改为角色视角
4. **情绪预判**："此时的他还不知道，等待他的将是..." → 删除
5. **伏笔明示**："这个细节后来证明至关重要" → 删除
6. **命运暗示**："命运的转折点就在此刻" → 删除
7. **对比剧透**："如果他选择了另一条路..." → 删除
8. **信息不对称检测**：每场对话是否有信息差？（详见04_技法_潜台词.md）
9. **叙述者越权**：叙述者替角色做判断 → 改为角色感知
10. **时间线剧透**：提及未来事件 → 删除
11. **因果倒置**：先说结果再说原因 → 调整为顺叙
12. **主题先行**：开篇点题 → 删除或后置
13. **角色评价**：叙述者评价角色 → 改为通过他人视角呈现
14. **读者引导**："读者可能已经注意到..." → 删除
15. **悬念破解**：刚设悬念立刻解释 → 拉开距离

## 核心 Gate 6：情节信息含金量与注水率检测

> 拦截AI最隐蔽的"高级废话"——文字优美但零信息推进。

### 检测项（~6项）
1. **哲学思辨型废话**：角色长段内心哲学思考但不推动剧情 → 压缩至2-3句或删除
2. **感官堆叠无推进**：纯环境/感官描写但角色未行动、未决策 → 压缩或删除
3. **心理漫游无增量**：角色回忆/联想但不提供新信息 → 压缩
4. **对话空转**：对话漂亮但双方关系/信息/目标无变化 → 删除或压缩
5. **重复信息**：同一信息用不同方式表述2+次 → 保留最佳版本
6. **语义连贯性**（朱雀A融入）：段落间语义承接是否自然

### 脱水两步法
1. 标记所有"零推进"段落
2. 压缩为1-2句关键信息或删除（受删除比例上限约束）

## 核心 Gate 7：网络梗检测（V6.0.1新增 — advisory级）

> **定位**：advisory级检测，不阻断输出，仅在执行确认字段中报告。
> **触发条件**：题材为都市/职场/系统/轻小说/二次元/爽文时自动执行；悬疑/历史/严肃文学跳过。
> **素材库**：`references/网络梗素材库.md`（S/A/B/C时效分级 + 题材路由）
> **核心约束**：网络梗只能用于角色对话/内心吐槽/围观反应，**禁止进入叙述者总结层**。

### 检测项（~6项，advisory级）
1. **密度检测**：每章网络梗数量统计
   - 核心章≤2处 / 日常章≤3处 / 过场章≤1处
   - 超出→advisory报告（不阻断，提示作者审视）
2. **堆砌检测**：连续2处以上网络梗且语义重复
   - 命中→blocking级（这是唯一blocking项，防止AI堆砌变模板）
3. **题材冲突检测**：悬疑/历史/严肃文学题材中使用网络梗
   - 命中→blocking级（题材路由封锁，违反=出戏）
4. **叙述者越权检测**：网络梗出现在叙述者总结层（非角色对话/内心吐槽）
   - 命中→blocking级（叙述者用网络梗总结是AI味典型特征）
5. **时效性标记**：B/C级网络梗在长连载（>50章）中标记过期风险
   - 命中→advisory报告（提示作者考虑替换为S/A级或新梗）
6. **角色口吻匹配**：网络梗是否符合角色声口（青少年/市井角色可用，严肃角色慎用）
   - 不匹配→advisory报告（提示作者调整角色口吻或更换表达）

### 执行规则
- advisory级发现：记录在执行确认字段，不阻断定稿
- blocking级发现（堆砌/题材冲突/叙述者越权）：必须修正后才可定稿
- 题材路由封锁章节（悬疑/历史/严肃）：跳过本Gate，不检测
- 网络梗素材库未加载时：跳过本Gate，在执行确认字段标注"未加载素材库，跳过检测"

### 与其他Gate的关系
- Gate 2（句式去套路）：网络梗不视为套路句式，独立检测
- Gate 3（心理描写残留）：内心吐槽中的网络梗不视为心理直给，独立检测
- Gate 6（信息含金量）：网络梗不视为零推进，独立检测
