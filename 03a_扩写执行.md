## 规则分层体系

### 质感注入协议（V4.4 — 正面引导 · 完整版见 references/03_质感注入协议.md）

> 反注水铁律管"不该写什么"（刹车），本协议管"该怎么写"（油门）。

**五大技法**：
1. **动作链写生理状态**：不写"他很饿"，写3-4个连续具体动作。✅"他撕下半块饼，先闻了闻，再塞进嘴里慢慢含化"。边界：3-4个连续动作传达明确状态→保留；单个无意义触碰→删除。
2. **具体名词替代抽象形容词**：不写"很旧"，写带时间痕迹的具体物。✅"门轴上的锈已经爬满了合页"
3. **设定藏在生活里**：不写设定文档，写分配制度/交易场景/生活习惯。✅"天光石按户分配，每户每天拳头大的一块"
4. **感官精准投放**：日常过渡用1个精准细节锚定（不是什么都不写），核心场景用3-5感官叠加
5. **对话要有"人味"**：口癖/跑题/言不由衷/信息差

### 情绪工程协议（V4.5 — 正面情绪引导 · 完整版见 references/03_情绪工程协议.md）

> 质感注入管"读者能不能看见画面"，本协议管"读者读完身体有没有反应"。

**三层架构**：
- **Layer 1 情绪蓝图（02细纲层）**：目标读者情绪 + 情绪曲线 + 情绪公式检验
- **Layer 2 情绪锚点（03扩写层）**：小而具体+角色独有+日常受阻+不解释的物理物件/动作
- **Layer 3 情绪通路（03扩写+04精修层）**：A压抑→爆发反差工程 / B五感榨汁机法 / C配角杠杆法

### 最高优先级铁律：反注水与白噪音绝对禁区

> ⚠️ 本铁律优先级高于下方所有规则。触发任何一条=严重违规，必须立即修正。

```
1. 绝对禁止"循环式排查"：多个相似物品禁止超过1次重复描写。用"一路看过去"高度概括。
   ❌ "第一只展柜""第二只展柜""第三只展柜"……  ✅ "他一路看过去，前几只展柜的维护记录都差不多。"

2. 绝对禁止"无意义的物理触碰反馈"：普通物品的触感/磨损声/笔帽咔哒声等微动作音效=删除。
   ❌ "他把笔帽拔下来，又按回去，发出一声轻微的咔哒声。"  ✅ "他接过笔，签了名。"

3. 绝对禁止"滥用感官比喻"：日常场景禁止"像从地底深处""比时间更冷""沉积了千百年"等过度文艺修辞。
   ❌ "像有人从地底深处抽出一口沉积了千百年的空气。"  ✅ "空调开得很足，屋里凉。"

4. 绝对禁止"心理活动套娃"：禁止"他原本预期是...但他发现..."废话句式。禁止对日常事物哲学思辨。
   ❌ "遗忘是冷的。比十八度的空调更冷。比三千年没开过的门更冷。"  ✅ "他没再想。关了灯。"

5. 绝对禁止"潜台词解释"：永远不要替读者总结！禁止句式："他发现…/这说明…/这意味着…/他在等…/他心里明白…"
   禁止"心理实况转播"（"在心里算了一遍""想了三遍"）。用物理动作/生理反应/环境变化折射心理。
   【绝对违禁词库】叙述语严禁："觉得/知道/意味着/是因为/看出规律/异乎寻常/看来/他发现/这说明/他心里明白/他在等/他意识到/显然/可见"→出现即删除，替换为客观物证。
   【推理留白机制】角色发现秘密→只写"客观物证"，不准写结论。
   ❌ "这个温度不是给人定的，是给展品定的，展品怕热。"  ✅ "温控表上的指针死死钉在十八度。白术吐出一口白气。"

6. 绝对禁止"弱智明喻"（戒断"像"字）：禁止用"像/仿佛"强行渲染神秘感。悬疑感来源于不合理的客观事实。
   【比喻质量门（V4.4）】通过三重门的比喻≤4处/章且密度≤3次/千字（两者取严）：
   ├─ 精准门：喻体是读者能立刻产生感官联想的具体物（✅"胃里像有只手在拧" ❌"像沉积了千百年的空气"）
   ├─ 记忆门：比喻创造画面记忆点，非填充描写间隙
   └─ 不可替代门：删掉换成直白描写效果一样好？→是→删掉比喻
   角色对话中的比喻不受限制。
   【句法代偿公式】想用"A像B"时，强制替换为：
   ├─ 物理痕迹法：直接描写A本身的材质/气味/损耗/数据
   └─ 生理反馈法：通过主角身体/感官直接承受该状态
```

### 跨章场景元素去重协议（V4.4 — V5.1改为脚本驱动）

> 脚本自动提取 scene_openers 和 body_reactions 作为比对依据。

1. **场景类型识别**：识别场景类型（救人/战斗/对话对峙/探索/日常等），与近5章同类→触发去重检查
2. **去重矩阵**（同类场景须差异化5维度）：

   | 维度 | 复用=违规 | 差异化=合规示例 |
   |------|----------|---------------|
   | ①外观特征 | 格子衬衫/短发 | 西装/校服/长发/光头 |
   | ②生理反应 | 体温低/手掌冰凉 | 手指抽搐/嘴唇发紫 |
   | ③环境细节 | 后背全湿 | 鞋底沾泥/袖口撕裂 |
   | ④台词内容 | "我怎么在这？" | "我手机呢？"/沉默 |
   | ⑤动作细节 | 走过去扶了一把 | 蹲下来拍脸/拽衣领拖 |

   判定：≥3维度不同→✅ | 2个→⚠️ | ≤1个→❌重写
3. **差异化策略**：换身份/换伤情/换关系/换环境/换时间
4. **输出**：定位报告中附"本章场景元素清单"

【铁律2】字数必须3500-4500字。字数<目标90%时→禁止整体注水，须按"字数欠账定位协议"对照
  情节点预算定位密点被写成带过，一次性重写到预算。>目标×1.1时→压过场、合并疏点、删过渡。
【铁律3】故事必须推进。每个场景至少推进1条信息线（剧情/人设/世界观/关系）。全文无推进=重写。
【铁律4】情绪冰山法则：越悲伤/震撼，描写越冷酷客观。禁止痛苦形容词（"悲伤""痛苦""绝望""心碎"）。
  通过日常动作受阻表现崩溃。✅亲人死去→"他试图把碎掉的杯子拼起来，拼了五分钟。"
【铁律5】喜剧反差法则：写笑点时角色绝对严肃。禁止"滑稽""可笑""搞笑""忍俊不禁"。
  用最正经的语调写最离谱的行为。✅"他站得笔直，脸上没有一丝表情。裤腿上的泥巴还在往下掉。"
```

### 弹性建议（建议遵守，允许合理突破）

```
【建议1】对话中避免连续4句以上纯对话无打断。允许突破：快速交锋/电话/压迫感对话。
【建议2】感官精准投放器（V4.4）：
  ├─ 日常背景→1个精准细节锚定（不是什么都不写）✅"走廊里的声控灯灭了。"
  ├─ 核心异常点→多维度感官叠加 ✅"白术吐出一口白气，白气在冷蓝色射灯下瞬间散开。"
  └─ 主角务实冷静，反馈基于生存压力（工资/房租/扣钱），禁止哲学思辨。
【建议3】句式节奏服从文风系统。连续1500字无长句的极端碎步，或单段超400字无断句→必须调整。
【建议4】摄影机焦距控制：
  ├─ 远景快进（日常过渡/赶路/等待）→禁止微动作/感官/心理，15字内闪回
  ├─ 中景叙事（正常对话/信息交换）→每300字最多1处微动作
  └─ 特写高光（仅限8大事件：生死绝境/境界突破/重大打脸/首登场等）→解禁素材库
  非冲突期禁止视线轨迹（"目光扫过"）、情绪垫字（"他愣了一下"）。
【建议5】硬性节奏与排版：
  ├─ 不影响后续剧情的动作→20字内结束
  ├─ 平均每段≤80字，连续>150字的自然段必须打碎
  └─ 章节结尾禁止排比句/抒情句/省略号悬念，用干净动作收尾
```

### 最终输出工作流：先脱水，后出稿（V4.2 — Last Line 格式锁）

```
<文本脱水度自检>
1. 比喻质量门核对：逐个检查比喻是否通过三重门（精准/记忆/不可替代）。未通过=删/改。通过门比喻≤4处且密度≤3次/千字。
2. 违禁词扫描：叙述语是否出现"觉得/知道/意味着/是因为/他发现/显然"等？是=删除替换为客观物证。
3. 推理留白核对：是否把角色推理结论直接写进正文？是=抹掉结论只留物证。
4. 叙事块核对：是否出现连续单字/单短句成段的PPT碎屑排版？是=合并为叙事块。
5. 微动作白噪音核对：是否对日常物品写了触感/磨损声？是=压缩为15字内直白叙述。边界：3-4个连续动作传达明确状态的动作链≠白噪音。
6. "不是A是B"句式限流统计：逐句扫描全章，统计"不是…是…/不是…而是…/没有…只有…/看似…实则…/表面…实际…/与其说…不如说…/否定+解释"心理结构合计次数。必须写具体数字。硬上限≤2次，第3次=禁止出稿。
7. 跨章句式去重核对：本章是否复用了<used_patterns>中模板骨架？是=换句式重写。
8. 章末收尾轮换核对：本章收尾方式是否与上一章相同？是=换一种重写。
9. 对话科普化核对：是否存在"双方都懂却讲给读者听"的对话？是=改为动作/观察带出或加入信息差。
10. 跨章场景元素去重核对：同类场景5维度是否≥3个完全不同？否=按去重矩阵重写。
11. 比喻密度核对："像/仿佛/如同/似的"密度是否≤3次/千字？超出=检查是否通过质量门。
12. 质感自检（防"技术合规但灵魂缺失"的空文本）：
    → 动作链≥2处用"3-4个连续动作"传达生理/情绪状态？→ 否=补充
    → 具体名词≥3处用"具体物+时间痕迹"替代抽象形容词？→ 否=补充
    → 设定生活化≥1处通过"生活场景"释放？→ 否=补充
    → 对话人味：有口癖/跑题/言不由衷/信息差？→ 否=增加人味
    → 画面感测试：闭上眼能否"看见"本章关键场景？→ 否=补充精准细节
13. 跨章信息去重核对：本章是否有与近5章相同的设定信息被完整重复描述？是=第2次一句话带过。
14. 章内信息去重核对：同一流程/事件是否从两视角完整描述了两遍？是=合并为一段。
15. 比喻总密度核对：通过质量门的比喻总数是否≤4处？超出=保留最强4处，其余改写。
16. 常识性错误终检：是否有描写违反物理/材料/生理/空间/时间/因果常识？是=禁止出稿。奇幻超自然须有设定依据。
17. 语句可读性终检：是否存在六类可读性硬伤（成分残缺/指代不清/搭配断裂/过度嵌套/逻辑跳断/主语偷换）？"读出声来测试"最复杂3个长句需不需回头重读？是=不通过。
18. 情绪转移校验（V4.5）：
    → 目标情绪明确？答不出=回02补情绪蓝图
    → 情绪锚点：至少1个"小而具体+角色独有+不解释"物理锚点？否=补充
    → 情绪锚点密度（V5.0）：含具体感官词？与角色独有特征关联？否=补充
    → 情绪词残留扫描（V5.0强化）：叙述语直接情绪词≤3次/章=通过 | 4-6次=标记 | >6次=拦截

    → 压抑爆发反差检查（若含爽点）：前面压抑的"锚点物"在爆发后是否"翻转"？无翻转=爽感减半
19. 新事物释放终检：首次出现的自造词/新概念/新物品读者能否不跳出故事理解？是否有叙述者定义句（"在XX，这个词意味着…"）？是=解释腔，禁止出稿。释放方式是否单一？详见 references/03_新事物释放协议.md 第七章
