# 04 标点收尾与角色指纹校验（V5.4 — 从04完整版拆分）

> **加载条件**：章末收尾/对话密集场景/每章必检
> **加载方式**：Read `references/04_标点收尾.md`

## 标点统计与修正

### 标点限额（每章）

| 标点 | 上限 | 超限处理 |
|------|------|---------|
| 省略号（……） | ≤5 | 替换为句号/破折号/动作描写 |
| 破折号（——） | ≤4 | 替换为逗号/句号/独立句 |
| 感叹号（！） | ≤15 | 替换为句号+动作/语气词 |

### 标点超限修正策略

**省略号超限**：
```
❌ "你……是不是……"她声音发抖……
✅ "你是不是——"她声音发抖，咽了口唾沫。
```

**破折号超限**：
```
❌ 他站起来——椅子倒了——没人去扶——空气凝固了
✅ 他站起来，椅子倒了。没人去扶。空气凝固了。
```

**感叹号超限**：
```
❌ "住手！"他大喊！"你疯了吗！"
✅ "住手。"他喊了一声，声音不大，但所有人都停了。"你疯了吗。"
```

## 结尾去升华

> 章末最后3句禁止"升华式"收尾。

### 禁止的结尾模式

| 类型 | 禁止示例 | 原因 |
|------|---------|------|
| 哲理升华 | "这不仅是勇气的考验，更是命运的抉择" | 空洞+AI味 |
| 感悟总结 | "他终于明白了，所谓的成长就是……" | 叙述者替读者下结论 |
| 时间跳瞰 | "从此以后，他的生活再也回不去了" | 跳出叙事时间线 |
| 命运修辞 | "命运的齿轮开始转动" | 套话+零信息 |
| 情绪标签 | "他心中充满了前所未有的决心" | 心理直给 |

### 推荐的结尾模式

| 类型 | 示例 | 要点 |
|------|------|------|
| 动作收尾 | 他把刀放回桌上，转身走了。门没关。 | 具体动作+留白 |
| 感官收尾 | 雨还在下。他听见远处传来一声狗叫。 | 感官细节+悬置 |
| 对话悬念 | "你明天还来吗？"她问。他没有回答。 | 对话+未回应 |
| 物件特写 | 桌上的茶已经彻底凉了，杯壁上的水雾干了一半。 | 物件变化映射情绪 |
| 断裂收尾 | 他刚迈出第一步，手机响了。 | 突然中断+钩子 |

## 角色语言指纹校验

### 遮名测试

> 遮住角色名，能否通过对话辨识出是谁？

**测试方法**：
1. 从本章对话中随机抽取3句，遮住说话者名字
2. 逐一检查：仅凭对话内容/语气/用词，能否判断是谁说的？
3. 不能判断 → 该角色需要增加口癖/句式特征

**口癖建立清单**：
| 维度 | 示例 |
|------|------|
| 口头禅 | "话又说回来"/"你懂我意思吧"/"得了吧" |
| 句式偏好 | 短句/长句/反问句/条件句 |
| 用词范围 | 书面语/口语/方言/行业术语 |
| 语气词 | "嘛"/"呗"/"啊"/"哈" |
| 信息组织 | 先结论后原因/先原因后结论/只说一半 |

### 口癖指纹法

> 每个主要角色必须有1-2个独有语言习惯。

**指纹建立规则**：
1. 主角和前3名配角必须有口癖指纹
2. 口癖在全书中保持一致（除非有角色成长理由变化）
3. 口癖不可太明显（"哈哈哈哈"不算口癖）
4. 口癖应在自然对话中流露，不是每次说话都带

### 情绪通路校验

> 角色的情绪变化是否有可追溯的因果链？

**校验规则**：
1. 从章首到章末，标记主角的情绪变化点
2. 每个变化点必须有"触发事件"（不可无原因变化）
3. 情绪变化幅度必须与触发事件匹配（小事件不可引发大情绪）
4. 禁止"平静→狂怒"无铺垫跳变（至少需要1个中间情绪状态）

**情绪通路模板**：
```
[情绪起点] → [触发事件1] → [中间情绪] → [触发事件2] → [情绪终点]
平静 → 同伴受伤 → 紧张 → 敌人嘲讽 → 愤怒
```

**违反示例**：
```
❌ 他很平静地喝着茶。突然他暴怒了，把桌子掀了。
   （无触发事件，无中间情绪）

✅ 他很平静地喝着茶。门被踹开的时候，他手里的杯子停了一下。
   来人说了句话。他放下杯子。杯底磕在桌面上，声音很响。
   然后他站起来，把桌子掀了。
   （触发事件：门被踹开+来人的话 / 中间情绪：停顿+放杯子的力度）
```
