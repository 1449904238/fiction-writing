# 题材钩子映射表（V5.4 — 24题材全覆盖）

> 02a细纲生成时根据 `novel_meta.genre` 启用对应题材的钩子和爽点结构。
> 使用方式：02a 读取本表对应题材段落 → 将其钩子类型纳入"章尾钩子/章中钩子"选型池 → 爽点结构作为节奏波峰设计依据 → 行为逻辑约束写入细纲生成行为限制。
> 与 `references/scene-trigger-map.md`、`references/方法论综合.md` 协同。

### 钩子类型使用说明

本表的"特色钩子类型"可同时用于章末钩子和章中钩子。章首钩子建议优先使用以下类型：
- 悬念延续型（上一章末钩子的直接延续）
- 冲突爆发型（开篇即冲突）
- 信息投放型（开篇即投放关键信息）

章末钩子建议优先使用本表列出的题材特色钩子类型，以增强题材辨识度。
3章内不重复同类钩子（详见 02b 细纲质检 Gate 3）。

## 1. 玄幻

### 特色钩子类型
1. **境界突破悬停** — 突破关键时刻切断，留"能否成功"悬念。例：丹田异象刚起，章末断在"成或败，就在此一举"。
2. **秘境将启** — 限时秘境/遗迹开启倒计时，制造紧迫感。例："三日后天火秘境开启，名额仅剩三枚。"
3. **宿敌压制** — 强敌登场以绝对实力碾压，留"如何翻盘"悬念。例：对方一掌震碎护体灵光，主角吐血倒飞。
4. **神秘传承** — 获得上古传承/血脉觉醒的瞬间留钩。例：识海中浮现一道苍老声音："吾之传承，有缘者得。"

### 爽点结构
扮猪吃虎→被小看→展露冰山一角→众人震惊→更大的危机压来。核心是"压制-反转-再压制"的螺旋升级，每3-5章一个爽点波峰，波峰之间用修炼/探索过渡填充。

### 行为逻辑约束
修仙界弱肉强食，主角可狠但不能无脑嚣张；实力差距>2个大境界时必须用智谋/底牌而非硬刚；突破须有代价（资源/机缘/顿悟），禁止无理由跳级；杀人夺宝须符合该世界道德逻辑，不可现代圣母心态。

## 2. 都市

### 特色钩子类型
1. **身份反差揭底** — 主角隐藏身份被逼迫到不得不亮明。例：众人嘲笑穷小子，直到他的黑卡刷不出限额。
2. **打脸连环** — 嘲讽者依次被打脸，最后一个最强。例：前女友、前老板、豪门公子接连栽跟头。
3. **权势碾压预告** — 抛出"你知道我是谁吗"但暂不揭晓。例：电话那头传来一个让所有人色变的声音。
4. **日常危机突袭** — 平静生活被突发事件打破。例：门铃响起，门外站着两个穿制服的人。

### 爽点结构
低调装逼→被挑衅→轻描淡写解决→围观者震惊→留下更神秘印象。核心是"信息差爽点"，主角知道而反派不知道的优越感，每章或隔章一个打脸点。

### 行为逻辑约束
都市背景须接地气，金钱/权力/人脉要有合理来源（重生预知/系统/隐藏身份等金手指须早交代）；主角不可全知全能，须有世俗羁绊（家人/工作/债务）；打脸后须有后续反扑，不能一帆风顺。

---

## 3. 悬疑

### 特色钩子类型
1. **尸体/线索异常** — 发现与常理矛盾的关键证据。例：死者手表停在死亡时间之后两小时。
2. **密室/不可能犯罪** — 制造"怎么可能"的困惑。例：门窗从内反锁，尸体却在房间中央。
3. **第二死者预告** — 凶手留下"下一个是谁"的暗示。例：桌上多了一张照片，照片上的人还活着……暂时。
4. **可靠叙述者崩塌** — 主角发现自己记忆/感知可能被篡改。例：他确信昨天见过的人，所有人说从未存在。
5. **倒计时危机** — 限时内必须破解否则后果发生。例："还剩12小时，第三个孩子就会死。"

### 爽点结构
疑点抛出→错误推理（红鲱鱼）→新证据推翻→真相反转→更大的谜。核心是"认知颠覆"，每次以为接近真相就被推翻，3-5章一个小反转，一卷一个大反转。

### 行为逻辑约束
线索必须公平展示，禁止结尾天降新证据；凶手动机须合理且提前埋线；主角推理须基于已知信息，不可"灵光一闪"破案；恐惧用克制笔法，留白胜过直写血腥。

## 4. 言情

### 特色钩子类型
1. **误会即将揭穿** — 真相差一步揭晓时被打断。例：他刚要解释，她已经转身走进雨里。
2. **心动瞬间悬停** — 暧昧气氛达到顶点时切断。例：两人距离不到一寸，门突然被推开。
3. **情敌/阻碍登场** — 第三方介入打破平衡。例：未婚妻挽着别人的手臂出现。
4. **身份秘密暴露风险** — 隐藏的身份/过去有被发现危险。例：她随手翻开那本相册，脸色变了。
5. **虐点反转预告** — 甜蜜后抛出即将到来的虐。例："如果有一天我骗了你，你会恨我吗？"

### 爽点结构
暧昧拉扯→小甜蜜→误会/阻碍→虐心→和解升级。核心是"情绪过山车"，甜虐交替，每次和解后感情更深一层；甜后必虐、虐后必甜，避免单一情绪疲劳。

### 行为逻辑约束
感情发展须有铺垫，禁止一见钟情直接在一起（除非设定支持）；虐须有合理原因，不可为虐而虐；主角情感选择须符合人设，不可工具化；对话要有潜台词，少直白告白，多用动作/环境外化情感。

## 5. 科幻

### 特色钩子类型
1. **技术失控预警** — 科技产物出现非预期行为。例：AI第一次在没有指令时主动开口。
2. **宇宙级威胁揭示** — 抛出超越个体认知的危机。例：望远镜里，那颗星星在一秒内消失了。
3. **意识/身份质疑** — 主角无法确认自己是否还是"自己"。例：他找到自己的备份，备份说"你才是复制品"。
4. **文明遗迹谜题** — 发现超出理解的外星造物。例：舱门打开，里面是一张人类的面孔。
5. **倒计时灾难** — 物理法则层面的倒计时。例："小行星将在72小时后撞击，我们有三次窗口。"

### 爽点结构
发现异常→科学推演→技术突破/应用→解决当前危机→揭示更深层宇宙真相。核心是"认知边界拓展"，每次解决问题都打开新认知维度，层层揭开世界真相。

### 行为逻辑约束
科幻设定须内部自洽，物理规则一旦确立不可随意违反；技术解决须基于已建立的设定，禁止天降黑科技；硬科幻注重推演过程，软科幻注重人文思考；新概念释放遵循 `references/03_新事物释放协议.md`。

---

## 6. 历史

### 特色钩子类型
1. **历史拐点干预** — 主角站在改变历史的关键节点。例：诏书还未盖上玉玺，他的手停在半空。
2. **朝堂暗流涌动** — 表面平静下的权力博弈。例：宴会上所有人都在笑，只有他注意到太子没动筷子。
3. **命运倒计时** — 已知历史事件逼近，主角能否改变。例："三天后就是靖难之役，他还没决定站哪边。"
4. **身份错位危机** — 穿越/卧底身份有暴露风险。例：有人喊了他的本名，满堂皆惊。
5. **史实伏笔回响** — 真实历史事件以伏笔形式埋入。例：那个被贬的小官，日后会成为首辅。

### 爽点结构
知晓历史走向→提前布局→改变小事件→引发蝴蝶效应→见证大历史改写。核心是"信息差+布局爽感"，用后世知识碾压当代，每卷一个历史节点改写。

### 行为逻辑约束
尊重历史大框架（架空可放宽），细节须考据；主角不可凭一己之力随意颠覆大势，须借势借力；历史人物刻画须立体，非脸谱化；权谋须有逻辑链，不可靠主角光环。

---

## 7. 无限流

### 特色钩子类型
1. **副本规则残缺** — 关键规则缺失，须用命去试。例："规则只说不能回头，没说回头会怎样——上一个回头的人没了。"
2. **倒计时死亡** — 副本内限时任务失败即死。例："还剩3分钟，门还没打开。"
3. **NPC异常** — 非玩家角色行为偏离设定。例：那个本该重复台词的NPC，突然叫出了他的真名。
4. **队友背叛预告** — 暗示队伍中有内鬼。例：通关条件是"活到天亮的人数≥3"，可他们有5个人。
5. **副本关联谜题** — 不同副本间的隐藏联系。例：这个副本的Boss，长得和上个副本里死掉的新人一模一样。

### 爽点结构
进入副本→试错摸规则→找到破局点→反杀/通关→获得奖励并发现更大的局。核心是"解谜+生存"双重爽感，每个副本是一个完整的起承转合。

### 行为逻辑约束
规则一旦公布不可暗改，须公平；死亡须有真实代价，不可无限复活（除非设定支持且有限制）；主角智慧体现在信息整合而非全知；队友须有独立人格，非工具人。

---

## 8. 规则怪谈

### 特色钩子类型
1. **规则自相矛盾** — 两条规则冲突，须判断哪条是真的。例："规则3：永远相信管理员。规则7：管理员已经死了。"
2. **违规后果延迟** — 触发违规但后果未即时显现。例：他看了看镜子里的自己，数了数——六根手指。
3. **规则补丁异常** — 新出现的规则暗示前规则有漏洞。例：墙上多了一行字："补充规则：请不要数自己的手指。"
4. **安全区崩塌** — 以为安全的地方不再安全。例："规则说卧室是绝对安全的——但规则没有说哪个是卧室。"
5. **认知污染渐进** — 主角感知逐渐不可靠。例：他开始分不清，刚才那句话是规则说的，还是自己想的。

### 爽点结构
阅读规则→分析矛盾→试探验证→发现真规则→逃出生天（或发现更深的怪谈）。核心是"逻辑推理+恐惧递增"，恐惧随认知不确定性上升而上升。

### 行为逻辑约束
规则须可推理，禁止无逻辑的天降死亡；恐惧来自认知不确定性而非血腥；主角须主动推理，不可纯靠运气活；每条规则都要在文中回收其含义，不可废规则。

---

## 9. 系统流

### 特色钩子类型
1. **系统升级/变异** — 系统本身发生变化带来新功能或新限制。例："检测到宿主濒死，系统进入暴走模式。"
2. **任务倒计时** — 限时任务失败有严厉惩罚。例："任务：24小时内击杀目标。失败惩罚：抹杀。"
3. **隐藏任务触发** — 意外触发稀有任务。例：他只是随手扶了那个老人，系统弹出金色提示框。
4. **系统vs系统** — 遇到其他系统宿主。例：对方的系统提示音和他的不一样。
5. **系统bug/漏洞** — 主角发现可利用的系统漏洞。例：如果把任务面板和商城同时打开……

### 爽点结构
获得系统→完成小任务→升级解锁→碾压同级→系统背后的秘密浮现。核心是"成长可视化+即时反馈"的爽感，每次升级都有明确数据反馈。

### 行为逻辑约束
系统规则须明确且一致，不可临时加设定；奖励须有梯度，禁止一步登天；系统不能替代主角思考，关键决策须人做；系统来源须有伏笔，不可永远是个黑箱。

## 10. 御兽

### 特色钩子类型
1. **御兽进化悬停** — 御兽进化关键时刻切断。例：蛋壳裂开一道缝，金光从缝隙里透出来——然后卡住了。
2. **契约失败/反噬** — 契约过程出现意外。例：契约阵法亮起，但光芒的颜色不对。
3. **隐藏血脉觉醒** — 御兽暴露真实血脉。例：所有人都说这是最普通的灰狼，直到它打了第三个喷嚏，长出了角。
4. **御兽联赛对手预告** — 强劲对手登场。例：卫冕冠军的御兽，是一头传说级的……
5. **御兽沟通异常** — 御兽传递了不该知道的信息。例：小鸟突然说了一句人话，内容是明天的彩票号码。

### 爽点结构
获得御兽→培养→对战获胜→进化→挑战更强。核心是"养成+对战"双重爽感，御兽越强主角越有面子，每次进化是一个大爽点波峰。

### 行为逻辑约束
御兽须有独立性格，非纯工具；进化须有铺垫（材料/历练/羁绊），禁止突变；对战须有策略性，不可纯靠等级碾压；御兽死亡须有情感重量，不可随意复活。

## 11. 末世

### 特色钩子类型
1. **安全区沦陷预告** — 安全区出现崩坏迹象。例：围墙外，尸潮的嘶吼声比昨天近了一公里。
2. **物资危机倒计时** — 关键物资即将耗尽。例："净水还够三天，最近的补给点在八十公里外。"
3. **变异体进化** — 怪物出现新能力。例：那只丧尸会开门了。
4. **人性考验设局** — 被迫在道德与生存间抉择。例：救这孩子，就要丢掉一半物资；不救，他活不过今晚。
5. **末世真相碎片** — 发现灾变起因的线索。例：实验室的最后一行日志写着："我们以为在拯救世界。"

### 爽点结构
危机降临→收集资源/队友→建立据点→击退威胁→扩张势力。核心是"生存压迫感+逆袭建立秩序"，从被动求生到主动掌控。

### 行为逻辑约束
资源稀缺须贯穿始终，不可物资自由；末世人性复杂，不可全员圣母或全员恶人；主角成长须付出代价（队友牺牲/道德困境）；变异体须有合理进化逻辑，不可随意加能力。

## 12. 种田

### 特色钩子类型
1. **丰收/产出悬念** — 关键作物/产品即将收获。例：那棵变异灵稻明天就熟了，可今晚有人来"借种"。
2. **产业链升级预告** — 即将解锁新生产模式。例：第一炉砖烧出来了，村里的路终于能修了。
3. **外部威胁觊觎** — 势力/贼人盯上了成果。例：县令的师爷"恰好"路过，眼睛盯着新建的磨坊。
4. **技术突破卡点** — 卡在关键工序上。例：酒曲的配方还差一味药，满窖的粮食在等着。
5. **邻里关系转折** — 关键人物态度转变。例：一直刁难他的村长，今天破天荒送来一坛酒。

### 爽点结构
发现问题/需求→想办法→小成功→带动周边→规模扩大。核心是"建设感+获得感"，看得见的进步，每章/每几章一个产出里程碑。

### 行为逻辑约束
发展须循序渐进，不可一步现代化；技术须有合理来源（穿越知识也要受限于材料/人力）；人际关系是核心，不可只搞生产不写人；节奏偏慢热，重在生活质感。

## 13. 重生

### 特色钩子类型
1. **前世记忆触发** — 遇到前世关键事件/人物的提前预知。例：他记得，明天这家公司会发布公告，股价翻三倍。
2. **蝴蝶效应显现** — 改变小事引发意料外连锁反应。例：他只是没走那条路，结果救下了本该死的人——可那人，前世是反派。
3. **遗憾弥补悬念** — 即将改变前世最大的遗憾。例：前世她死在今天，这一次他提前三个小时到了。
4. **重生秘密暴露风险** — 有人察觉他"未卜先知"。例："你怎么知道会地震？"她盯着他的眼睛。
5. **前世敌人重逢** — 宿敌提前出现。例：那个前世毁了他一切的人，此刻还是个不起眼的小职员。

### 爽点结构
预知关键节点→提前布局→改变命运→蝴蝶效应新挑战→更大的棋局。核心是"信息差碾压+遗憾弥补"，每次利用预知都是一个爽点。

### 行为逻辑约束
预知须有局限（记忆不全/蝴蝶效应改变后续），不可全知全能；重生须有核心动机（复仇/弥补/避祸）；改变历史须承担新后果；前世记忆须随时间推移逐渐失效或偏差增大。

---

## 14. 穿越

### 特色钩子类型
1. **原身秘密** — 穿越后发现原身有隐藏身份/债务/仇家。例：刚睁眼，门外就有人来讨债，数目大得吓人。
2. **现代知识降维** — 用现代知识惊艳土著的瞬间。例：他写下几个符号，老炼丹师的手开始抖。
3. **回归危机** — 穿越"穿帮"风险。例：她随口哼了一首地球的歌，有人问这是哪国的民谣。
4. **世界规则冲突** — 现代认知与新世界规则碰撞。例：他想用法律解决问题，对方拔出了刀。
5. **金手指觉醒/限制** — 穿越buff揭示或有条件限制。例：系统能鉴定一切，唯独鉴不了她自己。

### 爽点结构
穿越落地→摸清处境→用现代知识破局→建立根基→卷入更大格局。核心是"认知降维+文化碰撞"爽感。

### 行为逻辑约束
穿越者须有适应期，不可瞬间融入；现代知识应用须受限于新世界条件（材料/规则/信仰）；原身的人际关系须处理，不可当不存在；金手指须有限制防失衡。

---

## 15. 快穿

### 特色钩子类型
1. **任务世界规则揭示** — 进入世界后才知道真正的任务条件。例：系统说要攻略男主，没说男主是条龙。
2. **原主烂摊子** — 接手原主留下的烂局。例：原主刚把女主推下水，全村都在抓她。
3. **剧情偏离预警** — 主角行为导致原剧情崩坏。例：男主本该爱上女主，现在他看她的眼神……不太对。
4. **世界崩塌倒计时** — 改变太大导致世界不稳定。例：系统警告：剧情偏离度87%，超过90%世界将重置。
5. **跨世界伏笔** — 不同世界间的隐藏联系。例：这个世界的大反派，和上个世界的路人甲长得一样。

### 爽点结构
进入世界→接手烂局→翻盘打脸→完成任务→发现跨世界主线。核心是"短平快爽点+单元剧节奏"，每个世界5-15章。

### 行为逻辑约束
每个世界须有完整起承转合，不可烂尾；主角人设可随世界微调但内核不变；任务完成须有合理逻辑，不可硬凑HE；世界间过渡须流畅，有主线串联。

---

## 16. 游戏

### 特色钩子类型
1. **隐藏副本/彩蛋发现** — 意外触发稀有内容。例：他随手点了那块墙砖，整面墙翻转了。
2. **装备/技能悬念** — 关键掉落即将揭晓。例：Boss倒下，地上那道光是金色——传说级？
3. **赛事/排行挑战预告** — 强敌/赛事逼近。例：全服第一发了条广播："那个新来的，我在竞技场等你。"
4. **游戏bug/机制利用** — 发现机制漏洞。例：如果把这两个技能的释放间隔控制在0.3秒……
5. **游戏与现实交汇** — 游戏内事件影响现实（或反之）。例：游戏里获得的伤口，现实中隐隐作痛。

### 爽点结构
进入游戏→升级打怪→获得稀有奖励→PVP碾压→解锁更高层级内容。核心是"成长数据化+即时奖励反馈"。

### 行为逻辑约束
游戏机制须自洽，数值平衡要合理；主角强须有依据（操作/策略/运气），不可纯靠运气掉神装；游戏与现实的关系须有规则；NPC须有区分（脚本NPC vs 智能NPC）。

---

## 17. 电竞

### 特色钩子类型
1. **关键局悬停** — 比赛胜负手时刻切断。例：基地只剩最后100血，他的手指悬在R键上。
2. **黑马逆袭预告** — 默默无闻的选手即将爆发。例：解说还在念他的ID念错，他已经拿了三杀。
3. **战术博弈揭示** — 双方战术博弈的真相。例：原来前三局的送人头，都是为了这一局的BP陷阱。
4. **宿敌对决前夕** — 老对手重逢预告。例：抽签结果出来，八强赛第一个对手——是三年前那个人。
5. **职业生涯危机** — 伤病/年龄/战队矛盾威胁职业生涯。例：医生说他的手腕最多还能撑一个赛季。

### 爽点结构
组队/磨合→小组赛出线→遭遇强敌→战术博弈→决赛夺冠。核心是"团队热血+操作燃点+逆袭"。

### 行为逻辑约束
比赛须有专业感，术语/操作描写须真实；角色须有职业素养与个人弱点；战队配合是核心，不可主角一人C全场；胜负须有悬念，不可场场碾压。

---

## 18. 娱乐圈

### 特色钩子类型
1. **黑料即将爆出** — 危机公关倒计时。例：明早八点，那段视频就会登上热搜第一。
2. **角色争夺悬念** — 重要角色的归属未定。例：导演说最后两个候选人，明天试镜定生死。
3. **综艺直播意外** — 直播中突发状况。例：镜头扫过，弹幕突然炸了——他背后那个人是谁？
4. **粉丝反转预告** — 舆论即将逆转的伏笔。例：那条被骂上热搜的微博，三小时后会出现一条关键评论。
5. **业内大佬青睐** — 大人物抛出橄榄枝但条件未明。例：影帝递来一张名片："有兴趣来我下部戏吗？"

### 爽点结构
低谷/被黑→抓住机会→作品爆红→打脸黑粉→登顶。核心是"舆论反转+作品实力打脸"。

### 行为逻辑约束
娱乐圈生态须真实（资本/流量/作品的关系）；主角成功须靠作品实力，不可纯靠运气/金主；舆论须有反转逻辑，不可强行洗白；感情线须服务主线，不可喧宾夺主。

---

## 19. 盗墓

### 特色钩子类型
1. **机关触发悬停** — 踩中机关的瞬间切断。例：脚下的砖松了半寸，头顶传来石磨转动的声音。
2. **棺椁开启悬念** — 开棺前一刻切断。例：棺盖缓缓移开，缝隙里透出的不是腐臭，是……香味？
3. **诅咒/异象显现** — 触犯禁忌后异象初现。例：出去之后，他数了数队伍——多了一个。
4. **明器争夺设局** — 财宝引发人性博弈。例：金箱只有一件宝物，但有三双眼睛盯着。
5. **墓主身份谜题** — 墓主身份远超预期。例：这不该出现在这里的陪葬品，指向了一个只存在于传说里的朝代。

### 爽点结构
寻墓定位→进入墓道→破解机关→抵达主墓→明器到手+墓主秘密揭晓。核心是"探险解谜+机关惊险+历史悬疑"。

### 行为逻辑约束
机关须有历史合理性，可推理破解；危险须真实，队友可有伤亡；盗墓须有行业规矩（倒斗规矩/风水理论）；超自然元素须克制，能用科学/历史解释的优先。

---

## 20. 修仙

### 特色钩子类型
1. **渡劫悬念** — 天劫降临关键时刻切断。例：第九道天雷蓄在云中，他的护身法宝已经碎了三件。
2. **心魔劫** — 心魔考验中分不清真假。例：心魔化作他最爱的人，笑着说："你不是一直想放弃吗？"
3. **仙缘争夺** — 稀有仙缘多人争夺。例：那株万年灵药就在眼前，可三派弟子都到了。
4. **功法突破卡点** — 关键境界卡在最后一层。例：还差最后一丝感悟，他能感觉到那扇门，但推不开。
5. **飞升预告/失败案例** — 飞升的诱惑与恐怖并存。例：上一位飞升的大能，飞升台上只留下了一摊血。

### 爽点结构
修行苦修→机缘突破→碾压同辈→渡劫飞升→更高层次的挑战。核心是"逆天改命+境界碾压"，每个大境界是一个大爽点波峰。

### 行为逻辑约束
修仙须有"道"的内核（求道/问道/证道），不可纯打怪升级；境界突破须有心性匹配，境界高心性低会走火入魔；天道须有规则（因果/劫数），不可随意逆天；长寿带来的孤独/执念须有刻画。

---

## 21. 架空

### 特色钩子类型
1. **历史拐点悬停** — 关键历史事件即将发生时断章。例：诏书宣读前一刻，篡位者按住了玉玺，"再等一等"。
2. **权力更迭预告** — 皇位/政权变动暗示。例：老皇帝咳出的血染红了奏折，殿外禁军已换了防。
3. **架空设定冲突** — 架空规则与现实冲突爆发。例：议会的法案与圣旨同时下达，地方官不知该听谁的。
4. **人物命运分叉** — 主角站在历史十字路口。例：一边是勤王保驾，一边是拥立新君，他的手按在两封密信上。
5. **时代洪流倒计时** — 大变革倒计时压迫感。例："三月后漕运断绝，京城的粮食只够撑到立秋。"

### 爽点结构
历史参与感 + 架空设定优越感 + 命运改写快感。核心是"以小搏大、以一人撼时代"的格局感，每卷一个架空历史节点被改写或确认。

### 行为逻辑约束
架空设定须自洽，历史改编须有锚点，不可纯架空无参照；权力运作须有制度逻辑（官僚/军镇/世家/商贾），不可靠主角一人翻云覆雨；历史大势的改写须付出代价，不可轻飘飘翻盘；架空王朝的政治经济文化须内部一致，避免穿帮。

---

## 22. 奇幻

### 特色钩子类型
1. **魔法体系揭示** — 新魔法规则/限制即将揭晓。例：禁咒卷轴展开到最后一行，他读出那个被抹去的代价。
2. **异族文明接触** — 首次接触非人类文明。例：森林深处传来歌声，那不是人类的声带能发出的音。
3. **古老预言应验** — 预言开始应验的征兆。例：石碑上第三道裂纹亮起，长老的手开始发抖。
4. **世界规则崩塌** — 奇幻世界底层规则出现裂缝。例：重力在这一片土地失效了，魔药师的水银瓶飘了起来。
5. **命运织线交汇** — 多条命运线即将交汇。例：三位被不同神选中的少年，在同一夜梦见了同一座塔。

### 爽点结构
奇幻奇观探索 + 魔法力量获取 + 世界奥秘揭示。核心是"探索未知世界、解锁魔法力量、揭开世界真相"的三重爽感递进。

### 行为逻辑约束
魔法体系须有代价/限制（消耗/反噬/禁忌），不可无限施法；奇幻元素须服务剧情而非堆砌奇观，每场魔法都应推动情节或人物；种族文化须有差异逻辑（信仰/寿命/社会结构），不可披皮人类；世界观须自洽，神明/魔法/历史之间要有因果链。

---

## 23. 武侠

### 特色钩子类型
1. **武学突破卡点** — 内功/招式即将突破时断章。例：丹田真气运转到最后一关，经脉剧痛，他咬碎了满口牙。
2. **江湖恩怨爆发** — 门派/师门仇恨即将清算。例：二十年前的血衣还压在箱底，仇人的弟子今日上了山门。
3. **绝学失传悬念** — 失传武学重现江湖的线索。例：市井醉汉随手一掌，竟是失传百年的"惊鸿照影"起手式。
4. **义气抉择两难** — 义与利的终极抉择。例：兄弟递来一封密信，打开就是富贵，不打开就是义气——他拔出了刀。
5. **天下大势变局** — 武林格局即将洗牌。例：武林盟主暴毙，五剑山庄连夜撤出江湖令，各派按剑相向。

### 爽点结构
武学精进 + 快意恩仇 + 江湖地位提升。核心是"武学成长、恩怨分明、江湖名望"的三线并进，每个武学突破或恩怨了结都是爽点波峰。

### 行为逻辑约束
武学须有体系（内功/外功/轻功/暗器/毒术），不可无脑打斗；江湖恩怨须有因果，不可凭空结仇或莫名化敌为友；高手须有宗师气度，胜负须有武学逻辑而非纯靠主角光环；武功境界须循序渐进，奇遇突破须有铺垫与代价。

---

## 24. 同人

### 特色钩子类型
1. **原作角色互动悬念** — 原作角色即将登场/对决/合作的瞬间切断。例：门被推开，进来的人穿着他只在漫画里见过的制服。
2. **剧情偏离期待** — 原作剧情即将被改写的关键节点。例：原作中这一战他本该赢——可这次，对手多了一张底牌。
3. **彩蛋埋设** — 原作中的伏笔/梗/隐藏设定以彩蛋形式埋入。例：墙上那张褪色的合影，站在C位的人原著里从未出现过。
4. **原作角色OOC预警** — 原作角色即将做出反常行为（须有合理原因）。例：那个永远冷静的人，第一次失手打翻了茶杯——因为他看见了门口的人。
5. **平行世界交汇** — 不同原作世界的角色相遇（综漫同人）。例：对方的剑法他认得，可这个世界根本不该有剑客。

### 爽点结构
原作角色互动 + 剧情改写快感 + 彩蛋发现惊喜。核心是"原作情结的满足与超越"——读者既想看到熟悉的角色和剧情，又期待意想不到的新变化。每卷至少1次原作名场面的致敬或改写，作为爽点波峰。

### 行为逻辑约束
原作角色性格内核不可颠覆性OOC，须有原著依据或合理成长驱动；原作世界观核心规则不可随意魔改，力量体系/社会结构须保持一致；原创角色不可喧宾夺主（戏份≤50%），须服务于原作角色成长或世界拓展；原作角色声口须还原（口癖/语气/用词），让原作读者"一听就是ta"；改写原作剧情须有蝴蝶效应逻辑链，不可为改而改。
