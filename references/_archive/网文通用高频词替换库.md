# 网文通用高频词替换库

> V3.1 — 针对AI写作中最常出现的"高频模板词"，提供多级替换方案。新增11类（说/道、想/觉得、移动动词、程度副词、其他感官、时间过渡词、身体动作、情绪反应公式、不禁/不由得、似乎/仿佛、一股量词模板）。按使用频率从高到低排列。共25类，覆盖AI写作7大重灾区。

---

## 一、"看着"类（视觉动作）

**问题**：AI每段必有"他看着/她看着/他看着"，一章出现20+次。

| 替换方案 | 语气/氛围 | 示例 |
|:---------|:---------|:-----|
| 盯着 | 专注/审视 | "他盯着对方，不放过任何一个细节" |
| 凝视 | 深情/凝重 | "他凝视着远处的山峦" |
| 打量 | 评估/陌生 | "她上下打量了他一眼" |
| 扫视 | 快速/巡视 | "目光扫过在场每一个人" |
| 瞥了一眼 | 随意/不屑 | "他瞥了一眼，懒得再看" |
| 斜睨 | 轻蔑 | "她斜睨着他，嘴角带着嘲讽" |
| 瞪 | 愤怒 | "他瞪大眼睛，不敢置信" |
| 俯视 | 居高临下 | "他俯视着跪在地上的对手" |
| 仰望 | 敬仰 | "他仰望那座巍峨的宫殿" |
| 目光所及之处 | 全景式 | "目光所及之处，皆是一片狼藉" |

**零视觉动作替代**（最高级）：
- "他的视线落到……"
- "那个方向——他循声望去——"
- "不需要看也知道……"

## 十五、"说/道"类（对话标签）【AI第一高频词】

**问题**：AI每章"他说""她说""xxx道"出现50+次，严重僵硬。

### A. 按情绪替换

| 替换方案 | 情绪 | 示例 |
|:---------|:-----|:-----|
| 反问 | 质疑/逼迫 | "'那你觉得呢？'他反问。" |
| 质问 | 愤怒/追责 | "'谁干的！'她厉声质问。" |
| 低声道 | 谨慎/温柔 | "他低声道：'别出声。'" |
| 脱口而出 | 急切/冲动 | "'不是我！'他脱口而出。" |
| 忍不住开口 | 憋不住 | "她终于忍不住开口了。" |
| 轻声说 | 温柔/私密 | "她轻声说：'我相信你。'" |
| 闷声道 | 压抑/委屈 | "他闷声道：'知道了。'" |

### B. 按方式替换

| 替换方案 | 方式 | 示例 |
|:---------|:-----|:-----|
| 插嘴 | 打断 | "'等一下——'他插嘴道。" |
| 接话 | 接续 | "她立刻接话：'我也是这么想的。'" |
| 补了一句 | 补充 | "临走又补了一句：'小心点。'" |
| 话锋一转 | 转折 | "话锋一转：'不过有个条件。'" |
| 把话题拉了回来 | 拉回正题 | "他把话题拉了回来：'说正事。'" |
| 自言自语 | 独白/嘀咕 | "'怎么会这样……'他自言自语。" |
| 嘟囔 | 小声/抱怨 | "她嘟囔了一句，谁也没听清。" |

### C. 零对话标签（最高级）

- 用动作替代："他放下杯子。'你说什么？'"
- 用神态替代："她眼眶红了。'我没事。'"
- 直接裸对白（连续对话不用任何标签）
- 用沉默替代："他没有回答。——沉默就是答案。"

## 十六、"想/觉得/认为"类（心理动词）【AI第二高频词】

**问题**：AI每段心理描写都是"他想""她觉得""他认为"。

### A. 按思考方式替换

| 替换方案 | 语境 | 示例 |
|:---------|:-----|:-----|
| 暗忖 | 内心琢磨 | "他暗忖：此人不可信。" |
| 心道 | 简洁内心独白 | "她心道：麻烦了。" |
| 转念一想 | 思维转折 | "但转念一想，又觉得不对。" |
| 念头闪过 | 一闪而过的想法 | "一个念头闪过：难道是他？" |
| 心底有个声音 | 直觉/挣扎 | "心底有个声音在说：别信。" |
| 隐约察觉到 | 模糊认知 | "他隐约察觉到了不对劲。" |
| 脑海中浮现 | 回忆/联想 | "脑海中浮现出当年的画面。" |
| 猛然意识到 | 突然醒悟 | "他猛然意识到自己被骗了。" |

### B. 按态度替换

| 替换方案 | 态度 | 示例 |
|:---------|:-----|:-----|
| 认定 | 坚信 | "他认定对方在说谎。" |
| 断定 | 确定判断 | "她断定此事有蹊跷。" |
| 猜测 | 不确定 | "他猜测对方另有目的。" |
| 预感 | 直觉 | "她预感今天要出事。" |
| 怀疑 | 不信/质疑 | "他开始怀疑对方的动机。" |
| 隐约觉得 | 模糊感觉 | "隐约觉得哪里不对劲。" |
| 以他的经验来看 | 理性推导 | "以他的经验来看，这不合常理。" |

### C. 零心理动词（最高级）

- 直接内心独白用引号：'不对。这事不对。'
- 用身体感受替代心理："脊背发凉。"（替代"他觉得害怕"）
- 用行为暗示心理：他停下了脚步。（替代"他犹豫了"）

## 二十三、"不禁/不由得/忍不住"类（情绪触发模板）

**问题**：AI写情绪触发必用——"他不由得退了一步""她忍不住红了眼眶""他不禁愣住了"，一章8-12次。

| 替换方案 | 风格 | 示例 |
|:---------|:-----|:-----|
| 直接删掉，用动作接 | 最干净 | ❌"他忍不住笑了" → ✅"他笑了。" |
| 到底还是 | 强调经过挣扎 | "到底还是开了口" |
| 鬼使神差地 | 被无形力量驱使 | "鬼使神差地，她伸出了手" |
| 不自觉地 | 中性/轻度 | "她不自觉地屏住了呼吸" |
| 下意识地 | 本能反应 | "他下意识地摸了摸口袋" |
| 条件反射般 | 快速本能 | "条件反射般侧身躲开" |
| 鬼使神差 | 说不清原因 | "鬼使神差，他回头看了一眼" |
| 一个没忍住 | 口语/坦诚 | "一个没忍住，笑出了声" |
| 到底没忍住 | 努力克制但失败 | "到底没忍住，眼泪掉了下来" |

## 二十四、"似乎/仿佛/好像/隐隐"类（模糊描写模板）

**问题**：AI对不确定描写全用这几个词——"似乎有什么不对""仿佛在暗示什么""隐隐感到不安"，一章6-10次。

### A. 按语气替换

| 替换方案 | 语气 | 示例 |
|:---------|:-----|:-----|
| 透着几分 | 肯定中带模糊 | "这笑容透着几分古怪" |
| 隐约透着 | 视觉模糊感 | "眼底隐约透着一丝不耐" |
| 说不上来为什么 | 直觉型 | "说不上来为什么，他就是觉得不对" |
| 不知怎的 | 自己也困惑 | "不知怎的，今天的心跳一直很快" |
| 某种 | 确定性+模糊对象 | "空气里弥漫着某种说不清的紧张" |
| 说不上来的 | 难以名状 | "说不上来的违和感" |
| 莫名 | 无法解释 | "莫名的不安""莫名的烦躁" |
| 总感觉 | 口语/直觉 | "总感觉有人在盯着他" |
| 怪怪的 | 口语/日常 | "气氛怪怪的" |
| 不对头 | 口语/直觉 | "这事不对头" |

### B. 零模糊词（最高级）

- ❌ "似乎有什么东西在暗中盯着他" → ✅ "有人在盯着他。他看不到是谁，但他知道。"
- ❌ "仿佛整件事都是一个局" → ✅ "整件事——从头到尾——都是一个局。"
- ❌ "隐隐觉得不安" → ✅ "后颈的汗毛竖起来了。"
- 策略：直接写具体的感觉信号（汗毛/寒毛/鸡皮疙瘩/脊背发凉），不写"似乎/仿佛"

| 一股暖流 | 像喝了一口烈酒 | "喉咙像被烈酒烫了一下" |
| 一股怒火 | 火噌地窜上来了 | "火噌地就窜上来了" |
| 一股怒火 | 血一下子涌上头顶 | "血一下子涌上了头顶" |
| 一股杀意 | 杀气如有实质 | "杀气如有实质，刺得皮肤生疼" |

> **使用建议**：
> - 每章按"寻找高频词→替换→检查"三步操作，不要一步到位
> - 不要一次性替换所有同类词，保留2-3处原词作为风格差异
> - 替换的最高境界是"直接删掉，什么都不写"，而不是用一个新词代替一个旧词
