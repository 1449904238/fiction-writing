# Part L: 文风系统

> 融合自 oh-story-claudecode 项目。文风是决定读者能否沉浸的核心要素。

## L.1 对标文风提示

确定对标作品后，按以下维度提取文风特征：

```
【对标文风提取框架】

1. 句式风格
├─ 对标作品倾向短句还是长句？
│   ├─ 短句为主（≤15字）→ 节奏快，适合爽文/动作/悬疑
│   ├─ 中句为主（15-30字）→ 节奏适中，适合都市/言情
│   └─ 长句为主（30+字）→ 节奏慢，适合文艺/历史/史诗
├─ 对标作品段落长度？
│   ├─ 短段为主（1-3行）→ 轻快，适合手机阅读
│   └─ 中长段为主（4-8行）→ 厚重，适合沉浸阅读
└─ 对标作品是否常用"口语化"表达？
    ├─ 是 → 文风偏"说书人"口吻
    └─ 否 → 文风偏"文学性"叙述

2. 感官侧重
├─ 对标作品最常用的感官是什么？
│   ├─ 视觉为主 → 描写偏"画面感"
│   ├─ 触觉/嗅觉为主 → 描写偏"沉浸感"
│   └─ 听觉为主 → 描写偏"节奏感"
└─ 输出：你的文风感官优先级排序

3. 情绪密度
├─ 对标作品情绪表达的频率？
│   ├─ 高频（每段都带情绪）→ 情绪驱动型，偏感性
│   ├─ 中频（关键处带情绪）→ 平衡型，张弛有度
│   └─ 低频（大多数冷静叙述）→ 情节驱动型，偏理性
└─ 输出：你的文风情绪密度定位

4. 对话风格
├─ 对标作品对话占比？
│   ├─ 高（>50%）→ 对话驱动，角色互动为主
│   └─ 低（<30%）→ 叙述驱动，情节推进为主
├─ 对话长短？
│   ├─ 短对话为主（1-2句/轮）→ 简洁，快节奏
│   └─ 长对话为主（3-5句/轮）→ 深度，信息量大
└─ 输出：你的文风对话策略

5. 叙述者距离
├─ 对标作品的叙述者离角色多远？
│   ├─ 贴身（自由间接引语，融入角色视角）→ 代入感强
│   ├─ 中距（第三人称有限视角）→ 平衡
│   └─ 远距（全知视角，偶尔跳出来评论）→ 古典说书人感
└─ 输出：你的文风叙述者距离
```

### 对标文风提取操作步骤（5步法）

**第1步：选段采样**
从对标作品中选取3-5个连续章节（含1个爽点高潮章+1个日常过渡章+1个情感/转折章），确保覆盖该作品的主要场景类型。

**第2步：逐维度标注**
对采样段落逐一标注5个维度的特征：
- 句式风格：统计短/中/长句比例，记录段落平均行数
- 感官侧重：标记每段使用了哪些感官（视/触/嗅/听/味），统计频率
- 情绪密度：统计每段中情绪词/情绪动作的出现次数
- 对话风格：计算对话占比，标注对话平均长度
- 叙述者距离：判断是自由间接引语、有限视角还是全知视角

**第3步：归纳模式**
将标注结果汇总，找出对标作品最稳定的文风模式（如"短句+触觉优先+高频情绪+简洁对话+贴身视角"）。

**第4步：设定偏差容忍度**
为每个维度设定允许的偏差范围（如句式风格允许±10%偏移），超出范围则标记为文风偏离。

**第5步：写入项目设定**
将提取结果写入《00_小说设定文档》的 `style_profile` 字段，供后续扩写和精修步骤参照。

### 提取案例：某都市爽文（都市·打脸流）

**对标作品**：《都市之巅峰归来》（虚构示例）
**采样章节**：第12-14章（含打脸高潮+日常过渡+情感转折）

**提取结果**：

| 维度 | 提取结果 | 具体数据 |
|------|----------|----------|
| 句式风格 | 短句为主（≤15字），段落1-3行 | 短句占比68%，中句25%，长句7%；平均段落2.1行 |
| 感官侧重 | 触觉>视觉>听觉 | 触觉描写占42%，视觉31%，听觉19%，嗅觉/味觉8% |
| 情绪密度 | 高频，用行动表达 | 平均每段1.8个情绪标记点；70%通过动作/身体反应体现 |
| 对话风格 | 短对话为主，简洁有力 | 对话占比55%；平均每轮1.4句；无超过3句的长独白 |
| 叙述者距离 | 贴身（自由间接引语） | 80%段落融入主角视角，无全知评论 |

**归纳模式**：短句快节奏 + 触觉驱动 + 行动化情绪 + 简洁对话 + 贴身视角

**偏差容忍度**：
- 句式：短句占比不低于55%
- 感官：触觉不低于30%
- 情绪密度：每段不少于1个标记点
- 对话占比：45%-65%
- 叙述距离：贴身视角占比不低于70%

## L.2 文风优先级系统

```
爽文/打脸流（番茄核心）：
├─ 优先级1：句式节奏（短句为主，快节奏）
├─ 优先级2：对话风格（简洁有力，不废话）
├─ 优先级3：情绪密度（高频，但用行动表达）
├─ 优先级4：感官侧重（触觉/身体感受优先）
├─ 优先级5：叙述者距离（贴身，代入感）

悬疑/灵异流：
├─ 优先级1：感官侧重（嗅觉/听觉创造不安感）
├─ 优先级2：情绪密度（低频但精准，制造压抑）
├─ 优先级3：句式节奏（中短句为主，偶尔长句制造窒息感）
├─ 优先级4：对话风格（长对话为主，信息量大）
├─ 优先级5：叙述者距离（中距，让读者比角色多知道一点）

言情/情感流：
├─ 优先级1：对话风格（长对话，有拉扯感）
├─ 优先级2：情绪密度（高频，情感细腻）
├─ 优先级3：叙述者距离（贴身，感受角色内心）
├─ 优先级4：感官侧重（触觉/温度优先，创造亲密感）
├─ 优先级5：句式节奏（中长句为主，节奏舒缓）

升级/成长流：
├─ 优先级1：句式节奏（升级时短句爆发，日常时中句）
├─ 优先级2：情绪密度（升级时高频，日常时低频）
├─ 优先级3：感官侧重（战斗时触觉/身体感受，日常时平衡）
├─ 优先级4：对话风格（战斗时简洁，日常时有深度）
├─ 优先级5：叙述者距离（贴身，跟随主角视角）

科幻/无限流：
├─ 优先级1：感官侧重（视觉为主，想象力优先）
├─ 优先级2：对话风格（长对话，信息量大，解释设定）
├─ 优先级3：句式节奏（中句为主，稳定输出信息）
├─ 优先级4：情绪密度（中频，平衡）
├─ 优先级5：叙述者距离（中距，保持一定客观性）
```

### 跨题材优先级冲突处理规则

当作品同时具备多个题材标签（如"都市+悬疑"、"玄幻+言情"）时，文风优先级可能冲突。按以下规则处理：

**规则1：以核心爽点驱动题材为准**
- 判断标准：作品的主线爽点类型（打脸→爽文优先级；解谜→悬疑优先级；情感拉扯→言情优先级）
- 示例："都市+悬疑"但核心爽点是打脸揭密 → 以爽文优先级为主，悬疑优先级为辅

**规则2：按场景类型动态切换**
- 打脸/战斗/升级场景 → 适用爽文优先级
- 调查/探索/恐怖场景 → 适用悬疑优先级
- 感情/回忆/独处场景 → 适用言情优先级
- 切换信号：场景类型变化时，句式节奏和感官侧重随之调整

**规则3：感官侧重就高不就低**
- 当两个题材对同一感官有不同优先级要求时，取更高优先级
- 示例：爽文要求触觉优先，悬疑要求嗅觉优先 → 战斗场景用触觉，探索场景用嗅觉

**规则4：叙述者距离以核心题材为准**
- 叙述者距离是最难中途切换的维度，全程以核心题材的设定为准
- 示例：核心是爽文 → 全程贴身视角，即使在悬疑段落也不切换为远距

## L.3 文风一致性检查

每5章检查：句式风格、感官侧重、情绪密度、对话风格、叙述者距离是否保持一致。

### 文风一致性检查操作示例

**检查时机**：每写完5章后执行一次。

**操作流程**：

1. **抽样**：从最近5章中随机抽取3个段落（1个爽点段+1个日常段+1个情感段）

2. **逐项核对**（以"都市爽文"文风为例）：

| 检查项 | 标准 | 检查方法 | 判定 |
|--------|------|----------|------|
| 句式风格 | 短句占比≥55% | 统计抽样段落中≤15字句子占比 | 低于55%→标记偏离 |
| 段落长度 | 平均≤3行 | 统计抽样段落平均行数 | 超过4行→标记偏离 |
| 感官侧重 | 触觉≥30% | 统计感官描写中触觉占比 | 低于30%→标记偏离 |
| 情绪密度 | 每段≥1个标记 | 统计情绪词/动作出现次数 | 0个→标记偏离 |
| 对话占比 | 45%-65% | 计算对话字数/总字数 | 超出范围→标记偏离 |
| 叙述距离 | 贴身视角≥70% | 判断视角类型占比 | 低于70%→标记偏离 |

3. **偏离处理**：
- 1-2项偏离 → 在下一章写作时针对性修正
- 3项以上偏离 → 暂停写作，重新校准文风设定
- 叙述距离偏离 → 最严重，必须立即修正（此维度最难通过局部调整修正）

4. **记录**：将检查结果写入《上下文管理模板》的文风追踪栏，持续监控趋势。
