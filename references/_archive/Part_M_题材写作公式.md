# Part M: 题材写作公式

> 融合自 oh-story-claudecode 项目。21种主流网文题材的核心公式。

## M.1 题材写作公式速查

```
1. 都市爽文（赘婿/神医/神豪/战神）
   公式：被低估 → 冲突触发 → 身份/能力揭露 → 打脸 → 更大冲突
   关键：身份反差越大越爽，打脸层次递进

2. 都市修真/修仙
   公式：修真者入都市 → 用修真手段解决都市问题 → 引起修真界注意 → 双线冲突
   关键：修真手段与都市规则的碰撞

3. 都市异能
   公式：获得异能 → 隐藏身份 → 异能暴露危机 → 更大异能组织出现
   关键：隐藏身份与使用异能的矛盾

4. 系统流
   公式：获得系统 → 完成任务 → 升级 → 系统功能解锁 → 系统背后的秘密
   关键：任务设计要有趣，系统不能是万能外挂

5. 重生流
   公式：重生回到过去 → 利用先知优势 → 改变命运 → 蝴蝶效应 → 新的危机
   关键：先知优势要逐步消耗，蝴蝶效应制造新冲突

6. 穿越异世界
   公式：穿越 → 适应新世界 → 利用现代知识 → 在新世界崛起 → 寻找回家之路
   关键：现代知识与异世界规则的碰撞

7. 玄幻修仙
   公式：入门 → 修炼 → 突破 → 宗门/家族冲突 → 飞升/更高世界
   关键：升级体系要清晰，每次突破要有代价

8. 玄幻废柴逆袭
   公式：被看不起 → 意外获得机缘 → 逆袭 → 证明自己 → 成为最强
   关键：被看不起的阶段要让读者憋屈

9. 悬疑灵异
   公式：异常事件 → 调查 → 线索碎片 → 误导 → 真相反转 → 更大的阴谋
   关键：线索要分层释放，每次接近真相都要付出代价

10. 盗墓探险
    公式：获得线索 → 组队 → 进入古墓 → 机关/怪物 → 宝物/真相 → 逃脱
    关键：团队成员各有功能，要有牺牲和背叛

11. 无限流
    公式：进入副本 → 理解规则 → 策略/战斗 → 通关 → 获得奖励 → 副本背后的秘密
    关键：副本设计要有创意，规则要清晰

12. 恐怖惊悚
    公式：日常 → 异常 → 恐惧升级 → 逃生/对抗 → 暂时安全 → 更大的恐怖
    关键：恐怖要层层递进

13. 推理侦探
    公式：案件发生 → 调查 → 线索 → 推理 → 错误结论 → 新线索 → 真相
    关键：线索必须公平

14. 言情（现代）
    公式：相遇（冲突/误会）→ 被迫相处 → 逐渐了解 → 情感升温 → 危机 → 和解
    关键：误会不能太刻意

15. 言情（古代/宫斗）
    公式：入宫/入府 → 生存危机 → 结盟/树敌 → 步步为营 → 逆转局势 → 登顶
    关键：计谋要有层次

16. 言情（霸总/豪门）
    公式：契约/意外 → 同居/共事 → 矛盾冲突 → 互相了解 → 情感转折 → 危机 → HE
    关键：男主外冷内热，女主独立有主见

17. 纯爱/双男主
    公式：相遇 → 对立 → 合作 → 互相理解 → 情感转折 → 危机（身份/世俗）→ 坚定
    关键：情感发展要自然

18. 竞技/体育
    公式：低谷 → 遇到教练/队友 → 刻苦训练 → 比赛 → 失败 → 再战 → 胜利
    关键：比赛描写要专业

19. 电竞
    公式：组建战队 → 训练 → 比赛 → 团队矛盾 → 磨合 → 夺冠
    关键：游戏设定要具体

20. 历史/架空
    公式：穿越至古代 → 适应环境 → 利用知识 → 卷入政治/军事 → 改变历史
    关键：历史细节要真实

21. 末世/生存
    公式：末世降临 → 生存 → 建立据点 → 资源争夺 → 人性考验 → 重建文明
    关键：人性在极端环境下的表现
```

## M.2 题材融合规则

```
可融合的题材（互补型）：
├─ 都市 + 修真 = 都市修真（热度最高）
├─ 重生 + 穿越 = 重生异世界（热度高）
├─ 系统 + 任何题材 = 系统流变体（万能搭配）
├─ 悬疑 + 灵异 = 灵异探案（经典组合）
├─ 言情 + 古装 = 古言（经典组合）
├─ 末世 + 重生 = 重生末世（热度高）
└─ 玄幻 + 废柴逆袭 = 标准玄幻升级流

谨慎融合（冲突型）：
├─ 言情 + 硬科幻 → 情感线和科幻线容易互相干扰
├─ 爽文 + 深度文艺 → 节奏冲突
├─ 悬疑 + 搞笑 → 氛围冲突
└─ 恐怖 + 言情 → 需要极强的平衡能力

融合原则：
├─ 主题材占70%以上
├─ 融合不是简单叠加，要找到"化学反应"
├─ 如果融合导致节奏混乱 → 放弃融合
└─ 番茄平台：一个主题材+一个辅题材是最佳配置
```
