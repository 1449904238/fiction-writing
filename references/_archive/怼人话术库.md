# 网文怼人话术库

> **⚠️ 使用警告（V2.7 清洗）**：多为直接辱骂或网络流行梗，缺乏讽刺、阴阳怪气、冷暴力、潜台词等高级冲突表达。角色容易同质化为"喷子"。
> **转化建议**：升级为"冲突对话库"——保留高智商讽刺、职场暗讽、亲密关系中的伤害性表达；删除纯辱骂和网络烂梗；增加"沉默""回避""转移话题"等非语言冲突手段。真正的冲突不在"怼人"，而在"话里有话"。
> **互补素材库**：需要潜台词设计 → `人物声口对话风格库.md`（潜台词对照表）；需要情绪的身体呈现 → `情绪颗粒度心理描写库.md`（压抑型外化+反转型外化）。

> V2.0 — 按挑衅攻击性梯度从低→高排列。新增阴阳怪气/捧杀、古风/文言怼人、能力嘲讽细化（按智力/眼光/实力/人品/出身五维分类）。每种话术标注推荐人物性格和适用场景。

本素材库禁止大模型进行无差别的随机概率抽取。在调用本库词汇前，必须读取《00_小说设定文档》中的 `novel_meta.core_tags`（核心题材标签）与 `Part_L_文风系统` 确定的风格定位，进行自动化分类剪裁：

### 【分类栅栏路由表】
1. ⚔️ [修仙/玄幻/武侠标签]：
   - 唯一解锁：怼人话术库中的"古风/文言怼人"模块；"高冷型"模块中的古风变体
   - 严厉封锁：现代网络流行语吐槽（"你脑子进水了""你咋不上天呢"）、知乎风怼人话术
2. 🏢 [都市/神豪/战神/职场标签]：
   - 唯一解锁：怼人话术库中的"智力/能力/眼光嘲讽"现代流模块；"阴阳怪气/捧杀"模块；"嘴碎型"日常互怼
   - 严厉封锁：任何带"汝/尔/竖子/蝼蚁/本座"等古风文言怼人模块（除非角色设定为穿越者）
3. 🕵️ [悬疑/灵异/恐怖标签]：
   - 唯一解锁：怼人话术库中的"冷漠/拒绝对话"高冷型模块（保持压抑感）
   - 严厉封锁：搞笑吐槽类、捧杀类（破坏悬疑氛围）
4. 💕 [言情/情感标签]：
   - 唯一解锁：怼人话术库中的"嘴碎型"日常互怼、"阴阳怪气"轻度模块
   - 严厉封锁：过度攻击性的"打脸专用"高攻击模块（破坏情感基调）

> **鉴权失败处理**：若无法确定题材标签，仅使用"冷漠/拒绝对话"和"嘴碎型"两大通用模块，禁止使用题材敏感模块（古风文言/高攻击打脸等）。

## 一、冷漠/拒绝对话（高冷型·不战而屈人之兵）

| 话术 | 人物性格 | 适用场景 |
|:-----|:---------|:---------|
| "关你屁事。" | 高冷/孤僻 | 被无关者打听隐私 |
| "哦。" | 冷漠/不屑 | 对方炫耀/挑衅时 |
| "所以呢？" | 不耐烦 | 对方长篇大论指责 |
| "跟你有关系吗？" | 疏离 | 被多管闲事 |

| "你高兴就好。" | 讽刺性敷衍 | 懒得争辩时的终结技 |
| "你配吗？" | 高傲 | 对方不自量力时 |
| "说够了吗？" | 压制性平静 | 爆发前的压抑阶段 |
| "你也就这点本事了。" | 轻蔑 | 对手底牌尽出后 |
| "然后呢？就这？" | 失望式嘲讽 | 对方全力一击却不过如此 |

---

> **使用建议**：
> - 不是每段对话都需要怼人，控制在每章0-2处高强度怼人（爽文除外）
> - 角色性格决定话术风格——高冷角色用"冷漠类"，嘴碎角色用"吐槽类"
> - 怼人话术的成功感来自于"实力碾压+精准戳痛处"，不是单纯的脏话多
> - 反派/配角被打脸后的话术要用"求饶类"，不要写"反派到死都嘴硬"的套路
> - 现代题材避免过于古风的怼人话术（如"竖子尔敢"），除非是特定文风
> - 阴阳怪气/捧杀类话术杀伤力比直怼更强，适合高智商/腹黑型主角
> - 古风怼人话术只适合古言/玄幻/武侠题材，现代文禁用
> - 能力嘲讽应选对方最在意的维度——戳痛点比泛泛骂人效果好十倍
