# 小说创作成语库（优化版）

> **⚠️ 使用警告（V2.7 清洗）**：大量四字成语适合议论文而非小说叙事。生僻成语在对话和心理描写中会"掉书袋"，缺乏具象画面感。
> **转化建议**：优先使用有画面感的成语（如"噤若寒蝉""目眦尽裂"），剔除抽象说理类；增加"成语拆开用""反用""化用"的改写意识。
> **互补素材库**：需要情绪的具体身体呈现 → `情绪颗粒度心理描写库.md`；需要环境氛围 → `场景氛围环境渲染库.md`；需要感官细节 → `感官细节库.md`。
>
> **使用说明**：本成语库按**小说创作场景**分类，每个成语标注**常用度评分**（★★★★★最常用 → ★较少用），便于快速查找。解释精简，使用场景聚焦小说写作。

## 🛑 素材库安全鉴权与语境路由系统

本素材库禁止大模型进行无差别的随机概率抽取。在调用本库词汇前，必须读取《00_小说设定文档》中的 `novel_meta.core_tags`（核心题材标签）与 `Part_L_文风系统` 确定的风格定位，进行自动化分类剪裁：

### 【分类栅栏路由表】
1. ⚔️ [修仙/玄幻/武侠标签]：
   - 唯一解锁：成语库中的"修炼升级"类（境界突破/修为变化/顿悟心魔）；"计谋算计"类古风谋略成语
   - 严厉封锁：现代网络流行语化用的成语、纯都市职场类成语
2. 🏢 [都市/神豪/战神/职场标签]：
   - 唯一解锁：成语库中的"权力地位"类、"尴尬窘迫"类、"社会观察"类
   - 严厉封锁：任何带"丹田/罡气/飞升/蝼蚁/汝/尔"等严重脱离现代社会语境的仙侠专用成语
3. 🕵️ [悬疑/灵异/恐怖标签]：
   - 唯一解锁：成语库中的"恐惧"类、"悬念紧张"类、"转折变化"类
   - 严厉封锁：轻松诙谐类、插科打诨类成语（破坏恐怖氛围）
4. 💕 [言情/情感标签]：
   - 唯一解锁：成语库中的"爱情"类、"思念"类、"喜悦"类、"尴尬窘迫"类
   - 严厉封锁：过度血腥暴力的战争杀伐类成语

> **鉴权失败处理**：若无法确定题材标签，仅使用"人物刻画"和"情感表达"两大通用类，禁止使用题材敏感类（修炼升级/权力地位等）。

## 十一、使用指南

### 1. 如何查找成语

1. **按场景查找**：根据你要写的场景（人物、情感、情节、环境、对话）选择对应章节
2. **按常用度查找**：优先使用★★★★★和★★★★的成语，它们更常用更自然
3. **按主题查找**：使用索引快速定位到具体子类

### 2. 如何使用评分

- **★★★★★**：最常用，几乎任何场景都适用
- **★★★★**：常用，适合大多数场景
- **★★★**：一般常用，特定场景效果更好
- **★★**：较少用，但有独特效果
- **★**：较少用，慎用

### 3. AI使用建议

1. **分类检索**：AI可以根据分类快速找到合适成语
2. **评分筛选**：优先推荐高评分成语
3. **场景匹配**：根据使用场景选择最贴切的成语
4. **避免重复**：同一场景避免使用过多成语
