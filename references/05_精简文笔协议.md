# 精简文笔协议

> **核心定位**：本文件是04精修师与05去AI味精修师的"减法工具"——挤干小说里的水分，让内容更有张力、更抓读者。
> 与"改写范例库"（AI味→自然表达映射）和"网文通用高频词替换库"（词汇替换）互补——本协议管"怎么删"，改写范例库管"怎么改"，替换库管"换什么词"。
>
> **痛点对症**：AI写作有一个顽固的本能——"多写一点更安全"。AI怕信息丢失所以堆砌前缀、怕读者不懂所以上帝视角解释、怕情绪不到位所以重复强调。结果就是"字数达标但信息密度塌陷"——读者读起来像在水里跑步。
> 本协议提供3个精确的"脱水手术"，让AI学会"少写即是多写"。
>
> **核心原则**：多写不等于丰满，少写也不等于干瘪。真正的精简，是删掉那些没有营养的"脂肪"，留下的就全都是"肌肉力量"。

## 二、第一招：删掉心理活动的万能前缀

### 2.1 核心原理

> **新手写人物心理时，总喜欢加"他心里想、她暗自思忖、他不禁感到十分震惊"这类前缀。**
> **这些前缀不仅冗余，还会在读者和剧情之间形成"屏障"——破坏代入感。**

**为什么这些前缀是屏障？**
- 读者本来可以"直接进入角色内心"——但"他心里想"提醒读者"这是叙述者在转述角色的心理"→ 读者被推回旁观者位置
- 好的心理描写让读者"成为角色"（第一人称体验），差的心理描写让读者"听别人讲角色"（第三人称转述）

### 2.2 万能前缀黑名单

以下前缀在叙述语中**禁止使用**（对话中可保留，人在说话时确实会说"我想"）：

| 前缀类型 | 禁止使用的具体前缀 | 示例 |
|---------|-----------------|------|
| 直接心理前缀 | 他心里想/她心里想/他在心里想/她暗自想 | "他心里想，这家伙怎么可能这么强？" |
| 思忖类前缀 | 她暗自思忖/他暗忖/她心道/他心说 | "她暗自思忖，这件事没那么简单。" |
| 感到类前缀 | 他不禁感到/他不由得感到/他忽然感到 | "他不禁感到十分震惊和不可思议。" |
| 觉得类前缀 | 他觉得/她感觉/他意识到/她察觉到 | "她觉得这个人的眼神让她很不舒服。" |
| 知道类前缀 | 他知道/她明白/他清楚/她晓得 | "他知道，这次他真的完了。" |
| 发现类前缀 | 他发现/她发现/他忽然发现/她终于发现 | "她忽然发现，自己一直在被骗。" |

> ⚠️ 与04精修师"绝对违禁词库"的关系：违禁词库覆盖"觉得/知道/意味着/是因为/看出规律/异乎寻常/看来/他发现/这说明/他心里明白/他在等/他意识到/显然/可见"，本协议扩展了"暗自思忖/心道/心说/不禁感到/不由得感到/忽然感到/察觉到/清楚/晓得/终于发现"等违禁词库未覆盖的前缀。

### 2.3 删除前缀后的改写方法

**方法1：直接内心独白（删除前缀，保留心理内容）**
- 原句："看到叶凡一拳打碎了测试石，这个家伙怎么可能这么强？他不禁感到十分震惊和不可思议。"
- 改后："测试石轰然爆碎。王腾倒吸一口凉气：怎么可能？这小子什么时候变得这么强了！"
- 操作：①删除"他不禁感到十分震惊和不可思议" ②添加生理反应"倒吸一口凉气" ③心理内容改为直接内心独白

**方法2：生理反应替代（用身体反应代替心理告知）**
- 原句："她心里想，这次面试肯定又失败了。"
- 改后："她盯着面试官合上的简历，指甲掐进了掌心。"
- 操作：①删除"她心里想" ②用动作+生理反应承载心理

**方法3：对话/动作自然流露（心理通过言行自然呈现）**
- 原句："他暗自思忖，这个价格太高了，但他又不好意思开口还价。"
- 改后："他看了看价签，嘴张了张，又合上了。"
- 操作：①删除"他暗自思忖" ②心理内容转为动作+犹豫的身体语言

### 2.4 删除前缀的操作流程

```
【万能前缀删除三步法】

Step 1：扫描 + 标记
  全章扫描，标记所有"心理前缀+心理内容"的组合
  前缀类型：想/思忖/感到/觉得/知道/发现/意识到/察觉/明白/清楚

Step 2：判断 + 选择
  心理内容能否通过动作/对话/生理反应自然呈现？
    → 能：删除前缀+心理内容，替换为动作/生理反应（方法2/3）
    → 不能（心理内容有不可替代的信息量）：删除前缀，保留心理内容，改为直接内心独白（方法1）
  
  判断标准：这段心理内容是"情节信息"（读者需要知道）还是"情绪渲染"（读者不需要被告知）？
    → 情节信息：保留，改为直接内心独白
    → 情绪渲染：删除，替换为动作/生理反应

Step 3：改写 + 自检
  改写后读一遍——如果删掉前缀后读者仍然能感受到角色的心理，改写成功
  如果删掉后感觉少了什么 = 替换的动作/生理反应不够具体，需要更精准
```

### 2.5 删除前缀的注意事项

- **对话中保留**：角色在对话中说"我想""我觉得"是自然的，不删除
- **关键信息保留**：如果心理内容包含"读者必须知道的情节信息"，用直接内心独白保留，只删前缀
- **不要过度删除**：每章允许1-2处"经过前缀的内心独白"（只限于角色独处、内心有重大转折时），但必须短（≤20字）
- **与04精修师铁律1的协同**：铁律1已禁止"他觉得/他发现/这说明"等——本协议在此基础上，进一步禁止"暗自思忖/心道/心说/不禁感到/不由得感到/忽然感到"等铁律1未覆盖的前缀

## 三、第二招：用主观视角代替上帝视角的环境描写

### 3.1 核心原理

> **新手喜欢用"上帝视角"堆砌环境细节——比如写高档餐厅，就罗列"水晶吊灯、暗金色壁纸、手工地毯"。**
> **但这些内容和主角无关，属于"无效描写"——读者看网文是来看剧情，不是来看"室内装修设计"。**

**两种环境描写的本质区别**：

| 维度 | 上帝视角环境描写（❌） | 主观视角环境描写（✅） |
|------|-------------------|---------------------|
| 视角 | 全知叙述者站在空中俯视 | 通过角色的眼睛/耳朵/皮肤感受 |
| 内容 | 客观罗列环境中的所有元素 | 只写角色注意到的、在乎的元素 |
| 效果 | 读者像在看"室内装修清单" | 读者像在"亲身体验这个环境" |
| 信息效率 | 低——80%的环境信息对剧情/人设无用 | 高——每个环境细节都在"塑造人设"或"推动剧情" |
| 水字数风险 | 高——容易写成"环境说明书" | 低——每个环境细节都有"功能性" |

### 3.2 主观视角环境描写的具体技法

**技法1：环境融合进人物动作**
- 不是"房间里有水晶吊灯、暗金色壁纸、手工地毯"
- 而是"他踩在软绵绵的手工地毯上，总觉得自己的运动鞋和这里格格不入"
- 环境细节通过"踩"的动作带出，同时塑造了人物的局促感

**技法2：环境融合进人物情绪**
- 不是"餐厅很豪华，但主角心情不好"
- 而是"他扫了一眼餐厅那盏落了微尘的水晶吊灯，眉头微皱，直接收回了目光"
- 同一个环境（水晶吊灯），不同角色/不同心情看到的是不同的东西

**技法3：环境融合进人物身份**
- 穷小子视角："他局促地踩在软绵绵的手工地毯上，总觉得自己的运动鞋和这里格格不入，头顶那盏晃眼的水晶吊灯，仿佛在无声地嘲笑他的寒酸。"
- 千亿总裁视角："他扫了一眼餐厅那盏落了微尘的水晶吊灯，眉头微皱，直接收回了目光。"
- 同一个环境，不同身份的人看到不同的细节、产生不同的反应

**技法4：环境只写"角色此刻在乎的"**
- 角色在紧张→环境只写"门/窗/逃生路线"（安全相关）
- 角色在寻找某人→环境只写"人脸/体型/衣着"（目标相关）
- 角色在享受→环境只写"气味/温度/触感"（舒适相关）
- 不在角色当前注意力范围内的环境细节，一律不写

### 3.3 主观视角环境描写的操作流程

```
【主观视角环境描写三步法】

Step 1：确定角色视角
  问自己：当前视角是谁的？（主角/配角/反派？）
  问自己：这个角色此刻在关注什么？（安全/找到某人/享受/逃离？）
  问自己：这个角色的身份/性格/过往是什么？（这会决定他注意到什么）

Step 2：筛选环境细节
  从环境的所有细节中，只选"角色此刻会注意到的"
  筛选标准：
    → 角色在关注安全 → 只写与安全相关的细节（门/窗/逃生路线/可疑的人）
    → 角色在寻找某人 → 只写与人相关的细节（人脸/体型/衣着/声音）
    → 角色在感受氛围 → 只写感官细节（气味/温度/光线/声音），但必须带角色主观滤镜
    → 角色在评估环境 → 只写与角色身份相关的细节（修理工看锈/厨师看食材/商人看价格）

Step 3：融合进动作/情绪/对话
  环境细节不是单独一段，而是融入角色的动作/情绪/对话中
  原句："餐厅很大，装修豪华。水晶吊灯散发着柔和的光。"
  改后："他推开门，头顶的水晶吊灯晃了一下——他下意识缩了缩脖子。"
  （环境细节"水晶吊灯"通过"推门"动作带出，同时带出角色的局促）
```

### 3.4 上帝视角描写的"何时可以例外"

以下情况允许使用上帝视角环境描写（但必须克制）：

1. **新场景首次登场**：读者需要知道"这是哪里"，但只写2-3个最关键的细节（不是全部）
2. **环境本身有剧情功能**：环境不是背景而是"道具"（如密室/陷阱/关键地点），需要写清楚结构
3. **环境变化暗示剧情**：环境发生了"变化"（灯灭了/门开了/声音停了），需要描写变化本身

> 即使在这些例外情况下，环境描写也要尽量"角色化"——通过角色的眼睛来写，而不是全知叙述者。

## 四、第三招：合并情绪词与对话框

### 4.1 核心原理

> **新手常陷入"情绪+对话+动作"的套娃式复读——"他心里非常愤怒，猛地一拍桌子，愤怒地大吼道：你给我滚出去！"**
> **同一句话里，情绪词（愤怒）、动作（拍桌子）、对话（滚出去）三重表达同一个意思，像"复读机"。**
>
> **解决方法：能用台词和动作表达的情绪，绝对不用形容词说出来。**

### 4.2 "情绪+对话+动作"套娃模式识别

**套娃模式**：情绪形容词 + 动作 + 情绪副词 + 对话标签 + 对话内容

```
标准套娃示例：
"他心里非常愤怒，猛地一拍桌子，愤怒地大吼道：你给我滚出去！"

拆解：
  情绪词×2："非常愤怒" + "愤怒地"
  动作×1："猛地一拍桌子"
  对话标签×1："大吼道"
  对话内容×1："你给我滚出去"
  
→ 同一情绪被表达了4次！读者被反复告知"这个人很愤怒"。
```

**套娃变体**：
- 变体1："他冷冷地看了她一眼，冷冷地说：我知道了。"（"冷冷"×2）
- 变体2："她开心地笑了，高兴地说：太好了！"（"开心"+"高兴"+"太好了"+"笑"——同一情绪4次）
- 变体3："他沉默了一会儿，低声说：……"（"沉默"+"低声"+"……"——可以但"沉默"和"低声"任选其一即可）

### 4.3 合并方法

**方法1：只留对话+动作，删情绪词**
- 原句："他心里非常愤怒，猛地一拍桌子，愤怒地大吼道：你给我滚出去！"
- 改后：""你给我滚出去！"他猛地一拍桌子，震得茶杯叮当响。"
- 操作：①删除"他心里非常愤怒" ②删除"愤怒地" ③"大吼道"→对话内容+动作（对话本身就够有张力了） ④增加"震得茶杯叮当响"（物理后果增强张力）

**方法2：只留对话，删动作和情绪词**
- 原句："她冷冷地看了他一眼，冷冷地说：我知道了。"
- 改后：""我知道了。"她没看他。"
- 操作：①删除"冷冷地看了他一眼" ②删除"冷冷地说" ③"没看他"=冷，但不说"冷"

**方法3：只留动作，删情绪词和对话标签**
- 原句："她开心地笑了，高兴地说：太好了！"
- 改后："她笑了，没说话，只是把通知书翻来覆去看了三遍。"
- 操作：①删除"开心地"和"高兴地" ②删除"说：太好了！" ③用一个"反复看"的动作替代对话

**方法4：对话+动作分开，不在同一句里重复**
- 原句："他沉默了一会儿，低声说：我不知道。"
- 改后："他沉默了一会儿。'我不知道。'"
- 操作：①"沉默"和"低声"选其一（两者都表达"低情绪"） ②对话和动作分两段，不在同一句里叠加

### 4.4 合并操作流程

```
【情绪词合并三步法】

Step 1：识别套娃
  扫描每个对话段落，检查是否出现"情绪词+动作+情绪副词+对话"的叠加
  关键词："愤怒地/冷冷地/开心地/高兴地/激动地/温柔地/平静地/淡淡地/低声/高声"

Step 2：选择最强表达
  从"情绪词/动作/对话"中，选择"最有冲击力"的1-2个保留
  选择优先级：对话内容 > 动作 > 情绪词
  
  示例："她愤怒地一拍桌子，愤怒地说：你太过分了！"
    → 对话"你太过分了！"最有冲击力 → 保留
    → 动作"一拍桌子"有冲击力 → 保留
    → 情绪词"愤怒地"×2 → 删除

Step 3：增强保留项
  保留对话和动作后，增强它们的"表达力"
  - 对话本身够强→删动作，让对话"裸奔"（更有冲击力）
  - 动作本身够强→删对话，让动作"说话"（更含蓄但更有力）
  - 两者都强→保留，但中间用"。"分隔，不放同一句
```

### 4.5 情绪词合并的注意事项

- **对话标签保留规则**：如果对话标签是"说/问/答"等中性词，可以保留；如果是"愤怒地吼道/冷冷地说/开心地笑道"则删除情绪修饰
- **动作保留规则**：如果动作本身有独特性（不是"拍桌子"这种通用动作），保留；如果动作是"万能动作"（拍桌子/叹气/皱眉），删除或替换为独特动作
- **与04精修师铁律1的协同**：铁律1禁止"情绪词直给"（"他很愤怒"），本协议进一步禁止"情绪词+动作+对话"的套娃复读——即使情绪词被"藏"在动作和对话中，也不能叠加使用
- **与05 Gate 3的协同**：Gate 3检测"心理直给"，本协议在Gate 3检测之前，先在04精修阶段做"情绪词合并"——将"告知读者情绪"改为"让读者自己感受情绪"

## 五、精简文笔自检清单

```
【精简文笔自检清单 — 精修/去AI味时逐项核对】

第一招：万能前缀
  □ 全章是否有"他心里想/她暗自思忖/心道/心说"等前缀？
    → 有 = 删除前缀，改为直接内心独白或生理反应
  □ 全章是否有"不禁感到/不由得感到/忽然感到"等前缀？
    → 有 = 删除前缀，改为生理反应
  □ 删除后读者是否仍能感受到角色心理？
    → 不能 = 替换的动作/生理反应不够具体，需要更精准

第二招：主观视角环境描写
  □ 全章是否有"上帝视角"的环境罗列（水晶吊灯/暗金色壁纸/手工地毯）？
    → 有 = 改为通过角色眼睛/动作/情绪带出
  □ 环境描写是否服务于"塑造人设"或"推动剧情"？
    → 没有 = 删除或融入角色视角
  □ 环境描写是否"这个角色独有"？（换一个角色，环境描写会不同）
    → 不会 = 环境描写没有视角锚定，需要重写

第三招：情绪词合并
  □ 全章是否有"情绪词+动作+情绪副词+对话"的套娃？
    → 有 = 只保留对话+动作，删情绪词和情绪副词
  □ 对话标签是否有情绪修饰（愤怒地吼道/冷冷地说/开心地笑道）？
    → 有 = 删除情绪修饰，或改为中性"说"+"。"+动作
  □ 同一情绪是否被表达了≥2次？
    → 有 = 只保留最有冲击力的1次表达
```
