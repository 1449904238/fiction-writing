# 04 质感深化模块（V5.4 — 从04完整版拆分）

> **加载条件**：高情绪场景/战斗场景/SA级场景/核心记忆点场景
> **加载方式**：Read `references/04_质感深化.md`

## 质感深化三阶段完整流程

### 阶段1：质感诊断

扫描03初稿的质感空缺：

1. **细写等级核对**（V2.4 — 详见 references/03_质感注入协议.md 第六章）：
   - S/A级场景被写成快进（欠账）→ 补质感
   - C级场景被写成特写（注水）→ 压缩

2. **感官叠加审计**：
   - 核心场景只有"1个精准细节"但应有"3-5个感官叠加"？→ 升级
   - 过渡场景有"3+感官叠加"？→ 压缩为核心场景的1-2个

3. **对话锋利度审计**：
   - 对话虽然有人味但还不够"锋利"？→ 风格化
   - 角色A和B的对话互换不影响理解？→ 需增加口癖区分

4. **动作链个性化审计**：
   - 动作链虽然正确但太"工整"？→ 个性化
   - 所有角色的战斗描写用同一套动词？→ 差异化

5. **视角锚定审计**：
   - 具体名词精准但缺乏"这个角色独有的视角"？→ 视角锚定
   - 是否存在"上帝视角"的环境罗列？→ 主观化

### 阶段2：风格化质感注入

**核心场景升级法**：
```
03初稿："空调开得很足，屋里凉。"（1个细节，合格但平淡）
04深化："空调开得很足，屋里凉。白术吐出一口白气，白气在冷蓝色射灯下瞬间散开。
         她听见自己的心跳——不对，那不是心跳，是楼下车库的卷帘门在响。"
         （3感官叠加+1个感官错位）
```

**感官错位原则**：核心场景必须包含1个"不是A而是B"的感官错位——角色以为听到/看到/感觉到A，实际上是B。这创造紧张感和真实感。

**对话风格化清单**：
| 角色类型 | 独有语言习惯示例 |
|---------|----------------|
| 军人/武者 | 短句+命令式+省略主语 |
| 学者/谋士 | 长句+引用+假设推理 |
| 商人/市井 | 砍价式表达+利益量化 |
| 少年/热血 | 感叹号+绝对化措辞+比喻泛滥 |
| 老者/沧桑 | 省略号+回忆穿插+言不及义 |

**动作个性化原则**：
- 同一动作，不同角色做出来不一样
- 示例：开门 → 军人：踹开/学者：推门后先观察门框/少年：直接推

### 阶段3：记忆点打磨

**记忆点公式**：出人意料 + 具体到可以画出来 + 与角色内心强关联

**记忆点类型**：
1. **反常动作**：角色做了一个与其性格不符的动作（如冷酷杀手在雨中给流浪猫撑伞）
2. **感官锚点**：一个反复出现但有变化的物件（如父亲的旧手表，每次出现时间不同）
3. **沉默时刻**：在应该说话的时候不说话（如葬礼上唯一没哭的人）
4. **精确数字**：一个异常精确的数字（如"他数了一下，一共137级台阶"）
5. **错位感知**：角色感知到了"不该感知到"的东西（如闻到已故之人的气味）

**记忆点密度**：
- 每章至少1个记忆点
- 高潮章2-3个
- 记忆点之间至少间隔500字（避免堆砌）

## 感官锚点深度穿插

> 从03接收的"感官锚点"在04做深度打磨。

**穿插规则**：
1. 核心物件/场景在全章出现3-5次
2. 每次出现必须有变化（光线/角度/角色的感知/使用状态）
3. 最后一次出现必须与角色情绪产生共振
4. 禁止"物件的独白"——必须通过角色感知呈现

**示例**：
```
第1次（开篇）：桌上那把刀，刀鞘上缠着发黑的旧布。
第2次（中段）：他坐下时手肘碰到了刀——刀鞘冰凉，比屋里任何东西都凉。
第3次（高潮）：他抽出刀，旧布散开。刀刃上有一道豁口，那是三年前留下的。
第4次（结尾）：他把刀放回桌上。豁口朝着窗户，光从豁口里漏进来。
```

## 闲笔与呼吸感（过渡章可选，高潮章不建议）

> V5.2.0降级为过渡章可选技法。

- 每章15-25%的非推进性文字（环境/日常/闲聊）
- 高潮章压缩至5-10%
- 闲笔必须服务角色塑造或氛围营造，不可纯粹水字数
- 闲笔内容应与主线有隐喻/对照关系
