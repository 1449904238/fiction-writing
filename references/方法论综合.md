## 开篇方法与决策树

# Part K: 开篇方法与决策树

> 融合自 oh-story-claudecode 项目。不同类型的小说需要不同的开篇策略。

## K.1 开篇5种方法

```
方法1：危机切入（推荐指数：⭐⭐⭐⭐⭐）
├─ 适用：爽文、悬疑、都市、玄幻
├─ 做法：直接从主角身处危机中开始
├─ 优势：最快抓住读者，前100字制造紧张
├─ 示例：主角被追杀/被困绝境/面临死亡威胁
├─ 注意：危机不能太复杂，让读者一眼看懂"主角有危险"
└─ 番茄适配：最推荐的开篇方式，符合"前500字必有爽点"原则

方法2：日常异常（推荐指数：⭐⭐⭐⭐）
├─ 适用：悬疑、灵异、都市、科幻
├─ 做法：从日常场景开始，但有一个不合理的细节
├─ 优势：循序渐进，让读者在熟悉中感受异常
├─ 示例：正常上班→发现工位上多了一件不属于自己的东西
├─ 注意：异常必须在500字内出现，否则就是"平淡日常"
└─ 番茄适配：适合氛围型作品，但需要较快揭示异常

方法3：金手指觉醒（推荐指数：⭐⭐⭐⭐）
├─ 适用：升级流、系统流、修仙、玄幻
├─ 做法：开场直接展示主角获得金手指/外挂
├─ 优势：直接满足读者期待，快速建立期待感
├─ 示例：主角获得系统/传承/特殊能力/重生
├─ 注意：觉醒过程要快（≤500字），觉醒后立刻展示效果
└─ 番茄适配：系统流/升级流首选，配合"爽点验证"效果最佳

方法4：对话开场（推荐指数：⭐⭐⭐）
├─ 适用：言情、都市情感、轻小说
├─ 做法：从一段抓人的对话开始
├─ 优势：快速建立角色关系，制造信息差
├─ 示例：一句让人摸不着头脑但很好奇的话
├─ 注意：对话必须蕴含冲突或悬念，不能是日常寒暄
└─ 番茄适配：言情类可考虑，但要注意对话后尽快进入剧情

方法5：结果前置（推荐指数：⭐⭐⭐⭐）
├─ 适用：悬疑、推理、复仇、都市
├─ 做法：先展示一个震撼的结果，再倒叙解释过程
├─ 优势：制造强烈的好奇心，"他怎么会变成这样？"
├─ 示例：主角站在仇人坟前→倒叙他是怎么走到这一步的
├─ 注意：结果必须足够震撼，让读者极度想知道"为什么"
└─ 番茄适配：适合复仇流/悬疑流，开局就制造最大悬念
```

## K.2 开篇决策树

```
问：你的小说主打什么？
├─ 主打"爽感"（打脸/碾压/升级）
│   └─ → 方法1（危机切入）或 方法3（金手指觉醒）
│       问：主角开局是强者还是弱者？
│       ├─ 强者 → 方法1：危机中展示实力，让人震惊
│       └─ 弱者 → 方法3：金手指觉醒，让人期待变强
│
├─ 主打"悬念"（悬疑/推理/灵异）
│   └─ → 方法2（日常异常）或 方法5（结果前置）
│       问：悬念是"过程之谜"还是"结果之谜"？
│       ├─ 过程之谜（怎么发生的）→ 方法5：先展示结果
│       └─ 结果之谜（发生了什么）→ 方法2：从异常开始调查
│
├─ 主打"情感"（言情/都市情感/纯爱）
│   └─ → 方法4（对话开场）或 方法2（日常异常）
│       问：核心冲突是"关系"还是"事件"？
│       ├─ 关系冲突 → 方法4：从对话/互动切入
│       └─ 事件冲突 → 方法2：从改变日常的事件切入
│
└─ 主打"新奇"（科幻/无限流/架空）
    └─ → 方法2（日常异常）或 方法5（结果前置）
        问：新奇感来自"世界观"还是"情节"？
        ├─ 世界观新奇 → 方法2：从日常中展示异常规则
        └─ 情节新奇 → 方法5：从震撼结果入手
```

## K.3 开篇通用禁忌

```
1. 不写大段环境描写
2. 不写角色起床/洗漱/吃饭
3. 不写大段回忆/背景介绍
4. 不写没有冲突的对话
5. 不写读者"看不懂"的设定
```

## 文风系统

# Part L: 文风系统

> 融合自 oh-story-claudecode 项目。文风是决定读者能否沉浸的核心要素。

## L.1 对标文风提示

确定对标作品后，按以下维度提取文风特征：

```
【对标文风提取框架】

1. 句式风格
├─ 对标作品倾向短句还是长句？
│   ├─ 短句为主（≤15字）→ 节奏快，适合爽文/动作/悬疑
│   ├─ 中句为主（15-30字）→ 节奏适中，适合都市/言情
│   └─ 长句为主（30+字）→ 节奏慢，适合文艺/历史/史诗
├─ 对标作品段落长度？
│   ├─ 短段为主（1-3行）→ 轻快，适合手机阅读
│   └─ 中长段为主（4-8行）→ 厚重，适合沉浸阅读
└─ 对标作品是否常用"口语化"表达？
    ├─ 是 → 文风偏"说书人"口吻
    └─ 否 → 文风偏"文学性"叙述

2. 感官侧重
├─ 对标作品最常用的感官是什么？
│   ├─ 视觉为主 → 描写偏"画面感"
│   ├─ 触觉/嗅觉为主 → 描写偏"沉浸感"
│   └─ 听觉为主 → 描写偏"节奏感"
└─ 输出：你的文风感官优先级排序

3. 情绪密度
├─ 对标作品情绪表达的频率？
│   ├─ 高频（每段都带情绪）→ 情绪驱动型，偏感性
│   ├─ 中频（关键处带情绪）→ 平衡型，张弛有度
│   └─ 低频（大多数冷静叙述）→ 情节驱动型，偏理性
└─ 输出：你的文风情绪密度定位

4. 对话风格
├─ 对标作品对话占比？
│   ├─ 高（>50%）→ 对话驱动，角色互动为主
│   └─ 低（<30%）→ 叙述驱动，情节推进为主
├─ 对话长短？
│   ├─ 短对话为主（1-2句/轮）→ 简洁，快节奏
│   └─ 长对话为主（3-5句/轮）→ 深度，信息量大
└─ 输出：你的文风对话策略

5. 叙述者距离
├─ 对标作品的叙述者离角色多远？
│   ├─ 贴身（自由间接引语，融入角色视角）→ 代入感强
│   ├─ 中距（第三人称有限视角）→ 平衡
│   └─ 远距（全知视角，偶尔跳出来评论）→ 古典说书人感
└─ 输出：你的文风叙述者距离
```

### 对标文风提取操作步骤（5步法）

**第1步：选段采样**
从对标作品中选取3-5个连续章节（含1个爽点高潮章+1个日常过渡章+1个情感/转折章），确保覆盖该作品的主要场景类型。

**第2步：逐维度标注**
对采样段落逐一标注5个维度的特征：
- 句式风格：统计短/中/长句比例，记录段落平均行数
- 感官侧重：标记每段使用了哪些感官（视/触/嗅/听/味），统计频率
- 情绪密度：统计每段中情绪词/情绪动作的出现次数
- 对话风格：计算对话占比，标注对话平均长度
- 叙述者距离：判断是自由间接引语、有限视角还是全知视角

**第3步：归纳模式**
将标注结果汇总，找出对标作品最稳定的文风模式（如"短句+触觉优先+高频情绪+简洁对话+贴身视角"）。

**第4步：设定偏差容忍度**
为每个维度设定允许的偏差范围（如句式风格允许±10%偏移），超出范围则标记为文风偏离。

**第5步：写入项目设定**
将提取结果写入《00_小说设定文档》的 `style_profile` 字段，供后续扩写和精修步骤参照。

### 提取案例：某都市爽文（都市·打脸流）

**对标作品**：《都市之巅峰归来》（虚构示例）
**采样章节**：第12-14章（含打脸高潮+日常过渡+情感转折）

**提取结果**：

| 维度 | 提取结果 | 具体数据 |
|------|----------|----------|
| 句式风格 | 短句为主（≤15字），段落1-3行 | 短句占比68%，中句25%，长句7%；平均段落2.1行 |
| 感官侧重 | 触觉>视觉>听觉 | 触觉描写占42%，视觉31%，听觉19%，嗅觉/味觉8% |
| 情绪密度 | 高频，用行动表达 | 平均每段1.8个情绪标记点；70%通过动作/身体反应体现 |
| 对话风格 | 短对话为主，简洁有力 | 对话占比55%；平均每轮1.4句；无超过3句的长独白 |
| 叙述者距离 | 贴身（自由间接引语） | 80%段落融入主角视角，无全知评论 |

**归纳模式**：短句快节奏 + 触觉驱动 + 行动化情绪 + 简洁对话 + 贴身视角

**偏差容忍度**：
- 句式：短句占比不低于55%
- 感官：触觉不低于30%
- 情绪密度：每段不少于1个标记点
- 对话占比：45%-65%
- 叙述距离：贴身视角占比不低于70%

## L.2 文风优先级系统

```
爽文/打脸流（番茄核心）：
├─ 优先级1：句式节奏（短句为主，快节奏）
├─ 优先级2：对话风格（简洁有力，不废话）
├─ 优先级3：情绪密度（高频，但用行动表达）
├─ 优先级4：感官侧重（触觉/身体感受优先）
├─ 优先级5：叙述者距离（贴身，代入感）

悬疑/灵异流：
├─ 优先级1：感官侧重（嗅觉/听觉创造不安感）
├─ 优先级2：情绪密度（低频但精准，制造压抑）
├─ 优先级3：句式节奏（中短句为主，偶尔长句制造窒息感）
├─ 优先级4：对话风格（长对话为主，信息量大）
├─ 优先级5：叙述者距离（中距，让读者比角色多知道一点）

言情/情感流：
├─ 优先级1：对话风格（长对话，有拉扯感）
├─ 优先级2：情绪密度（高频，情感细腻）
├─ 优先级3：叙述者距离（贴身，感受角色内心）
├─ 优先级4：感官侧重（触觉/温度优先，创造亲密感）
├─ 优先级5：句式节奏（中长句为主，节奏舒缓）

升级/成长流：
├─ 优先级1：句式节奏（升级时短句爆发，日常时中句）
├─ 优先级2：情绪密度（升级时高频，日常时低频）
├─ 优先级3：感官侧重（战斗时触觉/身体感受，日常时平衡）
├─ 优先级4：对话风格（战斗时简洁，日常时有深度）
├─ 优先级5：叙述者距离（贴身，跟随主角视角）

科幻/无限流：
├─ 优先级1：感官侧重（视觉为主，想象力优先）
├─ 优先级2：对话风格（长对话，信息量大，解释设定）
├─ 优先级3：句式节奏（中句为主，稳定输出信息）
├─ 优先级4：情绪密度（中频，平衡）
├─ 优先级5：叙述者距离（中距，保持一定客观性）
```

### 跨题材优先级冲突处理规则

当作品同时具备多个题材标签（如"都市+悬疑"、"玄幻+言情"）时，文风优先级可能冲突。按以下规则处理：

**规则1：以核心爽点驱动题材为准**
- 判断标准：作品的主线爽点类型（打脸→爽文优先级；解谜→悬疑优先级；情感拉扯→言情优先级）
- 示例："都市+悬疑"但核心爽点是打脸揭密 → 以爽文优先级为主，悬疑优先级为辅

**规则2：按场景类型动态切换**
- 打脸/战斗/升级场景 → 适用爽文优先级
- 调查/探索/恐怖场景 → 适用悬疑优先级
- 感情/回忆/独处场景 → 适用言情优先级
- 切换信号：场景类型变化时，句式节奏和感官侧重随之调整

**规则3：感官侧重就高不就低**
- 当两个题材对同一感官有不同优先级要求时，取更高优先级
- 示例：爽文要求触觉优先，悬疑要求嗅觉优先 → 战斗场景用触觉，探索场景用嗅觉

**规则4：叙述者距离以核心题材为准**
- 叙述者距离是最难中途切换的维度，全程以核心题材的设定为准
- 示例：核心是爽文 → 全程贴身视角，即使在悬疑段落也不切换为远距

## L.3 文风一致性检查

每5章检查：句式风格、感官侧重、情绪密度、对话风格、叙述者距离是否保持一致。

### 文风一致性检查操作示例

**检查时机**：每写完5章后执行一次。

**操作流程**：

1. **抽样**：从最近5章中随机抽取3个段落（1个爽点段+1个日常段+1个情感段）

2. **逐项核对**（以"都市爽文"文风为例）：

| 检查项 | 标准 | 检查方法 | 判定 |
|--------|------|----------|------|
| 句式风格 | 短句占比≥55% | 统计抽样段落中≤15字句子占比 | 低于55%→标记偏离 |
| 段落长度 | 平均≤3行 | 统计抽样段落平均行数 | 超过4行→标记偏离 |
| 感官侧重 | 触觉≥30% | 统计感官描写中触觉占比 | 低于30%→标记偏离 |
| 情绪密度 | 每段≥1个标记 | 统计情绪词/动作出现次数 | 0个→标记偏离 |
| 对话占比 | 45%-65% | 计算对话字数/总字数 | 超出范围→标记偏离 |
| 叙述距离 | 贴身视角≥70% | 判断视角类型占比 | 低于70%→标记偏离 |

3. **偏离处理**：
- 1-2项偏离 → 在下一章写作时针对性修正
- 3项以上偏离 → 暂停写作，重新校准文风设定
- 叙述距离偏离 → 最严重，必须立即修正（此维度最难通过局部调整修正）

4. **记录**：将检查结果写入《上下文管理模板》的文风追踪栏，持续监控趋势。

## 题材写作公式

# Part M: 题材写作公式

> 融合自 oh-story-claudecode 项目。21种主流网文题材的核心公式。

## M.1 题材写作公式速查

```
1. 都市爽文（赘婿/神医/神豪/战神）
   公式：被低估 → 冲突触发 → 身份/能力揭露 → 打脸 → 更大冲突
   关键：身份反差越大越爽，打脸层次递进

2. 都市修真/修仙
   公式：修真者入都市 → 用修真手段解决都市问题 → 引起修真界注意 → 双线冲突
   关键：修真手段与都市规则的碰撞

3. 都市异能
   公式：获得异能 → 隐藏身份 → 异能暴露危机 → 更大异能组织出现
   关键：隐藏身份与使用异能的矛盾

4. 系统流
   公式：获得系统 → 完成任务 → 升级 → 系统功能解锁 → 系统背后的秘密
   关键：任务设计要有趣，系统不能是万能外挂

5. 重生流
   公式：重生回到过去 → 利用先知优势 → 改变命运 → 蝴蝶效应 → 新的危机
   关键：先知优势要逐步消耗，蝴蝶效应制造新冲突

6. 穿越异世界
   公式：穿越 → 适应新世界 → 利用现代知识 → 在新世界崛起 → 寻找回家之路
   关键：现代知识与异世界规则的碰撞

7. 玄幻修仙
   公式：入门 → 修炼 → 突破 → 宗门/家族冲突 → 飞升/更高世界
   关键：升级体系要清晰，每次突破要有代价

8. 玄幻废柴逆袭
   公式：被看不起 → 意外获得机缘 → 逆袭 → 证明自己 → 成为最强
   关键：被看不起的阶段要让读者憋屈

9. 悬疑灵异
   公式：异常事件 → 调查 → 线索碎片 → 误导 → 真相反转 → 更大的阴谋
   关键：线索要分层释放，每次接近真相都要付出代价

10. 盗墓探险
    公式：获得线索 → 组队 → 进入古墓 → 机关/怪物 → 宝物/真相 → 逃脱
    关键：团队成员各有功能，要有牺牲和背叛

11. 无限流
    公式：进入副本 → 理解规则 → 策略/战斗 → 通关 → 获得奖励 → 副本背后的秘密
```

## M.2 题材融合规则

```
可融合的题材（互补型）：
├─ 都市 + 修真 = 都市修真（热度最高）
├─ 重生 + 穿越 = 重生异世界（热度高）
├─ 系统 + 任何题材 = 系统流变体（万能搭配）
├─ 悬疑 + 灵异 = 灵异探案（经典组合）
├─ 言情 + 古装 = 古言（经典组合）
├─ 末世 + 重生 = 重生末世（热度高）
└─ 玄幻 + 废柴逆袭 = 标准玄幻升级流

谨慎融合（冲突型）：
├─ 言情 + 硬科幻 → 情感线和科幻线容易互相干扰
├─ 爽文 + 深度文艺 → 节奏冲突
├─ 悬疑 + 搞笑 → 氛围冲突
└─ 恐怖 + 言情 → 需要极强的平衡能力

融合原则：
├─ 主题材占70%以上
├─ 融合不是简单叠加，要找到"化学反应"
├─ 如果融合导致节奏混乱 → 放弃融合
└─ 番茄平台：一个主题材+一个辅题材是最佳配置
```

## 命名协议

# 命名协议

> **核心定位**：名字不是标签，是世界观的一部分。一个好名字自带信息量——读者听到名字的瞬间，就能闻到这个人/地/物的味道。
> **痛点对症**：AI 取名"想当然"——主角叫"林逸"、反派叫"魔尊"、宝剑叫"天罡"，名字是空的，不承载任何设定信息，也不埋伏笔。本协议强制在取名前先思考"这个名字要承担什么功能"。
> **适用步骤**：00（设定架构，主要落地点）/ 01（大纲，地名/势力名）/ 02（细纲，新登场角色/物品）/ 03（扩写，临时命名）

---

## 一、核心原则：先定功能，再取名字

**一句话法则**：
> 取名之前先回答——这个东西在故事里干什么？它的特性是什么？它处在什么位置？它要不要埋伏笔？回答完这些，名字自己就浮出来了。

**反面教训**：先取一个"好听"的名字，再硬塞设定进去——名字和设定两张皮，读者记不住，作者自己也忘了为什么叫这个。

---

## 二、按重要度分级命名（核心校准机制）

名字的"用力程度"必须与它在故事中的权重匹配。重要角色多方面考量，路人随便取一个即可——把精力花在刀刃上。

| 重要度 | 适用对象 | 命名投入 | 考量维度 | 示例 |
|--------|---------|---------|---------|------|
| **P0 核心** | 主角/核心反派/CP对象/贯穿全书的金手指或关键道具/核心地点 | 多维度深度考量，选最合适的 | 功能+特性+位置+伏笔+隐喻，至少3维交叉 | 道诡"李火旺"——火+旺=偏执狂热，呼应认知错位主题 |
| **P1 重要** | 重要配角/中期反派/重要势力/重要法宝/重要场景 | 1-2个维度精准命中 | 特性或伏笔二选一，名要"立得住" | "净血"——两字交代部族最稀缺的资源 |
| **P2 次要** | 次要配角/过渡地点/普通道具 | 单维度即可 | 按性格/外貌/职业取，贴角色即可 | 一个贪财的商人叫"金三两" |
| **P3 路人** | 一次性NPC/无名小卒/普通物品 | 随便取/不取名 | 甚至可以"不用名字"——"那个秃头""穿红衣的" | "店小二""老张""那把刀" |

**执行规则**：
1. 取名前先在心里（或设定文档里）标注这个东西的 P 级别
2. P0 必须写明"为什么叫这个"（命名理据），P1 可省略，P2/P3 不需要
3. 禁止给路人取"重量级"名字——一个只出场一次的 NPC 叫"凌天"会让读者误以为他是重要角色（预期错位）
4. 禁止给核心角色取"路人感"名字——主角叫"张三"除非有刻意的反差设计（且需在设定中说明）

---

## 三、命名维度库（取名时从这些维度切入）

### 维度1：功能/作用
名字直接暗示这个东西在故事里干什么。
- 物品按功能：能照见过去的镜子叫"回光"，能锁住气运的锁叫"封运"
- 势力按职能：专门猎杀异类的组织叫"清异司"而非"天机阁"（天机阁看不出职能）
- ❌ 空洞：一把剑叫"天罡剑"（天罡和剑的功能无关）
- ✅ 贴功能：一把专门斩断因果的剑叫"断缘"

### 维度2：特性/特点
名字暗示这个东西的物理特性、性格特质或能力边界。
- 人按性格：急性子的叫"霍急"（霍=快），阴沉的叫"沈默"（沈=沉+默）
- 物品按外观：发红光的石头叫"赤髓"，带裂纹的玉叫"裂璧"
- ❌ 错位：一个莽撞粗人叫"谨言"（名字和性格打架，除非是反讽且后续有交代）

### 维度3：所处位置/来历
名字带地理或来历信息，让读者一听就知道它从哪来。
- 地名带地形：山间的关隘叫"夹石关"，水边的集镇叫"临水集"
- 人名带出身：北方游牧民族的角色名带草原语感，南方水乡的角色名带柔婉字
- 势力名带根据地：盘踞东境的叫"东垣"而非"星河会"（星河会看不出地盘）

### 维度4：剧情伏笔
名字本身埋着一条暗线，读到后面才恍然大悟。
- 人名藏身份：卧底叫"归衡"（归=回归本营，衡=制衡），结局揭晓时读者回看名字才反应过来
- 物名藏命运：一把最终会反噬主人的剑叫"噬主"——前期读者以为是夸张命名，后期成真
- 地名藏秘密：一个藏宝的洞叫"空骨窟"——空骨=前人死绝，窟里其实有宝
- ⚠️ 伏笔命名不可太直白（"噬主"是后期才懂的，前期不能让读者一眼看穿），需要"表面意思"和"深层意思"双层

### 维度5：隐喻/主题
名字呼应故事的主题或母题。
- 主题是"遗忘"：主角叫"忘川"，组织叫"无忆"
- 主题是"代价"：金手指叫"典当"——用能力=当掉一部分自己
- ⚠️ 隐喻命名是 P0 专属，P1 及以下不要硬上隐喻，否则全员都"有深意"=没深意

### 维度6：性格/外貌（P2-P3 专用快捷维度）
不重要的人/物，按一个显性特征随手取，贴角色即可。
- 按外貌：脸上有疤的叫"疤六"，瘦高的叫"竹竿"
- 按性格：爱笑的叫"笑面"，话多的叫"碎嘴张"
- 按职业：打铁的叫"铁匠老周"，算命的叫"瞎子李"
- 这类名字"不讲究"就是讲究——路人的名字就该有路人的质感

## 四、分品类命名方法

### 4.1 人物命名

| 角色类型 | 命名要点 | 反面案例 |
|---------|---------|---------|
| 主角(P0) | 名字要能"立住人设"——读者听到名字能联想性格/命运/主题。可用隐喻或伏笔维度 | 叫"林逸"——逸=飘逸，但和"挣扎求生"的剧情无关，空 |
| 核心反派(P0) | 名字最好带"压迫感"或"镜像感"——和主角名字形成对照 | 反派叫"暗魔"——太直白，像NPC |
| CP对象(P0-P1) | 名字的字义可和主角形成呼应/对照/补全 | —— |
| 配角(P1-P2) | 按性格或职业取，一个维度命中即可 | 配角叫"凌天"——名字比主角还霸气，喧宾夺主 |
| 路人(P3) | 随手取/不取名，"老X""小X""职业+姓"即可 | 路人叫"独孤求败"——用力过猛 |

**人物命名禁忌**：
- 禁止所有角色都用"复姓+单字"（独孤X、慕容X、轩辕X）——这是AI取名最大的套路
- 禁止名字和已有知名IP角色撞名（除非是致敬且有交代）
- 同一本书里禁止两个角色名字读音相近（"李寒"和"李涵"）——读者会混淆
- 名字要符合世界观语感：修仙世界用古风名，都市用现代名，不要串味

### 4.2 地点命名

| 地点类型 | 命名要点 | 示例 |
|---------|---------|------|
| 核心地点(P0) | 名字带"压迫感"或"神秘感"，且暗藏剧情 | "无终谷"——进去就出不来的暗示 |
| 重要场景(P1) | 带地形/功能信息 | "断龙岭"——地形如龙脊中断 |
| 过渡地点(P2) | 客观描述即可 | "三岔口""西市" |

**地点命名规则**：
- 地名要有"地理逻辑"：山间的带"岭/谷/关"，水边的带"渡/矶/浦"，不混用
- 同一区域的地名要有"统一语感"：北境的地名都偏粗犷（黑石、铁刃），南方的偏柔婉（烟雨、青荷）
- 地名不可太文艺而失真："落樱缤纷谷"不像地名，像景点

### 4.3 物品命名

| 物品类型 | 命名要点 | 示例 |
|---------|---------|------|
| 关键道具(P0) | 按功能+伏笔命名，名字本身是谜面 | "回光镜"——功能是照见过去，伏笔是它照出的不一定是真相 |
| 重要法宝(P1) | 按功能或外观命名，一个维度命中 | "赤霄"——赤色+冲霄，外观+气势 |
| 普通道具(P2-P3) | 按外观或材质直取 | "那把豁了口的柴刀" |

**物品命名规则**：
- 物品名优先按"功能"取（它能干什么），其次按"外观"取（它长什么样）
- 禁止所有法宝都用"XX剑/XX刀/XX珠"的固定句式——可换"器/骨/髓/引/符"等
- 名字要体现"等级感"：低阶的朴素（铁剑），高阶的带气场（斩岳），不要低阶物取高阶名

### 4.4 组织/势力命名

| 势力类型 | 命名要点 | 示例 |
|---------|---------|------|
| 核心势力(P0) | 名字暗藏其真实目的或秘密 | 表面行医实为暗杀的"悬壶堂"——悬壶=行医，堂=堂而皇之杀人 |
| 重要势力(P1) | 按职能+地域取 | "东缉事厂"——东=方位，缉事=职能 |
| 次要势力(P2) | 一个特征即可 | "铁匠帮" |

**势力命名规则**：
- 组织名要"看名字知性质"：杀手组织别叫"清风明月阁"（除非是伪装，且后续有交代）
- 同一世界的势力名要有"阶层统一感"：朝廷机构用"司/厂/卫"，江湖门派用"门/帮/会"，不混用
- 伪装型组织：表面名和真实职能不符是"设计"，不是"失误"——必须在设定文档中标注

### 4.5 能力/功法/金手指命名

| 能力类型 | 命名要点 | 示例 |
|---------|---------|------|
| 金手指(P0) | 名字要暗含"代价"或"边界"——好能力的名字自带限制感 | "典当"——用一次当掉一段记忆，名字就是代价 |
| 核心功法(P1) | 按效果或来源取 | "吐纳术"——吐故纳新，功能即名 |
| 普通技能(P2) | 直白即可 | "铁砂掌" |

**能力命名规则**：
- 金手指的名字最好"美好中藏着不安"——读者初听觉得厉害，细想觉得不对劲
- 禁止用"混沌/鸿蒙/太初/无极"等空大词给低阶能力命名——大词要用在真正的大能力上

## 五、AI取名反模式（必须规避）

| 反模式 | 表现 | 为什么错 | 修正 |
|--------|------|---------|------|
| 复姓泛滥 | 主角必"独孤/慕容/轩辕/端木" | AI偷懒的标志，读者一眼出戏 | 用单姓，或自创符合世界观的姓 |
| 空大词堆砌 | "凌天/破虚/灭世/弑神" | 名字和角色实际能力/地位不匹配 | 名字要匹配角色的实际量级 |
| 全员文艺 | 所有角色名都像诗人 | 失去阶层/性格区分 | 草莽有草莽的名，文人有文人的名 |
| 名实不符 | 名字暗示A，角色其实是B | 除非是刻意的伏笔/反讽，否则是失误 | 名字和设定对齐 |
| 撞梗 | 和热门IP角色同名/同音 | 读者出戏，联想跳到别的书 | 取名后自查是否撞知名角色 |
| 工具人名 | 配角叫"甲乙丙丁""路人A" | 暴露敷衍，读者觉得不被尊重 | 哪怕路人也给个"职业+姓" |

## 六、命名一致性校验

取名完成后，须执行以下校验：

```
【命名一致性自检】

1. 重要度匹配检查：
   → P0角色是否多维度考量了？是否在设定文档写了"命名理据"？
   → 是否有路人取了"重量级"名字（预期错位）？
   → 是否有核心角色取了"路人感"名字（除非刻意反差且已标注）？

2. 世界观语感一致性：
   → 所有角色名是否符合世界观的语言风格？（修仙古风/都市现代/科幻未来不串味）
   → 同一区域的地名是否有统一语感？

3. 混淆度检查：
   → 是否有两个角色名字读音相近？→ 是=改其中一个
   → 是否有角色名和地名撞名？→ 是=改其中一个

4. 撞梗检查：
   → 核心角色名是否和知名IP角色撞名/同音？→ 是=改

5. 伏笔命名隐蔽性检查（若用了伏笔命名）：
   → 表面意思是否成立、不突兀？（读者第一眼不会觉得奇怪）
   → 深层意思是否要到揭晓时才恍然？（不会一眼看穿）

6. 反模式扫描：
   → 是否复姓泛滥？是否空大词堆砌？是否全员文艺？
   → 命中 → 按修正表改
```

## 新事物释放协议

# 新事物释放协议

> **核心定位**：当故事里出现一个新东西（概念/物品/规则/能力/组织/地名/术语）时，读者必须能"接得住"——不能一脸懵地往下读。本协议定义"何时释放、释放多少、用什么方式释放"。
> **痛点对症**：本 skill 的反解释体系（Gate 11解释腔 / 冰山原则 / 潜台词拦截器 / 铁律5）极其强势，导致 AI 走向另一个极端——**该解释的也不解释了**，读者读到"净血""拓脉""煞"这类自造词时完全不知道是什么，爽点和恐惧感都无从谈起。本协议是反解释体系的"补丁"：它不推翻"禁止解释腔"，而是划清"必须释放"和"禁止解释"的边界。
> **适用步骤**：02（细纲，规划释放节奏）/ 03（扩写，执行释放）/ 04（精修，校验读者是否接得住）

---

## 一、核心原则：释放 ≠ 解释腔

**最关键的区分——这是本协议存在的意义**：

| | 释放（必须做） | 解释腔（禁止做） |
|---|---|---|
| **谁在说** | 角色通过对话/行动/反应带出，或场景本身呈现 | 叙述者跳出故事，像写百科词条 |
| **形式** | 融在剧情里，不打断节奏 | 独立成段的定义说明，打断节奏 |
| **读者感受** | "哦，原来是这样"（自然获得信息） | "作者在给我上课"（被喂饭） |
| **信号词** | 无（信息藏在行为/对话/场景里） | "在XX，这个词意味着…""所谓XX，就是…""XX，是一种…" |
| ❌ 反面 | 读者读到自造词一头雾水，爽点落空 | 叙述者停下来科普，节奏断裂 |

**一句话法则**：
> 新东西第一次出现时，读者必须能在"不跳出故事"的前提下理解它——要么前文已铺垫，要么当下场景让它自明，要么角色替读者问出来。绝不能让读者带着疑问硬读三页，也绝不能让叙述者停下来上课。

## 二、什么是"新事物"（释放对象）

| 类别 | 举例 | 读者需要知道什么 |
|------|------|----------------|
| **自造概念/术语** | "净血""拓脉""煞""染煞" | 这东西是什么，为什么重要，碰到它会怎样 |
| **特殊物品/道具** | "天光石""无面陶俑""回光镜" | 它干什么用的，谁在用，用了有什么代价 |
| **世界规则/制度** | "净血不出猎""按户分配天光石" | 规则是什么，违反会怎样，谁定的 |
| **能力/功法** | "典当""吐纳术" | 能干什么，代价是什么，边界在哪 |
| **组织/势力** | "清异司""悬壶堂" | 是干什么的，站在哪边，多强 |
| **地名** | "无终谷""断龙岭" | 哪里，什么特征，为什么去/不能去 |
| **身份/称谓** | "净血者""守夜人" | 这身份意味着什么，社会地位如何 |

> **不需要释放的**：一眼就能看懂的常识词（"客栈""铜钱""马"），或作者用日常词替代的（"那块石头"而非自造名）。一眼能懂的不解释，越重要的越要详细释放——把精力花在刀刃上。

## 三、按重要度分级释放（核心校准机制）

释放的"详细程度"必须与新事物在故事中的权重匹配。

| 重要度 | 适用对象 | 释放详细度 | 释放节奏 | 示例 |
|--------|---------|-----------|---------|------|
| **P0 核心** | 贯穿全书的核心设定/金手指/核心规则 | 详细释放——功能+代价+边界+禁忌都要交代清楚 | 分3-4章渐进释放（悬念→暗示→确认→展开） | "净血"——全书核心概念，须交代它是什么、为什么稀缺、违反禁忌会怎样 |
| **P1 重要** | 卷级重要设定/中期关键道具/重要势力 | 中等释放——交代功能+1个限制即可 | 1-2章内释放，仍须对话互动带出 | "天光石"——交代它是能源+按户分配的稀缺性 |
| **P2 次要** | 章节级设定/普通道具/过渡规则 | 简短释放——一句话点出功能即可 | 当章释放，一句话带过 | "铁砂掌"——"一掌拍下去，砖碎了"即可 |
| **P3 背景碎片** | 纯氛围词/一次性背景 | 不主动释放——靠上下文自明 | 散落融入，每章2-3句 | 某个地名只在对话里提一嘴，不解释 |

**执行规则**：
1. 出现新事物时，先判定它的 P 级别
2. P0 必须有"释放计划"（在哪几章、用什么方式、释放哪个层面），写入细纲
3. **一眼能懂的不释放**——"客栈"不需要解释是住店的地方
4. **越重要的越要早铺垫**——P0 概念第一次出现可以只给悬念不给答案，但不能让读者完全不知所云

## 四、八大释放技法（从这些方式中选最合适的）

> **核心原则**：不直接解释，而是通过剧情、人物、场景把信息"漏"给读者。根据剧情选择当下最自然的方式——不是所有东西都用同一种方法释放。

### 技法1：角色对话带出（最常用）
通过角色的嘴说出来，但必须有**信息差**——一方不知道、好奇、质疑，另一方回答。禁止"双方都懂却讲给读者听"的科普对话。

| ❌ 科普对话（解释腔） | ✅ 信息差对话（释放） |
|---------------------|---------------------|
| "净血，就是未染煞的纯血之人，在我们部族里比强壮更值钱。"（双方都懂，讲给读者） | "阿介才十六，还没拓脉，怎么能出猎？""他净血。净血不出猎，这是规矩。"（一方质疑，一方回答，自然带出） |

**要点**：必须有人"不知道"或"质疑"，让解释有合理性。若双方都懂，改为动作/观察带出。

### 技法2：角色反应带出
不解释这个事物是什么，而是写**角色碰到它时的反应**——读者从反应里推断它是什么。

| ❌ 直接解释 | ✅ 角色反应释放 |
|-----------|---------------|
| "煞是一种致命的污染，染煞者会失去理智。" | 沙九看到阿介的尸体，脸色变了："才十六岁，还没拓脉。"老长老没说话，把门关上了。（读者从"脸色变了""没说话"推断煞的可怕） |

**要点**：反应要"小而具体"——不是"他震惊了"，而是"他手里的碗掉了"。

### 技法3：生活场景呈现（设定藏生活里）
不写设定文档，写分配制度/交易场景/生活习惯，让读者通过"逛"来了解。

| ❌ 设定说明 | ✅ 生活场景释放 |
|-----------|---------------|
| "这个世界能量很稀缺，按户分配。" | "天光石按户分配，每户每天拳头大的一块。王家婶子总是提前两小时熄了石。"（分配制度+具体家庭+生活习惯） |

**要点**：写一个"分配/交易/使用"的具体场景，比写十句设定说明都管用。详见 `03_质感注入协议.md` 技法3。

### 技法4：前置铺垫（先埋钩子，后揭谜底）
新事物第一次出现时不解释，只给一个**引发好奇的切片**；几章后读者积累了足够线索，再让它"自明"。

| 节奏 | 做法 | 示例 |
|------|------|------|
| 第1次出现 | 只给现象+悬念，不给定义 | "门没有招牌。前台让他在没有格子的白纸上填名字，不填身份证号。"（读者：这是什么地方？） |
| 第2-3次出现 | 补充线索，读者开始拼图 | "纸是A4的，但比普通A4厚，摸上去有布的质感。"（读者：越来越奇怪） |
| 第4次出现 | 读者已能推断，确认即可 | "他终于知道那张纸为什么没有格子——格子会限制你能写下的东西。"（读者恍然） |

**要点**：前置铺垫适合 P0 核心设定——越重要的东西，越值得用3-4章慢慢释放。

### 技法5：后置揭秘（先见效果，后补原理）
先让新事物**发挥作用**（读者看到效果），后补它的原理/代价。读者先被效果震撼，再理解机制。

| ❌ 先解释后使用 | ✅ 先使用后揭秘 |
|---------------|---------------|
| "典当是一种能力，用一次会失去一段记忆……"然后他用了 | 他念了一个词，对手的动作停了。但第二天他醒来，想不起母亲的脸。（先见效果+代价，读者自己推断机制） |

**要点**：后置揭秘适合**能力/金手指**——先让读者爽到/惊到，再解释为什么。禁止"先说明书后使用"。

### 技法6：物件暗示
不解释规则，写一个**体现规则的物件/标识/文书**，让读者自己读懂。

| ❌ 直接解释规则 | ✅ 物件暗示释放 |
|---------------|---------------|
| "净血者被严禁参与高危猎杀，这是部族的铁律。" | 墙上钉着一张兽皮，上面画了三种人形，中间那种被红泥划掉了。旁边写着三个字：不出猎。（读者从兽皮告示推断规则） |

### 技法7：误解纠正
让一个角色**带着误解**接触新事物，被纠正的过程中，读者跟着一起理解。

| ❌ 直接解释 | ✅ 误解纠正释放 |
|-----------|---------------|
| "回光镜能照见过去，但照出的不一定是真相。" | "它能让我看见昨天？"他伸手去摸。老者把镜子挪开："能。但镜子也会撒谎。你以为你看见的是昨天，其实它照的是你最怕看见的。" |

**要点**：误解纠正自带信息差，是最自然的释放方式之一。

### 技法8：代价/后果展示
不解释这个事物是什么，写**使用它的代价/后果**——读者从代价反推它是什么。

| ❌ 直接解释 | ✅ 代价展示释放 |
|-----------|---------------|
| "典当会消耗记忆作为代价。" | 他用完能力后，摸了摸口袋——里面有张照片，但他想不起照片上的人是谁了。（读者从"想不起"反推代价=记忆） |

## 五、释放时机选择（前/同步/后/延时）

| 时机 | 适用场景 | 要点 |
|------|---------|------|
| **前置铺垫** | P0核心概念，读者不懂就无法理解后续剧情 | 先给悬念切片，不一次性说完 |
| **同步释放** | 当下场景必须懂才能看懂 | 用对话/反应/生活场景当章带出 |
| **后置揭秘** | 先看效果更震撼的能力/物品 | 先让读者惊到，再补原理 |
| **延时拼图** | 悬疑型设定，越晚揭晓越有张力 | 分3-4次散落线索，最后确认 |

**选择原则**：
- 剧情当下**必须懂才能看懂** → 同步释放（技法1/2/3/6）
- 剧情当下**不懂也不影响推进**，但越晚懂越有冲击 → 前置铺垫或延时拼图（技法4）
- 这个事物**先看效果更震撼** → 后置揭秘（技法5/8）
- 读者**会带着错误理解**接触它 → 误解纠正（技法7）

> **根据剧情选择最合适的方式**——不是所有东西都同一种方法。同一章里出现三个新事物，可以一个用对话带出、一个用反应暗示、一个用物件呈现，避免方法单一化。

## 六、与反解释体系的调和（关键章节）

> ⚠️ 本节解决"释放协议"与"Gate 11解释腔 / 冰山原则 / 潜台词拦截器 / 铁律5"的冲突。两者不矛盾——它们管的是不同的东西。

### 6.1 边界裁定表

| 情况 | 归属 | 判定 |
|------|------|------|
| 新概念第一次出现，读者完全不懂，需要信息 | **释放协议管辖**（必须释放） | 读者能不能在不跳出故事的前提下理解？能=合格 |
| 叙述者停下来写"在XX，这个词意味着…"的定义句 | **解释腔（禁止）** | 是叙述者在说还是角色/场景在说？叙述者=违规 |
| 角色对话带出设定，且有信息差 | **释放协议·技法1**（允许） | 双方都懂却讲给读者听=违规；一方好奇/质疑=合格 |
| 角色的心理结论"他知道/他发现/这意味着" | **潜台词拦截器（禁止）** | 这是心理直给，不是设定释放——两者不同 |
| 通过生活场景/物件/反应呈现设定 | **释放协议·技法2/3/6**（允许且鼓励） | 不打断节奏=合格 |
| 后置揭秘——先看效果后补原理 | **释放协议·技法5**（允许） | 原理通过场景/角色带出=合格；叙述者直接说=违规 |

### 6.2 一句话裁定法

> **问自己**：这段信息是"叙述者跳出来讲的"，还是"角色/场景自然漏出的"？
> - 叙述者讲的 → 检查是不是解释腔（看有没有"在XX，这意味着""所谓XX就是"信号词）
> - 角色漏出的 → 检查有没有信息差（双方都懂的科普对话=违规）
> - 场景漏出的 → 合格（生活场景/物件/反应呈现）

### 6.3 与各反解释规则的对齐

| 反解释规则 | 管什么 | 释放协议管什么 | 关系 |
|-----------|--------|--------------|------|
| Gate 11解释腔 | 禁止叙述者百科式定义 | 要求新事物通过角色/场景释放 | **互补**：Gate11禁"怎么不能释放"，释放协议说"该怎么释放" |
| 冰山原则 | 禁止把因果/心理结论直说 | 要求把设定信息藏进场景 | **一致**：都主张"呈现而非讲述" |
| 潜台词拦截器 | 禁止"他发现/这说明/他心里明白" | 要求通过物证/反应释放信息 | **一致**：都主张用客观事实替代主观结论 |
| 铁律5潜台词解释 | 禁止心理结论直给 | 要求新概念通过对话/场景带出 | **互补**：铁律5管心理，释放协议管设定 |
| F.2分层释放 | 设定3-4章渐进释放 | 本协议的P0节奏=分层释放 | **继承**：释放协议扩展了F.2，增加方法库 |

> **结论**：释放协议是反解释体系的"正面补丁"——反解释体系说"不要这样解释"，释放协议说"那该怎么让读者懂"。两者并行不悖。

## 七、释放自检清单（扩写后自检）

```
【新事物释放自检 — 与脱水自检/质感自检并行执行】

1. 新事物盘点：本章是否首次出现了自造词/新概念/新物品/新规则/新势力？
   → 是 → 逐个进入下列检查
   → 否 → 跳过本清单

2. 读者理解度检查（逐个新事物）：
   → 读者能不能在"不跳出故事"的前提下理解这个新事物？
   → 若读者会一脸懵地往下读 → 违规，必须补充释放
   → 若一眼就能懂（常识词）→ 不需要释放，跳过

3. 重要度匹配检查：
   → P0核心概念是否分3-4章渐进释放？（不能一次性倒完，也不能完全不释放）
   → P1是否在1-2章内用对话/场景带出？
   → P2是否一句话点出功能？
   → P3背景碎片是否只散落融入、不主动解释？

4. 方法选择检查：
   → 释放方式是否与剧情自然契合？（不是生硬塞进去的）
   → 同一章多个新事物是否用了不同释放方法？（避免方法单一）
   → 对话释放是否有信息差？（双方都懂的科普对话=违规）

5. 解释腔红线检查：
   → 是否有"在XX，这个词意味着…""所谓XX，就是…"等叙述者定义句？→ 有=违规
   → 释放的信息是"角色/场景漏出"还是"叙述者跳出来讲"？→ 后者=违规

6. 后置揭秘检查（若用了后置揭秘）：
   → 是否先展示了效果/代价，再补原理？
   → 原理是否通过场景/角色带出（而非叙述者直说）？

7. 跨章释放一致性（P0专用）：
   → 本章释放的信息是否与已埋设的伏笔/线索一致？
   → 是否与设定文档中的定义一致？
```

## 八、与流水线的集成

| 步骤 | 释放职责 |
|------|---------|
| **00 设定架构师** | 在设定文档中标注哪些是"读者陌生的自造概念"，标记 P 级别 |
| **02 细纲编写** | 为 P0 核心概念规划"释放节奏"（哪几章、用什么技法、释放哪个层面），写入细纲 |
| **03 扩写** | 执行释放——按细纲规划的方式把新事物漏给读者；即兴出现的新事物当场判定 P 级别并释放 |
| **04 精修** | 校验"读者是否接得住"——通读本章，标记读者可能看不懂的新事物，补充释放 |
| **05 去AI味** | Gate 11检查时，区分"解释腔（删）"和"合法释放（保留）"——不要误删合法释放 |

## 质感锚点库

# 质感锚点库

> **用途**：03扩写时参考"一段好文字长什么样"。
> 这些不是单句示例，而是**完整段落**——展示细节密度、动作链、感官投放的整体效果。
> **核心价值**：当前skill有大量单句级❌/✅对照，但缺少整段级正面锚点。
> AI需要看到"好的段落整体长什么样"，才能在生成时对齐这个标准。

---

## 一、动作链写生理状态

### 锚点1：饥饿（《夜无疆》第1章）

> 垃圾堆里翻出半块饼，霉斑绿得吓人。他小心撕下相对干净的那点，没急着咬，先放鼻子下闻了闻——还好，只是霉味，没馊。这才塞进嘴里慢慢含化，让那点可怜的甜味在舌头上多留一会儿。

**拆解**：
- 动作链：撕→闻→塞→含化（4个连续动作，有因果关系）
- 具体物：霉斑绿得吓人、相对干净的那点、可怜的甜味
- 质感来源：不写"他很饿"，让4个动作+3个具体物自己传达饥饿感

### 锚点2：寒冷（通用范例）

> 她把手缩进袖子里，指甲盖发白，脚趾在鞋里蜷起来。风从领口灌进去，她打了个哆嗦，脖子往衣领里缩了缩，没用——冷气贴着脊背往下淌。

**拆解**：
- 动作链：缩手→蜷脚→打哆嗦→缩脖子（4个连续动作）
- 具体物：指甲盖发白、冷气贴着脊背往下淌
- 质感来源：不写"她很冷"，让身体反应链传达寒冷

### 锚点3：紧张（通用范例）

> 他把杯子端起来又放下，放下又端起来，指尖在杯壁上敲了两下。茶已经凉了，他喝了一口，烫的——什么时候续的热水？他没注意到。

**拆解**：
- 动作链：端起→放下→端起→敲→喝（5个动作，重复体现紧张）
- 信息错位：他以为茶凉了，结果是烫的——紧张到感知失灵
- 质感来源：不写"他很紧张"，让无意义的重复动作传达焦虑

## 二、具体名词替代抽象形容词

### 锚点4：破旧（《夜无疆》风格）

> 门轴上的锈已经爬满了合页，推开的时候吱呀一声，声音在走廊里转了两圈才散。屋里没人，桌上搁着半碗剩饭，筷子横在碗沿上，饭上面结了一层硬皮。

**拆解**：
- 不写"房间很旧很破"，写：锈爬满合页、吱呀声、剩饭硬皮
- 每个具体物都带"时间痕迹"（锈、结硬皮）
- 4个具体物 = 4倍质感

### 锚点5：贫穷（通用范例）

> 他数了三遍口袋里的硬币，码得整整齐齐，连一分钱的边缘都对准了桌缝。十二块三。他盯着那堆硬币看了半天，把其中一块钱的拿起来，又放下。

**拆解**：
- 不写"他很穷"，写：数三遍、码整齐、对准桌缝、拿起来又放下
- 动作细节传达心理：拿起来又放下 = 舍不得花
- 12块3 = 精确数字增加真实感

### 锚点6：危险（通用范例）

> 刀刃上没有锈，映着火光，边缘薄得能看见后面的手指。他把刀翻了个面，另一面也一样——磨过的，而且磨得很用心。

**拆解**：
- 不写"这把刀很锋利很危险"，写：没有锈、映火光、薄得看见手指
- "磨得很用心" = 间接传达这把刀的主人不是善茬
- 具体物自带信息量

## 三、设定藏在生活里

### 锚点7：资源稀缺（《夜无疆》第1章）

> 天光石按户分配，每户每天拳头大的一块。王家婶子总是省着用，天黑前两小时就熄了石，一家人坐在黑暗里说话。她说："黑不可怕，瞎聊天时才听得出真心话。"

**拆解**：
- 分配制度（每户拳头大一块）→ 体现稀缺
- 具体家庭（王家婶子）→ 制度落地
- 生活习惯（提前熄石）→ 体现氛围
- 一句哲理（"黑不可怕"）→ 增加深度
- 四行字，世界活了

### 锚点8：交易体系（《夜无疆》第2章）

> 集市上不用金银，用"光时"。卖菜的赵老汉挂出木牌："一把野菜，换一刻钟天光石照明。"有人讨价还价："半刻钟！你这菜不新鲜。"赵老汉瞪眼："不新鲜？你点个火把照照！"——全场哄笑，永夜世界哪来的火把。

**拆解**：
- 交易体系（用光时）→ 世界观自然释放
- 人物性格（赵老汉瞪眼）→ 角色活起来
- 幽默感（哪来的火把）→ 世界观融入日常
- 设定不是"讲"出来的，是"逛"出来的

### 锚点9：社会等级（通用范例）

> 他走进去的时候，前台让他在一张没有格子的白纸上填名字，不填身份证号。纸是A4的，但比普通A4厚，摸上去有布的质感。他填完，前台看了一眼，没说话，把纸折了两折，放进抽屉——没锁。

**拆解**：
- 不写"这个组织很神秘很有钱"，写：白纸、不填身份证号、纸厚如布、折两折、没锁
- 5个具体细节 = 5个信息点
- "没锁" = 最有杀伤力的一个词（不怕你看，因为你看了也没用）

## 四、感官精准投放

### 锚点10：核心场景感官叠加（通用范例）

> 空调开得很足，屋里凉。白术吐出一口白气，白气在冷蓝色射灯下瞬间散开。她听见自己的心跳——不对，那不是心跳，是楼下车库的卷帘门在响，一下一下，闷闷的，像有人拿锤子敲地板。

**拆解**：
- 温度：屋里凉
- 视觉：白气、冷蓝色射灯
- 听觉：心跳→卷帘门→锤子敲地板（听觉递进）
- 感官错位：以为是心跳，其实是卷帘门——紧张到感官误判
- 3种感官+1个错位 = 核心场景质感密度

### 锚点11：日常过渡精准锚定（通用范例）

> 走廊里的声控灯灭了。他站了一会儿，灯又亮了。

**拆解**：
- 只写1个精准细节：声控灯灭→亮
- 不写"走廊很安静很黑"——灯灭自然传达黑暗，灯亮自然传达有人
- 2句话 = 过渡场景的质感锚定

### 锚点12：战斗动词链（通用范例）

> 他侧身，出拳，收拳，再出拳。拳风擦着对方下巴过去，没打中。对方抬腿扫过来，他往后撤半步，脚跟踩到地上的碎玻璃，滑了一下——就这一下，对方的拳头已经到了面门。

**拆解**：
- 动词链：侧身→出拳→收拳→出拳→撤步→踩到→滑→拳头到面门
- 8个动词，节奏极快
- "踩到碎玻璃滑了一下" = 战斗中的意外细节，增加真实感
- 不写"战斗很激烈"，让动词链自己传达速度感

## 五、对话有人味

### 锚点13：口癖+跑题（通用范例）

> "我说，你到底去不去？"老周把烟掐了，又点了一根，"楼下新开了一家羊肉汤，我昨天经过闻了一下，啧。"他砸吧了一下嘴，"那个味儿正。"
> 林川没接话。
> "你聋了？"
> "我在想。"
> "想什么想，吃碗羊肉汤还要想？你这个人就是——"
> "走吧。"
> 老周愣了一下："你答应了？"
> "我说羊肉汤。"

**拆解**：
- 口癖："我说" "啧" "你这个人就是"
- 跑题：从"去不去"扯到羊肉汤
- 言不由衷：林川说"我在想"但其实已经答应了
- 信息差：老周以为林川答应去办事，其实林川只说去吃羊肉汤
- 人味来源：不是每句话都在推进情节，但每句话都在体现性格

### 锚点14：言不由衷+微动作（《夜无疆》风格）

> "我才不稀罕。"她把手里的东西往桌上一搁，搁完又觉得不对，拿起来塞进兜里，动作快得像是怕被人看见。

**拆解**：
- 言不由衷：嘴说"不稀罕"，手却塞进兜里
- 微动作有情绪含义：塞进兜里+怕人看见 = 其实很在意
- 不写"她嘴上说不稀罕，其实很想要"——让动作和台词的矛盾自己说话

### 锚点15：信息差+装懂（通用范例）

> "你知道那个地方在哪吧？"
> "知道。"
> "那你说，怎么走？"
> "你先往……"他顿了一下，"你先往南走。"
> "南边是墓地。"
> "对，就从墓地旁边绕过去。"
> 老周看了他一眼，没说话，把车钥匙扔给他："你来开。"

**拆解**：
- 信息差：他其实不知道路，但装知道
- 暴露：南边是墓地——显然瞎指的
- 老周的反应：没说话，扔钥匙——用行动替代揭穿
- 人味来源：真人就是会装懂，真人就是会看破不说破

## 六、综合锚点（多技法叠加）

### 锚点16：开篇综合（虚构范例·辰东风格）

> 秦铭最后一次见到太阳，是三年前。那时他十岁。现在他只知道，胃里像有只手在拧，拧得他直不起腰。
>
> 他蹲在墙根底下，后背贴着冰凉的石头。天又黑了——在这地方天一直是黑的，但到了晚上，黑得更彻底，像有人把最后一层纱也扯走了。他把手缩进袖子里，指甲盖发白。
>
> 远处传来狗叫，叫了两声就停了，大概是冻的。
>
> "秦铭。"隔壁的陆婶喊了一声，"过来喝口热的。"
> "不用。"他说。胃又抽了一下，他弯下腰，额头抵在膝盖上。
> "别逞强。"陆婶的门开了，一道暖光漏出来，照在他脚尖上。"你三天没吃东西了吧？"
> 他没回答。不是不想回答，是胃里那只手拧得太紧，他张不开嘴。

**拆解**：
- "三句定生死"开篇：最后一次见太阳+十岁+胃里像有手在拧
- 动作链写饥饿：蹲→贴→缩手→弯腰→抵膝盖→张不开嘴
- 具体名词写环境：冰凉的石头、最后一层纱、暖光照在脚尖上
- 对话有人味：陆婶的关心+秦铭的逞强（言不由衷）
- 感官叠加：触觉（冰凉/暖光）+听觉（狗叫两声停了）+视觉（黑更彻底）
- 不写一个"饿"字，但每个细节都在传达饥饿

**这就是"有质感的文字"和"空的文字"的区别。**
