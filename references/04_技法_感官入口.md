# 04 技法：感官入口与脏感注入（V5.4 — 从04完整版拆分）

> **加载条件**：环境转换场景/战斗场景/恐怖/悬疑场景
> **加载方式**：Read `references/04_技法_感官入口.md`

## 感官入口重定向

> V4.1废除全局优先级，采用二元模式：日常最小白描 / 核心多感官叠加。

### 二元模式规则

| 场景类型 | 感官策略 | 禁止 |
|---------|---------|------|
| 日常/过渡（C/D级） | 最小白描：1个视觉细节即可 | 禁止3+感官叠加（过度渲染） |
| 核心场景（S/A/B级） | 多感官叠加：3-5种感官+1个错位 | 禁止纯视觉开头（AI味标志） |

### 场景开头感官入口检测

**规则**：每个核心场景的开头3句中，至少包含2种感官（视觉+非视觉）。

**违反示例（纯视觉开头 — AI味）**：
```
❌ 阳光照进窗户，照亮了桌上的茶杯。茶杯旁边放着一本书。书翻到了第37页。
   （纯视觉堆砌，零情绪投入）
```

**修正示例（多感官入口）**：
```
✅ 茶已经凉了，杯壁上凝着一层水雾。他伸手去拿——指尖碰到瓷面的那一刻，
   才意识到自己的手也在抖。窗外有鸟叫，但那声音好像隔了一层什么东西。
   （触觉+听觉+视觉+本体感觉）
```

### 感官轮换原则

- 同一章内不连续使用同一种感官入口（避免感官疲劳）
- 推荐轮换顺序：听觉→视觉→触觉→嗅觉→味觉→本体感觉
- 战斗场景优先触觉+听觉（痛感+冲击声）
- 恐怖场景优先嗅觉+听觉（异味+不可见声源）

## 脏感注入

> 在关键场景中加入"不舒服"的生理细节，创造沉浸感。脏感 ≠ 恶心，而是"身体的不适感"。

### 脏感类型库

| 脏感类型 | 适用场景 | 示例 |
|---------|---------|------|
| 皮肤异物感 | 战斗后/紧张时 | 汗水沿着脊背往下淌，像有什么东西在爬 |
| 温度错位 | 恐惧/休克 | 他不觉得冷，但牙齿在打颤 |
| 呼吸阻碍 | 紧张/愤怒 | 吸气时喉咙里像堵了团棉花 |
| 视觉模糊 | 受伤/醉酒 | 灯光变成一团一团的光斑 |
| 听觉失真 | 爆炸后/极度紧张 | 耳朵里嗡嗡响，别人的声音像从水底传来 |
| 肠胃反应 | 恐惧/恶心 | 胃里翻了一下，酸水顶到嗓子眼 |
| 肌肉失控 | 极度疲惫/恐惧 | 手指张不开，像被冻住了 |

### 脏感使用规则

1. **每章1-2处脏感**（不可过多，否则变成恐怖小说）
2. **脏感必须服务情绪**——不是随机加入不适感，而是在角色情绪达到临界点时自然出现
3. **脏感与动作绑定**——不是"他感到恶心"，而是"胃里翻了一下，他咽了口唾液压下去"
4. **高潮章可增至3处**，但必须分布在不同的情绪节点
5. **过渡章不使用脏感**

## 感官锚点配置

> 从00设定中提取1-2个"感官锚点"（核心物件/场景特征），在04精修时做深度穿插。

**配置规则**：
1. 每章配置1-2个感官锚点（从03a细纲的"感官锚点"字段继承）
2. 锚点在章内出现3-5次，每次有变化
3. 锚点变化方式：光线/温度/角色感知/使用状态/位置
4. 最后一次出现必须与角色情绪共振
5. 禁止锚点"自述"——必须通过角色感知呈现
