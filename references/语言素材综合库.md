## 俗语惯用语

# 常用俗语、惯用语与顺口溜精选集（补充版）

> **⚠️ 使用警告（V2.7 清洗）**：很多是农耕时代产物（"庄稼""牲口"相关），与现代/都市/奇幻题材严重脱节。口语感过强，难以融入叙述语言。
> **转化建议**：转化为"角色限定语料"——明确标注适用角色类型（如"老年农村角色""市井商贩""喜剧配角"），并添加使用场景警告。现代/都市/奇幻题材中，禁止主角使用农耕俗语（除非是穿越/农村背景角色）。
> **互补素材库**：需要角色对话风格化 → `人物声口对话风格库.md`（身份语料包+潜台词对照表）。

> **⚠️ 使用警告（V2.7 清洗）**：大量四字成语适合议论文而非小说叙事。生僻成语在对话和心理描写中会"掉书袋"，缺乏具象画面感。

> **使用说明**：本成语库按**小说创作场景**分类，每个成语标注**常用度评分**（★★★★★最常用 → ★较少用），便于快速查找。解释精简，使用场景聚焦小说写作。

## 十一、使用指南

### 1. 如何查找成语

1. **按场景查找**：根据你要写的场景（人物、情感、情节、环境、对话）选择对应章节
2. **按常用度查找**：优先使用★★★★★和★★★★的成语，它们更常用更自然
3. **按主题查找**：使用索引快速定位到具体子类

### 2. 如何使用评分

- **★★★★★**：最常用，几乎任何场景都适用
- **★★★★**：常用，适合大多数场景

- **★**：较少用，慎用

### 3. AI使用建议

1. **分类检索**：AI可以根据分类快速找到合适成语
2. **评分筛选**：优先推荐高评分成语
3. **场景匹配**：根据使用场景选择最贴切的成语
4. **避免重复**：同一场景避免使用过多成语

# 歇后语精选集

> **⚠️ 使用警告（V2.7 清洗）**：歇后语自带"抖包袱"节奏，仅适用于特定喜剧角色或市井对话。滥用会破坏严肃叙事氛围，显得油腻或出戏。
> **转化建议**：转化为"角色限定语料"——明确标注仅适用于"喜剧配角""市井商贩""老年农村角色"的对话。严肃场景/主角台词/叙述语言中禁止使用。可考虑"歇后语化用"——只说前半句不说后半句，让读者自己补。
> **互补素材库**：需要角色对话风格化 → `人物声口对话风格库.md`（身份语料包+潜台词对照表）。

## 十五、"说/道"类（对话标签）【AI第一高频词】

**问题**：AI每章"他说""她说""xxx道"出现50+次，严重僵硬。

### A. 按情绪替换

| 替换方案 | 情绪 | 示例 |
|:---------|:-----|:-----|
| 反问 | 质疑/逼迫 | "'那你觉得呢？'他反问。" |
| 质问 | 愤怒/追责 | "'谁干的！'她厉声质问。" |
| 低声道 | 谨慎/温柔 | "他低声道：'别出声。'" |
| 脱口而出 | 急切/冲动 | "'不是我！'他脱口而出。" |
| 忍不住开口 | 憋不住 | "她终于忍不住开口了。" |
| 轻声说 | 温柔/私密 | "她轻声说：'我相信你。'" |
| 闷声道 | 压抑/委屈 | "他闷声道：'知道了。'" |

### B. 按方式替换

| 替换方案 | 方式 | 示例 |
|:---------|:-----|:-----|
| 插嘴 | 打断 | "'等一下——'他插嘴道。" |
| 接话 | 接续 | "她立刻接话：'我也是这么想的。'" |
| 补了一句 | 补充 | "临走又补了一句：'小心点。'" |
| 话锋一转 | 转折 | "话锋一转：'不过有个条件。'" |
| 把话题拉了回来 | 拉回正题 | "他把话题拉了回来：'说正事。'" |
| 自言自语 | 独白/嘀咕 | "'怎么会这样……'他自言自语。" |
| 嘟囔 | 小声/抱怨 | "她嘟囔了一句，谁也没听清。" |

### C. 零对话标签（最高级）

- 用动作替代："他放下杯子。'你说什么？'"
- 用神态替代："她眼眶红了。'我没事。'"
- 直接裸对白（连续对话不用任何标签）
- 用沉默替代："他没有回答。——沉默就是答案。"

| 脸涨得通红 | 愤怒/羞耻 | "他脸涨得通红，额角青筋直跳" |
| 脸上的笑容僵住了 | 突然被打断 | "脸上的笑容僵住了" |
| 脸色铁青 | 极度愤怒 | "他脸色铁青，指节捏得咔咔响" |
| 脸色一阵青一阵白 | 复杂/打击 | "她脸上青一阵白一阵，说不出话" |
| 面色沉了下来 | 不悦 | "他的面色沉了下来，周遭空气都冷了几度" |
| 表情管理彻底失控 | 现代/崩溃 | "她的表情管理彻底失控了" |
| 面不改色 | 镇定/老练 | "他面不改色，仿佛什么都没发生" |

### D. "心里一紧"类替换

| 替换方案 | 情绪 | 示例 |
|:---------|:-----|:-----|
| 心猛地缩了一下 | 心疼/恐惧 | "心猛地缩了一下——疼。" |
| 胸口发闷 | 压抑 | "胸口一阵发闷，他不得不解开领口" |
| 喉咙像被掐住了 | 恐惧/窒息 | "喉咙像被一只无形的手掐住了" |
| 全身紧绷 | 戒备/紧张 | "他全身紧绷，每一寸肌肉都绷到了极限" |
| 呼吸都停了一瞬 | 震惊/专注 | "呼吸都跟着停了一瞬" |
| 指甲掐进掌心 | 强忍 | "指甲掐进掌心，用疼痛让自己冷静" |
| 脊背绷直 | 紧张/戒备 | "脊背不自觉地绷直了" |

### E. 零情绪标签（最高级）

- ❌ "他心中一惊" → ✅ "那张脸——他认得的。二十年前死掉的人，正站在他面前。"
- ❌ "她心头一沉" → ✅ "她盯着那行字看了很久，没说话。"
- ❌ "他脸色一变" → ✅ "他的笑消失了。"
- ❌ "他心里一紧" → ✅ "手指不自觉地攥紧了桌角，指节发白。"

## 二十三、"不禁/不由得/忍不住"类（情绪触发模板）

**问题**：AI写情绪触发必用——"他不由得退了一步""她忍不住红了眼眶""他不禁愣住了"，一章8-12次。

| 替换方案 | 风格 | 示例 |
|:---------|:-----|:-----|
| 直接删掉，用动作接 | 最干净 | ❌"他忍不住笑了" → ✅"他笑了。" |
| 到底还是 | 强调经过挣扎 | "到底还是开了口" |
| 鬼使神差地 | 被无形力量驱使 | "鬼使神差地，她伸出了手" |
| 不自觉地 | 中性/轻度 | "她不自觉地屏住了呼吸" |
| 下意识地 | 本能反应 | "他下意识地摸了摸口袋" |
| 条件反射般 | 快速本能 | "条件反射般侧身躲开" |
| 鬼使神差 | 说不清原因 | "鬼使神差，他回头看了一眼" |
| 一个没忍住 | 口语/坦诚 | "一个没忍住，笑出了声" |
| 到底没忍住 | 努力克制但失败 | "到底没忍住，眼泪掉了下来" |
| 憋了半天还是 | 克制→失控 | "憋了半天，还是问出了那句蠢话" |
| 莫名其妙地 | 自己也不理解 | "莫名其妙地，他跟着走了出去" |
| 鬼知道为什么 | 自嘲式 | "鬼知道为什么，他就是信了" |

**最高级**：用外部刺激触发动作，替代"忍不住"
- ❌ "他忍不住回头看了一眼" → ✅ "身后有什么响了一下。他回头了。"
- ❌ "她忍不住哭了" → ✅ "眼泪掉下来了。她没去擦。"

## 二十四、"似乎/仿佛/好像/隐隐"类（模糊描写模板）

**问题**：AI对不确定描写全用这几个词——"似乎有什么不对""仿佛在暗示什么""隐隐感到不安"，一章6-10次。

### A. 按语气替换

| 替换方案 | 语气 | 示例 |
|:---------|:-----|:-----|
| 透着几分 | 肯定中带模糊 | "这笑容透着几分古怪" |
| 隐约透着 | 视觉模糊感 | "眼底隐约透着一丝不耐" |
| 说不上来为什么 | 直觉型 | "说不上来为什么，他就是觉得不对" |
| 不知怎的 | 自己也困惑 | "不知怎的，今天的心跳一直很快" |
| 某种 | 确定性+模糊对象 | "空气里弥漫着某种说不清的紧张" |
| 说不上来的 | 难以名状 | "说不上来的违和感" |
| 莫名 | 无法解释 | "莫名的不安""莫名的烦躁" |
| 总感觉 | 口语/直觉 | "总感觉有人在盯着他" |
| 怪怪的 | 口语/日常 | "气氛怪怪的" |
| 不对头 | 口语/直觉 | "这事不对头" |

### B. 零模糊词（最高级）

- ❌ "似乎有什么东西在暗中盯着他" → ✅ "有人在盯着他。他看不到是谁，但他知道。"
- ❌ "仿佛整件事都是一个局" → ✅ "整件事——从头到尾——都是一个局。"
- ❌ "隐隐觉得不安" → ✅ "后颈的汗毛竖起来了。"
- 策略：直接写具体的感觉信号（汗毛/寒毛/鸡皮疙瘩/脊背发凉），不写"似乎/仿佛"

## 二十五、"一股..."量词模板（万能容器）

**问题**：AI用"一股"当万能容器——"一股寒意""一股暖流""一股力量""一股怒火""一股杀意"，一章5-8次。

| 原词 | 替换方案 | 示例 |
|:-----|:---------|:-----|
| 一股寒意 | 凉气从脚底窜上来 | "凉气从脚底一直窜到天灵盖" |
| 一股寒意 | 寒气顺着脊背往上爬 | "寒气顺着脊背往上爬" |
| 一股暖流 | 热意在胸口化开 | "热意在胸口化开" |
| 一股暖流 | 像喝了一口烈酒 | "喉咙像被烈酒烫了一下" |
| 一股怒火 | 火噌地窜上来了 | "火噌地就窜上来了" |
| 一股怒火 | 血一下子涌上头顶 | "血一下子涌上了头顶" |
| 一股杀意 | 杀气如有实质 | "杀气如有实质，刺得皮肤生疼" |
| 一股杀意 | 周身散发出危险的气息 | "周身散发出危险的气息" |
| 一股力量 | 力道从腰腹间炸开 | "力道从腰腹间炸开，通过手臂轰了出去" |
| 一股力量 | 灵力奔涌而出 | "灵力奔涌而出，灌注掌心" |
| 一股香气/臭气 | 直接写嗅觉动词 | "腥臭扑鼻而来""暗香浮动" |
| 一股冲动 | 差点就…… | "差点就一拳砸过去了" |
| 一股失望 | 心凉了半截 | "心凉了半截" |
| 一股勇气 | 不知哪来的胆子 | "不知哪来的胆子，他上前了一步" |

**核心原则**：把抽象"一股X"变成**具体感官信号**——体温变化、肌肉反应、具体动作、环境互动。不是替换词，是换一个写法。

---

> **使用建议**：
> - 每章按"寻找高频词→替换→检查"三步操作，不要一步到位
> - 不要一次性替换所有同类词，保留2-3处原词作为风格差异
> - 替换的最高境界是"直接删掉，什么都不写"，而不是用一个新词代替一个旧词
> - 人物性格决定替换方向：高冷角色保留更多"沉默""冷眼"，活泼角色多用"吐槽类"
> - 第一人称视角中"看着"类替换减少2/3，第一人称本来就是主观视角
> - "说/道"是AI味最重的词，一章出现30+次即可判定为AI生成——优先替换此项
> - "心中一惊/心头一沉/脸色一变/心里一紧"是AI味第二大重灾区，一章合计出现10+次即为AI生成
> - 身体动作类替换（转身/低头/伸手/站起）一章合计出现15+次即需大换——优先用动作+情绪融合写法
> - "不禁/不由得/忍不住""似乎/仿佛/隐隐""一股X"三类合计出现15+次/章即需处理——属于AI万能过渡词/万能量词
> - 移动动词替换注意题材匹配：玄幻多用"掠/闪身"，都市多用"晃/蹭/溜"
