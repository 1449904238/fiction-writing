# 场景触发加载映射表（V5.2 — 唯一索引文件）

> 本文件是 `fiction-writing/references/` 目录下全部参考文件的**唯一索引**。
> 原 `references_index.md` 和 `素材库统一索引.md` 的内容已合并至本文件。

## 四、降级策略

2. 场景触发集降级为"AI 根据 prompt 中内嵌的精简版规则继续工作"
3. 在定位报告中标注 `[reference降级: 文件名]`

## 五、使用建议

1. **00 设定架构师**：加载 `00_命名协议.md` + `00_严谨性校验协议.md`（或对应综合文件部分）
2. **03 扩写前**：核心常驻集已内嵌；场景触发集按场景类型加载
3. **03b 质检**：加载 `03_检测器_综合.md`（一站式检测器规则）
4. **04 精修时**：加载 `04_节奏控制协议.md` + `04_交接协议.md`（或 `04_精修协议综合.md`）
5. **05 去AI味**：加载 `05_精简文笔协议.md` + `改写范例库.md` + `网文通用高频词替换库.md`
6. **开篇/新卷**：加载 `Part_K`（选开篇方法）+ `Part_L`（定文风）+ `Part_M`（查题材公式）
7. **伏笔管理**：02埋设时加载 `02_伏笔全生命周期管理.md`，03/04按表操作
8. **平台审稿**：09审稿时加载 `平台规则配置.md`
9. **素材库调用前**：必须先完成安全鉴权（读取 `novel_meta.core_tags` + `Part_L_文风系统`）
