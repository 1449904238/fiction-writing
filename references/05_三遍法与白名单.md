## 三遍法映射层

> 将11 Gate映射到三遍改写流程，与分级执行叠加。

### Pass 1：去泛化（对应核心Gate 1+2）

**目标**：消除AI最典型的"泛化表达"——禁用词、套路句式、对话腔调。

**执行内容**：
1. 禁用词全量扫描+替换（核心Gate 1）
2. 句式套路检测+修正（核心Gate 2的19项）
3. 对话腔调检测+修正（核心Gate 2的6项）

**Pass 1输出**：标注所有修改位置，生成"Pass 1修改清单"

### Pass 2：去书面化（对应核心Gate 3+4+5）

**目标**：消除"书面化"痕迹——心理直给、结尾升华、剧透式叙述。

**执行内容**：
1. 心理描写残留扫描（核心Gate 3）
2. 结尾去升华（核心Gate 4）
3. 剧透式叙述检测（核心Gate 5）

**Pass 2输出**：标注所有修改位置，生成"Pass 2修改清单"

### Pass 3：回自然感（对应核心Gate 6+可选Gate）

**目标**：让文本"回到自然"——信息含金量、节奏、意象、POV。

**执行内容**：
1. 信息含金量检测+脱水（核心Gate 6）
2. （深度模式）节奏打碎（可选Gate A）
3. （深度模式）意象疲劳检测（可选Gate B）
4. （深度模式）POV一致性（可选Gate C）

**Pass 3输出**：最终稿+AI味评分

### 三遍法与分级执行的叠加关系

| 分级 | Pass 1 | Pass 2 | Pass 3 |
|------|--------|--------|--------|
| 轻度 | ✅ 核心1+2 | ❌ 跳过 | ✅ 核心6 |
| 中度 | ✅ 核心1+2 | ✅ 核心3+4+5 | ✅ 核心6 |
| 重度 | ✅ 核心1+2 | ✅ 核心3+4+5 | ✅ 核心6+可选全部 |

## 白名单豁免机制

### 白名单文件

创建 `.deslop-whitelist` 文件（项目根目录），每行一个豁免规则：

```
# 角色口头禅豁免
角色A:不禁
角色B:缓缓

# 引用/仿写豁免
chapter-5:排比三连

# 用户标注保留
chapter-10:paragraph-3

# 第一人称独白豁免
chapter-1:第一人称段落
```

### 白名单匹配规则

1. 05执行Gate前先加载白名单
2. 检测到的AI味项与白名单匹配 → 跳过（标注"白名单豁免"）
3. 白名单匹配支持：角色名+禁用词、章节号+句式类型、章节号+段落号
4. 白名单不匹配 → 正常执行Gate

### 白名单更新

- 05执行后，用户可将被误判的项加入白名单
- 白名单文件由用户维护，05不自动修改
- 每卷结束建议审查白名单（部分豁免可能已不适用）

## 收敛终止协议

### 收敛规则

1. **最多3轮迭代**：每轮执行完整的6核心Gate
2. **连续2轮无改动**→ 终止（已收敛）
3. **`[需复核]`标记** → 移交用户处理
4. **删除比例超限** → 终止并报告

### 收敛判定标准

| 轮次 | 终止条件 |
|------|---------|
| 第1轮 | 修改项≤3 → 可终止 |
| 第2轮 | 修改项≤2 → 终止 |
| 第3轮 | 无论修改多少 → 强制终止 |

### 未收敛处理

如果3轮后仍有≥5处未处理：
1. 标记为"AI味顽固段落"
2. 在输出中列出具体段落和未处理原因
3. 建议用户：
   - 手动改写顽固段落
   - 或回退03a重新扩写该段落
   - 或将顽固段落加入白名单（如果是有意为之）

## 模型退化检测

### 退化信号

1. **修改幅度递增**：第2轮修改比第1轮更多 → 模型可能在"为改而改"
2. **修改质量递减**：第2轮的修改明显不如第1轮 → 模型注意力下降
3. **误判率上升**：白名单豁免项被重复标记 → 模型未读取白名单
4. **循环修改**：第2轮把第1轮的修改改回去 → 模型陷入循环

### 退化处理

1. 检测到退化信号 → 立即终止当前轮次
2. 保留最佳轮次的结果（通常是第1轮）
3. 在输出中标注"检测到模型退化，建议人工复核"
4. 建议用户切换模型或拆分为多次执行

## 修复效果验证闭环

### 验证流程

1. 05执行前：运行 `node scripts/check-ai-patterns.js` 获取AI味分数
2. 05执行后：再次运行获取新分数
3. 分数对比：
   - 分数下降≥20% → ✅ 有效
   - 分数下降10-20% → ⚠️ 部分有效
   - 分数下降<10% → ❌ 效果不佳，建议换模型或手动处理
4. 输出"AI味分数对比报告"
