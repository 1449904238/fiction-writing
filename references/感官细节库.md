# 感官细节库

> **定位**：解决"干瘪叙述"和"告诉而非展示"的核心痛点。不是词汇表，是按**场景→感官→具象表达**三层组织的实战工具。
> **使用原则**：不是选词填空，是按"角色此刻的身体会感知到什么"来筛选。同一场景不同角色感知到的感官细节不同。
> **与质感注入协议的协作**：质感注入协议管"什么时候该写感官"，本库管"用什么具体的感官词汇写"。

---

## 一、五感细分词表

> **核心理念**：不写"香/臭/疼/冷/吵"，写"霉味/铁锈味/灼痛/刺骨/嗡嗡声"。
> 每个词附带"适用场景"和"误用警告"，确保即取即用不出戏。

### 1.1 嗅觉（最强烈的记忆触发器）

| 气味 | 适用场景 | 误用警告 |
|------|---------|---------|
| 霉味 | 老宅/地下室/旧书/久未通风的房间 | 不宜用于高档场所 |
| 铁锈味 | 血腥/旧机械/废弃工厂/雨水打在铁上 | 不是所有"金属味"都叫铁锈 |
| 焦糊味 | 火灾/爆炸/电路短路/烤焦的食物 | 注意区分"焦香"（食物）和"焦臭"（灾祸） |
| 奶腥味 | 婴儿/哺乳期/奶制品腐败 | 仅限特定场景，滥用出戏 |
| 消毒水味 | 医院/实验室/刚打扫过的房间 | 暗示"不自然/被控制"的氛围 |
| 潮湿的泥土味 | 雨后/墓地/地下室/森林 | 可暗示"新生"或"腐朽"，取决于上下文 |
| 汗酸味 | 人群/紧张/战斗后/贫穷 | 不宜用于浪漫场景 |
| 檀香味 | 寺庙/老宅/修行者/祭祀 | 自动带出"宗教/古老"氛围 |
| 烟草味 | 老男人/酒馆/交谈后/失眠夜 | 注意区分"新鲜烟味"和"隔夜烟臭" |
| 廉价香水味 | 风尘女子/底层/强行掩饰 | 含阶层判断，使用需谨慎 |
| 煤油味 | 旧时代/贫穷/矿工/战争 | 强时代感，现代都市慎用 |
| 雨后空气味 | 转折/希望/新生/情绪释放 | 万能但不要滥用（每卷≤1次） |
| 血腥味 | 暴力/死亡/受伤/恐怖 | 强刺激，每章最多1次 |
| 药味 | 病弱/慢性病/老年/调理 | 可暗示"脆弱"或"被照顾" |
| 海腥味 | 码头/渔民/海边/远航 | 自动带出地理信息 |

### 1.2 触觉（最容易被忽略的感官）

| 触感 | 适用场景 | 误用警告 |
|------|---------|---------|
| 刺痛 | 伤口/寒冷/电流/尖锐物 | 比"疼"更精准，指明疼痛的"质地" |
| 钝痛 | 内伤/撞击/暗伤/内脏 | 区别于刺痛——"里面在疼"而非"表面在疼" |
| 灼痛 | 烧伤/烫伤/腐蚀/剧毒 | 比"火辣辣"更精准 |
| 胀痛 | 肿胀/充血/炎症/情绪压抑 | 暗示"里面有东西在膨胀" |
| 抽痛 | 神经性/间歇性/痉挛 | 暗示"一波一波地疼"，不是持续的 |
| 发麻 | 久坐/压迫/恐惧/寒冷 | 可暗示"失去控制"或"即将失去知觉" |
| 发痒 | 伤口愈合/虫爬/过敏/撒谎 | 伤口愈合时的痒=希望，虫爬的痒=恐惧 |
| 发紧 | 喉咙/胸口/皮肤/绷带 | 暗示"被束缚/被压迫" |
| 发烫 | 发烧/羞耻/愤怒/靠近热源 | 从身体内部发热≠外部热源 |
| 冰凉 | 恐惧/失血/死亡/金属/水 | 区分"外部凉"（摸到凉的）和"内部凉"（身体变凉） |
| 黏腻 | 汗水/血液/潮湿/腐烂 | 暗示"不舒服/不干净" |
| 粗糙 | 老茧/树皮/旧墙/砂纸/麻布 | 可用以暗示角色的"劳动/贫穷/古老" |
| 光滑 | 玻璃/瓷器/丝绸/冰面/石头 | 暗示"精致/冷/危险"（取决于上下文） |
| 柔软 | 毛发/棉布/皮肤/泥土/云 | 过度使用=AI味 |

### 1.3 听觉（节奏与氛围的引擎）

| 声音 | 适用场景 | 误用警告 |
|------|---------|---------|
| 嗡嗡声 | 电器/人群/耳鸣/昆虫 | 低频持续音，暗示"烦扰/不安" |
| 咔嗒声 | 门锁/开关/枪械/高跟鞋 | 短促清脆，暗示"动作/转折" |
| 沙沙声 | 树叶/纸张/脚步声/蛇 | 干摩擦音，暗示"隐秘/接近" |
| 滴答声 | 水/时钟/血/雨 | 重复节奏，暗示"时间流逝/等待" |
| 轰隆声 | 雷/爆炸/坍塌/重型机械 | 低频巨响，暗示"不可抗力/灾难" |
| 尖啸声 | 警报/刹车/风/惨叫 | 高频刺耳，暗示"危险/紧急" |
| 碎裂声 | 玻璃/陶瓷/冰/骨头 | 暗示"不可逆的破坏" |
| 闷响 | 撞击肉体/消音器/关门 | 低沉短促，暗示"被压制的暴力" |
| 回声 | 空旷/孤独/记忆/幻听 | 暗示"空/过去/不真实" |
| 静默 | 震惊/死亡/对峙/专注 | 不是"没有声音"，而是"声音的缺席"——比噪音更响 |

### 1.4 视觉（最丰富但最容易写成"上帝视角"）

> **核心原则**：视觉描写不罗列"有什么"，而是写"角色注意到了什么"。

| 视觉要素 | 进阶写法 | 示例 |
|---------|---------|------|
| 光线 | 不写"灯光很亮"，写"灯光刺得他眯起眼睛" | 光线通过角色的反应来呈现 |
| 颜色 | 不写"红色"，写"干涸的血迹是褐色的，不是红的" | 颜色通过"时间/状态/情绪"的滤镜来呈现 |
| 形状 | 不写"方形"，写"墙角那只箱子，棱角已经被撞圆了" | 形状通过"磨损/使用痕迹"来呈现 |
| 距离 | 不写"他离得很远"，写"他的脸只剩一个模糊的轮廓" | 距离通过"视觉清晰度"来呈现 |
| 运动 | 不写"他跑得很快"，写"他的衣角在身后拉成一条直线" | 运动通过"视觉残留"来呈现 |
| 细节 | 不写"房间很旧"，写"墙角的墙皮卷起来，像一张干透的嘴唇" | 细节通过"一个具体的物"来呈现 |

### 1.5 味觉（小说中最少用但最有冲击力）

| 味觉 | 适用场景 | 误用警告 |
|------|---------|---------|
| 铁锈味（血） | 口腔出血/被打/咬破嘴唇 | 血腥场景专用 |
| 苦味 | 毒药/中药/失败/嫉妒 | 可双关——物理苦和精神苦 |
| 酸味 | 腐败/呕吐/胃酸/醋 | 暗示"不适/变质/嫉妒" |
| 甜腥味 | 血+唾液/生肉 | 比纯"铁锈味"更令人不适 |
| 麻木（无味） | 震惊/脑损伤/中毒 | 失去味觉=失去感知能力 |

## 二、通感修辞集

> **核心理念**：跨感官比喻——用听觉描述触觉，用触觉描述视觉。打破感官壁垒，制造"陌生化"效果。
> **使用限制**：每章≤2处通感修辞，滥用=用力过猛。

### 2.1 听觉→触觉

| 通感 | 示例 | 效果 |
|------|------|------|
| 声音像触觉 | "她的声音黏腻得像融化的糖" | 声音有了"触感"——让人想擦掉 |
| 声音像温度 | "他的话冷得像冰碴子" | 声音有了"温度"——加强了"冷"的感觉 |
| 声音像重量 | "每个字都沉甸甸地砸在地上" | 声音有了"重量"——加强了"沉重"的感觉 |
| 声音像质地 | "他的笑声粗糙得像砂纸" | 声音有了"质地"——让人不舒服 |
| 声音像形状 | "尖锐的刹车声把空气撕开一道口子" | 声音有了"形状"——视觉化了无形的声波 |

### 2.2 视觉→触觉

| 通感 | 示例 | 效果 |
|------|------|------|
| 光线像触觉 | "光线尖锐地扎进眼睛" | 光有了"攻击性"——不适感 |
| 颜色像温度 | "墙上的红色冷得像凝固的血" | 颜色有了"温度"——打破"红色=热"的直觉 |
| 画面像重量 | "那片黑暗沉沉地压过来" | 视觉有了"重量"——黑暗不再是"空"的 |
| 视线像触觉 | "她的目光像针一样扎在他后背上" | 视线有了"物理存在感" |

### 2.3 嗅觉→视觉/触觉

| 通感 | 示例 | 效果 |
|------|------|------|
| 气味像颜色 | "那股霉味是灰绿色的" | 气味有了"颜色"——联想更丰富 |
| 气味像重量 | "血腥味沉沉地压下来" | 气味有了"重量"——压迫感 |
| 气味像触觉 | "消毒水的味道刺得他鼻梁发酸" | 嗅觉有了"触觉"——生理反应 |

## 三、身体反应映射表

> **核心理念**：情绪不说出来，让身体"说"出来。
> **核心原则**：紧张≠心跳加速（AI味），每一个情绪至少有3种身体反应可选，每种标注适用角色和场景。

### 3.1 恐惧/紧张（最常写但最容易套路化）

| 身体反应 | 适用角色 | 场景 | 击败AI味的方法 |
|---------|---------|------|--------------|
| 后颈汗毛竖起 | 所有角色 | 感觉到危险/被注视/超自然 | 比"心跳加速"更原始，更真实 |
| 吞咽时喉咙发紧 | 所有角色 | 需要说话但不敢/即将行动前 | 比"口干舌燥"更具体 |
| 指尖失去血色 | 所有角色 | 极度恐惧/寒冷 | 视觉化的恐惧——别人能"看到"的恐惧 |
| 胃里像有东西在翻 | 所有角色 | 恶心/恐惧/厌恶 | 比"胃里翻江倒海"更克制 |
| 膝盖发软 | 所有角色 | 惊吓/极度恐惧 | 注意：不能用"他腿软了"（AI味），要写"他不得不扶住墙" |
| 笑容僵在脸上 | 社交型角色 | 社交场合中的恐惧/尴尬 | 面部表情+恐惧的冲突 |
| 呼吸变浅 | 硬汉/军人 | 努力控制恐惧时 | 注意：不能写"他呼吸急促"，要写"他发现自己只吸了半口气，一直没呼出去" |
| 眼睛发直/瞳孔放大 | 所有角色 | 极度恐惧/震惊 | 注意：不能用"瞳孔放大"（医疗报告腔），要写"她的眼睛突然变得很黑" |

### 3.2 愤怒（最容易被写成"拍桌子"）

| 身体反应 | 适用角色 | 场景 | 击败AI味的方法 |
|---------|---------|------|--------------|
| 下颌发紧/牙关咬紧 | 所有角色 | 压抑愤怒 | 比"握紧拳头"更高级——愤怒在脸上 |
| 太阳穴突突跳 | 所有角色 | 极度愤怒 | 比"青筋暴起"更克制 |
| 声音变低/变稳 | 控制型角色 | 愤怒到极点反而平静 | 比"大吼"更有压迫感 |
| 手在发抖但动作很稳 | 军人/工匠/专业型 | 愤怒但不失控 | 冲突感——身体在愤怒，但训练在控制 |
| 视野变窄 | 战斗型角色 | 极度愤怒/战斗 | 注意：不写"他红了眼"，写"他只能看见那个人，周围的一切都模糊了" |
| 把东西放回原处时发出声响 | 压抑型角色 | 无法发作的愤怒 | 比"摔东西"更高级——愤怒在"控制"中泄露 |
| 笑出声来 | 疯子/反派/极度愤怒 | 愤怒到笑 | 高级写法——愤怒的反面不是平静，是笑 |
| 沉默着做一件重复的事 | 内敛型角色 | 愤怒无法发泄 | 比"默默握拳"更具体 |

### 3.3 悲伤（最容易被写成"眼泪"）

| 身体反应 | 适用角色 | 场景 | 击败AI味的方法 |
|---------|---------|------|--------------|
| 胸口发闷/像被什么压着 | 所有角色 | 压抑的悲伤 | 比"心里很难受"更具体 |
| 眼泪在眼眶里但没掉下来 | 所有角色 | 强忍悲伤 | 比"他哭了"更克制，但比"眼眶红了"更新鲜 |
| 突然觉得冷 | 所有角色 | 失去/噩耗 | 悲伤的"物理化"——悲伤让身体变冷 |
| 一直盯着某个东西但没在看 | 所有角色 | 麻木/发呆 | 注意：不能写"他发呆"，写"他盯着杯子看了很久，但杯子里的水一直没少" |
| 反复做同一个动作 | 内敛型角色 | 无法接受的悲伤 | 如"反复擦同一块玻璃""反复叠同一件衣服" |
| 没哭但声音变了 | 所有角色 | 强忍悲伤时说话 | 不写"他的声音哽咽"，写"他的声音听起来不像他自己的" |
| 突然不知道该把手放哪 | 所有角色 | 手足无措的悲伤 | 比"他不知所措"更具体 |

### 3.4 喜悦/兴奋（最容易被写成"笑了"）

| 身体反应 | 适用角色 | 场景 | 击败AI味的方法 |
|---------|---------|------|--------------|
| 脚尖在点地 | 年轻角色/活泼型 | 按捺不住的高兴 | 身体"泄露"了情绪 |
| 说话变快/变多 | 所有角色 | 兴奋/激动 | 不在内容而在节奏 |
| 反复看同一个东西 | 所有角色 | 收到好消息/礼物 | 如"把通知书翻来覆去看了三遍" |
| 走路时踩到了自己的脚 | 活跃型角色 | 极度兴奋 | 兴奋到"失控"——身体比大脑快 |
| 想笑但强行憋住 | 所有角色 | 不该高兴的场合 | 冲突感——高兴但不敢表现出来 |
| 身体突然变轻 | 所有角色 | 释放/解脱 | 不写"如释重负"，写"他站起来，感觉自己比平时轻了十斤" |

### 3.5 厌恶/恶心（最容易被写成"眉头一皱"）

| 身体反应 | 适用角色 | 场景 | 击败AI味的方法 |
|---------|---------|------|--------------|
| 舌根发酸 | 所有角色 | 恶心/想吐 | 比"胃里翻涌"更精准——恶心的"前兆" |
| 不想让那个东西碰到自己 | 所有角色 | 厌恶到拒绝接触 | 不写"厌恶"，写"他用两根手指捏起那个东西，胳膊伸得远远的" |
| 屏住呼吸 | 所有角色 | 臭味/厌恶 | 不写"他很厌恶那个味道"，写"他屏住呼吸，但那股味道已经钻进了鼻子" |
| 后退半步 | 所有角色 | 被恶心到/被吓到 | 身体比意识更快 |

## 四、使用指南

### 4.1 调用时机

- **03扩写**：写场景时，打开"五感细分词表"，确保每个场景至少覆盖2种感官（不只是视觉）
- **04精修**：诊断"只写了视觉"的场景，按"身体反应映射表"注入缺失的感官
- **05去AI味**：检查"心跳加速/手心出汗/眉头一皱"等AI套路化身体反应，替换为本库的差异化表达

### 4.2 感官分配原则

| 场景类型 | 主导感官 | 辅助感官 | 限制 |
|---------|---------|---------|------|
| 战斗/动作 | 触觉+听觉 | 视觉 | 嗅觉≤1处 |
| 对话/对峙 | 听觉+视觉 | 触觉 | 味觉禁用 |
| 抒情/回忆 | 嗅觉+视觉 | 触觉 | 听觉≤1处 |
| 恐怖/悬疑 | 听觉+触觉 | 嗅觉 | 视觉精简（看不见才恐怖） |
| 日常/过渡 | 视觉+听觉 | 嗅觉 | 触觉≤1处 |

### 4.3 与AI味的边界

| 这是AI味 ❌ | 这是人味 ✅ |
|------------|-----------|
| "他感到一阵恐惧" | "后颈的汗毛一根根竖起来" |
| "他心跳加速" | "他听见自己的心跳在耳朵里嗡嗡响" |
| "他手心出汗" | "他在裤子上蹭了蹭手，但蹭不干" |
| "他胃里翻江倒海" | "舌根突然发酸，他不得不咽了两次口水" |
| "他眉头一皱" | "他没有皱眉头，但眼睛眯了一下" |

## 五、速查卡

```
【感官细节速查卡 — 写场景时贴在屏幕上】

写任何场景前，先问3个问题：
① 这个场景的主导感官是什么？（战斗=触觉，对话=听觉，回忆=嗅觉）
② 除了视觉，至少还要写哪种感官？（每场景最少2种感官）
③ 角色的身体反应是"套路"还是"这个角色独有"的？
   （套路=心跳加速/手心出汗/眉头一皱 → 换成本库的差异化表达）

写恐惧/紧张时：
  → 不要写"心跳加速"（AI味），写"后颈汗毛竖起"或"吞咽时喉咙发紧"
写愤怒时：
```
