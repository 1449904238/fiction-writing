## N.0 题材判断（V5.2.0新增）

读取 `novel_meta.genre` 字段，按题材启用不同的开篇抓手和爽点要求：

| 题材 | 开篇抓手 | 爽点要求 |
|------|---------|---------|
| 玄幻/都市/系统 | 爽点（前500字必须有） | 每3章≥1个小爽点，每10章≥1个大爽点 |
| 悬疑/推理 | 悬念（前500字必须有1个未解之谜） | 信息揭示即爽点，不强制传统爽点 |
| 言情/情感 | 情感碰撞（前500字必须有1次关系张力） | 关系推进/反转即爽点 |
| 严肃文学 | 氛围锚点（前500字必须有1个沉浸感细节） | 不强制爽点，以情绪到达为标准 |
| 历史/军事 | 利益冲突（前500字必须有1次博弈） | 权谋胜利即爽点 |

> **题材混合时**：双题材混合取两题材要求的并集（更严格者优先）；主题材权重≥70%时以主题材要求为主。

## N.1 为什么这是番茄的核心？

番茄的编辑通常在**前3章（甚至前1章）**就决定生死。

**开篇没有爽点 = 必死**（注：此条针对玄幻/都市/系统题材；悬疑/言情等题材以各自开篇抓手为准）

爽点类型（以玄幻/都市/系统为主）：
- 打脸（主角被看不起 → 反转 → 对方傻眼）
- 金手指觉醒（获得外挂/系统/传承）
- 绝境翻盘（必死之局 → 意外破局）
- 复仇成功（仇人出现 → 主角出手 → 爽）
- 装逼（主角展现隐藏实力 → 众人震惊）

## N.2 爽感保护协议

```
[爽感保护协议 - N模块]

第1章强制检查：
├─ 是否有爽点事件？（玄幻/都市/系统题材）
│   ├─ 是 → 通过检查
│   └─ 否 → ⚠️ 警告："第1章无爽点，番茄签约风险极高！"
│       └─ 建议：
│           ├─ A. 将后文章节的爽点挪到第1章
│           ├─ B. 在第1章结尾加入"爽点预告"（主角获得某种暗示）
│           └─ C. 重新设计开篇，直接从爽点场景切入
│
├─ 爽点是否在前500字内出现？
│   ├─ 是 → ✅ 完美开篇
│   ├─ 否，但在前1000字内 → ⚠️ 可接受，但建议前移
│   └─ 否，超过1000字 → ❌ 警告："铺垫太长，读者已流失"
│
└─ 爽点是否有"情绪着陆"？
    ├─ 是 → ✅ 爽感完整
    └─ 否 → ⚠️ 建议："打完脸后，让主角有个'没意思'的反应，显得更酷"

前三章检查（黄金三章）：
├─ 第1章：必须有爽点（打脸/金手指/翻盘）— 题材适配后以各自开篇抓手为准
├─ 第2章：必须有新钩子（让读者想知道接下来会发生什么）
├─ 第3章：必须有更大的期待感（升级的爽感预告）
│
└─ 任一章节不满足 → ⚠️ 警告："黄金三章未达标，签约风险高"
```

## N.3 爽感熔断器

```
[爽感熔断器]

目的：防止爽点写太长，变成水字数

规则：
├─ 单个爽点描写不超过800字
│   └─ 超过 → ⚠️ 警告："爽点过长，建议精简"
│
├─ 爽点必须有"情绪着陆点"
│   └─ 示例：打完脸后，主角觉得没意思，然后走人
│   └─ 这样显得主角很酷，不care
│
├─ 爽点后必须有"新钩子"
│   └─ 示例：打脸后，主角发现更大的麻烦来了
│   └─ 让读者期待下一章
│
└─ 爽点不能连续出现
    └─ 连续爽点会麻木
    └─ 建议：爽点 → 小挫折 → 更大的爽点
```

### 爽点类型库
- **打脸**：被看不起→反转→对方傻眼。主角全程不解释，用行动说话
- **金手指觉醒**：绝境/意外→获得外挂→实力暴涨。觉醒过程要快，觉醒后立刻展示效果
- **绝境翻盘**：必死之局→意外破局→反杀。绝境要真绝，翻盘要合理（有伏笔）
- **复仇成功**：仇人出现→主角出手→爽。仇人要先嚣张，主角出手要狠，反应要淡
- **装逼**：主角展现隐藏实力→众人震惊。先让众人低估，主角不经意展露，反应要夸张

## N.5 爽感密度调节器

```
[爽感密度调节器]

章节类型与爽感密度：
├─ 第1章：超高密度（必须有爽点）
├─ 第2-3章：高密度（必须有钩子）
├─ 常规章节：中密度（每章至少一个小爽点）
├─ 过渡章节：低密度（可以平淡，但要有期待感）
└─ 高潮章节：超高密度（连续爽点，但要控制节奏）

爽感间隔规则：
├─ 连续爽点不超过3个
│   └─ 超过 → 爽感麻木
│   └─ 建议：爽点 → 小挫折 → 爽点
│
├─ 无爽点章节不超过2章
│   └─ 超过 → ⚠️ 警告："连续无爽点，读者可能流失"
│
└─ 大爽点间隔不超过5章
    └─ 超过 → ⚠️ 警告："爽感密度过低"

注：悬疑/言情/严肃文学等题材，以N.0题材判断中的爽点要求为准，
不强制传统爽感密度规则。
```
