# Part K: 开篇方法与决策树

> 融合自 oh-story-claudecode 项目。不同类型的小说需要不同的开篇策略。

## K.1 开篇5种方法

```
方法1：危机切入（推荐指数：⭐⭐⭐⭐⭐）
├─ 适用：爽文、悬疑、都市、玄幻
├─ 做法：直接从主角身处危机中开始
├─ 优势：最快抓住读者，前100字制造紧张
├─ 示例：主角被追杀/被困绝境/面临死亡威胁
├─ 注意：危机不能太复杂，让读者一眼看懂"主角有危险"
└─ 番茄适配：最推荐的开篇方式，符合"前500字必有爽点"原则

方法2：日常异常（推荐指数：⭐⭐⭐⭐）
├─ 适用：悬疑、灵异、都市、科幻
├─ 做法：从日常场景开始，但有一个不合理的细节
├─ 优势：循序渐进，让读者在熟悉中感受异常
├─ 示例：正常上班→发现工位上多了一件不属于自己的东西
├─ 注意：异常必须在500字内出现，否则就是"平淡日常"
└─ 番茄适配：适合氛围型作品，但需要较快揭示异常

方法3：金手指觉醒（推荐指数：⭐⭐⭐⭐）
├─ 适用：升级流、系统流、修仙、玄幻
├─ 做法：开场直接展示主角获得金手指/外挂
├─ 优势：直接满足读者期待，快速建立期待感
├─ 示例：主角获得系统/传承/特殊能力/重生
├─ 注意：觉醒过程要快（≤500字），觉醒后立刻展示效果
└─ 番茄适配：系统流/升级流首选，配合"爽点验证"效果最佳

方法4：对话开场（推荐指数：⭐⭐⭐）
├─ 适用：言情、都市情感、轻小说
├─ 做法：从一段抓人的对话开始
├─ 优势：快速建立角色关系，制造信息差
├─ 示例：一句让人摸不着头脑但很好奇的话
├─ 注意：对话必须蕴含冲突或悬念，不能是日常寒暄
└─ 番茄适配：言情类可考虑，但要注意对话后尽快进入剧情

方法5：结果前置（推荐指数：⭐⭐⭐⭐）
├─ 适用：悬疑、推理、复仇、都市
├─ 做法：先展示一个震撼的结果，再倒叙解释过程
├─ 优势：制造强烈的好奇心，"他怎么会变成这样？"
├─ 示例：主角站在仇人坟前→倒叙他是怎么走到这一步的
├─ 注意：结果必须足够震撼，让读者极度想知道"为什么"
└─ 番茄适配：适合复仇流/悬疑流，开局就制造最大悬念
```

## K.2 开篇决策树

```
问：你的小说主打什么？
├─ 主打"爽感"（打脸/碾压/升级）
│   └─ → 方法1（危机切入）或 方法3（金手指觉醒）
│       问：主角开局是强者还是弱者？
│       ├─ 强者 → 方法1：危机中展示实力，让人震惊
│       └─ 弱者 → 方法3：金手指觉醒，让人期待变强
│
├─ 主打"悬念"（悬疑/推理/灵异）
│   └─ → 方法2（日常异常）或 方法5（结果前置）
│       问：悬念是"过程之谜"还是"结果之谜"？
│       ├─ 过程之谜（怎么发生的）→ 方法5：先展示结果
│       └─ 结果之谜（发生了什么）→ 方法2：从异常开始调查
│
├─ 主打"情感"（言情/都市情感/纯爱）
│   └─ → 方法4（对话开场）或 方法2（日常异常）
│       问：核心冲突是"关系"还是"事件"？
│       ├─ 关系冲突 → 方法4：从对话/互动切入
│       └─ 事件冲突 → 方法2：从改变日常的事件切入
│
└─ 主打"新奇"（科幻/无限流/架空）
    └─ → 方法2（日常异常）或 方法5（结果前置）
        问：新奇感来自"世界观"还是"情节"？
        ├─ 世界观新奇 → 方法2：从日常中展示异常规则
        └─ 情节新奇 → 方法5：从震撼结果入手
```

## K.3 开篇通用禁忌

```
1. 不写大段环境描写
2. 不写角色起床/洗漱/吃饭
3. 不写大段回忆/背景介绍
4. 不写没有冲突的对话
5. 不写读者"看不懂"的设定
```
