## F1.1 为什么番茄主角要"流氓"？（仅适用于玄幻/都市/系统题材）

番茄读者喜欢：
- 嘴贱（会吐槽、会阴阳）
- 护短（对自己人好，对外人狠）
- 有点坏（不是圣母，会算计）
- 不纠结（想干就干，不内耗）

番茄读者讨厌：
- 太端着（装高尚）
- 太讲礼貌（唯唯诺诺）
- 太圣母（以德报怨）
- 太纠结（想半天不行动）

## F1.2 流氓逻辑约束

```
[流氓逻辑约束 - F1.2]

主角人设检查（仅玄幻/都市/系统题材启用）：
├─ 主角是否太"端着"？
│   └─ 是 → ⚠️ 建议："让主角刻薄一点，让他吐槽路人"
│
├─ 主角是否太讲礼貌？
│   └─ 是 → ⚠️ 建议："让主角少说'谢谢'，多说'算你识相'"
│
├─ 主角是否太圣母？
│   └─ 是 → ⚠️ 建议："让主角记仇，有仇必报"
│
├─ 主角是否太纠结？
│   └─ 是 → ⚠️ 建议："让主角想干就干，干了再说"
│
└─ 主角是否有"贱气"？
    └─ 否 → ⚠️ 建议："让主角偶尔犯贱，显得有人味"
```

### 流氓话术速查
| 场景 | ❌ | ✅ |
|------|----|----|
| 被看不起 | "我会证明自己的" | "哦。" / "说完了吗？" |
| 打脸后 | "这只是开始" | "就这？" / "没意思。" |
| 被感谢 | "这是我应该做的" | "别误会，我只是顺手。" |
| 面对求饶 | "我给你一次机会" | "你觉得我会信？" / "晚了。" |
| 被质疑 | "时间会证明一切" | "信不信随你。" / "关我屁事。" |
| 面对嘲讽 | 沉默忍辱负重 | 直接怼回去 / 阴阳怪气 / 无视 |

## F1.4 流氓行为库

```
[流氓行为库]

行为1：记仇
├─ 主角被得罪 → 必须报复
├─ 可以小报复（让对方吃瘪）
├─ 也可以大报复（让对方付出代价）
└─ 但不能不报复

行为2：护短
├─ 自己人被欺负 → 主角必须出头
├─ 出头要狠，让对方记住
└─ 但对自己人可以凶，对外人更凶

行为3：算计
├─ 主角不是傻子，会算计利益
├─ 可以占便宜，但要有底线
└─ 底线：不害无辜，不背信弃义

行为4：果断
├─ 想干就干，不犹豫
├─ 干了再说，不纠结
└─ 错了就认，不找借口

行为5：偶尔犯贱
├─ 主角可以有点贱，显得有人味
├─ 比如故意气人、恶作剧、嘴欠
└─ 但要有度，不能真讨厌
```

## F1.5 人设讨喜度检测

```
[人设讨喜度检测 - 每章检查]

检测项：
├─ 主角是否有吐槽/阴阳/嘴欠的时刻？
│   └─ 无 → ⚠️ 建议："加入主角的吐槽，增加人味"
│
├─ 主角是否有果断行动的时刻？
│   └─ 无 → ⚠️ 建议："主角想太多，让他直接干"
│
├─ 主角是否有护短的时刻？
│   └─ 无 → ⚠️ 建议："加入护短情节，增加好感"
│
├─ 主角是否有记仇/报复的时刻？
│   └─ 无 → ⚠️ 建议："主角太圣母，让他记仇"
│
└─ 主角是否有"装完逼就跑"的时刻？
    └─ 无 → ⚠️ 建议："加入装逼情节，增加爽感"

讨喜度评分：
├─ 高：有吐槽、有果断、有护短、有记仇
├─ 中：有部分，但不明显
└─ 低：太正经、太圣母、太纠结
```
