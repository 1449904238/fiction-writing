> **加载方式**：Read `references/04_技法_潜台词.md`

## 潜台词对话法

> 角色说的 ≠ 角色想的。读者读到的信息量 > 角色说出的字面信息量。

### 潜台词三层结构

```
层1（表面层）：角色实际说的话 — 推动剧情
层2（意图层）：角色想达到的目的 — 制造张力
层3（情感层）：角色真实的感受 — 创造共鸣
```

**示例**：
```
❌ 无潜台词：
  "我很担心你。"她说。（表面=意图=情感，零信息差）

✅ 三层潜台词：
  "你吃饭了吗？"她说。（表面：问候 / 意图：想留下来说话 / 情感：担心但不敢直说）
```

### 潜台词生成策略

| 策略 | 角色心里想A → 嘴上 | 示例 |
|------|-------------------|------|
| 正话反说 | 说A的反面 | 想说"我怕失去你" → "你走啊，谁拦你了" |
| 转移话题 | 说无关的B | 想说"我爱你" → "今天天气不错" |
| 用提问代替陈述 | 问关于A的问题 | 想说"你还好吗" → "你昨晚睡了几小时" |
| 用动作代替语言 | 做与A相关的动作 | 想说"别走" → 把对方的鞋藏起来 |
| 沉默 | 什么都不说 | 想说"我原谅你" → 沉默，但递过去一杯水 |
| 过度补偿 | 说A的夸张反面 | 想说"我很难过" → 笑着说"好事啊，恭喜" |

### 潜台词检测规则

1. 扫描所有对话，标记"角色直接说出了心里想的内容"的句子
2. 标记后检查：是否有前置铺垫允许直说？（情绪崩溃/醉酒/极端信任）
3. 无铺垫的直说 → 替换为潜台词策略
4. 有铺垫的直说 → 保留，但确保生理反应铺垫到位

### 潜台词例外清单（可以直说的场景）

| 场景 | 允许直说的条件 |
|------|--------------|
| 情绪崩溃 | 需有3章以上压抑铺垫 |
| 醉酒 | 需有醉酒的生理描写 |
| 生死关头 | 需有时间紧迫感铺垫 |
| 极端信任 | 两人关系已到"无需伪装"阶段 |
| 误导/谎言 | 角色故意直说假话（此时"直说"本身就是潜台词） |

## 内心矛盾外化法

> 把"纠结"变成可见的行为冲突。不是"他内心很矛盾"，而是让读者通过行为看到矛盾。

### 外化技法

| 技法 | 操作 | 示例 |
|------|------|------|
| 动作中断 | 角色开始做A，中途停下改做B | 他伸手去拿刀，手停在半空，又缩回来，拿起茶杯 |
| 物件交替 | 角色在两个物件间犹豫 | 他把信封翻过来又翻过去，拆了一半又封上 |
| 重复动作 | 角色反复做同一动作 | 他把烟点上又掐灭，点上又掐灭，第三次的时候烟碎了 |
| 矛盾指令 | 角色给出矛盾的指令 | "走吧。"顿了顿，"等等。"又顿了顿，"算了，走吧" |
| 环境映射 | 用环境变化映射内心 | 窗外的雨时大时小，他站在窗前，雨大时往后退，雨小时往前走 |

### 内心矛盾检测规则

1. 扫描"他犹豫了/他纠结/他不知道该不该"等心理直给词
2. 替换为上述外化技法
3. 确保外化动作与角色性格一致（军人不会"把信封翻来翻去"）
4. 矛盾外化后必须有"决定"——不能只展示矛盾不解决（除非是断章钩子）

## 对话信息不对称

> 不同角色掌握的信息量不同，通过对话中的"信息差"制造张力。

### 检测规则

1. 每场对话至少1处"信息不对称"（A知道B不知道的/双方都不知道的/双方知道但假装不知道的）
2. 标记"双方信息完全对等且无隐瞒"的对话 → 检查是否真的需要这场对话
3. 信息不对称的揭示必须有节奏——不能一次性全部暴露

### 对话零补丁规则

> 禁止用对话解释前文没写清楚的内容。

**违反示例**：
```
❌ "也就是说，你昨天去了李府，被李府的人打了，然后逃到了这里？"
   （用对话复述前文，说明前文没写清楚）
```

**修正**：
```
✅ "你身上那些伤——"她看了他一眼，没继续问。
   "不碍事。"他说。
   （前文已写清楚打斗过程，对话只暗示"她看到了伤"）
```
