# 质感注入协议

> **核心定位**：本文件是03扩写系统的"油门"——与"反注水铁律"（刹车）并行执行。
> 反注水铁律管"不该写什么"，本协议管"该怎么写"。
> **痛点对症**：当前skill只有刹车没有油门，AI为合规通过检查而写最少内容、用最安全句式，
> 导致"技术合格但灵魂缺失"的空文本。本协议通过正面引导，让AI知道"好文字长什么样"。

---

## 一、核心原则：质感 = 可感知的具体物

**AI写不出人味的根本原因**：AI倾向于写"抽象概括"（"他很饿""房间很旧""她很紧张"），而人类作者写"可感知的具体物"（"撕下半块饼先闻了闻""门轴上的锈爬满了合页""她把手缩进袖子里，指甲盖发白"）。

**一句话法则**：
> 如果读者闭上眼睛，无法在脑海中"看见"你写的画面——那就是空的。

---

## 二、五大质感注入技法

### 技法1：动作链写生理状态

**原理**：不写"他很饿/很冷/很痛"，写3-4个连续的具体动作，让动作链自己传达生理状态。

| ❌ 抽象概括 | ✅ 动作链 |
|-----------|---------|
| "他很饿" | "他撕下半块饼，先闻了闻，再塞进嘴里慢慢含化" |
| "她很冷" | "她把手缩进袖子里，指甲盖发白，脚趾在鞋里蜷起来" |
| "他疼得厉害" | "他弯下腰，手按住胃，额头抵在膝盖上，汗顺着鼻尖滴" |
| "她很紧张" | "她把杯子端起来又放下，放下又端起来，指尖在杯壁上敲了两下" |
| "他累极了" | "他靠着墙滑坐下来，后脑勺抵着墙面，眼睛闭上就没再睁开" |

**执行要点**：
- 动作链 = 3-4个连续动作，有先后顺序
- 每个动作必须是具体的、可观察的（不是"他动了动"这种模糊动作）
- 动作之间有因果关系（因为饿→所以撕→所以闻→所以塞→所以含化）

**关键区分——动作链 vs 微动作白噪音**：
- ✅ 动作链：3-4个动作传达一个明确的生理/情绪状态，有信息推进
- ❌ 微动作白噪音：对普通物品的无意义触碰（笔帽咔哒声/手指摩擦杯壁），无信息推进
- 判定标准：这组动作是否在传达角色的状态或推动情节？是→保留；否→删除

---

### 技法2：具体名词替代抽象形容词

**原理**：一个好名词的杀伤力 > 十个形容词。不写"很旧/很破/很贵"，写一个具体物让读者自己得出结论。

| ❌ 抽象形容词 | ✅ 具体名词 |
|-------------|---------|
| "房间很旧" | "门轴上的锈已经爬满了合页" |
| "她穿得很朴素" | "她穿着洗到发白的蓝布外套，领口的扣子换过一颗，颜色不一样" |
| "食物很糟糕" | "饼上的霉斑绿得发亮" |
| "这地方很穷" | "村里的路是土的，下了雨就是泥塘，鞋拔出来，脚还陷在里面" |
| "他很有钱" | "他掏车钥匙的时候，钥匙扣上挂着保时捷的标" |
| "这把刀很快" | "刀刃上没有锈，映着火光，边缘薄得能看见后面的手指" |

**执行要点**：
- 写"物"不写"感"——让物体自己说话
- 具体名词要带"时间痕迹"或"使用痕迹"（锈、磨损、发白、换过、沾着）
- 一个精准名词 > 三个形容词叠加

---

### 技法3：设定藏在生活里

**原理**：不写设定文档（"这个世界能量很稀缺"），写分配制度/交易场景/生活习惯，让读者通过"逛"来了解世界观。

| ❌ 设定说明 | ✅ 藏在生活里 |
|-----------|-------------|
| "这个世界能量很稀缺" | "天光石按户分配，每户每天拳头大的一块。王家婶子总是提前两小时熄了石" |
| "货币是光时" | "赵老汉挂出木牌：一把野菜，换一刻钟天光石照明" |
| "这里的人很穷" | "他数了三遍口袋里的硬币，码得整整齐齐，连一分钱的边缘都对准了桌缝" |
| "这个组织很神秘" | "门没有招牌。进去之后，前台让他在一张没有格子的白纸上填名字，不填身份证号" |

**四步公式**（世界观藏入生活的标准流程）：
1. 写一个**分配制度**（体现稀缺/等级/规则）
2. 写一个**具体家庭/个人**（让制度落地）
3. 写一个**生活习惯**（体现氛围）
4. 可选：写一句**有哲理的口语**（增加深度）

---

### 技法4：感官精准投放（重写版）

**旧规则的问题**："日常背景极简白描"导致AI什么都不写——文本变空。

**新规则**：日常背景用1个精准细节锚定，核心场景用3-5个感官叠加。

| 场景类型 | 感官投放策略 | 示例 |
|---------|-------------|------|
| 日常过渡 | 1个精准细节锚定（不是"极简白描"什么都不写，而是写1个最有特征的物） | "走廊里的声控灯灭了" |
| 对话场景 | 2个锚点（1个环境物 + 1个角色微动作） | "茶杯里的水还冒热气。她把杯子转了半圈。" |
| 紧张/核心场景 | 3-5个感官叠加（温度+触感+声音+视觉） | "空调开得很足，屋里凉。白术吐出一口白气，白气在冷蓝色射灯下瞬间散开。" |
| 战斗/爆发 | 动词链密集轰炸（5+连续动词） | "他侧身、出拳、收拳、再出拳，拳风擦着对方下巴过去" |

**关键区分——精准细节 vs 微动作白噪音**：
- ✅ 精准细节：一个能锚定场景氛围的具体物（"声控灯灭了"锚定"黑暗/安静"）
- ❌ 微动作白噪音：对普通物品的无意义触碰反馈（笔帽声/手指摩擦/门轴响）
- 判定标准：这个细节是否在锚定场景的核心氛围？是→保留；否→删除

---

### 技法5：对话要有"人味"

**原理**：AI写的对话像"信息传递工具"——每个人说话方式都一样，每句话都在推进情节。真人对话有口癖、有跑题、有答非所问、有情绪化。

| 维度 | ❌ 工具人对话 | ✅ 有人味的对话 |
|------|-------------|---------------|
| 说话方式 | 所有人用同一种句式说话 | 粗人说粗话，文人说文话，各有口癖 |
| 信息传递 | 每句话都在推进情节 | 允许说废话/跑题/答非所问 |
| 情绪 | 理性、直接 | 会赌气、会试探、会言不由衷 |
| 动作 | 纯对话无动作 | 对话中夹杂微动作（性格体现，非白噪音） |

**对话人味四要素**：
1. **口癖**：每个角色有1-2个口头禅或说话习惯（"我说……" "你懂什么" "嗐"）
2. **跑题**：真人说话会跑题、会打断、会扯到无关的事
3. **言不由衷**：嘴上说A，动作做B（"我才不稀罕"——手却攥紧了）
4. **信息差**：不是所有人都知道同样的事，有人问蠢问题，有人装懂

**关键区分——对话中微动作 vs 微动作白噪音**：
- ✅ 对话中微动作：体现角色性格或情绪（"她把头发别到耳后"体现紧张/害羞）
- ❌ 微动作白噪音：无意义的物理触碰（"他摸了摸杯子的边缘"无情绪含义）
- 判定标准：这个微动作是否在传达角色的情绪或性格？是→保留；否→删除

## 三、质感密度标准

### 每章质感投放预算

| 场景类型 | 质感投放要求 | 占比 |
|---------|-------------|------|
| 日常过渡 | 1个精准细节锚定场景 | 30-40% |
| 对话场景 | 2个锚点（环境+角色微动作）+ 对话有人味 | 20-30% |
| 核心/紧张场景 | 3-5个感官叠加 | 15-20% |
| 战斗/爆发 | 动词链密集轰炸 | 10-15% |

### 质感检查清单（扩写后自检）

```
【质感自检清单 — 与脱水自检并行执行】

1. 动作链检查：本章是否有至少2处用"3-4个连续动作"传达角色生理/情绪状态？
   → 否 = 质感不足，在关键场景补充动作链

2. 具体名词检查：本章是否有至少3处用"具体物+时间痕迹"替代抽象形容词？
   → 否 = 质感不足，把关键形容词替换为具体名词

3. 设定生活化检查：本章是否有至少1处世界观设定通过"生活场景"释放（非说明文）？
   → 否 = 质感不足，把设定说明改为生活场景

4. 对话人味检查：本章对话是否有口癖/跑题/言不由衷/信息差？
   → 否 = 对话太工具化，给角色增加人味

5. 画面感测试：闭上眼睛，能否在脑海中"看见"本章的关键场景？
   → 否 = 场景太空，补充精准细节

6. 反注水铁律兼容检查：本章的质感注入是否与反注水铁律冲突？
   → 动作链是否变成了微动作白噪音？（检查：是否在传达状态？）
   → 感官投放是否变成了感官轰炸？（检查：是否在核心场景？）
   → 如果冲突 → 调整投放精度，不是删除质感
```

## 四、与反注水铁律的关系

### 核心原则：油门和刹车不矛盾

| 反注水铁律（刹车） | 质感注入协议（油门） | 关系 |
|-------------------|---------------------|------|
| 禁止"无意义物理触碰反馈" | 技法1：动作链写生理状态 | 动作链有信息推进≠白噪音 |
| 禁止"滥用感官比喻" | 技法4：感官精准投放 | 精准投放≠轰炸，核心场景才叠加 |
| 禁止"心理活动套娃" | 技法1：动作链写生理状态 | 用动作折射心理≠心理实况转播 |
| 禁止"潜台词解释" | 技法5：对话有人味 | 人味对话有潜台词≠解释潜台词 |
| "日常背景极简白描" | 技法4：1个精准细节锚定 | 1个精准细节≠极简到什么都不写 |

### 冲突仲裁规则

当质感注入协议与反注水铁律冲突时，按以下规则仲裁：

1. **动作链 vs 微动作白噪音**：
   - 动作链有3-4个连续动作且在传达明确状态 → 保留（质感注入优先）
   - 单个无意义触碰（笔帽声/杯壁摩擦）→ 删除（反注水优先）

2. **感官叠加 vs 感官轰炸**：
   - 核心场景3-5个感官叠加 → 保留（质感注入优先）
   - 日常场景无差别感官铺陈 → 删除（反注水优先）

3. **比喻质量门 vs 比喻限流**：
   - 通过三重质量门的比喻 → 保留（质感注入优先）
   - 未通过的填充比喻 → 删除（反注水优先）

## 五、比喻质量门（替代旧的"≤2处"数量限制）

> **痛点对症**：旧规则"像/仿佛 ≤ 2处/章"导致两种极端——要么AI完全不写比喻（文字干瘪），
> 要么违规大量使用（比喻轰炸7.5/千字）。根本原因是只管数量不管质量。

### 三重质量门

每个比喻必须通过以下三重质量门才可保留：

**【精准门】** 比喻的喻体必须是读者能立刻产生感官联想的具体物
- ✅ "胃里像有只手在拧"（触觉，具体，读者立刻有身体联想）
- ❌ "像从地底深处抽出的沉积了千百年的空气"（抽象，堆砌，无感官联想）

**【记忆门】** 比喻必须创造画面记忆点，而非填充描写间隙
- ✅ "掌纹清晰可见，凸起处宛若山"（画面冲击，读者会记住这个画面）
- ❌ "空气像凝固了一样"（无记忆点，纯填充，删掉不影响任何东西）

**【不可替代门】** 如果删掉这个比喻换成直白描写，效果是否一样好？
- 如果一样好 → 删掉比喻，用直白描写（说明比喻是多余的）
- 如果比喻明显更好 → 保留（说明比喻不可替代）

### 执行规则

- 通过三重门的比喻：保留，但有**总密度软上限**——全章通过质量门的比喻≤4处/章
  - 即：即使每个比喻都通过了三重质量门，全章叙述文本中的比喻总数仍不得超过4处
  - 超过4处时：保留画面冲击力最强的4处，其余改写为【物理痕迹法】/【生理反馈法】
  - 对话中的比喻不计入总密度（人在说话时确实会用比喻）
- 未通过的比喻：删除或改写为【物理痕迹法】/【生理反馈法】
- **与旧规则的关系**：本质量门替代旧的"≤2处"硬上限。通过质量门且在总密度内的比喻不算违规。
- **角色对话中的比喻**：不受质量门和总密度限制（人在说话时确实会用比喻）

### 速判流程

```
写了一个比喻 → 
  ① 喻体是具体物吗？→ 否 → 删/改
  ② 有画面记忆点吗？→ 否 → 删/改  
  ③ 删掉换成直白描写一样好吗？→ 是 → 删/改
  → 三项全过 → 保留
→ 全章统计：保留的比喻是否≤4处？→ 否 → 删掉最弱的，改写为物理痕迹
```

## 六、细写决策协议（V2.4 新增 — 集中化"何时细写、何时不写"）

> **痛点对症**：本 skill 关于"何时细写、何时不写"的规则散落在多处——K.4三维镜头焦距（叙事与角色检测器）、F.1.2感官投放（描写与空间检测器）、本章质感密度标准——AI 执行时要在多个文件间跳转拼凑，容易顾此失彼。本节把"重要度→细写程度"的映射集中到一张决策表，作为扩写和精修时的第一参照。
> **核心原则**：不要直接写"谁做了什么"，而是写他的具体行为动作场景语言——**但只在需要的时候细写**。细写的目标是"身临其境、看到画面"，不是"任何时候都堆细节"。

### 6.1 细写决策表（重要度 → 细写程度）

写一个场景/动作前，先判定它的"细写等级"，再决定投入多少笔墨：

| 细写等级 | 触发条件（满足任一） | 细写程度 | 节奏模式 | 对应镜头 | 字数参考 |
|---------|-------------------|---------|---------|---------|---------|
| **S级 特写** | 生死绝境/境界突破/重大打脸/首登场/伏笔回收/情感关系转折/真相揭示/主角陷入绝境 | 快：动词链密集轰炸（战斗/逃命）<br>中：动作链+感官叠加（对峙/发现）<br>慢：全感官+情绪锚点（告别/觉醒） | 由角色行为决定：<br>逃命=快<br>对峙=中<br>告别=慢 | K.4 特写镜头 | 1200-1800字 |
| **A级 重点** | 冲突对话/线索发现/能力展示/新事物首次释放/角色对峙 | 快：快速对话+微动作（争论）<br>中：对话+2锚点（谈判）<br>慢：对话+环境渲染（交心） | 由场景张力决定 | K.4 中景镜头 | 800-1200字 |
| **B级 常规** | 正常信息交换/常规对话/普通移动/日常推进 | 快：直白叙述（信息交换）<br>中：1锚点定位（日常对话）<br>慢：环境+动作（漫步闲聊） | 由场景功能决定 | K.4 中景镜头 | 400-800字 |
| **C级 快进** | 赶路/等待/重复性日常/时间跳跃/非核心信息交代 | 15字内直白叙述——禁止感官/微动作/心理 | 越快越好 | K.4 远景快进 | ≤15字 |

**执行规则**：
1. 写每个场景前，先在预检面板里标注它的细写等级
2. **C级场景绝对禁止细写**——"他喝了口茶，继续听着"就是上限，不允许"指尖摩挲杯壁"
3. **S级场景必须细写**——只用15字带过=违规（浪费了高光时刻）
4. 等级和字数要匹配——S级场景写成400字=欠账，C级场景写成800字=注水

### 6.2 "画面感"达标标准（细写的目标）

细写的唯一目标是**让读者闭上眼能在脑海中看见画面**。判定一个场景是否"细写到位"：

```
【画面感测试】

闭上眼睛，回想刚写的这个场景——
→ 能"看见"具体的画面吗？（有物、有人、有动作）
  → 能 = 合格
  → 不能，只记得"谁做了什么" = 细写不足，回S/A级场景补质感

→ 画面里有"可感知的具体物"吗？（不是抽象形容词）
  → 有 = 合格
  → 只有"他很紧张/房间很旧" = 改为动作链/具体名词

→ 这个画面是"这个角色独有的"吗？（换个人不成立）
  → 是 = 合格（带角色视角）
  → 否 = 通用画面，加入角色职业/性格印记
```

### 6.3 "不细写"的红线（什么时候必须收）

细写不是越多越好——以下场景必须"收"，写多了就是注水：

| 必须收的场景 | 错误写法（细写了） | 正确写法（快进） |
|------------|------------------|----------------|
| 日常移动赶路 | "他穿过走廊，脚步声回荡，灯光忽明忽暗……" | "他去了后院。" |
| 重复性日常动作 | "他端起茶杯，摩挲杯壁，感受着温度……" | "他喝了口茶。" |
| 主角视线移动（非发现线索） | "他的目光扫过桌面，落在窗台，又转向门口……" | "他环顾四周。" |
| 角色情绪垫字 | "他愣了一下，眼中闪过一丝精芒……" | 直接写反应或台词 |
| 已交代过的信息再次出现 | 完整重复一遍设定说明 | 一句话带过 |

> **一句话**：S/A级场景要"写到读者看见画面"，C级场景要"快到读者不觉得拖"。把笔墨留给高光时刻，日常过渡越干脆越好。
