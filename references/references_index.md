## 一、03 质感系统

> 扩写系统的"油门系统"，与各检测器的"刹车系统"并行执行。03 扩写前必须先读取质感注入协议，扩写中参考质感锚点库与情绪工程协议。

### 03_质感注入协议.md
- **用途**：03 扩写系统的"油门"，正面引导如何写出有质感的文字（管"该怎么写"）。
- **适用步骤**：03 / 04
- **引用方式**：自动加载（03 扩写前必读）
- **关键内容**：
  - 核心原则：质感 = 可感知的具体物，不写抽象概括
  - 五大质感技法：动作链写生理状态 / 具体名词替代形容词 / 设定生活化 / 感官投放 / 对话人味
  - 比喻质量门 + 质感自检清单

### 03_情绪工程协议.md
- **用途**：03 扩写系统的第二台"油门"，工程化情绪转移而非模拟情绪表达（管"读者读完身体有没有反应"）。
- **适用步骤**：03 / 04
- **引用方式**：自动加载（与质感注入协议并行执行）
- **关键内容**：
  - 核心理论：模拟情绪表达 vs 工程化情绪转移
  - 公式：情绪冲击力 = 预期反差 × 合理度
  - 油门管"该怎么设计"，与刹车（铁律 4 情绪冰山 / 04 冰山原则 / 05 Gate3 心理外化）互补

### 03_质感锚点库.md
- **用途**：03 扩写时参考"一段好文字长什么样"的整段级正面锚点。
- **适用步骤**：03 / 04
- **引用方式**：按需调用（扩写时参考，精修时对标深化）
- **关键内容**：
  - 16 个完整段落范例（非单句对照）
  - 覆盖：动作链 / 具体名词 / 设定生活化 / 感官 / 对话 / 综合
  - 每个锚点附"拆解"，说明质感来源

## 二、检测器系列

> 扩写系统的"刹车系统"，检测并拦截各类写作问题。V4.0 规则分层：铁律 → 弹性建议 → 风格可选项。

### 03_检测器_伏笔与情绪.md
- **用途**：伏笔休眠-唤醒机制 + 多线节奏协同 + 情感资产账户系统。
- **适用步骤**：02 / 03 / 04
- **引用方式**：自动加载（03 扩写时检测）
- **关键内容**：
  - A. 伏笔四态模型：活跃 / 休眠 / 唤醒 / 已回收（含 Token 占用与触发条件）
  - B. 多线节奏禁止规则：双峰叠加 / 双谷塌陷 / 节奏冲突
  - C. 情感资产账户：每个逝者/关系设"情感调用额度"，超额触发新锚点建设
  - D / E. 情绪检测模块

### 03_检测器_爽点与对照表.md
- **用途**：商业爽点保护协议 + 爽感模式规则 + 呼吸节奏机制。
- **适用步骤**：02 / 03
- **引用方式**：自动加载（爽点触发事件时激活）
- **关键内容**：
  - NA. 爽点触发事件：打脸 / 升级 / 揭秘 / 逆袭 / 收获五类
  - 爽感保护模式：豁免双重动机约束、降低感官密度、允许碾压
  - 呼吸节奏：真实感=吸气，爽感=呼气；铺垫:释放 ≈ 2:1 或 3:1
  - 硬性熔断器：最长 800 字或 3 个自然段

### 03_检测器_描写与空间.md
- **用途**：描写模式高频预警 + 感官维度精准投放 + 空间构建检测。
- **适用步骤**：03 / 04
- **引用方式**：自动加载（03 扩写时检测）
- **关键内容**：
  - F. 描写高频模式预警：眼睛特写 / 物品状态重复 / 恐怖氛围词 / 时间过渡 / 纯视觉描写
  - F.1.2 感官二分法投放（V4.1 重构）：日常背景→极简白描；核心异常点→多维感官叠加
  - 废除旧的"气味 > 触觉 > 听觉 > 视觉"全局硬性优先级
  - G / H. 空间检测模块

### 03_检测器_基础规则.md
- **用途**：句式节奏底层极限拦截 + 硬性脱水压缩 + 段落排版控制。
- **适用步骤**：03 / 04
- **引用方式**：自动加载（03 扩写时检测）
- **关键内容**：
  - P. 全局底层极限拦截：禁 1500 字以上无长句飞灵化碎步；禁 400 字以上无断句 AI 大长段
  - P.1.5 硬性脱水压缩（V4.1 新增）：不影响剧情的动作 ≤20 字；日常动作超 20 字必压缩
  - 段落切分规则 + 执行约束清单

### 03_检测器_叙事与角色.md
- **用途**：动机模糊性与人性复杂度 + 感官密度动态调节器。
- **适用步骤**：03 / 04
- **引用方式**：自动加载（03 扩写时检测）
- **关键内容**：
  - J. 双重动机规则：表面动机（功利性）+ 深层动机（情感性），言行不一制造人性张力
  - J.2 动机显隐周期：显性期 70% / 隐性爆发期 30%
  - K. 感官密度动态调节器：高密度模式（新角色/伏笔回收/冲突等 8 触发条件）vs 低密度模式（信息交换/赶路等）
  - L / M. 叙事技法模块

## 使用建议

1. **03 扩写前**：自动加载 `03_质感注入协议.md` + `03_情绪工程协议.md`（油门），并加载全部检测器（刹车）。
2. **03 扩写中**：按需参考 `03_质感锚点库.md` 对标好段落；按场景调用语言素材库；写前查 `改写范例库.md` 预防层规避AI味句式（V2.3新增）。
3. **04 精修时**：参考质感系统深化，调用素材库填充质感；遇到AI味句式查 `改写范例库.md` 精修层找映射（V2.3新增）。
4. **05 去AI味**：按需调用 `改写范例库.md` + `网文通用高频词替换库.md`；读取04精修意图包后跳过已处理项（V2.3新增）。
5. **开篇 / 新卷**：调用 `Part_K` 选开篇方法，`Part_L` 定文风，`Part_M` 查题材公式。
6. **跨步骤交接**：03交付时附加扩写意图包，04交付时附加精修意图包（见 `04_交接协议.md`，V2.3新增）。
7. **伏笔管理**：02埋设伏笔时查 `02_伏笔全生命周期管理.md` 建立追踪表，03/04按表操作（V2.3新增）。
8. **平台审稿**：09审稿时加载 `平台规则配置.md` 对应平台规则（V2.3新增）。
9. **素材库调用前**：必须先完成安全鉴权（读取 `novel_meta.core_tags` + `Part_L_文风系统`）。
