# 改写范例库：AI味 → 自然表达 映射表

> V1.0 — 去 AI 味专用改写参考。整合 oh-story-claudecode 的 `anti-ai-writing.md` 改写范例 + 本地 11 Gate 检测项的可执行替换方案。
> 用途：05 去AI味精修师 Gate 1-11 检出 AI 味时，从此库查找自然表达替换。**不是素材填充库**，是"AI味→人味"的改写对照表。

---

## 使用规则

1. **改最少，效果最大**：能改一个词不改一句，能删一句不重写一段。没有问题的句子保留原句。
2. **改"怎么说"不改"说什么"**：剧情/人设/情节走向不动，只改表达方式。
3. **优先删除**：很多 AI 味句是废话（解释/注水/凑数），润色后照样冗余——先判能否删，再润色。
4. **不堆砌替换**：一次只用一个替换方案，不要把"他垂下眼"+"眯起眼"+"乐了"全用上。
5. **配合素材库**：替换后若需填充质感，调用成语库/怼人话术库等素材库（见素材库统一索引）。

## 一、情绪直给类（Gate 1 禁用词 + Gate 3 心理外化）

| AI 味原文 | 自然改写 | 原理 |
|----------|---------|------|
| 他感到一阵悲伤。 | 他蹲下身，捡起那块石头抛了抛，又放下了。 | 情绪转动作，悲伤用"无意义重复动作"承载 |
| 他心中涌起一股莫名的感动。 | 他别过脸，喉结动了一下。 | "涌起一股"是 AI 高频；用喉结动作暗示 |
| 他不由得紧张起来。 | 他的手摸到口袋里的烟盒，空了。 | "不由得"删；紧张转"找烟"的转移注意力 |
| 他忍不住笑了。 | 他嘴角抽了一下。 | "忍不住"删；笑用"嘴角抽"半抑制 |
| 他很愤怒。 | 他把杯子往桌上一顿，水溅出来。 | 情绪转破坏性动作 |
| 她很害怕。 | 她往后退了半步，背抵住墙。 | 恐惧转身体后退 |
| 他感到一丝失落。 | 他愣了一下，把手机放回口袋。 | "一丝"删；失落转"愣+放手机" |
| 他内心很矛盾。 | 他拿起又放下，放下又拿起。 | 矛盾转重复动作 |
| 他既期待又害怕。 | 他往前凑了凑，又缩回来。 | 矛盾情绪转"凑+缩"身体语言 |
| 他十分恐惧。 | 他盯着对手刀柄上缺了一根线的红缨，看了很久。 | 高压场景用"转移注意力"——失焦感比恐惧描写更人味 |

> **转移注意力机制**（高压场景专用，每章≤2处）：死亡威胁/重大打击/社死瞬间，禁止直接写恐惧/紧张/羞愧，强制角色注意力漂移到无关微小事物（蚂蚁/蜘蛛结网/车铃/水凉了）。

## 二、动作套路类（Gate 2 句式去套路 + Gate 7 意象疲劳）

| AI 味原文 | 自然改写 | 原理 |
|----------|---------|------|
| 深吸一口气 | 胸口起伏了一下 / 直接删 | "深吸一口气"90%无意义 |
| 眼中闪过一丝不易察觉的悲伤 | 他垂下眼 | "闪过一丝不易察觉"三重修饰堆砌 |
| 嘴角勾起一抹冷笑 | 他笑了一下，没到眼底 | "勾起一抹"是 AI 标志 |
| 缓缓开口 | 说 / 用动作引出对话 | "缓缓"是 AI 节奏词 |
| 微微皱眉 | 皱眉 / 眉头拧了一下 | "微微"删，直接动作 |
| 淡淡地说 | 说 / 语气平 | "淡淡"是 AI 腔 |
| 轻轻叹了口气 | 叹了口气 / 他"嗐"了一声 | "轻轻"删 |
| 不禁笑了 | 笑了 / 嘴角一歪 | "不禁"删 |
| 仿佛时间静止了 | 删整句 | 滥用比喻 |
| 空气仿佛凝固了 | 删整句 / 换具体：屋里安静得能听见钟表走针 | "凝固"万能比喻 |
| 他端起茶杯，吹了吹热气，抿了一口，放下杯子，手指敲了敲桌面，望向窗外，又收回目光。 | 他喝了口茶。 | 6个动作0信息推进，合并为1 |
| 铜钱发烫，硌着指腹。 | （换意象）他捏了捏袖口，又松开。 | 同一意象反复当情绪容器→换物件 |

## 三、对话腔调类（Gate 5 对话去腔调 + Gate 9 信息不对称）

| AI 味原文 | 自然改写 | 原理 |
|----------|---------|------|
| "你去哪了？" "查档案。" "查到什么？" "一些奇怪的东西。" | "你去哪了？" "……没去哪。" 他把外套挂好。"吃了吗？" | 问答太顺畅→打断+答非所问 |
| "谢谢你。" "不客气。" | （删礼貌用语，或改动作点头） | 太礼貌像采访 |
| "我要解释一下世界观……"（角色整段讲设定） | 改为动作/观察带出设定 | 角色当"科普嘴"是 AI 病 |
| "这意味着我们必须合作。" | "所以？" 他靠向椅背。 | "这意味着"是叙述者翻译腔 |
| 他说："……" 她问："……" 他答："……" | （去对话标签，用动作穿插） | 标签单一→50%用动作替代 |
| "我一定会证明自己的。" | "哦。" / "说完了吗？" | 太正经→让主角嘴贱 |
| "这只是开始。" | "就这？" / "没意思。" | 升华结尾→让主角冷淡 |

> **对话三症状**（Gate 5/9 共管）：①机械问答式，句间无情绪承接；②角色当"科普嘴"整段讲设定；③说话不分场合（高压场景插科打诨出戏）。

## 四、结尾升华类（Gate 6 结尾去升华）

| AI 味原文 | 自然改写 | 原理 |
|----------|---------|------|
| 三千年来，这是第一次…… | 删整段 | 排比升华 |
| 这不仅是……也是……更是…… | 删整段 | 三段排比 |
| 或许这就是人生吧。 | 删整句 / 换动作收尾 | 哲理总结 |
| 他终于明白了一件事…… | 删整句 / 换：他从那天起再也没提过那件事。 | "明白了一件事"是 AI 收尾标志 |
| 很多年后，当…… | 删整句（除非预设叙事框架） | 视角跳出 |
| 没有人知道，这一刻意味着什么。 | 删整句 | 上帝视角剧透 |
| 至于后来怎样——那是另一个故事了。 | 删 / 换真正的留白 | 刻意留白套路 |
| 他来的时候天是黑的，走的时候天还是黑的。 | 偶尔用，每卷≤2次 | 对称句式堆砌 |

**正确收尾方式**：戛然而止 / 物件承载 / 身体感受 / 环境收束 / 动作收尾 / 简短对话（≤10字）。

## 五、解释腔/上帝感类（Gate 11 + Gate 9 剧透式叙述）

| AI 味原文 | 自然改写 | 原理 |
|----------|---------|------|
| 之所以……是因为…… | 删，因果让读者从动作对话拼 | 解释因果 |
| 原来…… | 删 | 叙述者解释 |
| 这意味着…… | 删 | 叙述者翻译 |
| 换句话说…… | 删 | 叙述者翻译 |
| 她不知道的是…… | 删 | 上帝视角剧透 |
| 殊不知…… | 删 | 上帝视角剧透 |
| 仿佛预示着…… | 删 | 定性比喻 |
| 命运的齿轮开始转动 | 删 | AI 标志句 |
| 演得真好 | 删，证据留给读者判断 | 替读者定性 |
| 他就是这样薄情 | 删 / 改为角色此刻带偏见的瞬间感觉 | 替读者定性 |
| 关切得恰到好处 | 删评判，只写动作 | 隐蔽软评判（评判性副词） |
| 那点笑她看得分明 | 改为角色视角 | 剧透点破 |

> **Gate 9 vs Gate 11 区分**：Gate 9 管角色意图（"她说X其实她是Y"）；Gate 11 管叙述者解说（"之所以…是因为"）。两者并行不冲突。

## 六、句式套路类（Gate 2 否定翻转 + 万能状语）

| AI 味原文 | 自然改写 | 原理 |
|----------|---------|------|
| 不是恐惧，而是一种深沉的绝望 | 绝望从骨髓里渗出来。 | 否定铺垫+肯定翻转，最毒 AI 句式 |
| 他没有说话，只是默默地看着她。 | 他没说话。看着她。 | "没有X，只是Y"万能状语 |
| 声音不大，却带着不容置疑的威严。 | 声音不高，屋里却静下来。 | "声音不大，却带着"AI 标志 |
| 仿佛/犹如/宛若/如同 | 像 / 直接白描 | 文言腔过重 |
| 不容置疑/显而易见 | 删 / 用具体事实 | 书面化判断词 |
| 他知道，这一切都结束了。 | 删"他知道" | 直接告诉读者 |

## 七、重复语义四类（Gate 2 子规则，v1.4 新增）

| 类别 | 错误例 | 修法 |
|------|--------|------|
| 形容词重复 | 兴高采烈地笑着跑过来 | 笑着跑过来 |
| 近义词重复 | 非常重要的关键问题 | 关键问题 |
| 含义重复 | 我好饿，肚子咕咕叫 | 我好饿 |
| 主语/物品重复 | 上文"把药扔了一地"，下文"地上的药" | 只写"药片" |

---

## 八、修饰词清扫（Gate 2 子规则，v1.4 新增）

| 原文 | 清扫后 | 规则 |
|------|--------|------|
| 白色的药片 | 药片 | 颜色修饰无功能则删 |
| 飞驰的汽车 | 汽车 | 形容词无信息则删 |
| 手里那截链子 | 链子 | 指示代词多余则删 |
| 多年的衣服 | 旧衣服 | 保留含义换简洁 |
| 锋利的刀锋冷冽 | 刀 | 双重修饰堆砌 |

**原则**：一次只用一个形容词修饰或不修饰，不连用、不堆砌。删除后阅读不影响才删；含义流失则改成简洁名词。

## 与本地素材库的分工

| 资产 | 性质 | 用途 |
|------|------|------|
| **改写范例库（本文件）** | 改写对照表 | AI味→自然表达的替换参考（减法） |
| 成语库/歇后语库/俗语库/怼人话术库/打斗动作库 | 填充素材库 | 替换后需填充质感时调用（加法） |
| 网文通用高频词替换库 | 词汇替换 | AI高频词的多样化替换 |

> **改写范例库是"减法工具"**：去掉 AI 味后若文本变薄，再用素材库"加法"补质感。两者配合：先减后加，不要边减边加。
