# 情绪颗粒度/心理描写库

> **定位**：解决"情绪标签化"和"心理活动千篇一律"的核心痛点。不是"情绪词汇表"，是按**情绪光谱→外化行为→意识流句式**三层组织的心理描写系统。
> **核心原则**：不写"他很悲伤"，写"悲伤的哪个细分颗粒"——把"悲伤"拆解为哀恸/怅然/委屈/绝望/麻木/释然，每种配3-5个差异化表达。
> **与情绪工程协议协作**：情绪工程协议管"情绪冲击力设计"（预期反差×合理度），本库管"情绪的具体呈现"（用什么样的行为/句式来呈现情绪）。

---

## 一、情绪细分光谱

> **核心理念**：把"高兴/悲伤/愤怒/恐惧"四大基础情绪，拆解为可感知的细分状态。每个细分状态 ≠ 一个形容词，而是"身体状态+行为倾向+内心独白"的复合体。

### 1.1 悲伤光谱（从轻到重）

| 细分状态 | 身体状态 | 行为倾向 | 内心独白示例 | 与"悲伤"的区别 |
|---------|---------|---------|------------|--------------|
| 怅然 | 动作变慢，眼神放空 | 看着某个地方发呆，被叫到时反应慢半拍 | "好像少了点什么。" | 不是"哭"，是"空"——一种说不清少了什么的失落 |
| 委屈 | 喉咙发紧，鼻子发酸 | 想说话但说不出来，低头 | "为什么是我？" | 有"不公平感"——觉得被冤枉/被误解 |
| 心酸 | 胸口发闷，想叹气 | 反复看某个东西（照片/信/旧物） | "那时候……" | 有"回忆"触发——被过去的美好刺痛 |
| 伤痛 | 胸腔里像有东西在扯 | 不想说话，不想动，但睡不着 | "疼。" | 生理化的悲伤——悲伤已经转化为身体感受 |
| 哀恸 | 身体失去力气，可能瘫坐 | 哭/喊/沉默，行为失控 | "没有了。" | 失去"重要的人/物"——不可逆的失去 |
| 绝望 | 身体冰冷，不想动 | 放弃挣扎，不再尝试 | "算了。" | 失去"未来"——不只是失去了什么，而是不相信还能得到 |
| 麻木 | 什么感觉都没有 | 机械地做该做的事，没有表情 | "……"（没有内心独白） | 悲伤的终点——悲伤到不再悲伤 |
| 释然 | 身体变轻，想深呼吸 | 笑了一下（不是高兴，是"放下了"） | "就这样吧。" | 悲伤的出口——接受了，放下了 |

### 1.2 愤怒光谱（从控制到爆发）

| 细分状态 | 身体状态 | 行为倾向 | 内心独白示例 | 与"愤怒"的区别 |
|---------|---------|---------|------------|--------------|
| 不悦 | 眉心动了一下，眼神变锐 | 没说话，但动作变重了 | "……" | 还在"控制"阶段——不满但不想发作 |
| 烦躁 | 坐不住，想动 | 反复换姿势，叹气，看时间 | "烦死了。" | 有"不耐烦"——不是针对谁，是对"什么都烦" |
| 恼怒 | 牙关咬紧，呼吸变重 | 可能摔东西/拍桌子 | "他怎么敢——" | 有"被冒犯"感——针对特定的人/事 |
| 暴怒 | 身体发抖，视野变窄 | 可能失控——打人/砸东西 | 没有独白，只有冲动 | 失控——理性被情绪覆盖 |
| 冷怒 | 表情变冷，声音变稳 | 不动手，但说话像刀子 | "你再说一遍。" | 极怒的反面——不是爆发，是"冻结" |
| 迁怒 | 坐立不安，想找人发泄 | 对无辜的人发火 | "我又不是故意的——"（对无辜者） | 有"无理"感——角色自己也知道不该发火 |
| 积怨 | 长期绷紧，累 | 某件小事突然爆发 | "每次都是这样。" | 不是这一次，是"每次都这样" |

### 1.3 恐惧光谱（从不安到崩溃）

| 细分状态 | 身体状态 | 行为倾向 | 内心独白示例 | 与"恐惧"的区别 |
|---------|---------|---------|------------|--------------|
| 不安 | 总回头看，注意力分散 | 检查门/窗/锁，反复确认 | "好像有人在看我。" | 模糊的威胁——不知道具体怕什么 |
| 紧张 | 手心出汗，语速变快 | 喝水/清嗓子/摸脸 | "别出错别出错——" | 有"具体目标"——怕做不好/怕被发现 |
| 恐惧 | 身体僵硬，不想动 | 不敢出声，躲在某个地方 | "别过来——" | 威胁明确——知道怕什么 |
| 恐慌 | 身体失控，可能尖叫/跑 | 不顾一切逃跑/攻击 | 没有独白，只有"跑" | 失控——恐惧让理性失效 |
| 绝望恐惧 | 身体发软，不想跑了 | 等死/放弃 | "完了。" | 恐惧+绝望——知道逃不掉 |
| 后怕 | 安全后反而发抖 | 坐在地上，回想刚才 | "如果刚才……" | 延迟的恐惧——危险过去了，恐惧才来 |

### 1.4 喜悦光谱（从满足到狂喜）

| 细分状态 | 身体状态 | 行为倾向 | 内心独白示例 | 与"高兴"的区别 |
|---------|---------|---------|------------|--------------|
| 满足 | 动作变慢，想待着不动 | 不想说话，不想动 | "挺好的。" | 不是"开心"，是"够了"——不想要更多 |
| 欣慰 | 想叹气（不是叹气，是深呼吸） | 看着某个人/某个东西，嘴角上扬 | "总算……" | 有"等待"——等了很久终于等到了 |
| 雀跃 | 很想动，想跳 | 可能真的跳起来，可能哼歌 | "太好了！" | 有"突然"——不是一直高兴，是突然高兴 |
| 狂喜 | 大笑/大叫/可能哭 | 无法控制，行为夸张 | 没有独白，只有"啊——" | 极度——高兴到"失控" |
| 暗喜 | 想笑但憋住 | 偷偷笑，憋得很辛苦 | "嘿嘿。" | 有"隐蔽"——不能让别人知道自己在高兴 |
| 得意 | 下巴微抬，嘴角上扬但控制住 | 想炫耀但控制住 | "我就知道。" | 有"自我肯定"——我为自己的判断/能力高兴 |
| 成就感 | 累了但不想停 | 反复看自己的成果 | "我做到了。" | 有"付出"——高兴是因为"我做到了" |

## 二、情绪外化行为集

> **核心理念**：不写情绪词（"他很愤怒"），通过动作/环境/对话间接传递情绪。
> 每个行为 = "一个动作"而非"一个表情"——动作比表情更可信。

### 2.1 压抑型外化（情绪被"压住"时的行为）

| 情绪 | 外化行为 | 为什么有效 |
|------|---------|----------|
| 压抑愤怒 | 把东西放回原处时发出声响 | 愤怒在"控制"中泄露——比"摔东西"更高级 |
| 压抑悲伤 | 反复做同一件无意义的事 | 如"反复擦同一块玻璃"——悲伤在"重复"中泄露 |
| 压抑恐惧 | 检查不需要检查的东西 | 如"反复检查门锁"——恐惧在"确认"中泄露 |
| 压抑喜悦 | 想笑但强行憋住，笑变成咳嗽 | 喜悦在"控制"中泄露 |
| 压抑焦虑 | 把东西排整齐 | 如"把桌上的笔排成一条线"——焦虑在"秩序"中泄露 |

### 2.2 转移型外化（情绪被"转移"到别处）

| 情绪 | 外化行为 | 为什么有效 |
|------|---------|----------|
| 恐惧 | 突然注意到无关紧要的细节 | "墙上有一只蜘蛛在结网。他看了很久。"——恐惧被转移到蜘蛛上 |
| 悲伤 | 突然开始关心一件小事 | "他蹲下身，把碎掉的杯子拼了拼。"——悲伤被转移到碎片上 |
| 愤怒 | 做一件需要专注的小事 | "他把抽屉里的东西全部倒出来，又一件一件放回去。"——愤怒被转移到整理上 |
| 紧张 | 反复摆弄一个小物件 | "他把打火机按了又熄，熄了又按。"——紧张被转移到打火机上 |
| 尴尬 | 把注意力放在自己的穿着上 | "他低头检查自己的鞋带，鞋带系得很好，但他还是解开重新系了一遍。" |

### 2.3 反转型外化（情绪以"相反"行为呈现）

| 情绪 | 外化行为 | 为什么有效 |
|------|---------|----------|
| 极度愤怒 | 反而笑了 | 愤怒的反面不是平静，是笑——笑比愤怒更可怕 |
| 极度悲伤 | 反而没什么表情 | 悲伤到极点不是哭，是"什么都没有" |
| 极度恐惧 | 反而站住了不动 | 恐惧到极点不是跑，是"冻住" |
| 极度紧张 | 反而说话变慢 | 紧张到极点不是语速加快，是"每一个字都控制" |
| 极度高兴 | 反而哭了 | 高兴到极点不是笑，是流泪 |

## 三、意识流/内心独白句式

> **核心理念**：不同性格的角色，内心独白的"逻辑结构"完全不同。
> 不是"他想……"（万能前缀禁止），而是"角色的思维模式"决定内心独白的句式。

### 3.1 理性型（因果链式）

**思维特征**：因为A→所以B→但如果C→那么D→结论E
**句式特征**：多"如果/那么/但/所以"。句子长，逻辑清晰。

| 示例 | 角色类型 |
|------|---------|
| "如果他现在出去，他们会看到他。但如果他不出去，她就一个人了。她一个人，上次的事就会重演。他不能让它重演。" | 军人/侦探/策略型角色 |
| "门是锁着的。窗户也是。但灰尘不对——桌上的灰尘只有一半，另一半被什么擦掉了。有人来过。不是从门窗进来的。" | 调查型角色 |

### 3.2 敏感型（碎片联想式）

**思维特征**：A→让我想到B→B让我想到C→C又让我想到A
**句式特征**：多"像/好像/让我想起/以前……"。句子碎片化，跳跃式。

| 示例 | 角色类型 |
|------|---------|
| "那盏灯的颜色像她上次煮的汤。那次汤洒了，洒在桌上，她没擦。后来干了，黏在桌上。她的手现在也黏黏的。" | 感性/敏感/文艺型角色 |
| "他说话的时候一直看着窗外。窗外有什么？一棵树。树上有鸟。鸟飞走了。他还在看窗外。" | 观察型/敏感型角色 |

### 3.3 创伤型（闪回片段式）

**思维特征**：当下场景→触发创伤记忆→闪回→回到当下→但回不去
**句式特征**：过去和现在交织，不加"他想起/他回忆起"（禁止万能前缀），直接在叙述中切换。

| 示例 | 角色类型 |
|------|---------|
| "那个人走过来的时候，他闻到了消毒水的味道。病房的门开了。医生在说什么。他听不清。那个人还在走，已经走到他面前了。他眨了眨眼，消毒水的味道消失了。" | 有创伤史的角色 |
| "那个声音。那个声音和上次一样。上次他听到这个声音，然后——他掐了一下自己的手。现在。现在不是上次。" | 创伤后应激障碍（PTSD）型角色 |

### 3.4 直觉型（身体先于大脑）

**思维特征**：身体已经做出反应，大脑还没想明白
**句式特征**：不写思考过程，只写"身体做了什么"，然后"他意识到自己做了什么"

| 示例 | 角色类型 |
|------|---------|
| "他退了一步。他不知道自己为什么要退，但脚已经退了。" | 战斗型/直觉型角色 |
| "她没看那个人，但她已经把手机调成了静音。她不知道为什么要这样做，但手已经做了。" | 警觉型角色 |

### 3.5 自言自语型（碎片化自问自答）

**思维特征**：像在跟自己对话，句子碎片化，有问有答
**句式特征**：多"？"和短句，不连贯

| 示例 | 角色类型 |
|------|---------|
| "怎么办？不知道。跑？跑不了。打？打不过。那怎么办？……再说吧。" | 焦虑型/普通人在困境中 |
| "他刚才说什么？没听清。再说一遍？算了。不重要。真的不重要吗？应该不重要。" | 犹豫型角色 |

## 四、情绪场景速查（按"写作场景"索引）

### 4.1 写"失去"场景

```
不要写：他很悲伤。
要写：
  → 怅然："他坐在她常坐的那把椅子上，发现椅子已经不暖和了。"
  → 心酸："他把她的杯子翻过来，杯底还有一点水渍。他没擦。"
  → 哀恸："他跪在地上，没哭。他把手放在地上，像在摸什么——但地上什么都没有。"
  → 麻木："他站起来，把她的东西一件一件放进箱子里。动作很稳，像在打包一件普通的快递。"
```

### 4.2 写"愤怒"场景

```
不要写：他非常愤怒。
要写：
  → 不悦："他把茶杯放回桌上，声音比平时大了一点。"
  → 恼怒："他的下颌肌肉跳了一下，但他没说话。"
  → 冷怒："他说话的声音比平时轻，但每个字都清清楚楚。"
  → 暴怒："他站起来的时候，椅子倒在地上。他没去扶。"
  → 积怨："他低着头，听她说完。然后他把那张纸揉成一团——那张纸跟这件事没关系。"
```

### 4.3 写"恐惧"场景

```
不要写：他非常害怕。
要写：
  → 不安："他走了两步，回头看了一眼。再走两步，又回头看了一眼。"
  → 紧张："他清了一下嗓子。又清了一下。其实他嗓子不痒。"
  → 恐惧："他蹲在柜子后面，发现自己的手在抖。他按住手，但手还在抖。"
  → 恐慌："他跑了。他的脑子还没想好要跑，但腿已经跑了。"
  → 后怕："他坐在路边，哭了。不是因为刚才有多危险，是因为安全了。"
```

### 4.4 写"喜悦"场景

```
不要写：他非常高兴。
要写：
  → 满足："他靠在椅背上，闭上眼睛。不想动了。"
  → 欣慰："他看了很久，然后把那张纸叠好，放进口袋——不是随便放的，是对折了两次，放在最里面。"
  → 雀跃："他走路的时候脚尖在点地。他自己没发现。"
  → 暗喜："他转过身，偷偷笑了一下。转回来的时候，脸上又恢复了正常。"
  → 成就感："他看着那堆碎掉的零件，又看着自己拼好的机器。他笑了——不是对别人笑，是对自己笑。"
```

## 五、使用指南

### 5.1 调用时机

- **03扩写**：写情绪场景时，打开"情绪细分光谱"，从"悲伤→怅然/委屈/心酸/伤痛/哀恸/绝望/麻木/释然"中选择最精准的颗粒度
- **04精修**：诊断"情绪标签化"场景——出现了"他很愤怒/很悲伤/很害怕"→打开"情绪外化行为集"替换
- **05去AI味**：检测"他心里想/他觉得/他感到"等万能前缀+情绪词直给→打开"意识流句式"替换为角色的思维模式

### 5.2 情绪描写质量自检

```
【情绪描写自检 — 写完后逐处核对】

① 情绪有"颗粒度"吗？→ 不是"悲伤"，是"怅然/委屈/心酸/伤痛/哀恸/绝望/麻木/释然"中的一个
② 情绪是"说出来的"还是"做出来的"？→ 说出来的=删，用"动作"替代
③ 角色当下的行为是"这个角色"的行为吗？→ 军人愤怒≠商人愤怒≠文人愤怒
④ 内心独白符合角色的"思维模式"吗？→ 理性型=因果链，敏感型=碎片联想，创伤型=闪回
⑤ "情绪词+动作+情绪副词+对话"四重叠加了吗？→ 有=合并，只保留1-2个最强表达
```

## 六、速查卡

```
【情绪颗粒度速查卡 — 写情绪时贴在屏幕上】

写情绪前问自己：
① 这个情绪是"基本情绪"（悲伤/愤怒/恐惧/喜悦）还是"细分状态"？
   → 基本情绪=太粗糙，查"情绪细分光谱"找到最精准的颗粒度
② 这个情绪是"说出来的"还是"做出来的"？
   → 说出来的=删，查"情绪外化行为集"选一个动作
③ 这个角色的内心独白是"理性型/敏感型/创伤型/直觉型/自言自语型"？
   → 查"意识流句式"选对应的句式结构

写情绪后问自己：
① 删掉所有情绪词（悲伤/愤怒/恐惧/高兴），读者还能感受到这个情绪吗？→ 不能=重写
② 这个情绪表达是"所有角色通用"还是"这个角色独有"？→ 通用=不精准，重写
③ 同一个情绪是不是被表达了2次以上？→ 是=合并，只保留最强的1次
```
