## J.1 为什么动机要显隐？

番茄读者不喜欢：
- 主角想太多（内耗）
- 主角动机太复杂（看不懂）
- 主角太纠结（不爽）

番茄读者喜欢：
- 主角目标明确（知道要干嘛）
- 主角行动果断（说干就干）
- 主角动机简单（为钱/为仇/为女人）

## J.2 动机显隐周期表

```
[动机显隐周期表 - J模块]

显性期（前5章）：
├─ 主角目标必须极其简单粗暴
├─ 错误示范："我想探寻世界的真理，但又怕受伤，还要考虑师父的感受..."
├─ 正确示范："为了赏金，老子要弄死这个怪物。"
│
├─ 表面动机：钱、仇、女人、生存、面子
├─ 行动动机：弄死他、抢过来、逃出去、打他的脸
└─ 深层动机：隐藏起来，不要让读者觉得主角在装

隐性期（5章后）：
├─ 可以逐渐揭示深层动机
├─ 但不要长篇大论，用行动暗示
└─ 示例：主角为了钱杀人 → 但杀完不要钱 → 读者自己悟：原来他是为了...

周期规则：
├─ 第1-3章：纯表面动机（钱/仇/生存）
├─ 第4-5章：表面动机 + 行动动机
├─ 第6-10章：开始暗示深层动机
├─ 第10章后：深层动机逐渐浮现
└─ 注意：深层动机永远不要直接说出来，让读者自己悟
```

### 动机类型速查
| 类型 | 表面动机 | 行动 | 深层动机 |
|------|---------|------|---------|
| 利己型 | 为了钱/资源 | 抢/杀/争 | 证明自己/填补空虚 |
| 复仇型 | 为了报仇 | 弄死仇人 | 为了正义/守护 |
| 生存型 | 为了活下去 | 逃/打/拼 | 寻找意义 |
| 守护型 | 为了保护某人 | 挡/杀/争 | 弥补过去的遗憾 |
| 证明型 | 为了证明自己 | 挑战/超越/打脸 | 获得认可 |

## J.4 动机检查协议

```
[动机检查协议 - 每章检查]

第1-3章检查：
├─ 主角目标是否明确？
│   └─ 否 → ❌ 警告："主角目标不明，读者不知道他要干嘛"
│
├─ 主角目标是否简单？
│   └─ 否 → ⚠️ 建议："简化动机，用钱/仇/生存等表面动机"
│
├─ 主角是否在行动？
│   └─ 否 → ❌ 警告："主角只想不动，节奏拖沓"
│
└─ 主角是否在纠结？
    └─ 是 → ❌ 警告："主角太纠结，让他直接干"

第4-5章检查：
├─ 是否有行动动机？
│   └─ 否 → ⚠️ 建议："加入具体行动目标"
│
└─ 深层动机是否隐藏？
    └─ 否 → ⚠️ 建议："不要直接说深层动机，用行动暗示"

第6章后检查：
├─ 深层动机是否逐渐浮现？
│   └─ 否 → 可选（有些作品不需要）
│
└─ 是否用行动暗示而非直接说？
    └─ 否 → ⚠️ 建议："用行动暗示，不要让主角自己说"
```

## J.5 行动派主角塑造

```
[行动派主角塑造]

原则：想干就干，干了再说

错误示范（想太多）：
├─ 李四站在门口，犹豫要不要进去。
├─ 如果进去，可能会被发现；如果不进去，又错过机会。
├─ 他想起师父说过的话，又想起自己的承诺...
├─ 纠结了半天，他终于下定决心，推开了门。
└─ 问题：想太多，节奏拖沓

正确示范（行动派）：
├─ 李四一脚踹开门。
├─ 被发现了再说，先干了。
└─ 正确：直接行动，不纠结

行动派特征：
├─ 不犹豫：想干就干
├─ 不解释：干了再说
├─ 不后悔：错了就认
├─ 不纠结：选择就执行
└─ 不废话：用行动说话
```
