# 短篇写作技能（V2.0 — 长篇规则子集提取）

> 本技能为长篇流水线（03/04/05）的短篇适配版，适用于单篇1.5万-3万字的短篇/中短篇创作。
> 核心理念：保留长篇流水线中影响产出质量的核心规则，删除跨章管理机制。

## 适用范围
- 单篇字数：8K-15K字（中短篇）/ 15K-30K字（中篇，可分段执行）
- 不需要跨章衔接、滚动建纲、伏笔追踪表
- 一次完成全部创作，不分章循环

## 与长篇流水线的区别

| 维度 | 长篇流水线 | 短篇流水线 |
|------|-----------|-----------|
| 字数 | 100万+字 | 8K-30K字 |
| 步骤 | 7步（含项目初始化） | 3步 |
| 细纲 | 全书完整细纲（滚动建纲） | 单篇大纲（一次性） |
| 扩写+精修 | 分离(03→04) | 合并为单步 |
| 去AI味 | 11 Gate+三遍法 | 11 Gate+两遍法 |
| 自检项数 | 40项(P0+P1+P2) | 10项(仅P0) |
| 收敛轮数 | 3轮 | 2轮 |
| 衔接包 | 需要 | 不需要 |
| 滚动建纲 | 需要 | 不需要 |
| 跨章去重 | 需要 | 不需要 |
| 伏笔追踪 | 需要 | 不需要（短篇伏笔少，人工把控） |
| 频率 | 每章循环 | 单次完成 |

## 三步流程

### Step 1：定位（5分钟）

**核心问题**：这篇短篇的"一句话卖点"是什么？

```
【短篇定位模板】

1. 类型标签：悬疑/言情/都市/奇幻/恐怖/现实
2. 一句话梗概：[主角]在[情境]下，做了[某事]，导致[结果]
3. 情感目标：让读者感到[爽/虐/怕/甜/思]
4. 对标风格：[如：类似知乎盐言《XXX》的叙事风格]
5. 目标字数：8K-15K字（中短篇）/ 15K-30K字（中篇）

6. 核心设定（如有特殊世界观）：
   ├─ 规则1：
   ├─ 规则2：
   └─ 规则3：

7. 角色表（1主角+2-3重要配角）：
   ├─ 主角：[姓名][身份][性格核心]
   ├─ 配角1：[姓名][功能]
   └─ 配角2：[姓名][功能]
```

**定位补充（V2.0新增）**：

- **核心情绪定位**：这篇短篇要让读者读完后的核心感受是什么？
- **爆点定位**：短篇只需要1-2个爆点，确定位置
- **字数预算分配**：开头(15%)→发展(50%)→高潮(25%)→收尾(10%)

**标点规范**：与长篇一致（省略号≤5/破折号≤4/感叹号≤15）

---

### Step 2：写作（核心执行）

合并03扩写+04精修为单步执行，保留以下长篇核心规则。

**执行原则**：
1. 单次完成：一次性生成完整短篇正文
2. 自带精修：写作时直接应用长篇流水线中04精修师的技法（潜台词/留白/金句）
3. 字数控制：8K-30K字，单篇自包含
4. 起承转合：开头钩子 → 冲突升级 → 高潮反转 → 余韵收尾

**结构模板**：

```
├─ 开头（15%）：钩子+角色+场景
│   ├─ 第1句：必须抓住读者
│   ├─ 前200字：建立冲突或悬念
│   └─ 前500字：角色出现+核心问题提出
│
├─ 发展（50%）：冲突升级+信息释放
│   ├─ 3-5个情节推进点
│   ├─ 过程中逐步释放背景信息
│   ├─ 穿插小高潮/小反转
│   └─ 配角互动增加层次
│
├─ 高潮（25%）：核心冲突解决
│   ├─ 最大的反转/揭示/对抗
│   ├─ 所有伏笔回收
│   └─ 情绪达到顶点
│
└─ 收尾（10%）：余韵+收尾
    ├─ 情感释放或留白
    ├─ 呼应开头
    └─ 留给读者回味空间
```

**短篇字数控制**：
- 按结构分段生成，每段完成后检查字数
- 若总字数<8K：在"发展"段补充冲突层次
- 若>15K：压缩"发展"段的过渡场景
- 中篇（15K-30K）可分段执行，但不分章循环

#### 2.1 反注水最高铁律（6条全保留）
1. 禁止循环式排查（多个相似物品挨个描写超过1次）
2. 禁止无意义物理触碰反馈（普通物品的触感/磨损声/笔帽咔哒声）
3. 禁止滥用感官比喻（日常场景用"像从地底深处"等）
4. 禁止心理活动套娃（"他原本预期...但他发现..."）
5. 禁止潜台词解释（"他发现/这说明/这意味着"）
6. 禁止弱智明喻（用"像/仿佛"强行渲染氛围，全篇超过2处）

#### 2.2 否定排比三连零容忍
- "没有…没有…没有…" / "不是…不是…不是…" → 立即删除
- 否定排比+升华句（"而是…/只有…/那是一种…"）→ 立即删除整段
- 全篇容忍度：0次（对话中可保留）

#### 2.3 质感注入五大技法（全保留）
1. 动作链写生理状态：3-4个连续动作传达角色状态（非微动作白噪音）
2. 具体名词替代形容词："桌上放着半壶凉透的茶"而非"桌上放着茶具"
3. 设定藏在生活里：世界观通过日常场景释放，非说明文
4. 感官精准投放：日常极简白描 / 关键异常点多维叠加（二分法）
5. 对话人味：口癖/跑题/言不由衷/信息差

#### 2.4 冰山原则五大铁律（全保留）
1. 只写水面上的1/8，让读者感受到水面下的7/8
2. 角色的过去通过"碎片+矛盾"释放，非履历表
3. 情绪不直说，用物理锚点承载（"他攥着那张纸，指节发白"）
4. 关键信息通过"角色反应"暗示，非叙述者解释
5. 留白比填满更有力量——读者脑补的永远比写出来的好

#### 2.5 "不是A是B"句式限流
- 全篇≤3次（短篇比长篇章限度略宽）
- 超出 = 改用直接陈述/动作描写

#### 2.6 常识性错误零容忍
- 物理/材料/生理/空间/时间/因果常识检查
- 超自然行为必须有设定依据

**短篇精修规则子集说明**：
- 必用技法：感官入口重定向、潜台词对话法、反标签行为法（这3项是短篇质感的核心）
- 选技法：脏感注入（仅现实题材）、内心矛盾外化法（仅主角视角）、感官锚点深度穿插（仅关键场景）
- 跳过技法：叙述厚度保障中的"对话与叙述黄金比例"（短篇对话比例可灵活）、模块R情感收束（短篇高潮集中，不需要长线收束）

---

### Step 3：去AI味（合并05核心Gate）

保留以下核心Gate，三遍法简化为两遍：

#### Pass 1：去泛化（必做）
- Gate 1: 否定排比三连（0容忍）
- Gate 2: "不是A是B"句式（≤3次）
- Gate 3: 心理直给（情绪词扫描）
- Gate 4: 万能前缀（"他心里想/暗自思忖/不禁感到"）
- Gate 5: 解释腔（"这说明/显然/意味着"）
- Gate 6: 常识性错误

#### Pass 2：回自然感（必做）
- Gate 7: 意象重复（同一意象≥3次→换）
- Gate 8: 比喻质量门（三重门+总密度≤6处/全篇）
- Gate 9: 句式均匀度（检查连续段落是否句式雷同）
- Gate 10: 信息含金量（连续>200字信息零推进→脱水）
- Gate 11: 画面感测试（闭眼能"看见"关键场景）

> 短篇字数短，建议全文过11 Gate，而非抽样检测。

#### 收敛终止协议
- 2轮上限（短篇比长篇少1轮）
- 第1轮消除70%AI味（高收益）
- 第2轮消除15%AI味（中收益）
- 剩余15%：LLM固有特性，标记[需复核]移交用户
- [需复核]项给出3个选项：A.接受现状 B.人工改写（给方向） C.放弃修改

## 短篇自检清单（10项，P0级精简版）
1. 字数是否在目标范围内？
2. 核心情绪是否到达？（读完有身体反应）
3. 反注水6条铁律逐条检查
4. 否定排比三连0次
5. "不是A是B"≤3次
6. 常识性错误0个
7. 画面感测试通过
8. 对话有人味（口癖/跑题/信息差）
9. 比喻通过质量门且≤6处
10. 章末收尾干净（非排比/非抒情/非省略号）

## 短篇转长篇

短篇成功（读者反馈好/数据表现佳）后，可扩展为长篇：

1. 保留原短篇的核心设定和主角
2. 扩展世界观（增加规则复杂度）
3. 设计更长的冲突弧线
4. 进入长篇流水线从步骤2（大纲）开始

## 流水线契约

- 短篇写作消耗03+04合并步骤（精修在写作时直接完成）

- 短篇产出物存入项目目录的 `短篇/` 子目录（如适用）
- 不占用长篇流水线的衔接包链/伏笔追踪等长线机制
- V2.0核心变更：从长篇流水线提取核心规则子集，删除跨章管理机制，保留影响产出质量的核心规则
