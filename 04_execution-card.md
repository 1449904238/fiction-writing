# 04 精修执行卡片

> **V5.3 执行卡片 | 本卡片≤2000字，完整规则见 04_小说正文精修师.md**

## 3大核心原则（精修第一优先级）

### 原则1：潜台词拦截器（Show Don't Tell — 严禁解释）

禁止输出角色的"心理结论"或"状态定义"，永远不要替读者总结。

```
禁止句式："他发现…""这说明…""这意味着…""他在等…""他心里明白…"
禁止"心理实况转播"：禁止描写"在心里算了一遍""想了三遍"

【叙述语AI味预警词库】
A级（硬禁止，零容忍）：意味着/是因为/这说明/他发现/他心里明白/他在等/
    他意识到/显然/可见/看出规律/异乎寻常 → 出现即删除替换为客观物证
B级（软限制，每章≤2次）：觉得/知道/看来
    事实陈述豁免（如"他知道这条路通往北门"非心理直给）
C级（对话豁免）：所有A/B级词在对话（引号内）不计入限制

【推理留白】角色发现秘密→只写"客观物证"，不准写结论。
❌ "这个温度是给展品定的，展品怕热。"
✅ "温控表指针死死钉在十八度。白术吐出一口白气。"
```

### 原则2：比喻质量门（质量检验而非绝对封杀）

```
逐个评估全文"像/仿佛/好似/犹如/如同"句：
├─ 高质量（保留）：新鲜且精准/有叙事功能/角色口吻
├─ 低质量（替换）：陈词滥调/纯氛围渲染/解释性比喻
└─ 低质量比喻替换为：
   ├─ 物理痕迹法：直接描写A本身的材质/气味/损耗/数据
   └─ 生理反馈法：通过主角身体/感官直接承受该状态
例外：角色对话中的比喻可保留。
```

### 原则3：情绪词三级分级制（V5.3.1 — 与03a铁律4对齐）

```
| 级别 | 场景类型 | 情绪词配额 | 技法要求 |
|------|---------|-----------|---------|
| L1低强度 | transitional/accent | 0个 | 纯Show（动作/细节/环境） |
| L2中强度 | important | ≤1个/场景 | Show为主+1个情绪词锚点 |
| L3高强度 | core | ≤2个/千字 | 允许情绪直给+Show叠加 |

每章情绪词总量≤5个。违反配额→重写。

【04仲裁规则】04有权将L2降级为L1（删情绪词加Show），无权将L1升级为L3。
当03a产出L3场景但无情绪词时，04可补充1-2个情绪词。
仲裁方向：只能向更克制方向调整（L3→L2→L1），不得反向放宽。
```

## 质感深化三步法

```
Step 1 诊断：扫描03初稿质感空缺
├─ S/A级场景被写成快进（欠账）=补质感
├─ C级场景被写成特写（注水）=压缩
├─ 核心场景只有1个细节但应3-5感官叠加=升级
└─ 对话有人味但不够锋利/动作链太工整=个性化

Step 2 注入：在不改变情节的前提下深化质感
├─ 核心场景升级："1个精准细节"→"3-5感官叠加+1个错位感官"
├─ 对话风格化：给角色增加独有说话节奏（非口癖）
├─ 动作个性化：标准动作链→这个角色独有的动作习惯
└─ 视角锚定：具体名词带角色职业/性格视角

Step 3 打磨：每章至少1个"读完忘不掉"的画面
├─ 记忆点应为"物理画面"（动作/物件/感官瞬间），非情节事件
└─ 方法：找情感最强烈瞬间→用1个极简精准物理画面替代情绪描写
```

## 5大技法简表

| # | 技法 | 核心 | 示例 |
|---|------|------|------|
| 1 | 感官入口重定向 | 描写前先决定读者从哪个感官进入；日常极简/核心多维二分法 | ❌"他看到展柜" ✅"霉味先钻进鼻腔" |
| 2 | 脏感注入 | 世界不是无菌室，每章≥3处脏感细节 | 灰/锈/潮湿/油腻/磨损/气味 |
| 3 | 潜台词对话法 | 人类说话70%意思不在字面上；试探→回避→松动 | B不直接回答，做无关小动作，答非所问 |
| 4 | 内心矛盾外化 | 通过"言行不一"呈现冲突，全程不解释心理 | 推开对方→愣两秒→做不相干事→说五个字以内 |
| 5 | 感官锚点穿插 | 在情绪需要呼吸处自然放置，不跟着数字走 | 沉默时用环境声填补/暗涌时用小动作暴露 |

## 6项精修检测清单

> 从完整~25项精简。6项为精修底线保障。

```
□ 对话零补丁：所有"重话"（讽刺/揭穿/告白/决裂/金句）后是否紧跟心理
  描写或解释？是=删除解释，换行后用动作/环境/沉默替代。每处"重话"后
  必须留白至少1个自然段。

□ 解释性后摇：扫描全文"因此/所以/因为/其实/这意味着/换句话说/说到底
  /归根结底"→出现在叙述中一律删除。核心：删掉后读者仍能理解=多余。

□ "不是A是B"限流：扫描"不是…是…/看似…实则…/表面…实际…"合计≤2次/章
  （引用 check-ai-patterns.js 结果）。对话中不计入。

□ 感官入口二分法：日常场景1-2感官极简白描；核心场景≥2种感官多维叠加。
  核心场景视觉单一开场（"他看到…"）=标记并改写。

□ 脏感覆盖≥3处：每章至少3处脏感细节（灰尘/锈迹/油腻/潮湿/磨损/汗味）。
  关键场景（对峙/战斗/探索）必须≥1处。

□ 对话信息不对称：连续3轮以上"问答顺畅"=必须打断；至少1个角色在隐瞒/
  撒谎/言不由衷；每章≥2处信息不对称段落；≥1处"欲言又止"。
```
