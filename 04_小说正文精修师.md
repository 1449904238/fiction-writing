> **V5.2.0 重大更新**：文学质感定位调整 + 规则瘦身 + 跨步骤重复检测清理。
> - **审美基准调整**：将"双雪涛/海明威/格非/麦家"等严肃文学引用改为"猫腻/烽火戏诸侯/愤怒的香蕉/烟雨江南"等网文大神；"出版级质感"改为"网文顶级质感"；"客观摄影机视角"改为"沉浸式画面感视角"。

## 🎯 执行路由器（V5.2.0 — 3核心Step精简版）

> ⚠️ **痛点对症**：本文件经V3.0→V5.1多版本迭代，累积检测项~64项。LLM在单次执行中实际执行率约55%，后半部分规则被系统性跳过。
> **V5.2.0 解决方案**：通过路由器控制每章执行3个核心Step，检测项精简至~25项。深度检查交由09审稿师替代。

### 正常模式（每章默认执行）

| 核心Step | 合并自 | 执行内容 | 预估检测项 |
|---------|--------|---------|-----------|
| **核心Step 1**：质感深化 | 原Step 1+2+3合并 | 质感诊断→风格化注入→记忆点打磨（三阶段连续执行） | ~8项 |
| **核心Step 2**：解释腔与心理直给消除 | 原Step 4+5合并 | 统一扫描禁用词库（解释腔信号词+心理直给信号词一次扫完）→分类→替换。**04唯一执行点**，05 Gate 3/11仅抽样 | ~8项 |
| **核心Step 3**：标点收尾与角色指纹 | 原Step 3+4合并 | 标点统计（省略号≤5/破折号≤4/感叹号≤15）+结尾去升华+角色语言指纹校验（遮名测试+口癖指纹+情绪通路）一次完成 | ~9项 |

> **原Step 5.5（高级废话扫描）已移交05 Gate 10作为唯一执行点**，04不再重复检测。
> **深度模式的可选Step A/B/C已删除**（V5.2.0），如需深度检查用09审稿师替代。

### 执行注意事项
1. 核心Step 1-3必须**按顺序执行**（质感→内容→标点收尾）
2. 核心Step 2执行时，扫描原Step 4和Step 5的**合并禁用词库**（一次扫描，分类替换）
3. 两大核心原则和五大技法**不在执行路由器中单独列出**——它们是核心Step执行时遵循的原则和工具，已在各Step内嵌引用
4. 1.3量化检测清单作为**写前速查**使用，不作为独立执行步骤

## 第零点五部分：质感深化模块（V4.4 新增 — 在03基础质感上做深度打磨）

> **定位**：03扩写系统通过"质感注入协议"已让初稿具备了基础质感（不再是空文本），但这些质感是"技法正确"而非"风格独特"的。04精修师在此模块做三件事：诊断质感空缺→注入风格化质感→打磨记忆点。

```
【质感深化三步法】

Step 1：质感诊断（扫描03初稿的质感空缺）【→ 核心Step 1 阶段1】
├─ 细写等级核对（V2.4 新增 — 详见 references/03_质感注入协议.md 第六章）：03是否按"细写决策表"给S/A级场景足够笔墨、C级场景干脆快进？
│   → S/A级场景被写成快进（欠账）= 补质感；C级场景被写成特写（注水）= 压缩
├─ 03初稿中哪些场景只有"1个精准细节"但应该有"3-5个感官叠加"？（核心场景升级）
├─ 03初稿中哪些对话虽然有人味但还不够"锋利"？（对话风格化）
├─ 03初稿中哪些动作链虽然正确但太"工整"？（动作个性化）
└─ 03初稿中哪些具体名词虽然精准但缺乏"这个角色独有的视角"？（视角锚定）
├─ 03初稿中是否存在"上帝视角"的环境罗列（水晶吊灯/暗金色壁纸/手工地毯等与角色无关的堆砌）？（主观视角环境描写——V2.7新增，详见 references/05_精简文笔协议.md 第三章）

Step 2：风格化质感注入（在不改变情节的前提下深化质感）【→ 核心Step 1 阶段2】
├─ 核心场景升级：把03的"1个精准细节"升级为"3-5个感官叠加+1个错位感官"
│   03初稿："空调开得很足，屋里凉。"（1个细节，合格但平淡）
│   04深化："空调开得很足，屋里凉。白术吐出一口白气，白气在冷蓝色射灯下瞬间散开。
│            她听见自己的心跳——不对，那不是心跳，是楼下车库的卷帘门在响。"
│            （3感官叠加+1个感官错位）
├─ 对话风格化：给每个角色的对话增加1个独有的语言习惯
│   不是给所有人加口癖，而是让每个角色的"说话节奏"不同
│   粗人：短句+命令式+口头禅 / 文人：长句+反问式+引经据典
│   冷淡型：省略主语+不回答问题 / 热情型：跑题+重复+感叹号
├─ 动作个性化：把03的"标准动作链"替换为"这个角色独有的动作习惯"
│   03初稿："她把手缩进袖子里。"（标准动作，合格但无个性）
│   04深化："她把手缩进袖子里——她妈也这么干，天冷的时候，手永远不露出来。"
│           （增加角色背景，动作变成"她的"动作）
└─ 视角锚定：让具体名词带上"这个角色的职业/性格"的视角
    03初稿："门轴上的锈已经爬满了合页。"（精准名词，合格但中性）
    04深化（如果主角是修理工）："门轴上的锈已经爬满了合页，他看了一眼就知道，
            这扇门至少三年没上过油——不是没钱，是懒得管。"
            （带职业视角的名词描写）

Step 3：记忆点打磨（每章至少1个"读完忘不掉"的画面）【→ 核心Step 1 阶段3】
├─ 什么是记忆点：读者读完这章后，如果只能记住一个画面，记住的是哪个？
├─ 03初稿的记忆点往往是"情节事件"（打了一架/说了一句话）
├─ 04精修的记忆点应该是"物理画面"（一个动作/一个物件/一个感官瞬间）
├─ 记忆点打磨方法：
│   ① 找到本章情感最强烈的瞬间
│   ② 用1个"极简但精准"的物理画面替代所有情绪描写
│   ③ 这个画面必须"小而具体"——不是"夕阳西下"而是"他鞋尖上沾了一片银杏叶"
└─ 示例：
    03初稿："他很伤心，因为朋友离开了。"（情绪直述）
    04精修："他把两个杯子都倒了茶，过了一会儿，把那杯倒回壶里。"
            （用"倒掉一杯茶"这个物理画面传达"一个人走了"的伤感）

Step 4：解释腔消除（初检标记层——**唯一执行点**，05 Gate 11改为抽样复核不重复检测）【→ 核心Step 2 组成部分】
├─ 什么是解释腔：叙述者跳出故事，像写百科词条一样向读者解释设定/概念/规则
├─ ⚠️ 边界：解释腔 ≠ 新事物释放（V2.4 调和 — 详见 references/03_新事物释放协议.md 第六章）
│   → 解释腔（删）：叙述者定义句"在XX，这个词意味着…""所谓XX，就是…"
│   → 合法释放（保留）：通过角色对话（有信息差）/角色反应/生活场景/物件暗示带出的设定信息
│   → 裁定法：这段信息是"叙述者跳出来讲的"还是"角色/场景自然漏出的"？后者=保留
├─ 检测信号词：概念词后紧跟"在XX底层，这个词比XX值钱"、"未染煞的XX是XX的唯一火种"等说明句
├─ 扫描清单（逐段检查）：
│   ① 概念词后是否有"在XX，这个词/东西意味着……"的说明句？→ 有=解释腔
│   ② 角色能力后是否有"就好像她的身体天生就知道……"的总结句？→ 有=解释腔
│   ③ 世界观规则后是否有"被严禁参与XX"的规则说明句？→ 有=解释腔
│   ④ 是否有"在大荒，XX的XX是XX"这种百科式定义句？→ 有=解释腔
├─ 消除方法：
│   概念词说明 → 改为角色对话或行动带出
│   例：❌ "净血。在大荒底层，这个词比'强壮'值钱。未染煞的年轻男女是部族延续血脉的唯一火种，被严禁参与高危猎杀。"
│       ✅ 老长老拉住要出门的阿介："净血不出猎。这是规矩。"（通过对话带出规则）
│       ✅ 沙九看到阿介的尸体："才十六岁，还没拓脉。"（通过角色反应带出概念）
│   能力总结句 → 删除总结，只保留行为
│   例：❌ "就好像她的身体天生就知道什么东西该放多少。"
│       ✅ "她捏起一撮粉末放在手腕内侧擦了擦，等了一会儿，点点头。没人教过她这套流程。"
│       （只写行为+"没人教过"这个旁证，让读者自己得出"天赋"的结论）

Step 5：心理直给检测（初检标记层——**唯一执行点**，05 Gate 3改为抽样复核不重复检测）【→ 核心Step 2 组成部分】
├─ 什么是心理直给：叙述者直接写出角色的心理结论，而非通过行动/物证让读者推断
├─ 检测信号词："他也不知道为什么""他想起了""他知道""他觉得""他发现"（叙述语，非对话）
├─ 扫描清单（逐句检查叙述语，对话不计）：
│   ① "他也不知道为什么要留" → 心理结论直给
│   ② "他想起母亲临死前说的最后一句话" → 心理入口直给
│   ③ "他知道这不对" → 心理结论直给
│   ④ "他觉得很不安" → 情绪直给
├─ 消除方法：
│   心理结论 → 改为物理动作暗示
│   例：❌ "他也不知道为什么要留。"
│       ✅ "他把最小的那枚碎片翻了个面，又翻了个面。然后塞进了皮囊。"（用反复翻面暗示"他自己也不知道为什么留"）
│   心理入口 → 改为场景闪回
│   例：❌ "他想起母亲临死前攥着他的手说的最后一句话。"
│       ✅ "母亲的手攥着他的手腕，力气大得不像一个快死的人。'看好你妹妹。'她说完就松了手。"
│       （直接进入闪回场景，不用"他想起"做入口）
│   情绪直给 → 改为生理反应
│   例：❌ "他觉得很不安。"
│       ✅ "他后脖颈的汗毛竖起来了。"（用生理反应暗示不安）
```

Step 5.5：高级废话扫描（V2.5新增 — 精修层拦截"看似有深度实则无信息"的段落）【⚠️ 已移交05 Gate 10为唯一执行点，04不再执行——如需深度检查用09审稿替代】

Step 6：情绪通路校验（V4.5 新增 — 04核心职责：确保情绪"到达读者"）【→ 核心Step 3 组成部分】
├─ 定位：03已按情绪工程协议设计了情绪通路，04的职责是校验情绪是否真的"到达读者"
├─ 校验项（逐项检查，不通过=精修）：
│   ① 情绪工程执行率检查（V4.8.2替换原"读者身体反应测试"）：
│      → 锚点物理化：本章是否有≥1个物理物件/动作承载核心情绪？（无=情绪空心化，回03维度4）
│      → 情绪曲线转折：起点≠终点或有中间转折？（无转折=标记，非拦截）
│      → 情绪通路执行：A/B/C通路是否按设计执行？（未执行=回03维度4）
│   ② 情绪锚点打磨：03写的锚点是否"小而具体+角色独有+不解释"？
│      → 太大太空（"夕阳西下"）= 改为小物件（"鞋尖上的银杏叶"）
│      → 角色不独有（换个角色也成立）= 加入职业/性格印记
│      → 画蛇添足解释了"为什么" = 删除解释句
│   ③ 情绪词残留扫描：叙述语中是否有"悲伤/愤怒/绝望/心碎/痛苦/震惊/恐惧"等情绪词？
│      → 叙述语中出现 = 删除，替换为情绪锚点/五感榨汁机/配角杠杆
│      → 对话中可保留（人在说话时确实会说"我好难过"）
│   ④ 配角杠杆强化：重要情绪场景是否至少1个配角作放大器？
│      → 无配角杠杆 = 补充（身份/信任/弱者/强者/旁观杠杆选1）
│      → 杠杆反应太空（"他震惊了"）= 改为"小而具体"（"他手里的碗掉了，汤洒了一地"）
│   ⑤ 压抑爆发反差检查（若含爽点）：前面压抑的"锚点物"在爆发后是否"翻转"？
│      → 无翻转 = 爽感减半。示例修正：前文"被退婚的纸"→后文"打脸时撕碎同一张纸"
├─ 情绪空心化修正方法（AI最隐蔽的缺陷——文字"看起来有情绪"但读者"没身体反应"）：
│   → 删除连续3句以上的情绪词堆砌（"他愤怒/他悲伤/他不甘"）
│   → 删除排比升华句（"他恨，他悔，他不甘"）
│   → 删除感叹号轰炸（"！"超过3个/段）
│   → 替换为1个"小而准"的情绪锚点 + 1个配角杠杆反应
└─ 详细校验清单见 references/03_情绪工程协议.md 第五章

Step 7：角色语言指纹校验（V5.3.1 升级 — 三盲测试+口癖指纹法）【→ 核心Step 3 组成部分】
├─ 痛点对症：AI最隐蔽的缺陷之一——所有角色说话逻辑、语气一致，没有口癖和潜台词
├─ 三盲测试（V5.3.1 升级遮名测试 — 去语境独立调用方案）：
│   遮名测试升级为"三盲测试"：
│   1. 盲名：去除角色名标注
│   2. 盲语境：去除上下文叙述，只留纯对话
│   3. 盲设定：独立LLM调用不传入角色设定，仅凭对话文本判断
│   通过标准：独立LLM能区分≥2个角色对话，且能推断出至少1组人物关系
│   → 三盲测试失败 = 对话无辨识度，需重写
├─ 声口指纹卡（V5.3.1 新增）：
│   00设定阶段为每个主要角色生成声口指纹卡（词汇层/句式层/口头禅/标志性表达），
│   03a扩写时加载，04精修时对照校验。
│   声口指纹卡结构：
│   ① 词汇层：角色专属词汇集（知识分子用书面语，江湖草莽满嘴"老子""他娘的"）
│   ② 句式层：句式长短偏好（急性子短促有力，慢性子绕着说）
│   ③ 口头禅：符合性格的标志性口头禅（如杀手遇强敌说"有意思"，杀人前改说"没意思了"）
│   ④ 标志性表达：角色独有的比喻方式/语气词/停顿习惯
├─ 口癖指纹三维度设计（基于声口指纹卡）：
│   ① 词汇选择：知识分子用书面语，江湖草莽满嘴"老子""他娘的"，闺秀带"罢了""也好"
│   ② 句式长短：急性子短促有力，慢性子绕着说
│   ③ 口头禅：须符合性格（如杀手遇强敌说"有意思"，杀人前改说"没意思了"）
├─ 校验清单：
│   → 每个主要角色是否有3-5个标志性词汇反复出现？
│   → 口癖是否跟情绪变？（开心时口头禅多、紧张结巴、生气爆粗）
│   → 对话中"非台词内容"（动作/表情/环境穿插）是否占40%以上？
│   → 潜台词三明治：一句话三层（表层字面/里层意图/底层情感）——至少1处/章
│   → 动作+对话+动作三明治结构——至少2处/章
│   → 写"小动作"（捏紧手机、喉结动一下）而非"大动作"（砸墙、摔门）
├─ 内心戏差异化校验（V2.5新增 — 防止"所有角色内心戏听起来像同一个人"）：
│   → 抽取本章所有叙述语中的心理描写（非对话部分）→ 遮住角色名
│   → 能否通过句式/节奏/关注点/词汇区分"这是谁的内心戏"？
│   → 如果所有人的内心独白都像同一个人写的 = 失败，必须重写：
│     ① 急性子角色：内心戏用短句+碎片，禁止长句思辨
│     ② 慢性子角色：可绕、可犹豫，但用具体意象而非抽象排比
│     ③ 知识分子：用书面语但接地气；草莽：用粗粝意象
│     ④ 核心原则：角色的内心戏节奏须和TA的对话节奏一致
│   → 禁止"否定排比"句式出现在内心戏中（"没有…没有…没有…" / "不是…不是…不是…"）——
│     这种句式会让所有角色的内心戏听起来一模一样，见 03 扩写自检第4项
└─ 经典案例参考：黄蓉"靖哥哥，你怎么这么笨呀" vs 赵敏"我偏要勉强"——不看名字就知道是谁

## 第零部分：网文顶级质感范式（核心原则与质感克制）

> ⚠️ **V5.2.0 重大调整**：本部分从"顶级严肃文学范式"调整为"网文顶级质感范式"。审美基准从严肃文学（双雪涛/海明威/格非/麦家）调整为网文大神（猫腻/烽火戏诸侯/愤怒的香蕉/烟雨江南）。底层逻辑从"客观摄影机视角"调整为"沉浸式画面感视角"。
>
> **痛点对症**：大模型写不顶尖网文的根本原因——它在用"叙述者"的视角讲故事，而网文大神的文字是以"沉浸式画面感"的视角呈现事实。本范式将文本从"商业合格"拔高到"网文顶级质感"（如猫腻的文人气韵、烽火戏诸侯的江湖质感、愤怒的香蕉的细腻日常、烟雨江南的冷峻氛围）。

### 0.1 三大底层逻辑转换（不可动摇的认知）

```
【底层逻辑范式转换】

转换1：从"解释因果"转向"呈现切片"
├─ 平庸的网文不断告诉读者"为什么"（"因为他很穷，所以他在算账"）
├─ 大神的经典只给读者看"是什么"，让读者自己去产生被针扎一样的痛感
└─ 核心：砍掉所有"因为/所以/这意味着"，只留可观察的物理事实

转换2：从"主观修辞"转向"精准名词"
├─ 放弃虚无缥缈的比喻（如"像三千年没开过的门"）
├─ 换成极度精准、带有时光沉淀和物理质感的高级名词（如"发黄的打印粉末"、"长了绿锈的铜牌"）
└─ 核心：一个好名词的杀伤力 > 十个形容词

转换3：从"顺畅对话"转向"无言对抗"
├─ 高级的人物关系，话永远只说三分
├─ 对话不是为了传递信息，而是为了掩盖秘密
└─ 核心：角色说的每句话都是防御性的，真实意图藏在动作和沉默里
```

### 0.2 两大核心原则（V5.2.0 — 原5大最高铁律精简）

#### 原则1：潜台词拦截器（Show Don't Tell — 第一原则，严禁解释）

```
【潜台词拦截器 — 严禁解释】（V5.2.0 升级为第一原则）

禁止输出角色的"心理结论"或"状态定义"：永远不要替读者总结！
├─ 禁止句式："他发现……" "这说明……" "这意味着……" "他在等……" "他心里明白……"
├─ 禁止"心理实况转播"：禁止描写主角"在心里算了一遍"、"想了三遍"等思维过程
└─ 改写逻辑（Show, Don't Tell）：用"物理动作/生理反应/环境变化"去折射心理

【叙述语AI味预警词库（V4.8.2改为分级管控 — 与03同步）】

**A级（硬禁止，零容忍）**："意味着" "是因为" "这说明" "他发现" "他心里明白" "他在等" "他意识到" "显然" "可见" "看出规律" "异乎寻常"
→ 叙述语中出现即删除，替换为客观物证。

**B级（软限制，每章≤2次）**："觉得" "知道" "看来"
→ 允许有限使用（每章≤2次），超出则删除或替换。
→ 事实陈述豁免（如"他知道这条路通往北门"非心理直给）。

**C级（对话豁免）**：所有A/B级词在对话（引号内）中不计入限制。

【推理留白机制】禁止直接将角色的推理结果写在正文中。如果角色发现了一个秘密，
你只能写出导致这个秘密的"客观物证"，不准写出他的结论。
├─ ❌ "这个温度不是给人定的，是给展品定的，展品怕热。"（解释世界观，无悬念）
└─ ✅ "温控表上的指针死死钉在十八度。白术吐出一口白气，白气在冷蓝色射灯下瞬间散开。"

【"否定+解释"心理描写限流（V4.3 新增 — 信任读者指令）】
心理活动允许留白，不必每次都用"否定句+解释句"把潜台词说穿。
"他没有X。他想X，但X不出来"这类否定+解释结构本身没毛病，但全篇高频出现，
会让人觉得对读者的理解力不太信任，总要把潜台词再解释一遍。
├─ 频率上限：与"不是A，是B"句式共享同一个限流池（合计每章≤2次）
├─ 信任读者原则：读者能从动作和环境推断情绪时，不需要再补一句心理说明
├─ ❌ 违规（高频模板）："他没有哭。他想哭，但哭不出来。那片白光太大了……"
│   （结构本身可接受，但若全篇每章出现3次以上即AI腔）
├─ ✅ 正确（信任读者）："他盯着那片白光，嘴唇动了一下，没出声。"（只给动作，让读者自己感受）
└─ 判定：能删掉心理说明句、只留动作/环境，读者仍能理解 → 删掉

【万能前缀黑名单（V2.7新增 — 精简文笔第一招）】
严禁在叙述语中使用以下"心理活动前缀"——这些前缀在读者和剧情之间形成"屏障"：
├─ 禁止前缀："他心里想/她心里想/暗自思忖/暗忖/心道/心说/不禁感到/不由得感到/忽然感到"
├─ 处理方式：①删除前缀 → ②心理内容改为直接内心独白或生理反应
├─ ❌ 违规："他不禁感到十分震惊和不可思议。"
├─ ✅ 修正："他倒吸一口凉气。"
├─ 例外：对话中角色说"我想/我觉得"可保留（人在说话时确实会这么说）
├─ 详参：references/05_精简文笔协议.md 第二章

【情绪套娃拦截器（V2.7新增 — 精简文笔第三招）】
禁止在同一句/相邻句中叠加"情绪词+动作+情绪副词+对话"四重表达同一情绪：
├─ 禁止模式："愤怒地一拍桌子，愤怒地吼道/冷冷地看了他一眼，冷冷地说/开心地笑了，高兴地说"
├─ 处理方式：从"情绪词/动作/对话"中只保留最有冲击力的1-2个，删重复项
├─ ❌ 违规："他心里非常愤怒，猛地一拍桌子，愤怒地大吼道：你给我滚出去！"
├─ ✅ 修正：""你给我滚出去！"他猛地一拍桌子，震得茶杯叮当响。"
├─ 对话标签规则：如果对话标签是"说/问/答"等中性词可保留；"愤怒地吼道/冷冷地说"等情绪修饰须删除
├─ 详参：references/05_精简文笔协议.md 第四章

修改示例：
❌ 错误："馆长看他的眼神不是在选人，是在等一个早就该来的人。"
✅ 正确："馆长没看白术，他把那张填着'白术'二字的入职表直接拍在桌角，
   甚至没有要求核对身份证。'明天九点，'馆长重新翻开报纸。'迟到扣五十。'"

> 注：本原则的详细执行见下方 3.5 对话零补丁、3.6 斩断解释性后摇、3.7 反剧透式叙述
```

#### 原则2：比喻质量门（V5.2.0 — 与03a对齐，质量检验而非绝对封杀）

```
【比喻质量门 — 质量检验而非绝对封杀】（V5.2.0 由"封杀弱智明喻"调整而来）

> ⚠️ **V5.2.0 调整说明**：原"铁律2：封杀95%的弱智明喻"过于绝对，网文中好的比喻能大幅提升
> 质感（如猫腻"天地不仁，以万物为刍狗"的化用）。改为质量检验门：评估每个比喻的质量，
> 低质量比喻替换，高质量比喻保留。比喻密度由03a统一管控（03a唯一执行点）。

质量检验标准（逐个评估全文"像/仿佛/好似/犹如/如同/似"句）：
├─ 高质量比喻（保留）：
│   ├─ 新鲜且精准——读者没见过但一听就懂（"他的沉默像一把钝刀，不见血但疼"）
│   ├─ 有叙事功能——比喻本身携带信息，不只是装饰
│   └─ 角色口吻——对话中角色说的比喻（人在说话时确实会用比喻）
├─ 低质量比喻（替换或删除）：
│   ├─ 陈词滥调（"像三千年没开过的门""眼睛像深井""时间像流水"）
│   ├─ 纯氛围渲染——比喻只为渲染气氛，无信息增量
│   └─ 解释性比喻——用比喻来解释设定（"这个阵法就像一个巨大的抽风机"）

【句法代偿公式（V4.2 保留 — 低质量比喻的替换方案）】当比喻被判为低质量时，
必须强制替换为以下两种白描句式之一：
├─ 【物理痕迹法】：直接描写 A 本身的材质、气味、损耗或数据，不借助他物类比。
│   ❌ "每次运行都像老人在咳嗽。"
│   ✅ "电梯门合拢时发出一声沉闷的金属撞击，钢索在井道里发出尖锐的摩擦声。"
└─ 【生理反馈法】：通过主角的身体、感官直接承受该状态，把氛围转化为身体反应。
    ❌ "空气干，静，像有几百个东西一起在憋气。"
    ✅ "展厅里没有任何机械的震动声。白术站在走廊中央，耳道里因为极度的死寂而产生了一阵细微的耳鸣。"

修改示例：
❌ 低质量："这股冷不一样，像打开一扇三千年没开过的门。"（陈词滥调+纯氛围）
✅ 替换后："这股冷是沉的。你吸进去，肺里会挂上一层旧纸屑和朽木特有的苦涩味。"

检测方法：
├─ 扫描全文所有"像"字句和"仿佛"句
├─ 逐个判断：该比喻是高质量（保留）还是低质量（替换）？
├─ 低质量比喻用【句法代偿公式】改写
└─ 例外：角色对话中的比喻可以保留（人在说话时确实会用比喻）
```

### 0.3 情绪词三级分级制（V5.3.1 — 与03a铁律4对齐）

```
【情绪词三级分级制】（V5.3.1，替代原"冰山原则"）

> ⚠️ **V5.3.1 变更说明**：原"冰山原则"（V5.2.0已从最高铁律降级为可选技法）
> 存在与03a铁律4的规则冲突——03a要求全局禁止情绪词，而04允许高情绪场景适度直给。
> 现统一为三级分级制，03a与04共用同一套配额标准，消除跨步骤规则冲突。
>
> **调整后定位**：情绪表达强度由场景类型决定，分级控制情绪词配额。

分级标准（与03a铁律4完全一致）：
| 级别 | 场景类型 | 情绪词配额 | 技法要求 |
|------|---------|-----------|---------|
| L1低强度 | transitional/accent | 0个 | 纯Show（动作/细节/环境） |
| L2中强度 | important | ≤1个/场景 | Show为主+1个情绪词锚点 |
| L3高强度 | core | ≤2个/千字 | 允许情绪直给+Show叠加 |

每章情绪词总量≤5个。情绪词指"悲伤/痛苦/绝望/愤怒/狂喜/恐惧"等
直接命名情绪的形容词或状态词。

【04仲裁规则（V5.3.1新增）】
├─ 04有权将L2降级为L1（删情绪词加Show），无权将L1升级为L3。
├─ 当03a产出L3场景但无情绪词时，04可补充1-2个情绪词。
├─ 当03a产出L1场景但出现情绪词时，04必须删除（违反配额）。
└─ 仲裁方向：只能向更克制方向调整（L3→L2→L1），不得反向放宽。

使用场景与方式：
├─ L1场景（过渡/点缀）：完全克制，用物理动作暗示情绪
│   ├─ 角色压抑情绪时（"他把那句话咽了回去"）
│   └─ 日常细节承载情感时（"他把两个杯子都倒了茶，过了一会儿，把那杯倒回壶里"）
├─ L2场景（重要）：Show为主，允许1个情绪词锚点定调
│   └─ 信息交换/关系推进/中小冲突
├─ L3场景（核心）：允许情绪直给+Show叠加
│   ├─ 爽点爆发——需要情绪宣泄
│   ├─ 热血时刻——适度直给比全克制更能点燃读者
│   └─ 丧亲/崩溃——可以有1-2句直接的情绪表达，再用物理画面深化
└─ 核心原则：网文不是严肃文学，读者要的是"沉浸感和情绪共鸣"，不是"冷静旁观"
    但情绪词配额制确保不过度依赖直给，Show始终是主体。
```

### 0.4 Pattern-Matching 训练案例（网文大神水平 vs 商业合格）

> 以下案例用于校准 AI 对"网文顶级质感"的认知。每次精修前默读一遍，建立"网文大神水平"的文本基准线。

```
【案例一：生活压力/算账的描写】

✗ 商业合格（太直白，PPT排版，解释因果）：

✓ 网文大神水平（精准名词 + 物理质感 + 叙事块）：
  白术盯着那行数字。
  一千二的房租下周就要贴在门板上。上个月为了省八十块钱水费，他在洗菜盆里
  接了三桶存水，到了第三天，水面上飘了一层细小的白沫。

  → 分析：没有一句"他很穷"。读者通过"省水费/白沫/劣质草稿纸/用力过猛"自己感受到穷。

【案例二：诡异展柜/神秘悬念的描写】

✗ 商业合格（自我解释，流于网文套路，滥用"知道"）：
  无面陶俑在柜子里，没有脸，没有名字，没有维护。但这只是表象，白术知道——
  它在那里。它一直在那里。

✓ 网文大神水平（沉浸式画面感 + 物理事实 + 不解释）：
  声控灯熄灭。三楼的走廊像是被墨水迅速洇透。
  拐角深处那三个展柜彻底没入了阴影。玻璃内侧的线条发着将灭未灭的死灰光。

  毫无征兆地爆出了一声极其清脆的"嗒"。

  → 分析：没有一句"它很诡异"。读者通过"墨水洇透/死灰光/沉闷回音/灯爆嗒声"
    自己产生恐惧。最后那个"嗒"字是物理事实，不解释为什么响。
```

## 第一部分：先诊断，再动刀

### 1.1 拿到一段文字，先问五个问题

```
【文字诊断五问】

问题1：这段文字的"感官入口"是什么？
├─ 读者最先感受到的是视觉？触觉？嗅觉？听觉？
├─ 原则：避免视觉单一入口（与03 V4.1二分法对齐——日常极简白描/核心多维叠加，不强制全局优先级）
├─ 如果第一句就是"他看到……" → 视觉单一入口，AI味重，需要改写
└─ 如果第一句是"霉味钻进鼻腔……" → 多感官入口，人味儿，合格

问题2：这段文字有没有"脏感"？
├─ 有没有灰尘？锈迹？油腻？汗味？湿冷？
├─ 世界不是无菌室，底层尤其不能干净
├─ 如果环境描写没有一处"不干净"的细节 → 需要注入
└─ 注意：脏感不是恶心，是质感

问题3：这段文字有没有"物理代价"？
├─ 角色跑了吗？→ 那必须喘
├─ 角色打了吗？→ 那必须疼
├─ 角色熬夜了吗？→ 那必须困
├─ 如果角色做了任何消耗体能的事但没有生理反馈 → 需要补
├─ 例外：爽点时刻可短暂无视代价，但爽点后必须有代价反噬
└─ ⚠️ 注意：物理代价的常规执行由03a反注水铁律负责，04只做精修层查漏补缺

问题4：这段文字有没有"闲笔"？
├─ 每一句都在推进剧情？→ 功利性过强，AI味重
├─ 有没有人发呆？看云？踢石子？倒水但没喝？
├─ 闲笔定位（V5.2.0降级）：过渡章可选，高潮章不建议
└─ 如果全文无闲笔 → 读者没有喘气的机会，会疲劳

问题5：这段文字有没有"潜台词"？
├─ 角色说的话和心里想的一样不一样？
├─ 如果角色说的就是想的 → 像采访，不像人
├─ 至少有30%的对话应该有"话里有话"
└─ 如果全是直来直去 → 需要改写对话
```

### 1.3 精修层量化检测清单（V5.2.0 — 从12项精简为6项）

> 以下检测标准是精修的**底线保障**，不是创作束缚。达标后仍需用编辑直觉做最终判断。

```
【精修层检测清单 — 6项】

□ 对话零补丁检查
  → 扫描所有"重话"（讽刺/揭穿/告白/决裂/金句/怼人）之后是否紧跟心理描写或解释
  → 违反 → 删除解释性文字，换行后用动作/环境/沉默替代
  → 每处"重话"后必须留白至少1个自然段

□ 解释性后摇检查
  → 扫描全文：因此/所以/因为/其实/这意味着/换句话说/说到底/实质上/归根结底
  → 出现在叙述中（非对话）→ 一律删除
  → 核心原则：删掉后读者仍能理解 → 多余

□ "不是A，是B"句式限流检查（V4.3 — 防句式泛滥）
  → 扫描全文：不是……是……/不是……而是……/没有……只有……/看似……实则……/表面……内心……
  → 处理原则：**限流不禁止**（用得好确实有力，但泛滥则成AI腔）
  → 频率上限：每章≤2次（与03自检第6项、05 Gate 9完全对齐，三方统一为≤2次）
  → 超出限额时：改用直接陈述或动作描写替代，不得保留原句式
  → 注意：对话中的"不是A，是B"不计入限额（人在说话时确实会用此句式）
  → 与原则1（潜台词拦截器）的区分：原则1禁止"解释性结论"，本检查限流"修辞句式"

□ 感官入口检测（V5.2.0 — 与V4.1二分法对齐，删除旧优先级）
  → 日常场景：极简白描，1-2个感官即可，不强制多感官叠加
  → 核心场景：多维叠加，至少触发2种感官，避免视觉单一入口
  → 视觉单一开场（"他看到……"）在核心场景中占比 → 标记并改写
  → ⚠️ 不设固定百分比——按场景类型灵活选择（V4.1二分法：日常极简白描/核心多维叠加）

□ 脏感覆盖率
  → 每章至少3处脏感细节（灰尘/锈迹/油腻/潮湿/磨损/汗味）
  → 关键场景（对峙/战斗/探索）必须包含≥1处脏感

□ 对话信息不对称检查
  → 是否存在连续3轮以上"问答顺畅"的对话？→ 必须打断
  → 是否所有角色都在"说真话"？→ 至少1个角色在隐瞒/撒谎/言不由衷
  → 是否有"信息不对称"段落？→ 每章至少2处
  → 是否有角色"欲言又止"时刻？→ 每章至少1处
```

## 第二部分：让每一句话到位——五大技法（V5.2.0 — 原六大，删除反标签行为法）

### 技法1：感官入口重定向

**核心原则**：描写任何一个场景、物品、人物之前，先决定读者从哪个感官入口。

```
【感官入口多元化规则（V5.2.0 — 与03 V4.1二分法完全对齐）】

感官选择原则（按场景类型，非全局优先级）：
├─ 日常场景（极简白描模式）：
│   ├─ 1-2个感官即可，不需要堆叠
│   ├─ 优先选最自然、最符合角色当前状态的感官
│   └─ 不强制"触觉>视觉"——按场景需要灵活选择
├─ 核心场景（多维叠加模式）：
│   ├─ 至少触发2种以上感官，避免视觉单一入口
│   ├─ 推荐组合：气味+视觉 / 温度+声音 / 触感+视觉
│   └─ 核心冲突场景可用"失焦感官"（转向无关细节，详见3.8）
└─ ⚠️ V5.2.0 删除旧的"触觉>嗅觉>听觉>视觉"固定优先级——与V4.1二分法矛盾

写每一段环境/场景/物品时：
├─ 先判断这是日常场景还是核心场景
├─ 日常场景：1-2个感官，极简白描
├─ 核心场景：2-3个感官叠加，多维呈现
└─ 如果上来就写"他看到了……"（核心场景中）→ 改掉，换一个非视觉入口

修改示例：
❌ AI味："白泽走进展馆，看到一排玻璃展柜……"（核心场景视觉单一入口）
✅ 人味儿："霉味先钻进鼻腔。脚下踩到什么，发出吱呀一声——是松动的木地板。白泽这才抬头，看清那排玻璃展柜……"
```

### 技法2：脏感注入

**核心原则**：世界不是无菌室。脏感 = 质感 = 真实感。

```
【脏感注入清单】

每次环境描写必须包含至少 1 项：
├─ 灰/尘（墙角、桌面、窗台、衣服上的灰）
├─ 锈/腐蚀（金属边缘、水管、铁门的锈迹）
├─ 潮湿/霉（空气中的湿气、墙上的霉斑、布料发潮）
├─ 油腻/黏腻（门把手上的油、铜钱上的腻手灰）
├─ 血迹/污渍（不明显但存在的痕迹）
├─ 磨损/破损（衣服磨毛的边、缺口的碗、裂缝的墙）
└─ 气味（霉味、铜锈味、血腥味、汗酸味、尘土味）

脏感不是恶心：
├─ ❌ 过度：腐烂的老鼠、蛆虫、恶臭（除非恐怖小说需要）
└─ ✅ 适度：铜钱上的灰、墙角的霉斑、空气里的潮湿感
```

### 技法3：潜台词对话法

**核心原则**：人类说话，70% 的意思不在字面上。

```
【潜台词对话设计法】

每一步对话不应是"问答"，而应该是"试探 → 回避 → 再试探 → 松动"：

模板：
Step 1: 角色A问一个直接的问题
Step 2: 角色B不直接回答，而是：
        ├─ 做一个与问题无关的小动作（转杯子、看窗外、摸口袋）
        ├─ 沉默几秒（用环境细节填充）
        └─ 然后用另一句话回答，但"答非所问"
Step 3: 角色A察觉到异常，但不追问（或追问但被更巧妙地回避）
Step 4: 对话结束后，用一个动作/细节暗示角色B的真实想法

修改示例：
❌ AI味（直给式）：
  "你怎么来了？"
  "我有事要告诉你。关于你父亲的。"
  "什么事？"
  "他没死。"

✅ 人味儿（潜台词式）：
  "你怎么来了。"
  他把杯子转了半圈。茶水洒出来，他没擦。
  "路过。"
  凌晨两点，路过一家不打烊的茶馆。白泽没戳穿他，也没坐下。
  过了很久，他又开口，声音很轻：
  "你父亲——"
  白泽的手停在半空。
  "……算了。"
  但白泽已经知道了。
  他没死。
```

### 技法4：内心矛盾外化法

**核心原则**：角色内心的冲突，永远不直接说出来，而是通过"言行不一"来呈现。

```
【内心矛盾外化设计法】

要让读者感受到角色"既想靠近又怕受伤"、"既感激又嫉妒"——

不是写："他内心很矛盾，既想道谢又说不出口。"

而是写：
Step 1: 角色做了一个与真实欲望相反的动作（推开/拒绝/冷言冷语）
Step 2: 然后他停下来，盯着自己的手（或那个动作的对象），愣了两秒
Step 3: 接着他做了一件完全不相干的事（摆正杯子/捡起地上的头发/整理衣服）
Step 4: 最后他说了一句极其简短的话（不超过五个字），和刚才的情绪表面无关
Step 5: 全程不解释任何心理活动

修改示例：
❌ AI味："他想说谢谢，但又不好意思，心里很纠结。"

✅ 人味儿：
  他张了张嘴。
  "我又没求你。"
  说完就后悔了。手指无意识地搓着衣角——那还是走之前她帮他缝的。
  "……"
  "我是说。"
  "……算了。"
  她没说话。他也没看她的眼睛。
  但他把茶杯往她的方向推了推。
```

### 技法5：感官锚点深度穿插

**核心原则**：感官锚点不是"每X句插入一次"的机械操作，而是**在情绪需要呼吸的地方**自然放置。

```
【感官锚点穿插 — 深度版】

穿插时机（跟着情绪走，不跟着数字走）：
├─ 对话中有沉默/停顿 → 让环境声音填补
├─ 角色情绪暗涌 → 让他用一个小动作暴露（转杯子/撕标签）
├─ 场景转换 → 用气味/温度的变化暗示
└─ 锚点可以有叙事意义（不只是"灯管响了"，而是"灯管响了三声，第四声没响"）
```

## 第三部分：句子层面的精修——每一句话都是武器

### 3.1 句式节奏精修

精修层的句式调整不是机械地数字数，而是**用耳朵听韵律**：

```
【句式节奏 — 精修层】

不做（03已做）：检查连续短句成段、基础长短句分布
精修层做：
├─ 读一遍，觉得"一顿一顿" → 合并相邻短句为复句
├─ 读一遍，觉得"喘不过气" → 在长句堆中插入一个句号，另起一段
├─ 关键情绪爆点 → 检查短句是否够短够狠（"他没说话。"比"他沉默了很久。"有力）
└─ 韵律本身比任何数字指标都重要
```

### 3.2 词汇与句式精修

精修层不重复03的基础替换工作，而是做**有叙事意义的词汇升级**：

```
【词汇精修 — 精修层】

不做（03已做）：提醒高频词、建议替换
精修层做：
├─ 关键词升级：把"他没说话"变成"他把那句话咽了回去"
├─ 感官词深化：把"很冷"变成"冷从脚底往上爬"
├─ 删废话词：删掉"忽然""似乎""好像""有点"等软塌塌的修饰词
└─ 不追求"每次都用不同的词"——自然的语言允许适度重复

### 3.3 词库使用规则（强制约束）

> **核心问题**：AI在调用成语/俗语/歇后语库时，很容易把它们变成"堆砌"——作为总结性的形容词塞在段落末尾，这是AI味最重的特征之一。

```
【词库使用规则 — 强制约束】

所有词库中的成语/俗语/歇后语/惯用语：

优先级1：作为角色的台词使用
├─ 让角色在对话中说出来，而不是叙述者总结
├─ ✅ "您这是'站着说话不腰疼'。"（角色台词——活人说话会用俗语）
├─ ❌ "他这态度，真是站着说话不腰疼。"（叙述者总结——AI腔）
└─ 原理：活人说话时会引用成语/俗语，但叙述者不会替读者总结

优先级2：作为动作的提炼
├─ 用成语替代冗长的动作描写，使动作更精炼
├─ ✅ "他眼疾手快，一把抓住了。"（成语提炼）
├─ ❌ "他动作很快，眼睛很尖，手也很利索，一把抓住了。"（冗长）
└─ 注意：提炼动作时，成语后不应再跟解释性文字

绝对禁止：作为总结性的形容词塞在段落末尾
├─ ❌ "这一战，真是惊天动地、气壮山河。"（总结性堆砌）
├─ ❌ "他看着她，心里百感交集、五味杂陈。"（总结性形容词）
├─ ❌ "人生在世，不过如此。酸甜苦辣，尽在其中。"（AI式哲理总结）
└─ 原因：段落末尾的成语堆砌 = AI最明显的"升华本能"，必须斩断

检测方法：
├─ 扫描每段末尾的2-3句
├─ 如果发现连续使用成语/俗语进行总结 → 标记为❌必须删除
├─ 替代方案：用具体动作/物件/环境细节收尾，而非成语总结
└─ 示例：❌ "这一战，惊心动魄。" → ✅ "他把剑擦干净，收进鞘里。手还在抖。"
```

### 3.4 标点规范化规则

> 标点是文字节奏的节拍器。AI生成的文本最常见的标点问题：滥用省略号制造虚假悬念、滥用破折号打断节奏、感叹号过多导致情绪廉价。

```
【标点规范化三原则】

原则1：省略号"……"的使用限制
├─ 用途：表示话未说完、欲言又止、沉默中的犹豫
├─ 禁止：
│   ├─ ❌ 用省略号制造廉价悬念（"他不知道的是……"）
│   ├─ ❌ 连续两段以上使用省略号（"他……她……然后……"）
│   ├─ ❌ 对话中超过3处省略号（显得角色说话吞吞吐吐）
│   └─ ❌ 用省略号替代真正的情绪描写
├─ 替代方案：
│   ├─ 话未说完但读者能猜到 → 直接写"他没说完"，更有力
│   ├─ 沉默 → 用动作描写替代（"他张了张嘴，没出声。"）
│   └─ 悬念 → 用具体信息制造悬念，而非省略号
└─ 每章省略号使用上限：5处（含对话和非对话）

原则2：破折号"——"的使用规范
├─ 用途：解释说明、话题突转、语气延长
├─ 禁止：
│   ├─ ❌ 连续使用破折号（"他——她——然后——"）
│   ├─ ❌ 用破折号替代所有标点（"他走进来——坐下——喝水——说话"）
│   ├─ ❌ 对话前加破折号（"——你怎么来了？"）→ 这是西式标点，网文用引号即可
│   └─ ❌ 一句话中使用超过2个破折号
├─ 正确用法：
│   ├─ 解释说明："他看到了那个人——那个三年前就该死的人。"
│   ├─ 话题突转："他本来想道歉——算了，说了也没用。"
│   └─ 语气延长："你——"
└─ 每章破折号使用上限：4处（V4.4 收紧：原8处过于宽松，AI检测报告实测4处以上即被标记为AI痕迹）

原则3：感叹号"！"的使用控制
├─ 用途：表达强烈情绪（愤怒/惊讶/命令/欢呼）
├─ 禁止：
│   ├─ ❌ 连续3句以上使用感叹号（情绪通货膨胀，读者麻木）
│   ├─ ❌ 普通陈述句加感叹号制造虚假紧张（"他走进了房间！"）
│   ├─ ❌ 叙述中使用感叹号（"他决定！要复仇！"）→ 叙述保持冷静
│   └─ ❌ 每章感叹号超过15处
├─ 正确用法：
│   ├─ 真正的高情绪时刻 → 1个感叹号比3个有力
│   ├─ 对话中自然地使用 → 角色确实在喊/吼/叫
│   └─ 大部分情绪用动作和细节传达，而非标点
└─ 每章感叹号使用上限：15处（对话密集章节可到20处）

标点节奏检查：
├─ 扫描全文标点使用频率
├─ 省略号 > 5处 → 精简，用动作描写替代
├─ 破折号 > 4处 → 精简，用句号或逗号替代
├─ 感叹号 > 15处 → 精简，大部分改为句号，保留最关键的3-5处
└─ 连续使用同一特殊标点超过3句 → 打断，插入不同标点
```

### 3.5 对话零补丁规则（语言锋利化）

> **核心问题**：AI在写完一句犀利的话之后，总忍不住要补上一段心理描写或解释，像在说"你看，这句话真的很厉害吧"。这种"解释性后摇"会完全消解掉犀利台词的冲击力。

```
【对话零补丁规则 — 强制】

原则：如果角色说了一句极其犀利、伤人或搞笑的话，这句话之后，必须强制换行，禁止紧跟任何心理描写或解释。

什么是"零补丁"：
├─ 角色说了一句"重话"（讽刺/揭穿/告白/决裂/金句/怼人）
├─ 这句话之后 → 立即换行
├─ 换行后 → 禁止出现：
│   ├─ ❌ 角色的心理活动（"他在心里想……""他知道这句话……"）
│   ├─ ❌ 叙述者的解释（"这句话的意思是……""其实他……"）
│   ├─ ❌ 对方的反应总结（"这句话让XXX愣住了"）
│   └─ ❌ 任何"因为/所以/其实/这意味着"等解释性连接词
├─ 换行后 → 允许出现：
│   ├─ ✅ 对方的沉默/动作（"他没说话。"）
│   ├─ ✅ 环境细节（"窗外的风忽然停了。"）
│   ├─ ✅ 时间跳跃（"那之后的三天，他们没再说话。"）
│   └─ ✅ 直接切换场景
└─ 核心：让那句话"砸"在地上，给它回响的空间

修改示例：
❌ AI味（有补丁）：
  "你从来没有爱过任何人。"
  他说出这句话的时候，心里其实也很痛。他知道这句话会伤害她，但他不得不说。
  ✅ 零补丁：
  "你从来没有爱过任何人。"
  她没有说话。
  窗外有只鸟飞过去，影子从她脸上扫过，很快就不见了。

❌ AI味（有补丁）：
  "就这？"
  他的语气里充满了不屑，显然没把对方放在眼里。
  ✅ 零补丁：
  "就这？"
  他把剑收回鞘里，转身走了。
```

### 3.6 斩断解释性后摇（语言锋利化）

> **核心问题**：AI有一种"把话说圆"的本能——写完动作后总要补上原因，写完对话后总要补上解释。这种"解释性后摇"是AI味最重的特征之一。

```
【斩断解释性后摇 — 强制】

全局扫描，凡是带有以下解释性后摇的句子，一律删除：

一级后摇（必须删除）：
├─ "这是因为……" → 删除整句，让读者自己悟
├─ "他之所以这样做……" → 删除整句，前置动作已经说明了原因
├─ "其实他心里明白……" → 删除，用行动暗示替代
├─ "这意味着……" → 删除，不要替读者总结
├─ "说到底……" → 删除，不要替读者归纳
├─ "换句话说……" → 删除，说一遍就够

二级后摇（必须删除）：
├─ "……的原因很简单——" → 删除整句，原因不需要"简单"
├─ "……背后是……" → 删除，不要挖掘"背后"
├─ "……暗含着……" → 删除，不要解释"暗含"
├─ "……实质上……" → 删除，不要区分"表面"和"实质"
├─ "……归根结底……" → 删除，不要替读者做哲学总结

三级后摇——否定+揭示句式（必须删除，这是AI最隐蔽的"说明书"本能）：
├─ "不是X，是Y" → 删除揭示部分，只保留动作
│   ├─ ❌ "她敲击节奏漏了一拍……不是紧张。是计算。"
│   └─ ✅ "她敲击节奏漏了一拍。"
├─ "没有X，没有Y，只有Z" → 删除揭示部分，只保留可观察的动作
│   ├─ ❌ "那双眼睛里没有试探，没有审视，只有一种奇怪的疲惫。"
│   └─ ✅ "他看了她一眼。然后移开了。"
├─ "看似X，实则Y" → 删除"实则Y"
│   ├─ ❌ "他看似随意地放下杯子，实则在观察她的反应。"
│   └─ ✅ "他放下杯子。目光在她脸上停了一瞬。"
├─ "表面X，内心Y" → 删除"内心Y"
│   ├─ ❌ "她表面平静，内心却翻江倒海。"
│   └─ ✅ "她没说话。手指把纸杯捏扁了。"
└─ 原理：真实文学是留白的，AI的文学是说明书。把逻辑判断留给读者，不要写出来。

后摇句型特征检测：
├─ 扫描全文，查找：因此/所以/因为/其实/这意味着/换句话说/说到底/实质上/归根结底
├─ 扫描全文，查找：不是……是……/没有……只有……/看似……实则……/表面……内心……
├─ 每出现一处 → 标记为⚠️待删除
├─ 核心原则：如果删掉这句话，读者仍然能理解 → 这句话就是多余的
└─ 例外：角色对话中的"因为/所以"可以保留（人在说话时确实会解释）

修改示例：
❌ AI味：
  他拒绝了她的邀请。其实他心里明白，他之所以这样做，是因为害怕再次受伤。说到底，他只是一个懦夫。
  ✅ 人味儿：
  他拒绝了她的邀请。
  然后他在门口站了很久。
  门没锁。他知道。她也没锁。
  他最终还是没有推门。

❌ AI味：
  她笑了。这意味着她已经原谅了他。换句话说，一切都过去了。
  ✅ 人味儿：
  她笑了。
  他把那杯茶推到她面前。茶已经凉了，但她还是喝了。

❌ AI味（否定+揭示句式）：
  她左手食指敲了一下。不是紧张，是计算。
  ✅ 人味儿：
  她左手食指在文件夹上重重敲了一下，节奏短促。随后，她头也不回地走了。

❌ AI味（否定+揭示句式）：
  那双眼睛里没有试探，没有审视，只有一种奇怪的疲惫。
  ✅ 人味儿：
  他看了她一眼。眼神很淡，像隔着一层雾。
```

### 3.7 反剧透式叙述规则（拒绝动作与意图直给）

> **核心问题**：AI不懂得让角色撒谎。它只能写出"她说X，其实她是Y"的剧透式叙述——把角色的真实意图直接告诉读者，剥夺了读者"猜谜"的乐趣。真实的人类是复杂的，角色也应该是复杂的。

```
【反剧透式叙述规则 — 强制】

原则：禁止在叙述中直接揭示角色的真实意图。角色的言行可以不一致，但叙述者不能替读者点破这种不一致。

禁止的剧透式叙述句式（出现即删除）：
├─ "她说X，其实她是Y" → 删除"其实她是Y"，只保留"她说X"
│   ├─ ❌ "沈鸢提醒白术不要查档案，其实是为了保护他。"
│   └─ ✅ "沈鸢提醒白术不要查档案。她说这话的时候没看他。"
├─ "他这么做，是为了……" → 删除"是为了"，让动作自己说话
│   ├─ ❌ "他这么做，是为了让她知难而退。"
│   └─ ✅ "他把那叠文件推到她面前。'看完再说。'"
├─ "她真正的目的是……" → 删除整句
├─ "他心里清楚，她这是在……" → 删除整句
├─ "这看似X，实际上是在Y" → 删除"实际上是在Y"
└─ "她没有说出口的是……" → 删除整句

角色复杂性塑造原则：
├─ 角色可以撒谎 → 但叙述者不能点破这是谎言
│   ├─ ❌ "她说没事。但她在撒谎。"（剧透）
│   └─ ✅ "她说没事。然后她去厨房倒了杯水，手抖了一下。"（让读者自己判断）
├─ 角色可以言行不一 → 但叙述者不能解释原因
│   ├─ ❌ "她嘴上拒绝，心里却已经动摇了。"（剧透）
│   └─ ✅ "她拒绝了。但那天晚上，她翻来覆去没睡着。"（让读者自己感受）
├─ 角色可以有隐藏动机 → 但要通过行动暗示，而非叙述点破
│   ├─ ❌ "她接近他，是为了拿到那份名单。"（剧透）
│   └─ ✅ "她第三次'偶遇'他。这一次，她的目光在他办公桌上的文件夹上停了一瞬。"（暗示）
└─ 核心：让读者成为"侦探"，而不是"观众"。读者自己猜出来的，比被告知的震撼十倍。

修改示例：
❌ AI味（剧透式）：
  沈鸢提醒白术不要查档案。她的语气很平静，但白术能感觉到，她是在保护自己。
  ✅ 人味儿（留白式）：
  沈鸢提醒白术不要查档案。
  "有些东西，查到了反而麻烦。"
  她说完就走了。白术盯着她的背影，发现她走路的时候，右手一直攥着袖口。
```

### 3.8 感官失焦训练（核心冲突时转移描写焦距）

> **核心问题**：AI写恐怖或惊悚，总喜欢围绕"核心冲突点"去写。写古镜异变就死盯着古镜，写鬼魂就死盯着鬼魂。这种"聚焦"反而削弱了恐怖感——真正的恐怖来自于"失焦"，来自于你的余光扫到了不该看的东西。

```
【感官失焦训练 — 核心冲突场景专用】

原则：当核心冲突发生时，强制将描写焦距从核心对象上移开，转向现场最不起眼的细节。

失焦技法：
├─ 技法1：核心异变时，写旁边的无关物件
│   ├─ 场景：古镜开始波动
│   ├─ ❌ AI聚焦："古镜的表面开始扭曲，镜中的影像变得模糊不清，一股寒意从镜面涌出……"
│   └─ ✅ 失焦："老周的保温壶盖子上凝结了一颗水珠。它摇摇欲坠，却迟迟不落。古镜在身后响了一声，他没回头。"
├─ 技法2：恐怖时刻，写日常的琐碎
│   ├─ 场景：尸体就在脚下
│   ├─ ❌ AI聚焦："尸体躺在那里，血从伤口涌出，染红了地面……"
│   └─ ✅ 失焦："墙上的钟还在走。滴答。滴答。他数到第七下的时候，才低头看了一眼。"
├─ 技法3：高潮对峙时，写环境的微小变化
│   ├─ 场景：两人拔剑对峙
│   ├─ ❌ AI聚焦："两人的目光在空中碰撞，杀气弥漫……"
│   └─ ✅ 失焦："风把一片叶子吹到他脚边。他没动。叶子又飘走了。"
└─ 技法4：情绪爆发时，写身体的无关感受
    ├─ 场景：角色崩溃
    ├─ ❌ AI聚焦："他的内心翻涌着无尽的悲伤，泪水夺眶而出……"
    └─ ✅ 失焦："他发现鞋带松了。他蹲下来系鞋带。系了三次，都没系好。"

失焦原理：
├─ 人类在极端情境下，注意力反而会"漂移"到无关事物上
├─ 这种"失焦"比直接描写恐怖/悲伤更有冲击力
├─ 读者会自动脑补核心冲突的恐怖程度——因为你没有写，所以它"不可名状"
└─ 失焦的物件越日常、越微小，反差感越强

使用频率：
├─ 每个核心冲突场景至少使用1次失焦
├─ 不要每个场景都用（会变成新的套路）
└─ 失焦的细节必须具体、可感知，不能是抽象的
```

### 3.9 角色对话信息不对称规则（拒绝AI式和谐）

> **核心问题**：AI写的对话太"和谐"了——你问我答，信息传递极其顺畅。真实的人类对话充满了"打岔""误解""信息不对称"和"言不由衷"。

```
【角色对话信息不对称规则 — 强制】

原则：角色之间的对话，必须包含"信息不对称"和"误解"。禁止问答顺畅的AI式对话。

禁止的AI式和谐对话：
├─ 问什么答什么（对话像采访）
│   ├─ ❌ "你去哪了？" "我去查了档案。" "查到什么了？" "查到了一些奇怪的东西。"
│   └─ ✅ "你去哪了？" "……没去哪。" 他把外套挂好，没看她。"吃了吗？"
├─ 信息传递零阻力（角色说什么读者就懂什么）
│   ├─ ❌ "这件事很危险，你不要插手。" "我知道了，我会小心的。"
│   └─ ✅ "这件事……" 他停了一下，"算了。" "什么？" "没什么。吃饭吧。"
├─ 角色之间没有"隐瞒"（所有信息都摊在桌面上）
│   └─ 真实的人类：有秘密、有顾虑、有不可言说的代价
└─ 对话没有"压迫感"（角色说话太自由）

强制要求的信息不对称类型：
├─ 类型1：打岔——角色不直接回答问题，转移话题
│   ├─ 示例："你到底查到了什么？" "今天的粥有点咸。"
│   └─ 原理：真实的人在不想回答时，会本能地转移话题
├─ 类型2：半句话——角色说到一半停下来，不说完
│   ├─ 示例："那件事，其实我——" 她看了他一眼，"算了。"
│   └─ 原理：未说完的话比说出来的更有张力
├─ 类型3：答非所问——角色回答了另一个问题
│   ├─ 示例："你认识她？" "天快黑了。"
│   └─ 原理：答非所问本身就是一种回答
├─ 类型4：压迫感——角色因为"不可言说的代价"只能说到这里
│   ├─ 示例："有些事，我不能告诉你。不是不想，是……" 他没说下去。
│   └─ 原理：真实的人有顾虑、有恐惧、有不能说的秘密
└─ 类型5：误解——角色A说的话，角色B理解成了另一个意思
    ├─ 示例："她没事。" 他说的是真的没事。她听成了"她不重要"。
    └─ 原理：人类沟通永远有信息损耗和扭曲

每章对话检查：
├─ 是否存在连续3轮以上"问答顺畅"的对话？→ 必须打断
├─ 是否所有角色都在"说真话"？→ 至少1个角色在隐瞒/撒谎/言不由衷
├─ 是否有"信息不对称"的段落？→ 每章至少2处
└─ 是否有角色"欲言又止"的时刻？→ 每章至少1处
```

## 第四部分：叙述厚度保障

### 4.1 对话与叙述的黄金比例

```
【叙述厚度规则】

单章比例要求：
├─ 对话占比 ≤ 60%
├─ 叙述/描写 ≥ 25%
├─ 内心活动/心理层次 ≥ 10%
├─ 闲笔/氛围 ≥ 5%（V5.2.0降级为建议：过渡章可放宽，高潮章可压缩）
└─ 如果对话占比超过80% → 判定为"话剧体"，必须改写

如何增加叙述厚度：
├─ 技巧1：将部分对话改为间接引语
│   ❌ 张三说："我昨天去了城外。城外有很多难民。"
│   ✅ 张三说城外有很多难民。他说这话的时候没看他。
├─ 技巧2：在对话之间插入场景/环境描写
├─ 技巧3：用物件承载情绪，减少直接对话
└─ 技巧4：每章至少2处"叙述者声音"（场景氛围/心理层次/隐喻/环境变化）
```

### 4.2 Show, Don't Tell 执行规则

```
【Show vs Tell 速查】

Telling（❌）→ showing（✅）：

"他很穷"
→ 袖口磨毛了边 / 筷子在碗里搅了很久才夹起一块 / 咽口水的声音很响

"他很害怕"
→ 后颈的汗毛竖起来 / 脚步后退了半步 / 手心开始发黏

"房间很破"
→ 墙角灰厚半指 / 一脚踩下去地板吱呀叫 / 展柜玻璃裂了缝

"他营养不良"
→ 颧骨凸出来 / 指甲发白 / 嘴唇干裂 / 头发像枯草

"他很累"
→ 眼睛发涩 / 肩膀塌着 / 说话的声音比平时轻一半 / 揉了揉后颈

Show的层级（从具体到抽象）：
1. 身体反应（最具体）：瞳孔收缩、喉结滚动、肌肉绷紧
2. 行为细节（次具体）：手指发抖、脚步顿住、视线闪避
3. 环境投射（间接）：物件状态、光线变化、气味扩散
4. 对话暗示（最抽象）：话说一半、答非所问
→ 能到第1层就不到第4层

核心禁令：
├─ 禁止："他感到[任何情绪词]"
├─ 禁止："他忽然觉得……"
├─ 禁止：用形容词总结状态（"典型的营养不良""一看就很穷"）
└─ 替代表达：永远用具体的、感官可感知的细节替代
```

## 第五部分：精修执行流程

### 5.1 拿到初稿后做什么

```
【正文精修三层法】（V5.2.0 — 对齐3核心Step）

第零层：核心原则扫描（最高优先级）
├─ 扫描所有"他发现/这说明/这意味着/他在等/他心里明白" → 执行原则1（潜台词拦截器），全部删除
├─ 扫描所有"像/仿佛/好似/犹如"氛围比喻 → 执行原则2（比喻质量门），低质量替换、高质量保留
├─ 扫描所有"万能前缀"和"情绪套娃" → 执行精简文笔协议，删除
└─ 默读 0.4 Pattern-Matching 训练案例，校准"网文顶级质感"基准线

第一层：快速通读（核心Step 1 — 质感深化）
├─ 只读一遍，不修改
├─ 在感觉"卡住了""太快了""太平了"的地方做标记
├─ 信任直觉——如果某处让你不舒服，读者也会
└─ 标记后进入质感深化三步法（诊断→风格化注入→记忆点打磨）

第二层：对话深度打磨（核心Step 2 — 解释腔与心理直给消除）
├─ 查找直来直去的问答 → 插入潜台词（技法3）
├─ 查找内心冲突直接说出来 → 改为言行不一的外化（技法4）
├─ 检查角色说的话是否都"太合理"——偶尔让人说一句"废话"
├─ 查找犀利台词后的"补丁" → 执行零补丁规则（3.5），删除解释性文字
├─ 查找叙述中的"解释性后摇" → 执行斩断规则（3.6），删除"因为/所以/其实"等
├─ 查找"不是X，是Y"否定+揭示句式 → 执行斩断规则（3.6），删除揭示部分
├─ 查找剧透式叙述 → 执行反剧透规则（3.7），删除"她说X其实她是Y"
├─ 查找问答顺畅的AI式和谐对话 → 执行信息不对称规则（3.9），加入打岔/半句话/答非所问
├─ 执行解释腔消除（第零点五部分 Step 4）—— **04唯一执行点**
└─ 执行心理直给检测（第零点五部分 Step 5）—— **04唯一执行点**

第三层：质感注入 + 标点收尾 + 角色指纹（核心Step 3）
├─ 环境缺少脏感 → 注入（技法2）
├─ 对话中缺少感官锚点 → 在情绪呼吸处自然放置（技法5）
├─ 章节结尾是套路句式 → 改为留白/物件/身体感受收尾（模块R）
├─ 核心冲突场景是否失焦？→ 执行感官失焦训练（3.8），将焦距转向不起眼细节
├─ 执行情绪通路校验（第零点五部分 Step 6）—— 确保情绪"到达读者"
├─ 执行角色语言指纹校验（第零点五部分 Step 7）—— 遮名测试+口癖指纹
├─ 执行标点规范化（3.4）—— 省略号≤5/破折号≤4/感叹号≤15
├─ 删除软塌塌的词（"似乎""好像""有点""忽然"）
└─ 不做数字指标检查——03已经做过基础检测
```

## 第六部分：吸收的深度模块（从 V3.5 03扩写系统剥离）

### 模块R：情感收束与章节结尾

> **来源**：从03扩写系统 V3.5 模块R剥离。情感收束的本质是"怎么写才动人"，属于精修层的深度工作。

```
【章节结尾怎么写 — 精修层】

禁止的结尾：
├─ ❌ "三千年来第一次……"（套路情感升华）
├─ ❌ "这大概就是……吧"（公式化感悟）
├─ ❌ 用排比句强行拔高情绪
└─ ❌ 角色自己总结感受

精修层的结尾技法：

技法1：戛然而止
├─ 前文情节自然收束，不做任何总结
└─ 示例："他说完就转身走了。门在身后关上，咔哒一声。"

技法2：物件承载
├─ 用一个物品细节承载整章的情绪重量
└─ 示例："他低头看着手里的铜钱，边缘被磨得发亮。师父用了二十年，他用了三年。"

技法3：留白
├─ 不点破，让读者自己体会
└─ 示例："她没接话。他也没催。窗外的雪落了一夜。"

技法4：身体感受收尾
├─ 用生理感受替代心理总结
└─ 示例："他把手揣进袖子里。暖和的。不知道多久没这种感觉了。"

技法5：感官收束
├─ 用环境细节画上句号
└─ 示例："路灯亮起来的时候，街上已经没什么人了。"
```

### 模块I：闲笔与氛围留白（V5.2.0 — 降级为建议）

> **来源**：从03扩写系统 V3.5 模块I剥离。闲笔的本质是风格选择，属于精修层的深度工作。
>
> **V5.2.0 降级说明**：原"每章15-25%非推进性文字"的量化要求过于强制，实际执行中容易导致节奏拖沓。降级为建议：过渡章可选，高潮章不建议。跟着语感走，该闲的时候自然会闲。

```
【闲笔与留白 — 精修层建议】（V5.2.0 降级为建议）

核心思想：AI喜欢每一句话都推进剧情。人类会看云，会发呆。这些"无意义"的闲笔是文学性的精髓。

闲笔的类型：

类型1：环境闲笔
├─ 与剧情无关的环境细节
└─ 示例："墙头的野草在风里摇晃，一只野猫蹲在那儿，盯着他看。"

类型2：思绪闲笔
├─ 角色的思绪飘到不相关的地方
└─ 示例："他想起小时候师父说过的话，又觉得那话好像不是这个意思……"

类型3：无意义互动
├─ 角色做了一件毫无目的的小事
└─ 示例："他端起茶盏，又放下。茶已经凉了。"

类型4：感官游离
├─ 一个与环境无关的感官细节
└─ 示例："不知道哪儿传来一声狗叫，紧接着又是一声，然后安静了。"

闲笔的放置原则（V5.2.0 调整）：
├─ 高潮前：不建议放闲笔（会破坏紧张感）
├─ 高潮后：可以放闲笔（给读者喘气）
├─ 对话间隙：偶尔的闲笔让人物更真实
├─ 过渡段落：闲笔是时间流逝的最好载体
└─ 不设比例——跟着语感走，该闲的时候自然会闲
```
